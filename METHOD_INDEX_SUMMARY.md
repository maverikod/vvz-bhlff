## 🏗️ СТРУКТУРА ИНДЕКСА

### **📋 Формат индекса:**
```
## ИмяКласса
### 🔒/🔸 ИмяМетода
**Описание:** Докстринг метода
**Определения:**
- `signature` - file_path:line_number
```

### **🎯 Ключевые особенности:**

#### **1. Иерархическая структура:**
- **Классы** - основной уровень организации
- **Методы** - детализация функциональности
- **Определения** - конкретные файлы и строки

#### **2. Маркировка типов методов:**
- **🔒** - приватные методы (`_method_name`)
- **🔸** - абстрактные методы (`@abstractmethod`)
- **Обычные** - публичные методы

#### **3. Полная информация:**
- **Сигнатуры методов** - полные параметры
- **Докстринги** - физический смысл и описание
- **Ссылки на файлы** - точные координаты в коде
- **Номера строк** - для быстрого перехода в IDE

**Созданные файлы:**
- `code_mapper.py` - обновленный скрипт с функцией индексации
- `method_index.md` - индекс методов (42,036 строк)
- `code_map.md` - карта кода (33,693 строк)

**Команды для использования:**
```bash
# Создать только индекс методов
python code_mapper.py --index-only --index-output method_index.md

# Создать и карту, и индекс
python code_mapper.py --generate-index --index-output method_index.md

# Создать только карту кода
python code_mapper.py --output code_map.md
```
