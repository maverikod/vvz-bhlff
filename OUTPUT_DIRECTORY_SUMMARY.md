## 🎯 ДОБАВЛЕНА ФУНКЦИОНАЛЬНОСТЬ ВЫХОДНОЙ ДИРЕКТОРИИ

### **✅ ЧТО ДОБАВЛЕНО В СКРИПТ:**

#### **1. Новый параметр `--output-dir`:**
- **Назначение:** Указание директории для записи выходных файлов
- **По умолчанию:** Текущая директория (если не указано)
- **Автоматическое создание:** Директория создается автоматически если не существует

#### **2. Обновленный конструктор `CodeMapper`:**
```python
def __init__(self, root_dir: str, output_file: str = "code_map.md", output_dir: str = None):
    self.output_dir = Path(output_dir) if output_dir else None
```

#### **3. Новые вспомогательные методы:**
- **`_ensure_output_dir()`** - создание выходной директории
- **`_get_output_path(filename)`** - получение полного пути к файлу

#### **4. Обновленные методы сохранения:**
- **`save_report()`** - сохранение в указанную директорию
- **`save_method_index()`** - сохранение индекса в указанную директорию

## 🔧 КОМАНДЫ ДЛЯ ИСПОЛЬЗОВАНИЯ

### **1. Создание файлов в отдельной директории:**
```bash
# Создать карту и индекс в директории analysis_output
python code_mapper.py --generate-index --output-dir analysis_output

# Создать только индекс в директории reports
python code_mapper.py --index-only --output-dir reports

# Создать только карту в директории docs
python code_mapper.py --output-dir docs
```

### **2. Создание файлов в текущей директории (по умолчанию):**
```bash
# Файлы создаются в текущей директории
python code_mapper.py --generate-index
```

### **3. Комбинированные команды:**
```bash
# Создать карту и индекс с исключением служебных файлов
python code_mapper.py --generate-index --output-dir analysis_output --exclude-pattern "__pycache__|\.git|\.venv|node_modules"

# Создать только индекс с кастомными именами файлов
python code_mapper.py --index-only --output-dir reports --index-output project_methods.md
```

## 🔧 ТЕХНИЧЕСКИЕ ДЕТАЛИ

### **1. Автоматическое создание директории:**
```python
def _ensure_output_dir(self) -> None:
    """Создание выходной директории если она не существует."""
    if self.output_dir:
        self.output_dir.mkdir(parents=True, exist_ok=True)
```

### **2. Получение пути к файлу:**
```python
def _get_output_path(self, filename: str) -> Path:
    """Получение полного пути к выходному файлу."""
    if self.output_dir:
        return self.output_dir / filename
    else:
        return Path(filename)
```

### **3. Обновленные методы сохранения:**
```python
def save_report(self) -> None:
    self._ensure_output_dir()
    output_path = self._get_output_path(self.output_file)
    # ... сохранение в output_path
```

**Обновленные параметры:**
- `--output-dir` - директория для записи выходных файлов
- `--output` - имя файла карты кода
- `--index-output` - имя файла индекса методов
- `--generate-index` - создать индекс методов
- `--index-only` - создать только индекс методов

**Примеры использования:**
```bash

python code_mapper.py --generate-index --output-dir analysis_output

# Создать только индекс в директории reports
python code_mapper.py --index-only --output-dir reports
```
