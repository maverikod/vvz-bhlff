# Step 02: Ядро FFT решателя для фракционного оператора Рисса в 7D BVP Framework

## Цель
Реализовать высокоточный спектральный решатель для фракционного оператора Рисса в 7D пространстве-времени с интеграцией в BVP framework.

## Математическая основа

### 7D Фракционный оператор Рисса
```
L_β a = μ(-Δ)^β a + λa
```
где:
- μ > 0 - коэффициент диффузии
- β ∈ (0,2) - фракционный порядок
- λ ≥ 0 - параметр затухания
- (-Δ)^β - фракционный лапласиан в 7D пространстве M₇ = ℝ³ₓ × 𝕋³_φ × ℝₜ

### 7D Спектральная форма
В 7D k-пространстве:
```
â(k_x, k_φ, k_t) = ŝ(k_x, k_φ, k_t) / (μ|k|^(2β) + λ)
```
где |k|² = |k_x|² + |k_φ|² + k_t²

### BVP Integration
The FFT solver must integrate with BVP envelope equation:
```
∇·(κ(|a|)∇a) + k₀²χ(|a|)a = s(x,φ,t)
```

## Основные компоненты

### 1. 7D FFT Solver (core/fft_solver.py)
- Класс `FFTSolver7D` для решения стационарных задач в 7D
- Интеграция с BVP envelope solver
- Поддержка quench detection
- Обработка memory kernel
- Валидация входных данных

### 2. 7D Фракционный лапласиан (core/frac_laplacian.py)
- Класс `FractionalLaplacian7D` для вычисления (-Δ)^β в 7D
- Оптимизированные алгоритмы для различных β
- Кэширование спектральных коэффициентов
- Обработка особых случаев (k=0)
- Интеграция с BVP envelope equation

### 3. BVP Time Integrators (core/time_integrators.py)
- Класс `BVPTimeIntegrator` для динамических задач с memory kernel
- Поддержка quench detection в time evolution
- Экспоненциальная схема для BVP
- Адаптивный выбор шага по времени
- Контроль устойчивости BVP envelope

## Ключевые особенности

### 1. Точность вычислений
- Использование float64 для всех вычислений
- Строгая нормировка FFT
- Контроль численных ошибок
- Валидация результатов

### 2. Производительность
- Оптимизированные FFT планы
- Векторизованные операции
- Кэширование промежуточных результатов
- Поддержка параллельных вычислений

### 3. Обработка особых случаев
- Корректная обработка k=0 моды
- Валидация условия λ=0 при ŝ(0)≠0
- Обработка граничных случаев β→0, β→2
