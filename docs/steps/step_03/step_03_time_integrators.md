# Step 03: BVP Time Integrators для динамических задач в 7D

## Цель
Реализовать высокоточные временные интеграторы для решения динамических задач фракционного оператора Рисса в 7D BVP framework с поддержкой memory kernel и quench detection.

## Математическая основа

### 7D Динамическое уравнение с BVP
```
∂a/∂t + ν(-Δ)^β a + λa + ∑ⱼ γⱼ mⱼ = s(x,φ,t)
```
где:
- ν > 0 - коэффициент диффузии
- β ∈ (0,2) - фракционный порядок
- λ ≥ 0 - параметр затухания
- γⱼ, mⱼ - memory kernel parameters and variables
- s(x,φ,t) - источник в 7D пространстве

### Memory Variables Evolution
```
∂mⱼ/∂t + (1/τⱼ) mⱼ = source_terms
```

### 7D Спектральная форма
В 7D k-пространстве:
```
∂â/∂t + (ν|k|^(2β) + λ)â + ∑ⱼ γⱼ m̂ⱼ = ŝ(k,t)
```

## Основные компоненты

### 1. Базовый класс BVPTimeIntegrator (core/time_integrators.py)
- Абстрактный базовый класс для всех BVP интеграторов
- Общий интерфейс для временного интегрирования с memory kernel
- Система контроля ошибок и валидации
- Поддержка quench detection
- Интеграция с BVP envelope solver

### 2. BVP Экспоненциальный интегратор
- Класс `BVPExponentialIntegrator` для точного решения с memory kernel
- Аналитическое решение для гармонических источников в 7D
- Оптимальная точность для BVP задач
- Поддержка quench detection
- Интеграция с envelope solver

### 3. Crank-Nicolson интегратор
- Класс `CrankNicolsonIntegrator` для нелинейных задач
- Второй порядок точности по времени
- Безусловная устойчивость
- Эффективная реализация через FFT

### 4. Адаптивный интегратор
- Класс `AdaptiveIntegrator` с контролем ошибки
- Автоматический выбор шага по времени
- Контроль устойчивости
- Оптимизация производительности

## Алгоритмы

### 1. Экспоненциальная схема
Для гармонического источника s(x,t) = s₀(x)e^(-iωt):
```
â(k,t) = â₀(k)e^(-(ν|k|^(2β)+λ)t) + ŝ₀(k)/(ν|k|^(2β)+λ+iω)(1-e^(-(ν|k|^(2β)+λ+iω)t))
```

### 2. Crank-Nicolson схема
```
(â^(n+1) - â^n)/dt + (ν|k|^(2β) + λ)(â^(n+1) + â^n)/2 = (ŝ^(n+1) + ŝ^n)/2
```

### 3. Адаптивная схема
- Оценка локальной ошибки
- Автоматический выбор шага
- Контроль устойчивости
- Оптимизация производительности
