# Уровень A — BVP Framework Validation (полностью независимые, база)

**Цель уровня A**: Валидация BVP Framework как центрального фреймворка 7D теории материи.

**A0. BVP Framework Validation**
Цель: Валидация корректности решения уравнения огибающей BVP, квенч-детекции и вычисления импеданса.
Постановка: 7D BVP envelope equation в периодическом $\mathbb T^3$, спектральный (FFT) Riesz/фракц. Лапласиан.
Наблюдаемое/критерии: сходимость по сетке (×2: $N=128\to256\to512$), отсутствие анизотропии; энергобаланс ≤1–3%; корректность квенч-детекции.
Вариации: разные схемы дискретизации фракц. оператора, различные пороги квенчей.

**A1. BVP Envelope Solver Validation**
Цель: Проверка корректности решения уравнения огибающей BVP.
Постановка: Стационарное решение 7D BVP envelope equation с различными источниками.
Критерии: Инвариантность профилей при смене базовых масштабов; корректность U(1)³ фазовой структуры.

**A2. BVP Quench Detection Validation**
Цель: Валидация системы детекции квенчей BVP.
Постановка: Тестирование трёх порогов квенчей: амплитуда, детюнинг, градиент.
Критерии: Корректная детекция пороговых событий; точность определения зон квенчей.

# Уровень B — BVP Fundamental Properties (фундаментальные свойства поля, слабозависимые)

**Цель уровня B**: Исследование фундаментальных свойств BVP в однородной среде.

**B1. BVP Power Law Tails (Степенные хвосты BVP-модуляций)**
Цель: подтвердить $A(r)\propto r^{\,2\beta-3}$ при $\lambda=0$ для BVP envelope.
Постановка: точечный дефект (ядро), однородные параметры; стац. задача BVP envelope equation.
Наблюдаемое/критерии: линейный тренд в $\log A$–$\log r$, R²≥0.99 на ≥1.5 декады; отсутствие узлов в BVP envelope.
Вариации: $\beta\in[0.5,1.4]$, разные объёмы, разные ГУ (PML/большой тор).

**B2. BVP Spherical Nodes Absence (Отсутствие сферических узлов в BVP-огибающей)**
Цель: показать, что в однородной «безинтервальной» BVP среде узловые сферы не формируются.
Постановка: как B1, фокус на BVP envelope.
Критерии: число смен знака $\partial_r A$ ≤ 1; отсутствие периодических нулей $A(r)$ при росте $r$ в BVP envelope.

**B3. BVP Topological Charge (Топологический заряд через BVP-модуляции)**
Цель: стабилизация «частицы»-ядра через BVP topological charge.
Постановка: численно интегрировать $\oint\nabla\phi\cdot dl$ вокруг ядра в BVP envelope.
Критерии: $2\pi q$ с погрешностью ≤1%; устойчивость к гладким возмущениям BVP envelope.

**B4. BVP Zone Separation (Разделение зон через BVP)**
Цель: количественно отделить три зоны через BVP envelope analysis.
Постановка: посчитать индикаторы $N,S,C$ и радиусы $r_\text{core}, r_\text{tail}$ в BVP envelope.
Критерии: пороги, напр. ядро $N>3, S>1$; хвост $N<0.3, S<0.3$ в BVP envelope.

# Уровень C — BVP Boundaries and Resonators (границы, ячейки, последовательные резонаторы, зависят от B)

**Цель уровня C**: Исследование BVP и границ, резонаторных структур через BVP.

**C1. BVP Resonator Structures (Резонаторные структуры через BVP)**
Цель: рождение ячейки/псевдорезонатора через BVP envelope.
Постановка: сферическая оболочка с контрастом $\eta=\frac{|\Delta Y|}{\langle Y\rangle}$ в BVP framework.
Наблюдаемое/критерии: пики $Y(\omega)$ из BVP, локальные максимумы $A(r)$ между ядром и оболочкой в BVP envelope; порог $\eta^\*$ для появления первой моды.

**C2. BVP ABCD Chain (Цепочка резонаторов через BVP)**
Цель: принцип последовательных резонаторов через BVP framework.
Постановка: 2–5 вложенных оболочек (разные $\eta,\,\Delta r$) в BVP envelope.
Критерии: согласование $Y_{\text{tot}}(\omega)$ с ABCD-моделью через BVP (ошибка мод менее 5% по $\omega_n$, 10% по $Q_n$); пассивность $\Re Y \ge 0$.

**C3. BVP Quench Memory and Pinning (Квенч-память и пиннинг через BVP)**
Цель: статичность BVP envelope относительно дефекта.
Постановка: добавить диссипативно-памятный член (ядро/оболочка) в BVP framework.
Наблюдаемое/критерии: скорость дрейфа ячеек $v_{\text{cell}}$ по кросс-корреляции $I_{\rm eff}(t)$ в BVP envelope → $<10^{-3}L/T_0$; подавление биений.

**C4. BVP Mode Beating (Биения мод как BVP-модуляции)**
Цель: различить дрейф/«заморозку» решётки в BVP envelope.
Постановка: возбудить $\omega_1\approx\omega_2$ в BVP framework, измерить $\Delta\omega,\Delta k$.
Критерии: $v_{\text{cell}}\approx \Delta\omega/|\Delta k|$ без пиннинга и $\to 0$ при пиннинге в BVP envelope.

# Уровень D — BVP Multimode Superposition (многомодовое наложение и поля как огибающие, зависят от B–C)

**Цель уровня D**: Исследование BVP-многомодовости, наложения BVP-модуляций.

**D1. BVP Mode Superposition (Наложение BVP-модуляций)**
Цель: стабильность BVP «каркаса» при добавлении новых мод.
Постановка: последовательно добавлять возбуждения разных $\omega$ в BVP framework.
Критерии: карта максимумов $I_{\rm eff}$ в BVP envelope (Жаккар ≥0.8 до/после), пики $Y(\omega)$ из BVP сохраняют частоты в пределах 3–5%.

**D2. BVP Field Projections (Проекции полей через BVP)**
Цель: операциональная проверка проекции полей через BVP framework.
Постановка: три частотно-амплитудных окна в BVP envelope:
— «EM»: слабонелинейная зона хвоста BVP → проверка потоков $\propto\nabla\phi$;
— «сильное»: высоко-Q локальные моды у ядра в BVP;
— «слабое»: хиральные комбинации, малый Q/утечки в BVP.
Критерии: воспроизводимость характерных подписей (локализация/дальнодействие/анизотропия фазовых градиентов) в BVP envelope.

**D3. BVP Streamlines (Линии потока BVP)**
Цель: сопоставить «вихревые» линии с $\nabla\phi$ в BVP envelope.
Постановка: трассировка линий уровня $\phi$ и поля $\nabla\phi$ вокруг дефекта в BVP framework.
Критерии: устойчивость поля направления в BVP envelope, отсутствие материального транспорта (только фазовая геометрия).

# Уровень E — BVP Solitons and Defects (устойчивость и чувствительность, зависят от всего выше)

**Цель уровня E**: Исследование BVP-солитонов, солитонов как BVP-модуляций.

**E1. BVP Solitons (Солитоны как BVP-модуляции)**
Цель: устойчивость выводов в BVP framework.
Постановка: свип по $\beta,\ \kappa(\cdot),\ \chi'(\cdot),\ \chi''$, контрастам, размерам домена в BVP envelope.
Критерии: неизменность качественных выводов в BVP; диапазоны, где меняется класс поведения (карта фаз) в BVP envelope.

**E2. BVP Defect Dynamics (Динамика дефектов через BVP)**
Цель: реалистичность BVP framework.
Постановка: добавить слабую случайную неоднородность и шум источника в BVP envelope.
Критерии: робастность показателей из B–D в BVP framework (отклонения ≤10–15%).

**E3. BVP Interactions (Взаимодействия BVP-структур)**
Цель: исключить влияние границ в BVP framework.
Постановка: масштаб домена ×2, ×3 в BVP envelope.
Критерии: инвариантность метрик (хвост, $Y(\omega)$, $r_\text{core}$) в BVP envelope в пределах заявленной погрешности.

## Общие настройки и отчётность

* **Диапазоны параметров:** $\beta\in[0.5,1.4]$; контраст $\eta\in[0,0.5]$; память $\chi''$ от 0 до уровня «заморозки»; 1–5 оболочек.
* **Сетки:** $N^3=256^3$ базово, контроль $512^3$; радиальный 1D для сферических серий (ускорение ×50+).
* **Диагностика:** $A(r), \phi(r), \nabla A, \nabla\phi, Y(\omega), \{\omega_n,Q_n\}, v_{\text{cell}}$, индикаторы $N,S,C$, радиусы зон $r_\text{core},r_\text{tail}$.
* **Критерии пассивности:** $\Re Y(\omega)\ge0$, баланс мощности ≤3%.
* **Файнал:** таблица «эксперимент → выполнено/нет, метрики, графики», плюс сводная карта фаз по $(\eta,\chi'',\beta)$.
