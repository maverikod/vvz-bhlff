### 1.2. Фазовое состояние в Θ-терминах ⭐ НОВОЕ

**Введение:** Глава E3  
**Определение:**
```
S ≡ ({σ}, Π, Q, Σ-class, ν_Q)
```

**Ключевой сдвиг:**
- Классика: фаза = минимум свободной энергии
- Θ-теория: фаза = разрешённая геометрия связности поля

**Перевод классических фаз:**
- Твёрдое → малая подвижность Σ, высокая связность, редкие квенчи
- Жидкое → подвижная Σ, перестройки без каскадов
- Газ → слабая связность, быстрые перестройки интерфейсов
- Плазма → расширенный набор каналов/интерфейсов

**Ценность:**
- ✅ Унифицирует классические фазы через Θ-параметры
- ✅ Вводит состояние "без квенчей" как отдельный фазовый класс
- ✅ Объясняет BEC через экстремальную когерентность

**Проверка в БД:** Поиск по "фазовое состояние" - найдено только в 7d-106

### 1.3. Мягкие и жёсткие столкновения ⭐ НОВОЕ

**Введение:** Глава E4  
**Определение столкновения:**
- Не соударение тел, а **конфликт/сшивка интерфейсов**:
```
Σ_A ∩ Σ_B ≠ ∅ → перестройка каналов и/или запуск Q
```

**Критерии:**
- **Мягкое (adiabatic merge):** Σ перестраивается без лавины квенчей, σ сохраняется
- **Жёсткое (quench avalanche):** каскад Q, потеря связности, смена типа ячейки

**Ценность:**
- ✅ Классифицирует взаимодействия через топологию, а не силы
- ✅ Связывает с фазовыми состояниями (E4.5)
- ✅ Объясняет неэкспоненциальные взаимодействия через геометрические процессы

**Проверка в БД:** Поиск по "мягкое столкновение" - найдено только в 7d-106

### 1.4. Карта среды M(x,t) ⭐ НОВОЕ

**Введение:** Глава E1.1  
**Определение:**
```
M(x,t) ≡ {L(x,t), U(x,t), Q(x,t), Π(x,t)}
```

**Компоненты:**
- `L` - локальный масштаб/характерный размер ячейки
- `Q` - добротность (фрактальная поправка)
- `Π` - параметр(ы) ступени/проекции (каркас D3-D5)
- `U` - эффективный потенциал среды (в смысле функционала Θ-поля)

**Ценность:**
- ✅ Заменяет классические "плотность/температура" на Θ-параметры
- ✅ Тип ячейки = режим M + динамика границ Σ
