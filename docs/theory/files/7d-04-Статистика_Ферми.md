# 1. Общая идея статистики Ферми

1. **Фермионы** — это частицы со спином

   $$
   s = \tfrac{1}{2}, \tfrac{3}{2}, \dots
   $$

   У них волновая функция **антисимметрична** при перестановке двух одинаковых частиц:

   $$
   \Psi(\dots, x_i, \dots, x_j, \dots) = - \Psi(\dots, x_j, \dots, x_i, \dots).
   $$

2. Следствие: **принцип Паули** — в одном квантовом состоянии не может находиться более одной фермионной частицы.
   Это объясняет устойчивость материи: электроны не падают все в основное состояние, а заполняют уровни по одному.

3. **Статистическая формула распределения Ферми–Дирака** для средней оккупации уровня энергии \$\epsilon\$:

   $$
   f(\epsilon) = \frac{1}{\exp\!\left(\tfrac{\epsilon - \mu}{k_BT}\right) + 1}.
   $$

   где \$T\$ — температура, \$\mu\$ — химпотенциал.
   При \$T\to 0\$: все уровни ниже \$\mu\$ заняты, выше — пустые.

---

# 2. В твоей модели (7D фазовое пространство)

В документе прямо указано:

* **Барионы** описываются как топологические конфигурации \$U(x,\phi,t)\in SU(3)\$, где карта \$S^6\to SU(3)\$ задаёт барионное число \$B\in\mathbb Z\$.
* Для \$B=1\$ действует механизм **Finkelstein–Rubinstein (FR)**:

  * Конфигурационное пространство солитонов имеет нетривиальные петли.
  * При повороте всей конфигурации на \$2\pi\$ волновая функция получает фазу \$-1\$.
  * Это означает, что такие объекты обязаны подчиняться **ферми-статистике**: спин \$½\$, антисимметрия при перестановке.

Таким образом:

* **Фермионы** = топологические солитоны с \$B=1\$ (протон, нейтрон и любые барионы).
* Их статистика **не постулируется**, а выводится из топологии конфигурационного пространства.

---

# 3. Как это работает технически

1. Конфигурационное пространство \$Q\_B\$ солитонов с барионным числом \$B\$ имеет фундаментальную группу \$\pi\_1(Q\_B)\$.
2. Для \$B=1\$ эта группа \$\mathbb Z\_2\$. Значит, существует нетривиальная петля, при обходе которой волновая функция меняет знак.
3. Ограничения Finkelstein–Rubinstein накладываются на коллективные координаты (вращения, изоспин).
   ⇒ Из \$SU(2)\$ вращений возникает спиновое квантовое число \$½\$.
4. Следствие: **все барионы — фермионы**, даже если они построены из чисто классического поля.

---

# 4. Классическая статистическая картина

Для многих фермионов (например, в ядре или в белом карлике):

* Уровни энергии заполняются по принципу Паули.
* Появляется **ферми-энергия** \$E\_F\$, определяющая «поверхность Ферми».
* При \$T=0\$: все состояния с \$\epsilon\<E\_F\$ заняты.
* Давление в вырожденном газе носит квантовый характер (даже при \$T=0\$):

  $$
  P = \frac{2}{5}\,n\,E_F.
  $$

Это объясняет:

* устойчивость электронных оболочек атомов,
* устойчивость вырожденных звёзд (электронное и нейтронное вырождение).

# 1) Сектор барионов в вашей теории

**Поля и лагранжиан.** Рабочее пространство и поля заданы (фазовая подоснова Θ, барионный сектор $U\in SU(3)$, калибровочные проекции), а лагранжиан построен только из производных плюс WZW-топологический член; «массовых» термов нет: масса = энергия стационарного решения.  &#x20;

**Топологический заряд.** Барион — это конфигурация с отображением $S^6\!\to\!SU(3)$ степени $B\in\mathbb Z$; именно этот сектор помечен как «барионы (комбинации торов)». &#x20;

# 2) FR-ограничения (топологический вывод знака «минус»)

Пусть $Q_B$ — конфигурационное пространство поля $U$ с фиксированным барионным числом $B$.

* Для $B=1$ фундаментальная петля $\gamma\in\pi_1(Q_{B=1})$ соответствует **полному $2\pi$-повороту** конфигурации в обычном пространстве (или эквивалентной перестановке двух одинаковых солитонов по петле в $Q_{B=1}$).
* **Ограничения Finkelstein–Rubinstein**: квантуем состояние не как обычную функцию на $Q_{B=1}$, а как *сечение линейного расслоения* с нетривиальной голономией: при обходе $\gamma$ волновая функция меняет знак,

  $$
    \Psi \;\longrightarrow\; -\,\Psi\quad\text{при обходе } \gamma.
  $$

  Это зафиксировано в документе: при $2\pi$-вращении фаза равна $-1$ ⇒ спин $1/2$ ⇒ ферми-статистика.&#x20;

Интуитивно: FR превращает *скалярные* волновые функции на $Q_{B=1}$ в **проективные** представления вращений (двойная связность), т.е. спинорные. Потому любые «барионные» кванты в секторе $B=1$ автоматически ведут себя как фермионы.

# 3) Квантование коллективных координат

Пусть $U(x,t)\approx A(t)\,U_0(R(t)\,x)\,A(t)^\dagger$ — медленные вращения/изовращения устойчивого ядра $U_0$ (модульное многообразие). Взяв лагранжиан $\mathcal L[U]$ (SU(3)-кинетика + стабилизаторы + WZW), интегрируем по пространству и получаем **эффективное действие** для $R(t),A(t)$:

$$
S_{\text{eff}}=\int\!dt\;\Big(\tfrac12\,\mathcal I\,\Omega^2 + \tfrac12\,\tilde{\mathcal I}\,\tilde{\Omega}^2 \;\;+\;\; \underbrace{\mathcal A_i(\xi)\,\dot{\xi}^i}_{\text{Berry/WZW}}\Big),
$$

где $\xi$ — координаты на модульном многообразии, $\Omega,\tilde\Omega$ — угловые скорости; последний член — связность (Berry-фаза), индуцированная WZW-функционалом. Его **голономия ровно и реализует FR-знак** при $2\pi$-петле. Таким образом, допустимые квантуемые состояния — **спинорные**; спиновый оператор реализуется как генераторы правых/левых вращений на $Q_{B=1}$, причём условие FR запрещает целые спины и оставляет полуцелые. (Связь WZW с барионным числом и SU(3)-кинетикой зафиксирована в вашем лагранжиане. )&#x20;

# 4) Антикоммутация = фермионные операторы

Проективность представления вращений (знак $-1$ при $2\pi$) + FR-условие при перестановке двух одинаковых солитонов означает, что **операторы рождения/уничтожения кванта с $B=1$** должны удовлетворять

$$
\{a_{\alpha},a_{\beta}^\dagger\}=\delta_{\alpha\beta},\qquad
\{a_{\alpha},a_{\beta}\}=0=\{a_{\alpha}^\dagger,a_{\beta}^\dagger\}.
$$

Иначе невозможно реализовать одновременно спинорность и «минус» при перестановке. Таким образом, **антисимметрия многочастичной волновой функции** (и принцип Паули) — *логическое следствие* FR-квантования сектора $B=1$.

# 5) Микростатистика (распределение Ферми–Дирака)

Имея антикоммутирующие моды с одночастичными энергиями $\{\epsilon_k\}$ и химпотенциалом $\mu$, **великая каноническая сумма** для фермионов:

$$
\mathcal Z \;=\; \prod_k \bigl(1+e^{-\beta(\epsilon_k-\mu)}\bigr).
$$

Отсюда средняя оккупация уровня:

$$
\boxed{\;
\langle n_k\rangle \;=\; \frac{1}{e^{\beta(\epsilon_k-\mu)}+1}
\;}
$$

— стандартная формула Ферми–Дирака. Этот шаг — чисто статистико-механический, но он **обоснован** потому, что в секторе $B=1$ мы уже строго получили антикоммутацию из FR-топологии и структуры вашего лагранжиана (SU(3)-кинетика + стабилизаторы + WZW; «массовых» термов нет). &#x20;

# 6) Краткая связка шагов (в логике вашей модели)

1. **Поля/Лагранжиан** заданы; барионный сектор — $U\in SU(3)$, есть WZW-член; масса не вводится руками. &#x20;
2. **Топология**: $S^6\!\to\!SU(3)$, $B=1$ ⇒ конфигурационное пространство со $\mathbb Z_2$-петлёй.&#x20;
3. **FR-квантование**: волновая функция меняет знак при $2\pi$-повороте ⇒ спин ½ и ферми-статистика.&#x20;
4. **Операторы**: антикоммутация $\Rightarrow$ принцип Паули.
5. **Термодинамика**: из антикоммутации следует $\mathcal Z=\prod_k(1+e^{-\beta(\epsilon_k-\mu)})$ ⇒ распределение Ферми–Дирака.

# 1) Статистика фазовой подложки (без “частиц”)

## 1.1. Разделение масштабов

Пусть

$$
\tau_\phi \sim \frac{\ell}{c_\phi}\quad\text{(быстрая перестройка фазы)},\qquad
\tau_{\rm mat}\sim \frac{\ell}{c}\quad\text{(медленная материальная реакция)},\qquad
\varepsilon:=\frac{\tau_{\rm mat}}{\tau_\phi}\gg 1.
$$

Наблюдаем «подложку» со скользящим окном $\Delta t$ так, что

$$
\tau_\phi \ll \Delta t \ll \tau_{\rm mat}.
$$

За $\Delta t$ фаза совершает огромное число микрообновлений → действует ЦПТ (центральная предельная теорема) для сумм быстрых «микрошагов».

## 1.2. Гауссов функционал (коарс-грейдинг)

Обозначим поле фазы $\theta(x,t)$ (для краткости один фазовый канал). После усреднения по быстрым масштабам при фиксированной «жёсткости» подложки получается **квадратичный** (гауссов) эффективный функционал:

$$
\boxed{\;
\mathcal P[\theta]\ \propto\ \exp\!\Big(-\,\tfrac12\!\int\! dt\,d^3x\;\big[
\alpha_t(\partial_t\theta)^2 + \alpha_x\,|\nabla\theta|^2
\ +\ \ldots\big]\Big)
\;}
$$

где $\alpha_t\sim \alpha_t(\varepsilon)$, $\alpha_x>0$ выражают «инерцию»/«жёсткость» фазы. Точки — возможные более высокие производные (изгибная и «сверхжёсткая» добавки), которые улучшают ультрафиолет (но не меняют гауссовость при коарс-грейдинге).

Следствия:

* при фиксированном $t$ распределение полей $\theta(\cdot,t)$ — **многомерно нормальное**;
* при фиксированном $x$ инкременты $\theta(x,t+\Delta t)-\theta(x,t)$ — **нормальные** с дисперсией $\propto \Delta t$.

Это и есть «статистика поля-подложки»: **не распределения заселённости уровней**, а **гауссовская мера на конфигурациях** $\theta$.

## 1.3. Квантование быстрых волн подложки

Каноническая квантизация гауссова действия даёт линейную дисперсию $\omega_k^2 \approx v_\phi^2 k^2$ (с поправками от высоких производных) и стандартные моды:

$$
\theta(x,t)=\sum_k \frac{1}{\sqrt{2\omega_k V}}\big( a_k e^{-i\omega_k t+ikx}+a_k^\dagger e^{+i\omega_k t-ikx}\big),
$$

где операторы **коммутируют** (бозонный алгебраический сектор):
$[a_k,a_{k'}^\dagger]=\delta_{kk'}$, остальные нули. Это — свойство **подложки как непрерывной волновой среды**.

## 1.4. Откуда «дискретность» и «нормальное распределение»

За счёт $\varepsilon\gg 1$ подложка за очень узкое окно $\Delta t$ успевает перескочить через множество «порогов» фазы (фазовые обороты на $2\pi$): суммарное число «фазовых щелчков» $N_\Delta$ за $\Delta t$ велико и подчиняется **практически нормальному** закону (ЦПТ).
Дискретность наблюдаемых величин дальше появляется, когда **медленная** материальная подсистема «сэмплирует» накопленный целый счёт фазовых оборотов (фаз-слипы) — например, через условия согласования в переходной зоне. Так рождаются **кванты** как целые числа событий, хотя первичная статистика подложки остаётся гауссовой.

# 2) Два предельных режима из разрыва скоростей $c_\phi\gg c$

Рассмотрим два крайних случая того, **что именно** успевает «закрепить» медленная подсистема (реакция со скоростью $c$) из бешено быстро меняющейся фазы.

## 2.1. Предельный **бозе-волновой** режим (тривиальная топология, $B=0$)

Условия:

* наблюдаем/фиксируем лишь **линейные волновые флуктуации** подложки;
* **нет** топологического закрепления (ядра нет), только хвосты/линейные моды;
* окно наблюдения и связи с материей таково, что материал «видит» лишь **усреднённую волну**.

Тогда действуют результаты п.1.3:

* элементарные кванты — **бозоны** (коммутация);
* статистика многочастичной системы при термодинамическом описании даёт распределения **Бозе–Эйнштейна** (если вообще вводим тепловой ансамбль для этих волн).
  Здесь дискретизация — это кванты **волновых** мод гауссовой подложки.

## 2.2. Предельный **ферми-солитонный** режим (нетривиальная топология, $B=1$)

Условия:

* быстрые фазовые «щелчки» суммарно формируют **топологически защищённый узор** (ядро + переход + хвост);
* материальная подсистема успевает «захлопнуть» конфигурацию в **фиксированный топологический класс** (обмотка не развязывается плавно);
* включаются **FR-ограничения** для конфигурационного пространства: петля полного $2\pi$ даёт фазу $-1$.

Следствия:

* кванты этого сектора квантуются как **спиноры**, их операторы **антикоммутируют**;
* многочастичная антисимметрия → **ферми-статистика** (и принцип Паули);
* дискретизация — это уже **целочисленная топология** (степень отображения), отобранная медленной подсистемой из бурной фазы.

Интуитивно: подложка «кипит» быстро и непрерывно (гауссово), но **если** медленная часть успевает «снять слепок» в виде устойчивой обмотки, рождается ферми-объект; **если** успевает только «снять» линейную волну — получаем бозонный квант волны.

# 3) «Нормальное распределение» из узкого окна времени

Пусть за короткое $\Delta t$ быстрая фаза совершает $M$ условно независимых микрошагов $\delta\theta_j$ с конечной дисперсией. Тогда

$$
\Delta\theta=\sum_{j=1}^{M}\delta\theta_j\;\;\xrightarrow[M\to\infty]{\rm CLT}\;\; \mathcal N\big(0,\ \sigma^2 M\big).
$$

Далее:

* Если «звено реакции» может только **порогово** реагировать (например, как на целый сдвиг $2\pi$), то наблюдаемая переменная — это **целое** $n=\mathrm{round}(\Delta\theta/2\pi)$: дискретная величина, но её статистика **на больших числах — почти нормальная** (с коррекцией при малых счётах).
* Если же реакция видит только линейный отклик, остаётся **чистая гауссовость** (волновой режим).

# A) Подложка как упругая «скатерть» (статистика поля)

**1) Геометрия и переменные.**
Рабочее многообразие: $\mathcal M_7=\mathbb R^3_x\times\mathbb T^3_\phi\times\mathbb R_t$. Фазовая подоснова — три угловые фазы $\Theta\in U(1)^3$. Это «ткань» поля; частицы — её устойчивые конфигурации.&#x20;

**2) Энергетика/упругость (только производные).**
Эффективный функционал энергии поля содержит «градиентную жёсткость», «изгибную» и «сверхжёсткость»; никаких массовых членов (масса частиц — энергия стационара, а не параметр в $\mathcal L$). Прототип:

$$
\mathcal E[\Theta]\!=\!\tfrac{f_\phi^2}{2}|\nabla\Theta|^2
+\tfrac{\beta_4}{4}(\Delta\Theta)^2+\tfrac{\gamma_6}{6}|\nabla\Theta|^6+\cdots.
$$

(Именно такой подход фиксирован в планах лагранжиана: производные термы + стабилизаторы; «массовых» нет.)&#x20;

**3) Баланс «сжатие–разрежение».**
Устойчивость подложки обеспечивается компенсирующим балансом: локальное «сжатие» фазы компенсируется «разрежением» — по сути, это фазовая модуляция на упругой среде.&#x20;

**4) Разрыв скоростей.**
Фазовая связность передаётся со скоростью $c_\phi$ на **десятки порядков** выше $c$; материальная перестройка ограничена $c$. Это задаёт сильную иерархию времён.&#x20;

**5) Коарс-грейдинг ⇒ гауссова мера.**
Наблюдая поле на окнах $\tau_\phi\!\ll\!\Delta t\!\ll\!\tau_{\rm mat}$, мы усредняем быстрые фазовые микрошаги; в пределе ЦПТ получаем **гауссову** (квадратичную) меру на конфигурациях $\Theta$. То есть «подложка» статистически — не «газ частиц», а **упругая мембрана** с нормальной мерой на $\Theta$.

**6) Дисперсия/моды.**
Линеаризуя, получаем волновые моды $\omega_k\simeq c_\phi k$ (с правками от $\beta_4,\gamma_6$). Фазовая модуляция — базовое свойство среды (наблюдаемое как «хвосты» решений).&#x20;

**7) Откуда «дискретность» и «нормальность».**
За узкое $\Delta t$ быстрые фазовые «щелчки» (фаз-слипы по $2\pi$) накапливаются: счёт таких событий $N_{\Delta}$ велик и близок к **нормальному** закону (ЦПТ). Дискретность в наблюдении возникает не в самой подложке, а на **медленном чтении**: материальная подсистема фиксирует **целый** счёт событий (пороговый/топологический механизм).

# B) Бозе-волновой предел (тривиальная топология, $B=0$)

**1) Условие режима.**
Медленная подсистема «успевает» только усреднить **линейные волновые** флуктуации подложки; ядра/топологии нет: $B=0$.&#x20;

**2) Кинематика и кванты.**
Квантуем линейные моды $\Theta$; операторы порождения/уничтожения **коммутируют** (бозонная алгебра). Это кванты **волны подложки** (фотоны-подобные/калибровочные волны в проекциях).&#x20;

**3) Многочастичная статистика.**
Если вводим термодинамику для этих мод, максимум энтропии при ограничениях $(E,N)$ даёт **бозе-эйнштейнову** статистику заселения (стандартный результат для коммутирующих мод). Смысл: это статистика **проявлений** гауссовой подложки в тривиальном секторе.

**4) Нормальность в измерениях.**
Временнáя сумма амплитуд фазы за $\Delta t$ — нормальна (см. A-7). Если считывание пороговое по энергии/фазе, видим дискретные кванты волны; при линейном — наблюдаем гауссовы отклики.

# C) Ферми-солитонный предел (нетривиальная топология, $B=1$)

**1) Условие режима.**
Быстрые фазовые наслоения «схлопываются» в **устойчивую обмотку**: барионный сектор $U(x,\phi,t)\in SU(3)$, карта $S^6\!\to\!SU(3)$ со степенью $B=1$. Это уже **ядро + переход + хвост**.&#x20;

**2) Лагранжиан барионного сектора.**
Только производные (SU(3)-кинетика) + стабилизаторы (Skyrme-подобные) + топологический WZW-член; «массовых» термов нет (масса = энергия стационара).&#x20;

**3) FR-ограничение и спин.**
Конфигурационное пространство $Q_{B=1}$ имеет нетривиальную петлю; при полном $2\pi$-вращении волновая функция получает фазу $-1$. Отсюда — **спин $1/2$** и **минус при перестановке**.&#x20;

**4) Алгебра и статистика.**
Операторы солитонных квантов **антикоммутируют** $\Rightarrow$ принцип Паули. В термодинамике максимум энтропии при ограничениях $(E,N)$ даёт **ферми-диракову** заселённость уровней — уже как статистику **проявлений** (не подложки).&#x20;

**5) Нормальность в измерениях.**
Счёт событий, ведущих к формированию/перекиду обмотки за короткие окна, при больших интенсивностях также аппроксимируется нормальным (ЦПТ), но итоговая наблюдаемая дискретность — уже **топологическая** (целая степень отображения), поверх которой лежит ферми-алгебра.

# A) «Скатерть»: корреляторы и счёт фаз-слипов

### A1. Коарс-грейдед действие подложки (одна фаза для краткости)

Берём квадратичную (гауссову) аппроксимацию для быстрой фазы (без масс-члена, только производные):

$$
S_\Theta=\frac12\!\int\! dt\,d^3x\,\Big[\ \alpha_t(\partial_t\theta)^2+\alpha_x|\nabla\theta|^2+\beta_4(\Delta\theta)^2\Big],
$$

где $\alpha_t,\alpha_x>0$, $\beta_4\ge0$. Это — «упругая скатерть»: энергия — за счёт временных/пространственных производных (см. установку «только производные; жёсткости $f_\phi^2,\beta_4,\gamma_6$»).&#x20;

### A2. Двухточечный коррелятор

В Фурье ($\omega,\mathbf k$):

$$
\big\langle \theta\,\theta\big\rangle(\omega,\mathbf k)
=\frac{1}{\alpha_t\,\omega^2+\alpha_x\,k^2+\beta_4\,k^4}\,.
$$

Обратно во времени/пространстве — обычная «упругая» зелёная функция мембраны. Линеаризованная дисперсия:
$\ \omega^2 \simeq v_\phi^2 k^2+\tfrac{\beta_4}{\alpha_t}k^4,\quad v_\phi^2=\alpha_x/\alpha_t$
(фазовая скорость $v_\phi$ задаёт «сверхбыструю» связность поля; разрыв скоростей $c_\phi\gg c$ — постулат теории).&#x20;

### A3. Фаз-слипы как счётные события

Пусть $\varphi(t)=\theta(x_0,t)$ и отсчитываем целые обороты $\Delta N$ (скачки $\pm 2\pi$) за короткое окно $\Delta t$ при интенсивности $\lambda\propto c_\phi$:

$$
N_{\Delta t}\sim\mathrm{Poisson}(\lambda\,\Delta t),\qquad
N_{\Delta t}\ \approx\ \mathcal N\big(\lambda\Delta t,\ \lambda\Delta t\big)\quad(\lambda\Delta t\gg1).
$$

Это объясняет «почти нормальность» наблюдаемых сумм за узкое окно, при том что первичная среда — непрерывная «скатерть».

# B) Полный лагранжиан (секторы + связи, без масс-членов)

• фазовая подоснова $\Theta\in U(1)^3$ (три угловые фазы),
• барионный сектор $U\in SU(3)$ (топологические конфигурации),
• калибровочные проекции $A_\mu^{\rm em},\,W_\mu^a,\,G_\mu^A$.&#x20;

$$
\begin{aligned}
\mathcal L
&=\underbrace{\tfrac{f_\phi^2}{2}\,g^{MN}\partial_M\Theta\,\partial_N\Theta}_{\text{фазовая кинетика}}
\;+\;\underbrace{\tfrac{\beta_4}{4}\,\mathcal I_4[\Theta]}_{\text{изгибная жёсткость}}
\;+\;\underbrace{\tfrac{\gamma_6}{6}\,\mathcal I_6[\Theta]}_{\text{сверхжёсткость}}\\[2pt]
&\quad+\underbrace{\tfrac{F_2^2}{2}\,g^{MN}\mathrm{Tr}(L_M L_N)}_{\text{SU(3)-кинетика}},\quad L_M=U^\dagger\partial_M U\\[2pt]
&\quad+\underbrace{\tfrac{S_4}{4}\,\mathcal J_4[U]}_{\text{Skyrme-стабилизатор}}
\;+\;\underbrace{\tfrac{S_6}{6}\,\mathcal J_6[U]}_{\text{6-производн. стабилизатор}}
\;+\;\underbrace{\Gamma_{\rm WZW}[U]}_{\text{топологический член}}\\[2pt]
&\quad-\tfrac14 F_{\mu\nu}F^{\mu\nu}
-\tfrac14 W_{\mu\nu}^a W^{a\,\mu\nu}
-\tfrac14 G_{\mu\nu}^A G^{A\,\mu\nu}\\[2pt]
&\quad+\ \text{минимальные связи: }\ \partial\to D=\partial-ig\,A^{\rm proj}\,.
\end{aligned}
$$

* $\mathcal I_4,\mathcal I_6$ — локальные инварианты подложки (напр. $(\Delta\Theta)^2,\ |\nabla\Theta|^6$);
* $\mathcal J_4,\mathcal J_6$ — стандартные Skyrme-типа инварианты для $U$;
* $\Gamma_{\rm WZW}$ — топологический функционал (даёт барионность и FR-фазу при $2\pi$).
  **Массовых членов нет** (масса — энергия стационара).&#x20;

# C) Уравнения движения (Эйлер–Лагранж на $\mathcal M_7$)

Общее правило: $\ \displaystyle \frac{\delta\mathcal L}{\delta\Phi}-\partial_M\!\left(\frac{\delta\mathcal L}{\delta(\partial_M\Phi)}\right)=0$.

### C1. Фазовая подложка $\Theta$

Для прототипа с $\mathcal I_4=(\Delta\Theta)^2,\ \mathcal I_6=|\nabla\Theta|^6$:

$$
f_\phi^2\,\Box_7\Theta\ -\ \beta_4\,\Delta^2\Theta\ -\ \gamma_6\,\nabla_i\!\Big(|\nabla\Theta|^4\,\nabla_i\Theta\Big)
\ =\ J_{\rm proj}(\text{связи }D\Theta)\,,
$$

где $\Box_7=\partial_M\partial^M$, $\Delta$ — лапласиан по $x$, правые члены — от минимальных связей (EM/слаб./цвет. проекции), если они включены. Это и есть «упругая скатерть» с изгибной/сверхжёсткой коррекцией. (Принцип «только производные» соблюдён.)&#x20;

$$
\partial_M\!\Big(
F_2^2\,L^M
\ -\ S_4\,[L_N,\,[L^M,L^N]]
\ +\ S_6\,\mathcal K^M(L)
\Big)\ +\ \mathcal J_{\rm WZW}^M(L)=0,
$$

где $\mathcal K^M(L)$ — стандартные 6-производные вариации Skyrme-обобщения, $\mathcal J_{\rm WZW}^M$ — ток из WZW. С минимальными связями: $L_M\to L_M+ig\,A_M^{\rm proj}$ и добавляются проекционные токи. (Формат соответствует заявленному лагранжиану SU(3)+Skyrme+WZW.)&#x20;

### C3. Калибровочные поля

$$
\partial_\mu F^{\mu\nu}=j_{\rm em}^\nu(\Theta,U),\qquad
D_\mu W^{a\,\mu\nu}=j_W^{a\,\nu}(\Theta,U),\qquad
D_\mu G^{A\,\mu\nu}=j_g^{A\,\nu}(U),

где правые части — нётеровские токи проекций фазового/барионного сектора. (Границы/калибровки — Кулона/Лоренца по месту.)&#x20;

  +[L_P,[L_N,[L^N,L^M]]]

  (с симметризацией индексов по принятым константам $\alpha_{\times}$);

$$

## 3.2. Реальное пространство (асимптотики)

* В статике ($\omega=0$) при больших $r$: $\langle\theta(\mathbf r)\,\theta(0)\rangle\sim \frac{1}{4\pi f_\phi^2}\,\frac{1}{r}$ (если $\beta_4$ мал) с коррекцией $\propto e^{-r/\ell_4}$ при учёте $\beta_4$ (эффективная длина $\ell_4\sim\sqrt{\beta_4}/f_\phi$).
* Во времени при фиксированном $\mathbf r$: затухающие колебания с $\omega(k)$ (группа $v_g\simeq v_\phi$ на длинных волнах).

## 3.3. Фаз-слипы (счёт событий)

Пусть локально $\varphi(t)=\theta(x_0,t)$. Интенсивность «перекида» на $2\pi$ берём как $\lambda\sim c_\phi/\ell_\phi$ (эффективная фазовая «средняя длина волны» $\ell_\phi$). Тогда за $\Delta t$:

$$
N_{\Delta t}\sim\mathrm{Poisson}(\lambda\Delta t),
\quad
\mathrm{Var}(N_{\Delta t})=\lambda\Delta t,
\quad
N_{\Delta t}\approx\mathcal N(\lambda\Delta t,\lambda\Delta t)\ \ (\lambda\Delta t\gg1).
$$

**Наблюдаемая** дискретная величина (порог/сэмплинг материальной подсистемой):

$$
n=\mathrm{round}\!\Big(\frac{\Delta\varphi}{2\pi}\Big)
\quad\Longrightarrow\quad
\mathrm{Var}(n)\approx \frac{\mathrm{Var}(\Delta\varphi)}{(2\pi)^2}\ \ \text{на больших числах}.
$$

# 4) Предельные режимы (как следствие уравнений и статистики)

## 4.1. Бозе-волновой предел ($B=0$)

* Используем линейные решения для $\theta$ (с-ч. 2.1).
* Квантуем моды — коммутация $[a_k,a_{k'}^\dagger]=\delta_{kk'}$.
* Если вводим тепловой ансамбль → бозе-эйнштейново заселение (это **статистика проявлений**), а первичная статистика подложки — **гауссова** мера.&#x20;

## 4.2. Ферми-солитонный предел ($B=1$)

* Решаем уравнение для $U$ (с-ч. 2.2) с граничными условиями на степень $S^6\to SU(3)$.
* Наличие $\Gamma_{\mathrm{WZW}}$ $\Rightarrow$ FR-фаза $-1$ при $2\pi$ $\Rightarrow$ спин-½, антикоммутация мод, принцип Паули.
* Многотельная ферми-статистика (если вводим ансамбль) — **статистика проявлений** устойчивых обмоток ядра, не подложки. &#x20;

# 5) «Калькулятор» оценок (быстрые формулы)

Введём:

* фазовая скорость $c_\phi$ (или $v_\phi$);
* окно наблюдения $\Delta t$ (так, что $\tau_\phi\ll\Delta t\ll\tau_{\rm mat}$);
* характерная длина $\ell_\phi$ (масштаб модуляций).

**5.1. Дисперсия фазового инкремента**
$\ \Delta\varphi(t)=\varphi(t+\Delta t)-\varphi(t)$.
Для белого по времени драйва на линейной «скатерти»:

$$
\mathrm{Var}(\Delta\varphi)\ \simeq\ A\,\Delta t,\quad
A\ \propto\ \frac{1}{f_\phi^2}\int\!\frac{d^3k}{(2\pi)^3}\,\frac{1}{\omega_k}\,,
$$

где $\omega_k=\sqrt{v_\phi^2 k^2+\tfrac{\beta_4}{f_\phi^2}k^4}$; при ИК/УФ-срезах $k_{\min}\sim 1/L,\ k_{\max}\sim 1/\ell_\phi$ можно быстро оценить $A$ логарифмически.

**5.2. Интенсивность фаз-слипов**

$$
\lambda\ \sim\ \frac{c_\phi}{\ell_\phi},\qquad
\mathbb E[N_{\Delta t}]=\lambda\Delta t,\quad
\mathrm{Var}(N_{\Delta t})=\lambda\Delta t.
$$

**5.3. Вероятность «защёлкнуться» в $B=1$ за окно $\Delta t$**
Модельно: пусть нужен «критический» интеграл фазы/градиента $Q_c$ в локальном объёме $V_\star$ для рождения ядра. Тогда при гауссовой подложке:

$$
P_{B=1}\approx \exp\!\Big(-\frac{Q_c^2}{2\,\sigma_Q^2(\Delta t,V_\star)}\Big),
$$

где $\sigma_Q^2$ выражается через интеграл коррелятора $\langle\theta\theta\rangle$ по $V_\star\times[0,\Delta t]$. Чем больше $c_\phi$ и $\Delta t$, тем выше шанс (но переход стабилизируется только при наличии Skyrme+WZW-термов — иначе обмотка не устойчива).

**5.4. Критерий выбора режима (бозе/ферми)**

* Если $\Delta t$ и детали связи таковы, что $\sigma_Q\ll Q_c$ ⇒ $P_{B=1}$ ничтожна ⇒ видим **бозе-волновой** предел.
* Если $\sigma_Q\gtrsim Q_c$ и дополнительно выполняются виреальные условия стационара по $U$ ⇒ топология «схлопывается» ⇒ **ферми-солитонный** предел.

# I. 6-производный BPS-член: форма и вариация

## I.1. Определения (3+1 проекция наблюдаемой геометрии)

В «наблюдаемом» пространстве $\mathbb R^3_x$ барионный ток:

$$
\mathcal B^\mu=\frac{1}{24\pi^2}\,\epsilon^{\mu\nu\rho\sigma}\,\mathrm{Tr}(L_\nu L_\rho L_\sigma),\quad
L_\alpha=U^\dagger\partial_\alpha U,\ \ \alpha\in\{t,x_i\}.
$$

BPS-вариант шестого члена:

$$
\boxed{\ \ \mathcal L_{6}^{(\mathrm{BPS})}=+\frac{S_6}{6}\,\alpha_{\mathrm{BPS}}\,\mathcal B_\mu \mathcal B^\mu\ \ }.
$$

Он совместим с твоей «только производные + стабилизаторы + WZW» конструкцией и работает в одном пакете со Skyrme-термом $\mathcal J_4$ (см. твой план лагранжиана).&#x20;

## I.2. Вариация $\delta \mathcal L_{6}^{(\mathrm{BPS})}$

Используем $\delta U=U\,i\epsilon$, $\delta L_\mu = \partial_\mu\epsilon + [L_\mu,\epsilon]$.

$$
\delta(\mathcal B^\mu)=\frac{1}{8\pi^2}\,\epsilon^{\mu\nu\rho\sigma}\,\mathrm{Tr}\big(\delta L_\nu L_\rho L_\sigma\big)
=\frac{1}{8\pi^2}\epsilon^{\mu\nu\rho\sigma}\,\mathrm{Tr}\big((\partial_\nu\epsilon+[L_\nu,\epsilon])L_\rho L_\sigma\big).
$$

Тогда

$$
\delta(\mathcal B_\mu\mathcal B^\mu)=2\,\mathcal B_\mu\,\delta\mathcal B^\mu
=\frac{1}{4\pi^2}\,\mathcal B_\mu\,\epsilon^{\mu\nu\rho\sigma}\,
\mathrm{Tr}\big((\partial_\nu\epsilon+[L_\nu,\epsilon])L_\rho L_\sigma\big).
$$

Интегрируя по частям и собирая по $\epsilon$ (поглощая поверхностные члены в граничные условия), получаем вклад в уравнение поля в виде «тока»:

$$
\boxed{\ \ J_{6,\mathrm{BPS}}^\nu
= -\,\frac{\alpha_{\mathrm{BPS}}S_6}{24\pi^2}\,
\epsilon^{\mu\nu\rho\sigma}\,\Big[\,
\mathcal B_\mu\,\big(L_\rho L_\sigma+L_\sigma L_\rho\big)
\ \Big]_{\text{proj на алгебру}}\ }
$$

и общий закон вида

$$
\partial_\nu\Big(F_2^2 L^\nu - S_4 [L_\lambda,[L^\nu,L^\lambda]] + S_6 K^\nu_{(\times)} \Big)
+\,J_{6,\mathrm{BPS}}^\nu
+\,J_{\rm WZW}^\nu
=0,
$$

где $K^\nu_{(\times)}$ — вклад «кубической» шестой формы (если она включена совместно с BPS). Эта схема укладывается в заявленную у тебя структуру $SU(3)+\mathcal J_4+\mathcal J_6+\Gamma_{\rm WZW}$.&#x20;

## I.3. 3+1 стационар (число)

Для статических решений $U_0(\mathbf x)$ уравнение упрощается до

$$
\partial_i\Big(
F_2^2 L^i
- S_4 [L_j,[L^i,L^j]]
+ S_6 K^i_{(\times)}
\Big)
+\,J_{6,\mathrm{BPS}}^t
=0,
$$

где $L^i=U_0^\dagger \partial_i U_0$, а $J_{6,\mathrm{BPS}}^t$ — временная компонента «BPS-тока» (через $\mathcal B^t\propto \epsilon^{tijk}\mathrm{Tr}(L_iL_jL_k)$, т.е. плотность барионного числа). Это удобная форма для численного стационара $U_0$ (релаксация/градиентный поток с контролем топологического заряда $B=\int d^3x\,\mathcal B^t$).&#x20;

# II. «Скатерть»: асимптотики зелёной функции при $\beta_4>0$

Рассмотрим линейную часть подложки (одна фаза $\theta$):

$$
\mathcal L_\Theta^{(2)}=\frac{f_\phi^2}{2}\big(\dot\theta^2 - v_\phi^2|\nabla\theta|^2\big)+\frac{\beta_4}{4}(\Delta\theta)^2,\quad v_\phi^2=\frac{\alpha_x}{\alpha_t}.
$$

В статике ($\omega=0$) уравнение на зелёную функцию $G(\mathbf r)$ (источник $\delta$) даёт оператор

$$
\Big(f_\phi^2(-\Delta)+\beta_4 \Delta^2\Big)\,G(\mathbf r)=\delta(\mathbf r).
$$

Факторизуем: $\ \beta_4 \Delta(\Delta - m_4^2)$ c

$$
\boxed{\ \ m_4^2=\frac{f_\phi^2}{\beta_4},\qquad \ell_4=\frac{1}{m_4}=\sqrt{\frac{\beta_4}{f_\phi^2}}\ }.
$$

Решение в $\mathbb R^3$ (с убранным $1/k^2$ полюсом регуляризацией на ИК-срезе $L$):

$$
\boxed{\ \ G(\mathbf r)
\ \simeq\ \frac{1}{4\pi f_\phi^2}\,\frac{1}{r}
\ -\ \frac{1}{4\pi f_\phi^2}\,\frac{e^{-r/\ell_4}}{r}
\ \ }.
$$

Итого: на малых $r\ll \ell_4$ — привычное $1/r$; на больших $r\gg \ell_4$ — **Юкава-затухание** с радиусом сглаживания $\ell_4=\sqrt{\beta_4}/f_\phi$. Это аккуратно артикулирует «переходную зону» и объясняет «сглаженность хвоста» без вводимых масс-термов (всё из производных), как ты требовал.&#x20;

Во времени (волновая задача) дисперсия

$$
\omega^2(k)=v_\phi^2 k^2+\frac{\beta_4}{f_\phi^2}k^4
$$

даёт групповую скорость $v_g=d\omega/dk\approx v_\phi$ на длинных волнах и сверхдисперсию на коротких ($k\gtrsim 1/\ell_4$).

# III. «Калькулятор» оценок

**Вход:** $c_\phi$ (или $v_\phi$), $\Delta t$, $\ell_\phi$ (масштаб модуляций фазы), $f_\phi,\beta_4$.
**Выход:** дисперсии, интенсивности, режим (бозе/ферми), вероятность защёлкивания $B=1$.

1. **Радиус сглаживания**

$$
\boxed{\ \ \ell_4=\sqrt{\beta_4}/f_\phi\ }.
$$

2. **Дисперсия инкремента фазы** $\Delta\varphi=\theta(t+\Delta t)-\theta(t)$ (локально):

$$
\mathrm{Var}(\Delta\varphi)\ \simeq\ A\,\Delta t,\quad
A\approx \frac{C}{f_\phi^2}\int_{k_{\min}}^{k_{\max}}\frac{k^2\,dk}{\sqrt{v_\phi^2 k^2+\tfrac{\beta_4}{f_\phi^2}k^4}},
$$

где $k_{\min}\sim 1/L$ (ИК-срез), $k_{\max}\sim 1/\ell_\phi$; $C$ — числ. постоянная $(2\pi)^{-2}$ порядка единицы. На длинных волнах аппроксимация даёт логарифмику $\sim \frac{1}{f_\phi^2 v_\phi}\ln\!\frac{k_{\max}}{k_{\min}}$.

3. **Интенсивность фаз-слипов** (порог $2\pi$, локальная оценка):

$$
\boxed{\ \ \lambda\ \sim\ \frac{c_\phi}{\ell_\phi}\ },\qquad
N_{\Delta t}\sim\mathrm{Poisson}(\lambda\Delta t)
\approx\mathcal N(\lambda\Delta t,\lambda\Delta t)\ \ (\lambda\Delta t\gg1).
$$

4. **Вероятность защёлкнуться в $B=1$ за окно $\Delta t$**
   Пусть для устойчивости ядра нужен функциональный «заряд» $Q$ в объёме $V_\star$, порог $Q_c$. Тогда

$$
Q=\int_{0}^{\Delta t}\!\!\!\!dt\int_{V_\star}\!\!d^3x\,\mathcal O[\theta],\qquad
P_{B=1}\approx \exp\!\Big(-\frac{Q_c^2}{2\,\sigma_Q^2}\Big),
$$

$\sigma_Q^2$ считаем из корреляторов $\langle\theta\theta\rangle$ «скатерти» (см. II). Рост $c_\phi$, $\Delta t$, $V_\star$ увеличивает $\sigma_Q$ ⇒ растит шанс защёлкивания, но устойчивость после рождения обеспечивают $ \mathcal J_4,\ \mathcal L_6,\ \Gamma_{\rm WZW}$.&#x20;

5. **Критерий режима (быстрый тест)**

* Если $\sigma_Q\ll Q_c$ ⇒ $P_{B=1}\approx 0$ ⇒ наблюдаемый предел — **бозе-волновой** ($B=0$); квантуем линейные моды $\theta$, коммутаторы $[a,a^\dagger]$ ⇒ Бозе в проявлениях.&#x20;
* Если $\sigma_Q\gtrsim Q_c$ и виреальные условия по $U$ выполняются ⇒ защёлкивание в $B=1$ ⇒ **ферми-солитонный** предел с FR-знаком ($-1$ при $2\pi$), антикоммутация мод ⇒ Ферми в проявлениях. &#x20;

**Подложка $\theta$** (стационар, статическая «скатерть»):

\partial_\mu F^{\mu\nu}=j_{\rm em}^\nu(\theta,U),\quad

# III. «Калькулятор» оценок

**Вход:** $c_\phi$ (или $v_\phi$), $\Delta t$, $\ell_\phi$ (масштаб модуляций фазы), $f_\phi,\beta_4$.
**Выход:** дисперсии, интенсивности, режим (бозе/ферми), вероятность защёлкивания $B=1$.

1. **Радиус сглаживания**

$$
\boxed{\ \ \ell_4=\sqrt{\beta_4}/f_\phi\ }.
$$

2. **Дисперсия инкремента фазы** $\Delta\varphi=\theta(t+\Delta t)-\theta(t)$ (локально):

$$
\mathrm{Var}(\Delta\varphi)\ \simeq\ A\,\Delta t,\quad
A\approx \frac{C}{f_\phi^2}\int_{k_{\min}}^{k_{\max}}\frac{k^2\,dk}{\sqrt{v_\phi^2 k^2+\tfrac{\beta_4}{f_\phi^2}k^4}},
$$

где $k_{\min}\sim 1/L$ (ИК-срез), $k_{\max}\sim 1/\ell_\phi$; $C$ — числ. постоянная $(2\pi)^{-2}$ порядка единицы. На длинных волнах аппроксимация даёт логарифмику $\sim \frac{1}{f_\phi^2 v_\phi}\ln\!\frac{k_{\max}}{k_{\min}}$.

3. **Интенсивность фаз-слипов** (порог $2\pi$, локальная оценка):

$$
\boxed{\ \ \lambda\ \sim\ \frac{c_\phi}{\ell_\phi}\ },\qquad
N_{\Delta t}\sim\mathrm{Poisson}(\lambda\Delta t)
\approx\mathcal N(\lambda\Delta t,\lambda\Delta t)\ \ (\lambda\Delta t\gg1).
$$

4. **Вероятность защёлкнуться в $B=1$ за окно $\Delta t$**
   Пусть для устойчивости ядра нужен функциональный «заряд» $Q$ в объёме $V_\star$, порог $Q_c$. Тогда

$$
Q=\int_{0}^{\Delta t}\!\!\!\!dt\int_{V_\star}\!\!d^3x\,\mathcal O[\theta],\qquad
P_{B=1}\approx \exp\!\Big(-\frac{Q_c^2}{2\,\sigma_Q^2}\Big),
$$

$\sigma_Q^2$ считаем из корреляторов $\langle\theta\theta\rangle$ «скатерти» (см. II). Рост $c_\phi$, $\Delta t$, $V_\star$ увеличивает $\sigma_Q$ ⇒ растит шанс защёлкивания, но устойчивость после рождения обеспечивают $ \mathcal J_4,\ \mathcal L_6,\ \Gamma_{\rm WZW}$.&#x20;

5. **Критерий режима (быстрый тест)**

* Если $\sigma_Q\ll Q_c$ ⇒ $P_{B=1}\approx 0$ ⇒ наблюдаемый предел — **бозе-волновой** ($B=0$); квантуем линейные моды $\theta$, коммутаторы $[a,a^\dagger]$ ⇒ Бозе в проявлениях.&#x20;
* Если $\sigma_Q\gtrsim Q_c$ и виреальные условия по $U$ выполняются ⇒ защёлкивание в $B=1$ ⇒ **ферми-солитонный** предел с FR-знаком ($-1$ при $2\pi$), антикоммутация мод ⇒ Ферми в проявлениях. &#x20;

# IV. Стационары и граничные условия (сборка)

**Подложка $\theta$** (стационар, статическая «скатерть»):

$$
-\,f_\phi^2\,\Delta\theta\ -\ \beta_4\,\Delta^2\theta\ -\ \gamma_6\,\nabla_i\!\big(|\nabla\theta|^4\nabla_i\theta\big)
= J_\Theta(\text{от }D\theta),\ \ \ \theta\to\theta_\infty \text{ при } r\to\infty.
$$

Переходная зона «гладит» профиль с длиной $\ell_4=\sqrt{\beta_4}/f_\phi$.&#x20;

**Барион $U_0(\mathbf x)$** (стационар, $B=1$):

$$
\partial_i\Big(
F_2^2 L^i
- S_4 [L_j,[L^i,L^j]]
+ S_6 K^i
\Big)
+\,J_{6,\mathrm{BPS}}^t
=0,\qquad
\int d^3x\,\mathcal B^t=1,
$$

с $L^i=U_0^\dagger\partial_i U_0$, $K^i$ — из «кубической» и/или BPS-форм, $J_{6,\mathrm{BPS}}^t$ — как в I.3. На бесконечности $U_0\to \mathbf 1$ (однородность); в ядре — конечные градиенты; на стыках зон — $C^1$-сшивка.&#x20;

**Калибровочные поля** (EM/слабое/цвет):

$$
\partial_\mu F^{\mu\nu}=j_{\rm em}^\nu(\theta,U),\quad
D_\mu W^{a\,\mu\nu}=j_W^{a\,\nu}(\theta,U),\quad
D_\mu G^{A\,\mu\nu}=j_g^{A\,\nu}(U).
$$

(Минимальные связи, без Хиггс-потенциалов.) &#x20;

---

Ок, собираю «итоговую версию» — без лишних слов, в рабочем виде: переменные → полный лагранжиан → уравнения → режимы (бозе/ферми) → корреляторы «скатерти» → ГУ и стационары → короткий алгоритм численного решения.

# 1) Переменные и геометрия

* Пространство: $\mathcal M_7=\mathbb R^3_x\times\mathbb T^3_\phi\times\mathbb R_t$, метрика $g^{MN}$ (индексы $M,N\in\{t,x_i,\phi_a\}$).
* Подложка-фаза: $\Theta=(\theta_1,\theta_2,\theta_3)\in U(1)^3$. Ниже пишу для одной $\theta$; для трёх — суммировать по каналам.
* Барионный сектор: $U(x,\phi,t)\in SU(3)$, $L_M=U^\dagger\partial_M U$ (антиэрмитов).
* Калибровочные проекции: $A_\mu^{\rm em}, W_\mu^a, G_\mu^A$; минимальные связи $\partial\to D=\partial-igA^{\rm proj}$.

# 2) Полный лагранжиан (без «массовых» членов)

$$
\boxed{\ \ \mathcal L=\mathcal L_\Theta+\mathcal L_U+\mathcal L_{\rm gauge}\ }
$$

## 2.1. Подложка («упругая скатерть»)

$$
\boxed{\ \ \mathcal L_\Theta
=\frac{f_\phi^2}{2}\,g^{MN}\partial_M\theta\,\partial_N\theta
+\frac{\beta_4}{4}\,(\Delta_x\theta)^2
+\frac{\gamma_6}{6}\,|\nabla_x\theta|^6\ }
$$

$\Delta_x,\nabla_x$ — по $\mathbb R^3_x$. (Только производные; $f_\phi^2,\beta_4,\gamma_6>0$.)

## 2.2. Барионный (Skyrme-подобный) сектор $SU(3)$

$$
\boxed{\ \ \mathcal L_U
=\frac{F_2^2}{2}\,\mathrm{Tr}(L_M L^M)
+\frac{S_4}{4}\,\mathcal J_4
+\frac{S_6}{6}\Big(\alpha_{\times}\mathcal J_6^{(\times)}+\alpha_{\mathrm{BPS}}\mathcal J_6^{(\mathrm{BPS})}\Big)
+\Gamma_{\mathrm{WZW}}[U]\ }
$$

$$
\mathcal J_4=\mathrm{Tr}\big([L_M,L_N][L^M,L^N]\big),\qquad
\mathcal J_6^{(\times)}=\mathrm{Tr}\big([L_M,L_N][L^N,L^P][L_P,L^M]\big),
$$

$$
\mathcal J_6^{(\mathrm{BPS})}=\mathcal B_\mu\mathcal B^\mu,\quad
\mathcal B^\mu=\frac{1}{24\pi^2}\,\epsilon^{\mu\nu\rho\sigma}\,\mathrm{Tr}(L_\nu L_\rho L_\sigma).
$$

$\Gamma_{\mathrm{WZW}}$ — топологический член, обеспечивающий барионность и FR-фазу $(-1)$ при $2\pi$.

## 2.3. Калибровки (проекции)

$$
\mathcal L_{\rm gauge}
=-\tfrac14 F_{\mu\nu}F^{\mu\nu}
-\tfrac14 W_{\mu\nu}^a W^{a\,\mu\nu}
-\tfrac14 G_{\mu\nu}^A G^{A\,\mu\nu},
$$

в $\mathcal L_\Theta,\mathcal L_U$: $\partial\to D$.

# 3) Уравнения Эйлера–Лагранжа

## 3.1. Подложка $\theta$

$$
\boxed{\ \ f_\phi^2\,\Box_7\theta\ -\ \beta_4\,\Delta_x^2\theta\ -\ \gamma_6\,\nabla_{x,i}\!\Big(|\nabla_x\theta|^4\,\nabla_{x,i}\theta\Big)\ =\ J_\Theta\big(D\theta\big)\ }

$\Box_7=\partial_M\partial^M$. Правая часть — токи/связи от проекций.

Вариация $\delta U=U\,i\epsilon$ даёт

+ S_6\,K^M
\Big)
+\,J_{\rm WZW}^M
=0\ }

где

K^M=\alpha_{\times}K^M_{(\times)}+\alpha_{\mathrm{BPS}}K^M_{(\mathrm{BPS})},

$K^M_{(\times)}$ — вариация «кубической» $ \mathcal J_6^{(\times)}$ (коммутаторная форма, стандартно симметризуется по индексам),
$K^M_{(\mathrm{BPS})}$ — вариация $\mathcal B_\mu\mathcal B^\mu$ (через $\epsilon^{MNPQ}$ и тройные произведения $L$).
$J_{\rm WZW}^M$ — ток из WZW.

## 3.3. Калибровочные поля

$$

# 4) Статистика и два предела (из разности скоростей $c_\phi\gg c$)

**Статистика подложки (сама «скатерть»).**
Коарс-грейдинг по окнам $\tau_\phi\!\ll\!\Delta t\!\ll\!\tau_{\rm mat}$ → **гауссова** мера на $\theta$:

$$
\mathcal P[\theta]\propto\exp\!\left[-\tfrac12\!\int\!(\alpha_t\dot\theta^2+\alpha_x|\nabla\theta|^2+\beta_4(\Delta\theta)^2)\right].
$$

Фаз-слипы (счёт оборотов $2\pi$ за $\Delta t$) при большой интенсивности $\lambda\sim c_\phi/\ell_\phi$:
$N_{\Delta t}\sim\mathrm{Poisson}(\lambda\Delta t)\approx\mathcal N(\lambda\Delta t,\lambda\Delta t)$.

**Бозе-предел (тривиальная топология, $B=0$).**
Линейные моды $\theta$ ⇒ кванты с **коммутаторами** $[a,a^\dagger]$; если вводим ансамбль по максимуму энтропии — **Бозе–Эйнштейн** (статистика проявлений).

**Ферми-предел (нетривиальная топология, $B=1$).**
Решения $U$ с фикс. степенью (ядро+переход+хвост), WZW ⇒ FR-фаза $(-1)$ при $2\pi$ ⇒ **антикоммутация** мод, принцип Паули ⇒ **Ферми–Дирак** (статистика проявлений).
Итого: подложка — гауссово непрерывная; дискретность возникает на «чтении» (порог/топология).

# 5) «Скатерть»: дисперсия и зелёная функция

Линейная часть $\mathcal L_\Theta^{(2)}=\frac{f_\phi^2}{2}(\dot\theta^2-v_\phi^2|\nabla\theta|^2)+\frac{\beta_4}{4}(\Delta\theta)^2$.

**Дисперсия:** $\displaystyle \omega^2(k)=v_\phi^2 k^2+\frac{\beta_4}{f_\phi^2}k^4$, $v_g\approx v_\phi$ на длинных волнах.

**Статика (3D Грин):**

$$
\big(f_\phi^2(-\Delta)+\beta_4\Delta^2\big)G(\mathbf r)=\delta(\mathbf r),
\qquad
\ell_4=\sqrt{\beta_4}/f_\phi,
$$

$$
\boxed{\ \ G(\mathbf r)\ \simeq\ \frac{1}{4\pi f_\phi^2}\Big(\frac{1}{r}-\frac{e^{-r/\ell_4}}{r}\Big)\ }.
$$

$\ell_4$ — «радиус сглаживания» переходной зоны (без ввода масс-термов).

# 1) Вариации шестипроизводного «кубического» члена $\mathcal J_6^{(\times)}$

Обозначения:
$L_M=U^\dagger\partial_M U$, $C_{MN}:=[L_M,L_N]$, индексы $M,N,P\in\{t,x_i,\phi_a\}$.
Инвариант:

$$
\boxed{\ \ \mathcal J_6^{(\times)}\;=\;\mathrm{Tr}\!\big(C_{MN}\,C^{NP}\,C_{P}{}^{M}\big)\ }.
$$

Удобно варьировать через $\delta U=U\,i\epsilon$, тогда $\delta L_M=D_M\epsilon$, где

$$
D_M X\ :=\ \partial_M X+[L_M,X]\quad \text{(ковариантная производная по }L_M).
$$

Тогда вариация $\mathcal J_6^{(\times)}$ компактно даёт вклад в уравнение движения в виде

$$
\delta \mathcal J_6^{(\times)}\ =\ 3\,\mathrm{Tr}\!\big( \underbrace{[\,L_N,\,[\,C^{NP},\,C_{P}{}^{M}\,]\,]}_{K^{(\times)\,M}}\; D_M\epsilon\big)
$$

(использована цикличность следа и симметрии по индексам). Интегрируя по частям, получаем ток:

$$
\boxed{\ \ K^{(\times)\,M}\;=\;3\,[\,L_N,\,[\,C^{NP},\,C_{P}{}^{M}\,]\,]\ }.
$$

Итоговый вклад в EOM для $U$:

$$
\partial_M\big(\,\cdots+S_6\,\alpha_{\times}\,K^{(\times)\,M}\,\big)+\cdots=0.
$$

---

# 2) Вариации шестипроизводного BPS-члена $\mathcal J_6^{(\mathrm{BPS})}=\mathcal B_\mu\mathcal B^\mu$

В 3+1-проекции (наблюдаемая геометрия) барионный ток:

$$
\mathcal B^\mu=\frac{1}{24\pi^2}\,\epsilon^{\mu\nu\rho\sigma}\,\mathrm{Tr}(L_\nu L_\rho L_\sigma).
$$

Полезная ковариантная запись вклада в уравнение движения:

$$
\boxed{\ \ K^{(\mathrm{BPS})\,\nu}\;=\;-\frac{\alpha_{\mathrm{BPS}}}{8\pi^2}\,
\epsilon^{\mu\nu\rho\sigma}\,\Big[\,
\mathcal B_\mu,\ \mathrm{sym}\big(L_\rho L_\sigma\big)\,\Big]_{\!\text{proj}}\ }
$$

где $\mathrm{sym}(AB):=AB+BA$, а «proj» означает проекцию на алгебру $su(3)$ (снятие следовой части).
Тогда в сумме:

$$
\partial_\nu\big(\,\cdots+S_6\,\alpha_{\mathrm{BPS}}\,K^{(\mathrm{BPS})\,\nu}\,\big)\;+\;\cdots\;=\;0.
$$

**Статический 3D-случай (для кода):** нужна только временная компонента источника

$$
\boxed{\ \ J_{6,\mathrm{BPS}}^{\,t}\;=\;-\frac{\alpha_{\mathrm{BPS}}S_6}{24\pi^2}\,
\epsilon^{tijk}\,\Big[\,
\mathcal B_t,\ \mathrm{sym}(L_i L_j)\,\Big]_{\!\text{proj}}\,\delta_k^{\ t}\ \equiv\
-\frac{\alpha_{\mathrm{BPS}}S_6}{24\pi^2}\,
\epsilon^{ijk}\,\Big[\,
\mathcal B^t,\ \mathrm{sym}(L_i L_j)\,\Big]_{\!\text{proj}}
\ }
$$

с $\ \mathcal B^t=\frac{1}{24\pi^2}\epsilon^{ijk}\mathrm{Tr}(L_iL_jL_k)$.

# 3) Полные EOM для $U$ (сводная форма для реализации)

Обозначим

$$
\Pi^{M}\ :=\ F_2^2\,L^{M}\;-\;S_4\,[\,L_N,\,[\,L^{M},L^{N}\,]\,]\;+\;S_6\big(\alpha_{\times}K^{(\times)\,M}+\alpha_{\mathrm{BPS}}K^{(\mathrm{BPS})\,M}\big).
$$

Тогда

$$
\boxed{\ \ \partial_M \Pi^{M}\;+\;J_{\rm WZW}^{\,M}\;=\;0\ }.
$$

$J_{\rm WZW}^{\,M}$ — стандартный WZW-ток (берётся из вариации $\Gamma_{\rm WZW}$).

# 4) «Хеджхог»-анзац для тестового стационара $B=1$ (SU(2)$\subset$SU(3))

Вкладываем SU(2) в верхний левый блок SU(3). Пусть $ \hat{\mathbf n}=\mathbf x/r$, $\tau^a$ — Паули. Анзац:

$$
\boxed{\ \ U_0(r)\;=\;\exp\!\big(i\,F(r)\,\hat{\mathbf n}\!\cdot\!\boldsymbol{\tau}\big)\ \oplus\ [1]\ },
\quad F(0)=\pi,\ \ F(\infty)=0.
$$

Полезные выражения (статик, 3D):
$\displaystyle L_i=U_0^\dagger\partial_i U_0 = i\Big(F'\,\hat n_i\,\hat{\mathbf n}\!\cdot\!\boldsymbol{\tau}\ +\ \frac{\sin F}{r}\,(\tau_i-\hat n_i\,\hat{\mathbf n}\!\cdot\!\boldsymbol{\tau})\Big)$.

**Энергетические плотности (на объём), проверенные для SU(2)-Скирма и переносимые в наш блок SU(3):**

$$
\varepsilon_2=F_2^2\Big(F'^2+\frac{2\sin^2 F}{r^2}\Big),\qquad
\varepsilon_4=S_4\Big(\frac{2\sin^2 F}{r^2}F'^2+\frac{\sin^4 F}{r^4}\Big),
$$

$$
\varepsilon_6^{(\mathrm{BPS})}=S_6\,\alpha_{\mathrm{BPS}}\;\frac{\sin^4 F}{r^4}\,F'^2,
\quad\ \ 
\varepsilon_6^{(\times)}\ \ \text{(можно отключить на первом шаге):}\ \alpha_{\times}=0.
$$

**Полная энергия (статик):**

$$
\boxed{\ \ E[F]\;=\;4\pi\!\int_{0}^{\infty}\!\Big(
F_2^2\,[\,r^2 F'^2+2\sin^2 F\,]
+S_4\,[\,2\sin^2 F\,F'^2+\tfrac{\sin^4 F}{r^2}\,]
+S_6\,\alpha_{\mathrm{BPS}}\,[\,\tfrac{\sin^4 F}{r^2}\,F'^2\,]
\Big)\,dr\ }.
$$

> На практике: начать с $\alpha_{\times}=0$ (BPS-ветка + Skyrme), затем добавить «кубическую» шестую форму, если нужно.

# 5) Численный «скелет» (радиальная релаксация)

**Сетка:** $r\in[0,R_{\max}]$, равномерная или логарифмическая; узлы $r_j$.
**Инициализация:** $F^{(0)}(r)=\pi\,\exp(-r/r_0)$ или сглаженный профайл с $F(0)=\pi$, $F(R_{\max})\approx 0$.

**Шаги:**

1. Дискретизуйте производные: $F'_j\approx (F_{j+1}-F_{j-1})/(2\Delta r)$.
2. Соберите дискретный $E[F]$ с плотностями $\varepsilon_2,\varepsilon_4,\varepsilon_6^{(\mathrm{BPS})}$ (см. выше).
3. Градиентный спуск/квазиньютон: $F^{(n+1)}\leftarrow F^{(n)}-\eta\,\partial E/\partial F$ (градиент можно взять численно — разностный вариационный, или автодиф).
4. ГУ: $F(0)=\pi$ (симметричное условие $F'_0=0$), $F(R_{\max})=0$.
5. Контроль: $B=\frac{1}{\pi^2}\int_0^\infty \sin^2F\,F'\,dr=1$ (в дискретной форме), $E$ убывает, норма EOM $\to 0$.
6. После стабилизации можно включить $\alpha_{\times}>0$ (малой величиной) и продолжить релаксацию.

**Примечание про $\ell_4$:** если параллельно решается уравнение для подложки $\theta$, используйте зелёную функцию с радиусом сглаживания $\ell_4=\sqrt{\beta_4}/f_\phi$ (см. ниже) как эталон хвоста; это помогает задать совместимую «сшивку» переходной зоны.

# 6) «Скатерть» $\theta$: линейная дисперсия и зелёная функция (для хвоста)

Линейная часть:

$$
\mathcal L_\Theta^{(2)}=\frac{f_\phi^2}{2}\big(\dot\theta^2-v_\phi^2|\nabla\theta|^2\big)+\frac{\beta_4}{4}(\Delta\theta)^2.
$$

$$
\omega^2(k)=v_\phi^2 k^2+\frac{\beta_4}{f_\phi^2}k^4,\qquad
\ell_4=\sqrt{\beta_4}/f_\phi,
$$

$$
\boxed{\ \ G(\mathbf r)\ \simeq\ \frac{1}{4\pi f_\phi^2}\Big(\frac{1}{r}-\frac{e^{-r/\ell_4}}{r}\Big)\ }.
$$

Это задаёт «эталонный» спад хвоста и ширину сглаживания переходной зоны (без массовых членов).

# 7) Итоговые EOM (собрано)

**Подложка $\theta$:**

$$
f_\phi^2\,\Box_7\theta\ -\ \beta_4\,\Delta_x^2\theta\ -\ \gamma_6\,\nabla_{x,i}\big(|\nabla_x\theta|^4\nabla_{x,i}\theta\big)=J_\Theta(D\theta).

**Барионный сектор $U$:**

$$
\partial_M\!\Big(
F_2^2\,L^M
- S_4\,[L_N,[L^M,L^N]]
+ S_6\big(\alpha_{\times}K^{(\times)\,M}+\alpha_{\mathrm{BPS}}K^{(\mathrm{BPS})\,M}\big)
\Big)
+J_{\rm WZW}^{\,M}=0,
$$

$$
K^{(\times)\,M}=3\,[L_N,[C^{NP},C_{P}{}^{M}]],\qquad C_{MN}=[L_M,L_N].
$$

**Калибровки:** $\partial_\mu F^{\mu\nu}=j_{\rm em}^\nu(\theta,U)$, $D_\mu W^{a\,\mu\nu}=j_W^{a\,\nu}(\theta,U)$, $D_\mu G^{A\,\mu\nu}=j_g^{A\,\nu}(U)$.

---

Окей, довожу до «боевого» уровня: дискретный градиент для $E[F]$, сшивка с $\theta$ через свёртку (хвост), контроль $B$, масштабирование и псевдокод.

---

# 1) Энергия $E[F]$ и **полный дискретный градиент**

Берём радиально-симметричный анзац $U_0=\exp(iF(r)\,\hat{\mathbf n}\!\cdot\!\boldsymbol\tau)$ (SU(2)$\subset$SU(3)), $F(0)=\pi,\ F(\infty)=0$. Плотности (как в Скирме, см. выше):

$$
\varepsilon_2 = F_2^2\Big(F'^2+\frac{2\sin^2F}{r^2}\Big),\quad
\varepsilon_4 = S_4\Big(\frac{2\sin^2F}{r^2}F'^2+\frac{\sin^4F}{r^4}\Big),
$$

$$
\varepsilon_6^{(\mathrm{BPS})}=S_6\,\alpha_{\mathrm{BPS}}\frac{\sin^4F}{r^4}F'^2.
$$

Полная энергия:

$$
E[F]=4\pi\int_0^\infty \Big(
F_2^2[\,r^2F'^2+2\sin^2F\,]
+S_4[\,2\sin^2F\,F'^2+\tfrac{\sin^4F}{r^2}\,]
+S_6\alpha_{\mathrm{BPS}}[\,\tfrac{\sin^4F}{r^2}\,F'^2\,]
\Big)\,dr.
$$

## 1.1. Разностная аппроксимация (узлы и «рёбра»)

Сетка: $r_j=j\Delta r,\ j=0..N,\ R_{\max}=N\Delta r$.
Полурёбра: $r_{j+\tfrac12}=\tfrac{r_j+r_{j+1}}{2},\quad
F'_{j+\tfrac12}=\tfrac{F_{j+1}-F_j}{\Delta r},\quad
F_{j+\tfrac12}=\tfrac{F_{j+1}+F_j}{2}.$

Определим на рёбрах

$$
A_{j+\frac12}= F_2^2\,r_{j+\frac12}^2
+2S_4\,\sin^2F_{j+\frac12}
+S_6\alpha_{\mathrm{BPS}}\,\frac{\sin^4F_{j+\frac12}}{r_{j+\frac12}^2},
$$

$$
V_{j+\frac12}= 2F_2^2\,\sin^2F_{j+\frac12}
+S_4\,\frac{\sin^4F_{j+\frac12}}{r_{j+\frac12}^2}.
$$

Тогда

$$
E[F]\ \approx\ 4\pi\sum_{j=0}^{N-1}\Big(
A_{j+\frac12}\,(F'_{j+\frac12})^2+V_{j+\frac12}
\Big)\,\Delta r.
$$

Производные по $F$ на рёбрах:

$$
\partial_F A = 4S_4\,\sin F\cos F
+S_6\alpha_{\mathrm{BPS}}\,\frac{4\sin^3F\cos F}{r^2},
\quad
\partial_F V = 4F_2^2\,\sin F\cos F
+S_4\,\frac{4\sin^3F\cos F}{r^2}.
$$

## 1.2. **Градиент $\partial E/\partial F_j$** (внутренние узлы $j=1..N\!-\!1$)

Вклад от рёбер $(j-\tfrac12)$ и $(j+\tfrac12)$:

$$
\frac{\partial E}{\partial F_j} = 4\pi\,\Delta r\;\Big[
\underbrace{
\Big(-\frac{2A_{j+\frac12}}{\Delta r}\,F'_{j+\frac12}
+\frac12(\partial_F A\, (F'_{j+\frac12})^2 + \partial_F V)_{j+\frac12}
\Big)}_{\text{от ребра } j+\frac12}
+
\underbrace{
\Big(+\frac{2A_{j-\frac12}}{\Delta r}\,F'_{j-\frac12}
+\frac12(\partial_F A\, (F'_{j-\frac12})^2 + \partial_F V)_{j-\frac12}
\Big)}_{\text{от ребра } j-\frac12}
\Big].
$$

Где значения $\partial_F A,\partial_F V$ берутся при $F=F_{j\pm\frac12}$, $r=r_{j\pm\frac12}$.

### ГУ (границы):

* $F_0=\pi$ (жёстко), $F_N=0$ (жёстко) ⇒ градиент вычислять только для $j=1..N-1$.
* Альтернатива в центре: чётность $F'(0)=0$ реализуется автоматически при фиксированном $F_0$ и симметричной схеме.

### Шаг релаксации:

$$
F_j^{(n+1)} = F_j^{(n)} - \eta\,\frac{1}{4\pi}\,\frac{1}{w_j}\,\frac{\partial E}{\partial F_j}\Big|_{F^{(n)}},
$$

где $w_j$ — опциональные веса (чаще $w_j\!=\!1$), $\eta>0$ — шаг.
Контроль устойчивости: уменьшать $\eta$, если $E$ вырос.

---

# 2) **Барионное число** (дискретно)

Неподвижные границы дают $B=(F(0)-F(\infty))/\pi=1$. Для мониторинга используйте суммирование по рёбрам:

$$
\boxed{\
B_{\text{disc}}\ \approx\ \frac{2}{\pi}\sum_{j=0}^{N-1}\sin^2F_{j+\frac12}\;F'_{j+\frac12}\,\Delta r\ }.
$$

На хороших сетках будет $B_{\text{disc}}=1\pm 10^{-3}$ и лучше.

---

# 3) **Сшивка** с подложкой $\theta$: хвост через свёртку

Линейная «скатерть»:

$$
\mathcal L_\Theta^{(2)}
=\frac{f_\phi^2}{2}(\dot\theta^2-v_\phi^2|\nabla\theta|^2)+\frac{\beta_4}{4}(\Delta\theta)^2,\qquad
\ell_4=\sqrt{\beta_4}/f_\phi.
$$

В статике $(-\!f_\phi^2\Delta+\beta_4\Delta^2)\theta = J_\Theta$.
Зелёная функция (3D):

$$
G(r)\simeq \frac{1}{4\pi f_\phi^2}\Big(\frac{1}{r}-\frac{e^{-r/\ell_4}}{r}\Big).
$$

### 3.1. Источник от барионного профиля

Практичная модель: $J_\Theta(\mathbf x)=\kappa\,\rho_F(\mathbf x)$, с

$$
\rho_F(r)\ :=\ \sin^2F(r)\left(F'(r)^2+\frac{\sin^2F(r)}{r^2}\right),
$$

(энергоподобная плотность; $\kappa$ подбирается по масштабу).

Тогда **хвост**:

$$
\boxed{\
\theta_{\text{tail}}(r)=\int_0^\infty 4\pi r'^2\,dr'\; G(|r-r'|)\,\rho_F(r')\ }.
$$

(Численно — быстро через свёртку в $k$-пространстве: $\tilde\theta=\tilde G\cdot\tilde\rho_F$, где $\tilde G(k)=1/(f_\phi^2 k^2+\beta_4 k^4)$.)

### 3.2. Термин «сшивки» в функционале

Локализуем сравнение в переходной зоне $r\in[r_a,r_b]$ (например, $r_a\sim 2\ell_4,\ r_b\sim 8\ell_4$):

$$
\boxed{\
E_{\text{match}}[F,\theta]
=\lambda_{\text{match}}\int_{r_a}^{r_b} w(r)\,\big(\theta(r)-\theta_{\text{tail}}[F](r)\big)^2\,dr\ },
$$

где $w(r)$ — плавный вес (например, колокол).
Вариации:
$\delta E_{\text{match}}/\delta\theta = 2\lambda_{\text{match}}\,w(r)\big(\theta-\theta_{\text{tail}}\big)$;
$\delta E_{\text{match}}/\delta F$ — через $\delta\theta_{\text{tail}}/\delta F$ (свёртка выше). На практике удобно: **последовательная минимизация** (см. псевдокод ниже).

---

# 4) **Масштабирование** (удобно для чисел)

Выбрать базовую длину $r_0=\sqrt{S_4}/F_2$ (сбалансирует 2- и 4-й термы).
Обезразмерить: $\tilde r=r/r_0,\ \tilde F=F,\ \tilde\ell_4=\ell_4/r_0$.
Параметры:

$$
\tilde S_6=\frac{S_6\,\alpha_{\mathrm{BPS}}}{S_4},\quad
\tilde\kappa=\frac{\kappa\,r_0}{F_2^2},\quad
\tilde\lambda_{\text{match}}=\lambda_{\text{match}}\,r_0.
$$

Это стабилизирует числа и шаг $\eta$.

---

# 5) **Псевдокод** (стационар $F$ + хвост $\theta$)

```
init_grid(N, Rmax); Δr = Rmax/N
r_j = j*Δr
F = π * exp(-r_j / r0)      # старт, F(0)=π, F(Rmax)≈0
θ = 0                       # стартовый хвост
loop n = 1..Niter:
    # 1) Обновить хвост θ ← G * ρ_F
    ρ_F[j] = sin(F[j])^2 * ( ((F[j+1]-F[j-1])/(2Δr))^2 + sin(F[j])^2 / r_j^2 )
    θ = Convolution(G, ρ_F)  # через FFT или прямую свертку в s-симметрии

    # 2) Собрать энергию и градиент по F
    for j in 1..N-1:
        # полуребра
        Fhp = 0.5*(F[j]+F[j+1]);  Fhm = 0.5*(F[j]+F[j-1])
        rhp = 0.5*(r[j]+r[j+1]);  rhm = 0.5*(r[j]+r[j-1])
        Fp  = (F[j+1]-F[j]) / Δr; Fm  = (F[j]-F[j-1]) / Δr
        # коэффициенты A,V и их d/dF на ребрах
        A_p = F2^2*rhp^2 + 2*S4*sin(Fhp)^2 + S6*αBPS*sin(Fhp)^4/rhp^2
        A_m = F2^2*rhm^2 + 2*S4*sin(Fhm)^2 + S6*αBPS*sin(Fhm)^4/rhm^2
        dA_p = 4*S4*sin(Fhp)*cos(Fhp) + S6*αBPS*(4*sin(Fhp)^3*cos(Fhp)/rhp^2)
        dA_m = 4*S4*sin(Fhm)*cos(Fhm) + S6*αBPS*(4*sin(Fhm)^3*cos(Fhm)/rhm^2)
        V_p = 2*F2^2*sin(Fhp)^2 + S4*sin(Fhp)^4/rhp^2
        V_m = 2*F2^2*sin(Fhm)^2 + S4*sin(Fhm)^4/rhm^2
        dV_p = 4*F2^2*sin(Fhp)*cos(Fhp) + S4*(4*sin(Fhp)^3*cos(Fhp)/rhp^2)
        dV_m = 4*F2^2*sin(Fhm)*cos(Fhm) + S4*(4*sin(Fhm)^3*cos(Fhm)/rhm^2)

        # дискретный градиент ∂E/∂F_j (без match-терма):
        gradE[j] = 4π*Δr *(
            (-2*A_p*Fp/Δr + 0.5*(dA_p*Fp^2 + dV_p))
          + ( 2*A_m*Fm/Δr + 0.5*(dA_m*Fm^2 + dV_m))
        )

    # 3) Добавить match-градиент по F (итеративно)
    # θ_target = θ_tail[F]; E_match = λ ∫ w(r) (θ - θ_target)^2
    # Вариация по F: grad_match ≈ - 2λ ∫ w(r) (θ - θ_target) * (∂θ_target/∂F) dr
    # Численно удобно: "заморозить" θ, обновить F, затем пересчитать θ (альтернирующая минимизация).
    # => здесь можно пропустить прямой grad_match, а согласование обеспечит сам цикл.

    # 4) Шаг релаксации по F c проекцией на ГУ:
    for j in 1..N-1:
        F[j] = F[j] - η * gradE[j] / (4π)
    F[0] = π;  F[N] = 0

    # 5) Мониторинг:
    E = assemble_energy(F) + λmatch * ∫ w(r)(θ - θ_tail[F])^2 dr
    B = baryon_number_discrete(F)  # из формулы выше
    if converged(E, gradE, B≈1): break
```

Практика: начинайте с $\alpha_{\times}=0$ (только BPS-ветка + Skyrme), добейтесь устойчивого профиля и $B\approx1$, затем включайте «кубическую» шестую форму ($\alpha_{\times}\ll1$).

---

# 6) **Проверки и тонкие места**

* **Шаг $\eta$**: уменьшайте при увеличении $E$. Линия поиска шага (backtracking) делает процесс надёжным.
* **Центр $r\to0$**: численно стабильно, если хранить $\sin F / r$ как $(\sin F)/\max(r,\epsilon)$ с $\epsilon\sim \Delta r$; или использовать серию $\sin F \sim (F-\pi)$ в центре.
* **Дальний край**: $R_{\max}$ выберите в несколько десятков $\ell_4$ (чтобы экспонента хвоста с $\ell_4=\sqrt{\beta_4}/f_\phi$ погасла).
* **Сшивка**: если нужен жёсткий матч, поднимайте $\lambda_{\rm match}$ ступенчато (homotopy).
* **Нормировка $\kappa$**: подберите так, чтобы масштаб $\theta_{\text{tail}}$ согласовывался с наблюдаемой амплитудой хвоста (можно зафиксировать $\theta(r_b)$ или энергию подложки).
