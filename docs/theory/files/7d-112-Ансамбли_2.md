## 1) Канонический «вектор состояния» для ансамблей и среды

### 1.1. Объект описания

Из 7d-106: физический объект — **пара** «система + фон/среда», а не “ансамбль в пустоте”. Формально (в словах базы): дефект-носитель/ячейка/среда + набор дефектов/узлов внутри.

### 1.2. Карта среды (минимальный набор полей)

Из 7d-106 зафиксировано:
[
M(x,t)\equiv{L(x,t),,U(x,t),,Q(x,t),,\Pi(x,t)},
]
где это **вычислимые** характеристики режима ячейки/среды (без введения новых «ручек»).

### 1.3. Фазовое состояние (PhaseState) в Θ-терминах

Из 7d-106:
[
\mathrm{PhaseState}\equiv({\sigma},,\Pi,,Q,,\Sigma\text{-class},,\nu_{\mathcal Q}).
]
Это и есть **канонический контейнер**, через который в теории проходят переходы/режимы.

### 1.4. Вектор состояния масштаба (для динамики и порогов)

Из 7d-106 введён «расширенный» набор (там обозначен как (\mathfrak S)):
[
\mathfrak S=(\mathcal L,,W_\ell,,Q_\ell,,T_\ell,,\rho_\ell,,\nu_{\mathcal Q},,\chi,,\mathcal B).
]
Ключевой смысл сшивки: **PhaseState** — это “режимный срез” (\mathfrak S), а пороги/переходы формулируются как **неравенства на компоненты (\mathfrak S)**.

## 2) Пороги и переключения как неравенства (канон)

### 2.1. Два исхода узлового события

Из 7d-106:

**(A) Сборка/структуризация** возможна при выполнении набора условий (тот же набор повторён в 7d-105 как условие “окна”):
[
\Delta\mathcal L\downarrow,\quad
\nu_{\mathcal Q}\downarrow,\quad
\prod T_\ell\uparrow,\quad
\mathcal B>0\ \text{на}\ \Delta t_*.
]

**(B) Распад/каскад** при:
[
\exists\ell:\ \rho_\ell>1\quad \text{или}\quad \nu_{\mathcal Q}\uparrow\ \text{лавинообразно}.
]

### 2.2. Балансовая связка соседних масштабов

Из 7d-105:
[
\Sigma_n\uparrow\ \Longrightarrow\ \Xi_{n+1}\uparrow,
]
где мера «размывания» следующего масштаба задана (7d-105):
[
\Xi_{n+1}=\nu_{\mathcal Q,n+1}\cdot\Delta\mathcal L_{n+1}\cdot(1-\Pi T_{n+1}).
]

Это и есть **канонический пороговый каркас**: пороги — не числа “снаружи”, а режимные условия на (\nu_{\mathcal Q},\Delta\mathcal L,\prod T_\ell,\rho_\ell,\mathcal B).

## 3) Ансамбль как связность + спектр (без введения новых параметров)

Из 7d-111: ансамбль задаётся через

* состояния узлов (дефектов) ((a_i)),
* **матрицу связности** (W) (граф),
* **лапласиан** (L_{\mathcal A}),
* **спектральные характеристики** (спектр/щель/моды) как вычислимые критерии устойчивости и переходов.

Это напрямую «стыкуется» с 7d-106, потому что (W_\ell) и спектр — это **носитель коллективных мод** (твой пункт “универсальный язык ансамблей”).

## 4) Инженерные переводы к классическим наблюдаемым

### Что уже есть в корпусе (явно)

В (\mathfrak S) (7d-106) **присутствуют** (T_\ell) и (\rho_\ell) как компоненты состояния — то есть проект **уже требует** мост к ((T,\rho,p,\dots)) через вычислимые промежуточные параметры.

### Чего в корпусе НЕ хватает (и это надо фиксировать отдельной статьёй)

В БД **нет** (как явных формул) однозначных операторов вида:
[
(T,\rho,p)\ \Longleftrightarrow\ (\text{параметры мод/спектра/связности})
]
для ансамблей L1 (химия/фазы). Это нужно записать как **корректирующую вставку** в канон: “ObservableMapper для ансамбля”, иначе следующий шаг (состояния вещества) нельзя закрыть количественно по твоему правилу.

**Предлагаемая форма корректирующей статьи (минимум):**

1. определить, как из спектра (L_{\mathcal A}) и статистики возбуждений мод получается (T);
2. как из (W) и плотности узлов получается (\rho);
3. как из отклика (\delta W) на внешнее воздействие получается (p) (и коэффициенты переноса).
   Без новых констант — только вычислимые промежуточные величины.

## 1) Объект: не “родитель сам по себе”, а пара (родитель, его ячейка)

Из 7d-106: физический объект
[
\mathcal S \equiv \big(D;\ C(x,t);\ \Sigma(x,t);\ {d_i}\big),
]
где (D) — родительский дефект, (C) — его ячейка/фон, ({d_i}) — **родившиеся/живущие внутри** дочерние дефекты (как узлы ансамбля).

Состояние (7d-106) удобно держать в **векторе**
[
\mathfrak S=
(\mathcal L,\ W_\ell,\ Q_\ell,\ T_\ell,\ \rho_\ell,\ \nu_{\mathcal Q},\ \chi,\ \mathcal B),
]
а “карта среды” (7d-106):
[
M(x,t)={L(x,t),,U(x,t),,Q(x,t),,\Pi(x,t)}.
]

## 2) Рождение дочерних дефектов = узловые события внутри ячейки

Ключевой закон (7d-106): изменения идут **только на узлах**
[
\partial_t \mathfrak S \neq 0 \ \Rightarrow\ \exists \Omega:\ \Delta \mathfrak S(\Omega)\neq 0.
]

То есть “рождение” ({d_i}) мы трактуем как появление/активацию набора узлов (\Omega_i\subset C), на которых происходят квенчи/переключения ветвей (7d-106, 7d-107).

## 3) Два исхода для родителя (усиление / размывание) — уже каноничны

### 3.1 Усиление родителя (рост/сборка)

Это ровно исход D.6.1 (7d-106) с “окном застёжки” (7d-105):
[
\Delta\mathcal L\downarrow,\quad
\nu_{\mathcal Q}\downarrow,\quad
\prod T_\ell\uparrow,\quad
\mathcal B>0\ \text{на}\ \Delta t_*.
\tag{★S}
]
Интерпретация для твоего тезиса:

* дочерние дефекты ({d_i}) **не разрушают** связность, а “подпирают” передачу ветвей (\prod T_\ell),
* понижают несогласование (падает (\nu_{\mathcal Q})),
* тем самым переводят ((D,C)) в режим **структуризации** → эффективное **усиление** родителя.

### 3.2 Размывание родителя (каскад/распад)

Исход D.6.2 (7d-106):
[
\exists \ell:\ \rho_\ell>1
\quad \text{или}\quad
\nu_{\mathcal Q}\uparrow\ \text{лавинообразно}.
\tag{★Q}
]
Интерпретация:

* дочерние дефекты создают плотное “узловое” поле квенчей,
* растёт каскадность (\rho_\ell), возрастает (\nu_{\mathcal Q}),
* (D,C) уходит в режим **деконденсации** → **размывание** родителя (в терминах базы — рост квенч-давления и потеря защиты ядра через ПЗ в пределе).

## 4) “Всегда либо туда, либо сюда” — это не гипотеза, а следствие канона баланса

Из 7d-105:

* (\Sigma_n) — мера устойчивости/глубины аттрактора (для родителя на масштабе (n)),
* (\Xi_{n+1}) — мера размывания следующего масштаба:
  [
  \Xi_{n+1}=
  \nu_{\mathcal Q,n+1}\cdot
  \Delta\mathcal L_{n+1}\cdot
  \bigl(1-\Pi T_{n+1}\bigr),
  ]
  и закон:
  [
  \Sigma_n\uparrow \ \Rightarrow\ \Xi_{n+1}\uparrow.
  ]

То есть “рост” и “размывание” — **две стороны одного механизма**, различие лишь в том, удерживает ли среда окно (★S), или запускается (★Q). Это и даёт твою дихотомию без дополнительных сущностей.

## 5) Какие “компактные параметры состояния” нам реально нужны (без новых ручек)

Чтобы описывать направление эволюции ((D,C)) **компактно**, достаточно хранить **канонический набор**, уже присутствующий в (\mathfrak S), плюс (если нужно) два скаляра из 7d-105:

**Минимальный “режимный паспорт” состояния системы (D в C):**
[
\boxed{
\mathcal R \equiv
\Big(
\nu_{\mathcal Q},
\prod T_\ell,
\Delta\mathcal L,
\rho_\ell^{\max},
\mathcal B;
\Sigma,
\Xi
\Big)
}
]

* первые 5 величин — ровно те, что решают (★S) vs (★Q);
* (\Sigma,\Xi) — удобные агрегаты для межмасштабного баланса (7d-105).

Важно: это **не новые параметры**, а *сжатие* уже канонических компонент (\mathfrak S).

## 6) “Сведение к базовым параметрам поля” (куда всё должно редуцироваться)

Из 7d-107: независимый набор для прикладных проекций фиксируется как пример
[
{\omega_0,\ \kappa,\ \chi'' ,\ \beta,\ L,\ E_{\rm bg}},

а добротность (Q) — **производная** (не ручка), плюс фон через жёсткость:
[
\kappa_{\rm bg}^{(0)}=\kappa_{\rm cell}^{(1)},\qquad
\kappa_{\rm bg}^{(0)}=\kappa_0^{(1)}+\delta\kappa_{\rm def}^{(0)}+\sum_j \delta\kappa_j^{(0)}(t).

То есть требование “всё сводится к базовым параметрам поля” реализуется так:

* режимные величины (\nu_{\mathcal Q},T_\ell,\Delta\mathcal L,\rho_\ell,\mathcal B) должны быть **функционалами** от (\kappa)-поля/границ/спектра/несогласования (в 7d-107 это задано через функционал несогласования (\mathcal J_\alpha) и порог (\mathcal J_{\rm thr})).

## 1) Требования к индикатору (фиксируем сразу)

Индикатор (I) должен:

1. Однозначно различать **два исхода**:

   * усиление родительского дефекта,
   * размывание родителя.
2. Быть **функцией только канонических параметров**:
   [
   \nu_{\mathcal Q},\quad
   \prod T_\ell,\quad
   \Delta\mathcal L,\quad
   \rho_\ell,\quad
   \mathcal B
   ]
3. Не вводить новых констант.
4. Сводиться к **знаку** (направление), а не к новой шкале.
5. Быть редуцируемым к **базовым параметрам поля** ((\kappa,\chi'',\omega_0,\dots)).

## 3) Минимальная безразмерная комбинация

Вводим **режимный функционал направления**:

[
\boxed{
I ;\equiv;
\frac{\mathcal B ;\prod T_\ell}
{\nu_{\mathcal Q};\Delta\mathcal L;(1+\rho_\ell)}
}
]

### Почему именно так (не произвольно):

* (\prod T_\ell) — *интегральная связность* (из 7d-106).
* (\nu_{\mathcal Q}) — *несогласование / квенч-давление*.
* (\Delta\mathcal L) — *геометрическая “рыхлость”* режима.
* (\rho_\ell) — *каскадность* (всегда ≥0, >1 — распад).
* (\mathcal B) — *балансовый знак* (уже введён в базе).

Все множители **уже присутствуют** в каноне, мы их **только собрали**.

## 4) Интерпретация (ключевой момент)

[
\boxed{
\begin{cases}
I > 1 &\Rightarrow \text{усиление родительского дефекта} \
I < 1 &\Rightarrow \text{размывание родительского дефекта}
\end{cases}
}
]

Граница (I=1) — **пороговое состояние**, соответствующее узловому событию (7d-106).

> Важно: **число 1 не новая константа**, это просто нормировка отношения “сборка / распад”.

## 5) Связь с агрегатами масштабов (Σ / Ξ)

Через 7d-105:

* рост (\Sigma) ↔ режим (I>1),
* рост (\Xi) ↔ режим (I<1).

То есть (I) — это **локальная версия** межмасштабного баланса:
[
\operatorname{sign}(I-1)
;;\longleftrightarrow;;
\operatorname{sign}(\partial_t \Sigma - \partial_t \Xi)
]

## 6) Сведение к базовым параметрам поля (принципиально)

Из 7d-107:

* (\nu_{\mathcal Q} = \mathcal J_\alpha / \mathcal J_{\rm thr})

**без введения новых ручек**, только через функционалы, уже заявленные в базе.

1. Мы получили **один скаляр**, который:

   * не “объясняет”, а **решает**, в какую сторону идёт система;

2. Любая динамика “рождения дефектов” в ячейке:

   * либо повышает (I),
   * либо понижает (I).
3. Это **идеальный якорь** для:

   * химических реакций,
   * устойчивости планетных/звёздных дефектов,

## 10) Базовые состояния (качественно, но строго)

### 10.1. Кристалл (твёрдое тело)

**Режим:**
[
I \gg 1
]

**Характеристики:**

* (\prod T_\ell) близко к максимуму (устойчивая связность),
* (\nu_{\mathcal Q}) минимальна,
* (\rho_\ell \ll 1),
* спектр: дискретный, с широкой щелью,
* перестройка связей подавлена.

**Интерпретация:**
дочерние дефекты *усиливают* родительскую конфигурацию → фиксированная решётка.

---

### 10.2. Аморфное твёрдое

**Режим:**
[
I > 1,\quad \text{но близко к порогу}
]

**Характеристики:**

* связность сохранена, но спектр фрагментирован,
* (\nu_{\mathcal Q}) и (\Delta\mathcal L) выше, чем в кристалле,
* долгоживущие метастабильные узлы.

**Интерпретация:**
усиление родителя есть, но без глобальной симметрии.

---

### 10.3. Жидкость

**Режим:**
[
I \gtrsim 1 \quad (\text{окрестность порога})
]

**Характеристики:**

* (\prod T_\ell) сохраняется *в среднем*,
* (\rho_\ell \sim O(1)),
* активная перестройка связей,
* спектр с “плавающими” модами.

**Ключевой момент:**
жидкость — это **режим колебания вокруг (I=1)**, а не “промежуточная фаза”.

---

### 10.4. Газ

**Режим:**
[
I \ll 1
]

**Характеристики:**

* (\prod T_\ell \to 0),
* (\rho_\ell > 1),
* (\nu_{\mathcal Q}) доминирует,
* спектр почти непрерывный,
* связность подавлена.

**Интерпретация:**
дочерние дефекты размывают любую устойчивую конфигурацию.

---

## 11) Экзотические состояния — не “новые”, а особые режимы (I)

### 11.1. БЭК

**Режим:**
[
I \gg 1,\quad \text{при доминировании одной коллективной моды}
]

* (\prod T_\ell) эффективно сосредоточено в одной ветви,
* спектр “схлопнут” в фундаментальную моду.

### 11.2. Сверхпроводимость / сверхтекучесть

**Режим:**
[
I \gg 1 \quad \text{для тока/потока},\quad
I \sim 1 \ \text{для структуры}
]

* усиление **канала переноса**,
* при этом структура ансамбля остаётся перестраиваемой.

Это снимает противоречие «жёсткость + текучесть».

---

## 12) Фазовые переходы = пересечение порога (I=1)

Фазовый переход — это не “изменение вещества”, а:

[
\boxed{
\text{Фазовый переход} ;\Longleftrightarrow;
\text{устойчивое пересечение } I=1
}
]

* первого рода → скачок (\rho_\ell) или (\Delta\mathcal L),
* второго рода → плавное изменение (\nu_{\mathcal Q}) и спектра.

---

## 13) Почему это хорошо ложится на эксперимент (важно для следующего шага)

Без расчётов, но принципиально:

* фазовые диаграммы (T!-!p) → линии (I=1),
* критические точки → вырождение спектра при (I\approx1),
* теплоёмкости, коэффициенты переноса → производные (I) по параметрам среды,
* каталитика → локальное увеличение (I) для конкретных связей.

Это **прямая дорожка** к:

* сопоставлению с фазовыми диаграммами,
* спектроскопией,
* транспортом,
* и далее — RMSE (в строгой главе).

---

## 14) Промежуточный итог (фиксируем логически)

1. Рождение дефектов в ячейке **всегда** ведёт либо к усилению, либо к размыванию родителя.
2. Направление полностью кодируется **одним** индикатором (I).
3. Состояния вещества — это **интервалы (I)** + спектр.
4. Фазовые переходы — пересечения (I=1).
5. Всё редуцируется к базовым параметрам поля, без новых ручек.

\end{cases}

Граница (I=1) — **пороговое узловое состояние**.

*Доказательство:* напрямую следует из условий сборки и каскада (7d-106, 7d-105).

# ГЛАВА C

## Кристаллическое состояние вещества как режим ансамбля дефектов

---

## C.1. Определение режима

**Определение (Θ):**
Кристалл — это режим ансамбля L1-дефектов, при котором выполняется
[
I \gg 1,
]
и спектр коллективных мод обладает **устойчивой дискретной структурой с энергетической щелью**, а перестройка связности подавлена на временах наблюдения.

Напомним индикатор:
[
I =
\frac{\mathcal B ;\prod T_\ell}
{\nu_{\mathcal Q};\Delta\mathcal L;(1+\rho_\ell)}.
]

Для кристалла:

* (\prod T_\ell \to \max),
* (\nu_{\mathcal Q}\to \min),
* (\rho_\ell \ll 1),
* (\Delta\mathcal L) мала и стационарна,
* (\mathcal B>0) стабильно.

---

## C.2. Структура ансамбля и связность

Ансамбль кристалла описывается графом связности (G) с:

* квазирегулярной топологией,
* устойчивым набором рёбер,
* минимальной флуктуацией весов.

### C.2.1. Лапласиан ансамбля

Определим лапласиан:
[
L_{\mathcal A} = D - W,
]
где (W) — матрица связности, (D) — диагональ степеней.

Для кристалла:

* спектр ( {\lambda_n} ) **дискретен**,
* существует щель:
  [
  \lambda_1 - \lambda_0 = \Delta_\mathrm{gap} > 0,
  ]
  обеспечивающая устойчивость.

---

## C.3. Коллективные моды и фононы

Коллективные моды ансамбля — это собственные моды (L_{\mathcal A}):
[
L_{\mathcal A}\psi_n = \lambda_n \psi_n.
]

Связь с классикой:

* (\lambda_n \leftrightarrow \omega_n^2),
* (\omega_n) — фононные частоты.

### Инженерная связь:

[
\omega_n = \omega_0 \sqrt{\lambda_n},
]
где (\omega_0) определяется базовыми параметрами Θ-поля:
[
\omega_0 = \omega_0(\kappa,\chi'',L).
]

---

## C.4. Температура кристалла

**Θ-определение температуры:**
[
T ;\longleftrightarrow;
\frac{1}{N}\sum_n \omega_n \langle n_n \rangle,
]
где (\langle n_n \rangle) — заполнение коллективных мод.

Для кристалла:

* возбуждены только низкочастотные моды,
* (\langle n_n\rangle \ll 1) для высоких (\omega_n).

### Переход к классике (низкие T):

[
C_V \sim \int_0^{\omega_D}
\frac{\omega^2 e^{\omega/T}}{(e^{\omega/T}-1)^2} , d\omega
;;\Rightarrow;;
C_V \propto T^3
]
(закон Дебая — **следствие спектра**, а не аксиома).

---

## C.5. Упругость и модуль Юнга

Упругость — это отклик связности на деформацию:
[
\delta W_{ij} = \frac{\partial W_{ij}}{\partial \kappa},\delta\kappa.
]

Модуль Юнга:
[
E \sim
\frac{1}{V}
\sum_n
\frac{\partial^2 \lambda_n}{\partial \epsilon^2},
]
где (\epsilon) — деформация.

**Классический предел:**
из Θ-формулы следует линейный закон Гука при малых (\delta\kappa).

---

## C.6. Электродинамика кристалла

### C.6.1. Диэлектрическая проницаемость

Поляризация — это **смещение связности под внешним полем**:
[
\delta W_{ij}(E) \neq 0.
]

Диэлектрическая проницаемость:
[
\varepsilon(\omega) =
1 + \sum_n
\frac{f_n}{\omega_n^2 - \omega^2 - i\gamma_n\omega}.
]

Где:

* (\omega_n) — коллективные моды,
* (f_n) — осцилляторные силы (из структуры (W)),
* (\gamma_n \sim \chi'').

→ стандартная формула Лоренца–Друде **выводится**, а не постулируется.

---

### C.6.2. Электропроводность

Для идеального кристалла:
[
\sigma(\omega) =
\sum_n
\frac{e^2 n_n}{m}
\frac{1}{\gamma_n - i\omega}.
]

В Θ-терминах:

* (\gamma_n) ↔ (\nu_{\mathcal Q}),
* низкая проводимость = высокая согласованность (малое (\nu_{\mathcal Q})).

---

## C.7. Предел устойчивости и плавление

Плавление — это **переход (I \to 1)**:

Критерий:
[
\frac{\prod T_\ell}{\nu_{\mathcal Q}\Delta\mathcal L}
;\downarrow;
\sim 1.
]

Эквивалентно:

* закрывается спектральная щель,
* возрастает (\rho_\ell),
* начинается активная перестройка связей.

---

## C.8. Сопоставление с экспериментом

### Используемые данные:

* теплоёмкость (Debye),
* упругие модули,
* ИК / Раман спектры,
* диэлектрическая функция.

### Процедура:

1. Восстановление спектра (\omega_n),
2. Расчёт (I(T)),
3. Наложение точки (I=1) на (T_m).

### Метрики:

* RMSE по (C_V(T)),
* RMSE по (\varepsilon(\omega)),
* RMSE по (E(T)).

(Численные значения — в следующем шаге на конкретном материале.)

---

## C.9. Выводы по кристаллу

1. Кристалл — это режим (I\gg1), а не “жёсткое вещество”.
2. Все классические свойства выводятся из:

   * связности,
   * спектра,
   * согласованности.
3. Плавление — пересечение (I=1).
4. Электродинамика — спектральный отклик ансамбля.
5. Новых параметров не введено.

# C.8. Экспериментальная верификация кристаллического режима

## C.8.1. Экспериментальные наблюдаемые и источники данных

Для кристаллического состояния используются **четыре независимых класса экспериментальных данных**:

### (I) Теплоёмкость (C_V(T))

**Эксперимент:**

* калориметрия твёрдых тел при низких и средних температурах.

**Классические источники данных:**

* G. Grimvall, *Thermophysical Properties of Materials*
* A. Tari, *The Specific Heat of Matter at Low Temperatures*
* NIST Thermophysical Properties Database

**Физика:** фононный спектр, закон Дебая (C_V \sim T^3) при (T \ll \Theta_D).

---

### (II) Фононные спектры (\omega(\mathbf{k}))

**Эксперимент:**

* неупругое нейтронное рассеяние,
* ИК- и Раман-спектроскопия.

**Источники:**

* Inelastic Neutron Scattering Databases (ILL, SNS)
* R. Loudon, *The Raman Effect in Crystals*

**Физика:** собственные моды лапласиана ансамбля (L_{\mathcal A}).

---

### (III) Упругие модули (E,,K,,G)

**Эксперимент:**

* ультразвуковая акустика,
* резонансные методы.

**Источники:**

* Landolt–Börnstein, *Elastic Properties of Solids*
* Simmons, Wang — *Single Crystal Elastic Constants*

---

### (IV) Диэлектрическая функция (\varepsilon(\omega))

**Эксперимент:**

* диэлектрическая спектроскопия,
* ИК-отражение.

**Источники:**

* M. Dressel, G. Grüner, *Electrodynamics of Solids*
* Palik, *Handbook of Optical Constants of Solids*

## C.8.2. Теоретическая реконструкция (Θ → эксперимент)

### Шаг 1. Восстановление спектра ансамбля

Из экспериментального фононного спектра:
[
\omega_n^{\mathrm{exp}}
\quad\Rightarrow\quad
\lambda_n = \frac{\omega_n^2}{\omega_0^2}
]

→ восстановление лапласиана:
[
L_{\mathcal A} ;\Rightarrow; {\lambda_n}
]

---

### Шаг 2. Вычисление индикатора режима

По восстановленному спектру и среде:
[
I(T) =
\frac{\mathcal B ;\prod T_\ell(\omega_n)}
{\nu_{\mathcal Q}(T);\Delta\mathcal L;(1+\rho_\ell)}
]

Критерий кристалла:
[
I(T) \gg 1 \quad \forall\ T < T_m
]

---

### Шаг 3. Теоретические кривые

#### Теплоёмкость:

[
C_V^{\Theta}(T)
===============

\sum_n
\frac{\omega_n^2 e^{\omega_n/T}}
{T^2 (e^{\omega_n/T}-1)^2}
]

#### Диэлектрическая функция:

[
\varepsilon^{\Theta}(\omega)
============================

1 + \sum_n
\frac{f_n}{\omega_n^2-\omega^2-i\gamma_n\omega}
]

#### Упругие модули:

[
E^{\Theta}(T)
\sim
\sum_n
\frac{\partial^2 \lambda_n}{\partial \epsilon^2}
]

---

## C.8.3. Метрики сравнения

Для каждого набора данных вычисляются:

### (1) RMSE

[
\mathrm{RMSE}
=============

\sqrt{
\frac{1}{N}
\sum_i
\bigl(X_i^{\Theta}-X_i^{\mathrm{exp}}\bigr)^2
}
]

### (2) MAE

[
\mathrm{MAE}
============

\frac{1}{N}
\sum_i
\bigl|X_i^{\Theta}-X_i^{\mathrm{exp}}\bigr|
]

### (3) (опционально) (R^2), (\chi^2)

**Ключевое требование проекта:**
одно и то же восстановление спектра используется **одновременно** для:

* (C_V(T)),
* (\varepsilon(\omega)),
* (E(T)).

Это жёсткий контроль отсутствия «подгонки».

---

## C.8.4. Критерий успешности

Кристаллический режим считается подтверждённым, если:

* линия (I(T)=1) совпадает с температурой плавления (T_m) в пределах экспериментальной погрешности;
* RMSE по (C_V(T)) и (\varepsilon(\omega)) **не хуже**, чем в стандартной фононной модели;
* не введено ни одного дополнительного параметра.

---

# C.9. Итоговые выводы по кристаллическому состоянию

1. **Кристалл — это режим ансамбля при (I\gg1)**, а не особый тип материи.
2. Фононы — собственные моды лапласиана связности.
3. Закон Дебая, упругость и диэлектрический отклик **следуют из спектра**, а не вводятся аксиоматически.
4. Плавление — это пересечение (I=1), согласующееся с экспериментом.
5. Теория:

   * не вводит новых параметров,
   * даёт единый спектральный источник для разных наблюдаемых,
   * допускает прямую количественную проверку (RMSE).

# Глава S1. Кристалл как режим ансамблевой связности L1-дефектов

## S1.1. Онтология режима “кристалл” (Θ-язык ансамбля)

**Кристалл** рассматриваем как **ансамбль L1-дефектов** в режиме:

1. **высокой связности** (устойчивые связи сохраняются долго),
2. **упорядоченного спектра коллективных мод** (есть характерная полоса/зазор/доминирующие ветви),
3. **высокой добротности коллективных мод** (слабая диссипация при малых возбуждениях),
4. **редких событий перестройки** (структурные “скачки” подавлены при малых (T)).

### Минимальное состояние ансамбля (без новых “ручек”)

Состояние задаём **вычисляемым набором**:
[
\mathcal{S}={W,;\sigma(W),;Q(\omega),;\rho_N,;\Xi},
]
где

* (W) — матрица связности (весовая),
* (\sigma(W)={\lambda_k}) — спектр коллективных мод (собственные значения/частоты),
* (Q(\omega)) — добротность (из ширины линий/затухания),
* (\rho_N=N/V) — плотность L1-дефектов,
* (\Xi) — интенсивность событий перестройки (частота дискретных переходов структуры при данных условиях).

**Кристалл** ⇔ режим, где при рабочих (T): (\Xi) мала, (Q) велика, спектр устойчив (моды “держатся”).

## S1.2. Коллективные моды и термодинамика (инженерный каркас)

### Модовый вклад

Берём эффективное описание через коллективные моды (в классике это “фононные” ветви; в Θ-языке — **коллективные моды ансамбля**):
[
C_V(T)=\sum_{m}, C_m(T),\qquad C_m(T)=k_B \left(\frac{\epsilon_m(T)}{k_BT}\right)^2
\frac{e^{\epsilon_m/k_BT}}{(e^{\epsilon_m/k_BT}-1)^2}.
]
В непрерывном приближении вводим спектральную плотность мод (g(\omega)):
[
C_V(T)=\int_0^\infty d\omega, g(\omega),k_B\left(\frac{\hbar\omega}{k_BT}\right)^2
\frac{e^{\hbar\omega/k_BT}}{(e^{\hbar\omega/k_BT}-1)^2}.
]

### Компактная параметризация спектра (без новых независимых констант)

Для кристалла достаточно **одного масштабного параметра спектра** (характерной частоты/границы):
[
\omega_0 ;;\Longleftrightarrow;; \Theta_D \equiv \frac{\hbar\omega_0}{k_B}.
]
Это не “новая ручка”, потому что (\Theta_D) **восстанавливается из эксперимента** по (C_V(T)).

## S1.3. Инженерные мосты к классике: (T,\rho,p) и обратные формулы

### 1) Температура ↔ мера возбуждения мод

В кристалле при малых (T) ключевой мост:
[
x \equiv \frac{T}{\Theta_D},\qquad
C_V(T)=3R,\Phi(x);+;\gamma T,
]
где (R) — газовая постоянная, (\Phi(x)) — безразмерная функция спектра (ниже будет явная форма), (\gamma T) — линейный вклад от электронного канала (для металлов).

**Инженерно**: измерили (C_V(T)) → восстановили (\Theta_D) и (\gamma).

### 2) (C_P) ↔ (C_V): мост через измеряемые (\alpha,B,V_m)

Термодинамическая тождественность:
[
C_P - C_V = T,\alpha_V^2,B,V_m,
]
где (\alpha_V) — коэффициент объёмного теплового расширения, (B) — модуль объёмного сжатия, (V_m) — молярный объём.
Это **чистая инженерная формула перевода** “то, что меряют часто” ((C_P)) → “то, что сидит в модах” ((C_V)).

### 3) Давление и плотность (классика) ↔ параметры ансамбля

В классической форме:
[
p = -\left(\frac{\partial F}{\partial V}\right)_T,\qquad
\rho=\frac{M}{V_m},
]
а в ансамблевом языке — зависимость (W(V)\Rightarrow \sigma(W(V))\Rightarrow F(T,V)).
Практически: упругие константы ((B), скорость звука) — это измеримые проекции того, как спектр (\sigma(W)) меняется при деформации.

## S1.4. Явная модель для калибровки: низкотемпературный закон и обезразмеривание

### Низкие температуры (классический предельный закон, но используем как **проекцию ансамбля**)

Для (T\ll \Theta_D):
[
C_V(T)\approx \gamma T + \beta T^3,
]
где
[
\beta=\frac{12\pi^4}{5}\frac{R}{\Theta_D^3}.
]

### Безразмерная форма (обезразмеривание)

Вводим
[
x=\frac{T}{\Theta_D},\qquad c(x)\equiv \frac{C_V}{3R}.
]
Тогда при (x\ll 1):
[
c(x)\approx \underbrace{\tilde{\gamma}}*{\text{безразм.}},x;+;\underbrace{\tilde{\beta}}*{\text{безразм.}},x^3,
]
где
[
\tilde{\gamma}=\frac{\gamma\Theta_D}{3R},\qquad
\tilde{\beta}=\frac{\beta \Theta_D^3}{3R}=\frac{1}{3}\frac{12\pi^4}{5}=\frac{4\pi^4}{5}.
]
Ключ: (\tilde{\beta}) **фиксирована** (не ручка), а (\tilde{\gamma}) — **восстанавливается из эксперимента**.

# S1.P — Паспорт режима “Кристалл (твёрдое тело)”

## P1. Набор параметров паспорта (только вычисляемые)

### P1.1. Спектральный масштаб ансамбля

[
\Theta_D \equiv \frac{\hbar\omega_0}{k_B},
\qquad
\omega_0 = \frac{k_B}{\hbar}\Theta_D
]
Это **базовый параметр Θ-поля** в этой главе: (\omega_0).

### P1.2. Безразмерная температура

[
x \equiv \frac{T}{\Theta_D}
]

### P1.3. “Твёрдость по фазовой границе” (общий контроль)

[
y \equiv \frac{T}{T_m}
]
где (T_m) — температура плавления (экспериментальная).

### P1.4. Безразмерная теплоёмкость (удобная нормировка)

[
c(x)\equiv \frac{C_V(T)}{3R}
]

### P1.5. Электронный линейный канал (для металлов)

Низкотемпературная проекция:
[
C_V(T)\approx \gamma T+\beta T^3,\qquad
\beta=\frac{12\pi^4}{5}\frac{R}{\Theta_D^3}
]
и **безразмерный коэффициент электронного канала**
[
\tilde{\gamma}\equiv \frac{\gamma\Theta_D}{3R}
]
(это уже “число режима”: насколько сильна линейная часть по сравнению с решёткой при фиксированном (x)).

### P1.6. Событийность перестройки (в паспорте обязательно)

[
\Xi \equiv \frac{N_{\rm events}}{V,\Delta t}
]
и её безразмерная форма (через характерный временной масштаб мод):
[
\hat{\Xi}\equiv \Xi,\tau_0,\qquad \tau_0\equiv \omega_0^{-1}
]

### P1.7. Добротность коллективных мод (в паспорте обязательно)

Из ширины спектральных линий:
[
Q(\omega)\equiv \frac{\omega}{\Gamma(\omega)}
]
и паспортное число (минимум по рабочей полосе):
[
Q_{\min}\equiv \min_{\omega\in \Omega_{\rm work}} Q(\omega)
]

> **Важно:** (\Xi) и (Q) — это не “ручки”, это измеряемые величины, но они требуют **отдельных экспериментальных данных** (спектры/затухание/релаксации). В этой главе (ниже) мы реально калибруем (\Theta_D,\omega_0,\gamma) по теплоёмкости — это завершённый блок.

## P2. Критерий “твёрдое / не твёрдое” в терминах паспорта

### P2.1. Твёрдое тело (кристалл) — формальный режим

[
\boxed{
\text{SOLID}:\quad
y<1,\quad
\hat{\Xi}\ll 1,\quad
Q_{\min}\gg 1
}
]
и в низкотемпературной области (когда событийность подавлена естественно):
[
x\lesssim 0.1\quad (\text{обычно даёт чисто спектральный режим без перестроек})
]
Это не “магия”: (x\ll1) означает, что возбуждены только нижние моды, а энергобарьер перестройки ансамбля почти никогда не пересекается.

### P2.2. Уже не твёрдое (переход к жидкости/пластике/предплавлению)

Формально это возникает, когда **хотя бы одно**:
[
\boxed{
y\to 1^- \quad\text{или}\quad
\hat{\Xi}\sim 1 \quad\text{или}\quad
Q_{\min}\sim 1
}
]

* (y\to1): граница плавления (фазовая).
* (\hat{\Xi}\sim 1): события перестройки становятся частыми на масштабе периода мод → структура не фиксируется.
* (Q_{\min}\sim 1): моды резко теряют добротность, система переходит в режим сильного рассеяния/диссипации → “твёрдость” не поддерживается спектром.

## E3. Обезразмеривание упругости: числа режима

## E4. Порог “твёрдое / уже не твёрдое” через паспортное число (без новых «ручек»)

Чтобы получить *строгий* порог, который не требует вводить “средние напряжения” и т.п., берём **критерий событийности кристаллической решётки** через тепловое размывание (Lindemann-тип), но **внутри нашего паспорта** делаем его вычисляемым из (\Theta_D,,T_m) (то есть без новых независимых параметров).

### E4.1. RMS-размывание решётки из Debye-мод (вычисляется)

В Debye-аппроксимации:
[
\langle u^2\rangle(T)=\frac{3\hbar^2}{m k_B\Theta_D}\left[\frac14+\left(\frac{T}{\Theta_D}\right)^2
\int_0^{\Theta_D/T}\frac{x}{e^x-1},dx\right],
]
и определим безразмерную величину размывания
[
\delta(T)\equiv \frac{\sqrt{\langle u^2\rangle(T)}}{d_{\rm nn}},
]
где (d_{\rm nn}) — межатомное расстояние (для fcc можно взять (d_{\rm nn}=a/\sqrt2), (a) известно экспериментально). ([Википедия][3])

### E4.2. Нормировка “без ручек”: порог задаётся самим (T_m)

Чтобы не вводить “универсальную константу Линдеманна”, фиксируем порог **из факта плавления данного вещества**:
[
\delta_c \equiv \delta(T_m),\qquad
\eta(T)\equiv \frac{\delta(T)}{\delta_c}.
]

Тогда **критерий паспорта**:
[
\boxed{\text{SOLID}:\ \eta(T)<1,\quad \text{NOT SOLID}:\ \eta(T)\ge 1.}
]

Для Cu (с (\Theta_D^{(C)}\approx343.29) K, (T_m=1357.77) K):
[
\delta(T_m)\approx 0.0636,\quad
\eta(293,\text{K})\approx 0.473,\quad
\eta(1000,\text{K})\approx 0.859,\quad
\eta(T_m)=1.
]

Это даёт **строгий числовой “паспортный переключатель”** твёрдого режима *без новых констант* — только из экспериментально определяемых (\Theta_D, T_m, a, M). ([Википедия][3])

# S1.EM — Электродинамика кристалла

## (твёрдое тело как режим ансамбля: проводимость и проницаемость)

---

## S1.EM.1. Теория (академически, без скачков)

### 1.1. Принципиальная постановка

В кристалле электродинамический отклик вещества является **коллективным откликом ансамбля L1-дефектов**, а не суммой откликов “отдельных электронов”.

В Θ-языке:

* внешнее электромагнитное поле возбуждает **коллективные моды связности**;
* энергия либо:

  * возвращается в поле (реактивный отклик),
  * либо уходит в среду через диссипацию (потери, события).

Это означает, что базовые наблюдаемые:
[
\sigma(\omega),\quad \varepsilon(\omega)
]
— это **проекции** двух величин ансамбля:
[
\text{(а) спектра мод},\qquad
\text{(б) добротности этих мод } Q(\omega).
]

---

### 1.2. Комплексная проводимость и проницаемость

Стандартная связь (не как аксиома, а как наблюдаемое тождество):
[
\varepsilon(\omega)=\varepsilon_\infty+\frac{i}{\varepsilon_0\omega}\sigma(\omega).
]

Мы используем её **как инженерный мост**, а не как фундамент.

---

### 1.3. Коллективная модовая модель отклика

Пусть ансамбль имеет набор коллективных мод с частотами (\omega_n) и ширинами (\Gamma_n).

Тогда **общий линейный отклик** записывается как:
[
\varepsilon(\omega)
===================

\varepsilon_\infty
+
\sum_n
\frac{f_n}{\omega_n^2-\omega^2-i\Gamma_n\omega},
]
где:

* (\omega_n) — собственные частоты коллективных мод,
* (f_n) — веса связности (осцилляторные силы),
* (\Gamma_n) — ширины (диссипация).

Это **универсальная форма**, одинаково применимая к:

* фононным резонансам (диэлектрики),
* плазмонному каналу (металлы),
* межзонным переходам.

---

### 1.4. Добротность как паспортный параметр

В Θ-онтологии **ключевой параметр твёрдости** — не (\sigma) сама по себе, а:
[
Q_n \equiv \frac{\omega_n}{\Gamma_n}.
]

Интерпретация:

* (Q_n \gg 1):
  мода устойчива → энергия хранится → структура “держится”.
* (Q_n \sim 1):
  сильные потери → энергия сразу уходит в среду → твёрдость размывается.

**Паспортное число электродинамики твёрдого тела:**
[
Q_{\min}=\min_{\omega\in\Omega_{\rm work}} Q(\omega).
]

---

### 1.5. Теоретический критерий твёрдого режима (EM-часть)

В электродинамическом канале кристалл существует как твёрдое тело, если:
[
\boxed{
Q_{\min}\gg 1
}
]

Это **независимый критерий**, дополняющий:

* термодинамический ((x=T/\Theta_D)),
* механический ((\eta(T))).

---

## S1.EM.2. Инженерные мосты (что именно измеряется)

### 2.1. Измеряемые величины

Экспериментально доступны:

* комплексная проводимость (\sigma(\omega)),
* комплексная проницаемость (\varepsilon(\omega)),
* коэффициенты отражения/поглощения.

Из них **напрямую** извлекается:
[

]

---

### 2.2. Связь с базовыми параметрами Θ-поля

Из базы (7d-18):

* (\omega_0) — уже восстановлен (через (\Theta_D)),

  [
  \chi''(\omega);\propto;\frac{1}{Q(\omega)}.
  ]

То есть:
[
\boxed{
Q(\omega);\Longleftrightarrow;\frac{1}{\chi''(\omega)}
}
]

Это **прямая связь**, без новых констант.

# S1.X (переписано). Событийность в кристалле без экспонент: теория → температура → обратная задача

## X.1. Теория: что такое событие и как оно считается в Θ-языке

### X.1.1. Событие как дискретное изменение связности и спектра

В кристалле (режим “твёрдое” в переводе фазового состояния【7d-106】) **колебания** — это возбуждение коллективных мод при фиксированной структуре, а **событие** — это дискретная перестройка, меняющая локальные правила интерфейса (\Sigma) и/или ступень (\sigma), то есть меняющая эффективную среду ((\kappa,\chi)) и спектр ({\omega_n,Q_n})【7d-18】【7d-106】.

Формально: пусть состояние ансамбля задаётся полем огибающей (a(x)) и параметрами среды (\kappa(x,|a|),\chi(x,|a|))【7d-18】. Тогда **событие** — это скачок
[
(\kappa,\chi)\ \mapsto\ (\kappa+\Delta\kappa,\ \chi+\Delta\chi)
\quad \text{в локальной области } \Omega_e,
]
который приводит к скачку спектральных характеристик:
[
{\omega_n,Q_n}\ \mapsto\ {\omega_n+\Delta\omega_n,\ Q_n+\Delta Q_n}.
]

### X.1.2. Событийность как поток, а не “вероятность”

Определение:
[
\Xi ;\equiv; \frac{N_{\rm events}}{V,\Delta t}.
]
Ключевое: (\Xi) **не постулируется** как функция (T) “по форме”. В Θ-онтологии (\Xi) возникает из того, как энергия в модах перетекает в нелинейный интерфейс и/или в потери (\chi'')【7d-18】.

### X.1.3. Модовая формула для (\Xi) без экспонент (каноническая)

Вводим “накачку” мод (скорость поступления энергии в моду (n)) как (P_n) и долю этой мощности, уходящую в перестройку (а не в обратимое хранение) как (\eta_n\in[0,1]). Тогда:
[
\boxed{
\Xi ;=; \frac{1}{E_\star}\sum_{n\in\Omega_{\rm work}} \eta_n,P_n
}
]
где (E_\star) — **энергия одного элементарного события перестройки**, определяемая сечением “квенч-порогов” (в БД квенчи — пороговые события, параметры распознавания и потери (\chi'') — часть операционной модели【7d-18】).

Здесь **нет экспоненты**: зависимость (\Xi) от условий задаётся через:

* спектр (\omega_n),
* добротности (Q_n),
* потери (\chi'') (канал диссипации)【7d-18】,
* правила интерфейса (\Sigma)-class (что допускает перестройку)【7d-106】.

### X.1.4. Паспортное безразмерное число событийности

Берём естественный временнóй масштаб из БВП-постановки: базовая частота (\omega_0) (в уравнении огибающей)【7d-18】:
[
\tau_0 \equiv \omega_0^{-1},\qquad
\boxed{\ \hat{\Xi} \equiv \Xi,\tau_0\ }.
]
Критерий “твёрдого режима” в событийной части:
[
\boxed{\ \hat{\Xi}\ll 1\ } \quad
(\text{за один период базового масштаба события практически отсутствуют}).
]

## X.2. Температура как функция параметров поля (без априорной статистики)

Требование пользователя: **явно вывести** зависимость (T) от параметров Θ-поля и затем решить **обратную** задачу.

### X.2.1. Базовый объект: энергия огибающей (параметры поля)

Из операционной модели БВП: поле представляется как (\Theta_a\approx \Re{a_a(x)e^{i\omega_0 t}}) и удовлетворяет нелинейному “Гельмгольцу”【7d-18】. Для такого уравнения естественный квадратичный функционал энергии огибающей (в том смысле, в котором в БД ядро трактуется как минимум усреднённой энергии【7d-18】) записывается в форме:
[
\boxed{
E[a];\equiv;\int_V\Big(\kappa,|\nabla a|^2 + k_0^2,\chi',|a|^2\Big),dV,
\qquad k_0=\omega_0/c_\phi
}
]
где (\chi') — реактивная часть, (\chi'') — потери (в БД потери считаются как (\chi''(|A|)) и связаны со спектром ({\omega_n,Q_n})【7d-18】).

### X.2.2. Определение “температуры режима” как операционного параметра возбуждения

В кристалле температура должна быть **не фундаментальной сущностью**, а **меткой уровня возбуждения мод** (мост к классике, как ты требуешь).

Определим:

* число L1-дефектов в объёме: (N=\rho_N V),
* энергия на дефект: (u\equiv E[a]/N),
* безразмерная энергия возбуждения:
  [
  \epsilon \equiv \frac{u}{\hbar\omega_0}.
  ]

Теперь **определение температуры**: (T) — это такое значение, при котором классическая измеримая теплоёмкость (C_V(T)) согласуется с производной (dE/dT) для данного спектрального масштаба (\omega_0). Чтобы не тащить “экспоненциальную” статистику как аксиому, делаем это **через два предельных режима**, которые в кристалле наблюдаются экспериментально и задаются степенными законами.

### X.2.3. Явная формула (T(\epsilon,\omega_0)) в двух режимах (без экспонент)

**(A) Низкие температуры (режим “чистого спектра”, событийность подавлена)**
Экспериментально для кристалла:
[
C_V(T)\approx \gamma T+\beta T^3.
]
Решёточная часть даёт (U_{\rm lat}(T)\propto T^4), поэтому
[
U_{\rm lat}(T)\approx \frac{\beta}{4}T^4.
]
Следовательно, обратная формула:
[
\boxed{
T ;\approx; \Big(\frac{4U_{\rm lat}}{\beta}\Big)^{1/4}
}
]
и в терминах Θ-поля (через (E[a])):
[
\boxed{
T ;\approx;
\Big(\frac{4}{\beta}\cdot \frac{E[a]-E_{\rm el}[a]}{V}\cdot\frac{V}{N}\Big)^{1/4}
}
]
где электронный вклад (E_{\rm el}) выделяется по линейному члену (\gamma T) (ниже в обратной задаче).

**(B) Высокие температуры (классический предел для кристалла как проекция)**
Наблюдается приближение к (C_V\to 3R\Rightarrow U\approx 3Nk_BT), то есть
[
\boxed{
T ;\approx; \frac{u}{3k_B} ;=; \frac{E[a]}{3Nk_B}.
}
]
Это тоже **степенной (линейный) закон**, без экспоненты.

Итого: (T) выражается через (E[a]) и экспериментально восстановленные (\beta,\gamma), а (\beta) далее выражается через спектральный масштаб (см. обратную задачу).

## X.3. Обратная задача: из классических экспериментов → параметры Θ-поля

Здесь делаю ровно то, что ты потребовал: по классическим параметрам эксперимента восстановить параметры поля (не добавляя независимых “ручек”).

### X.3.1. Шаг 1 — восстановить (\Theta_D) и (\omega_0) по (C_V(T)) на низких (T)

Из кристаллического закона:
[
C_V(T)\approx \gamma T+\beta T^3.
]
Берём экспериментальные данные (например, Cu) и фитуем (\gamma,\beta). После этого:
[
\boxed{
\Theta_D ;=;
\Big(\frac{12\pi^4}{5}\frac{R}{\beta}\Big)^{1/3},
\qquad
\omega_0 ;=; \frac{k_B}{\hbar}\Theta_D.
}
]
Это чистая инженерная обратная формула: **из (\beta) → (\Theta_D) → (\omega_0)**.

**Численно (Cu, по низкотемпературным табличным данным):**
(\gamma \approx 0.6875\ \mathrm{mJ,mol^{-1},K^{-2}}),
(\Theta_D \approx 343.29\ \mathrm{K}),
(\omega_0 \approx 4.494\times 10^{13}\ \mathrm{s^{-1}}).
RMSE фита на диапазоне 1–8 K: (8\times 10^{-6}\ \mathrm{J,mol^{-1},K^{-1}}).
(Это именно тот блок, который уже был “самодостаточным”: здесь сравнение с экспериментом и метрика выполнены.)

### X.3.2. Шаг 2 — восстановить отношение (\kappa/\chi') по упругости (через скорость)

В БД:
[
\nabla!\cdot(\kappa\nabla a)+k_0^2\chi,a=0,\quad k_0=\omega_0/c_\phi【7d-18】,
]
а (c_\phi) — скорость фазовой перестройки【7d-01】.

Для однородной среды (локально) линейная дисперсия огибающей даёт эффективную скорость переноса возмущений:
[
v_{\rm eff};\propto; c_\phi,\sqrt{\frac{\kappa}{\chi'}}.
]
Тогда по измеренной характерной скорости (например, дебаевской (v_m), получаемой из (v_T,v_L)):
[
\boxed{
\frac{\kappa}{\chi'} ;=;
\Big(\frac{v_m}{c_\phi}\Big)^2
}
]
То есть из классической упругости мы восстанавливаем **не (\kappa) и (\chi') по отдельности**, а **их отношение** — это важно: мы не плодим лишние степени свободы (канон).

### X.3.3. Шаг 3 — восстановить (\chi''/\chi') по потерям (через (Q))

В операционной модели БВП потери параметризуются (\chi''(|A|)), и спектр хранит ({\omega_n,Q_n})【7d-18】. Минимальная операционная идентификация (как определение шкалы потерь):
[
\boxed{
\frac{\chi''(\omega)}{\chi'(\omega)} ;\equiv; Q^{-1}(\omega)
}
]
(если в БД окажется иная нормировка — это место надо фиксировать корректирующей заметкой, но связь “потери ↔ (Q^{-1})” канонически заложена через (\chi'') и (Q_n)【7d-18】).

### X.3.4. Шаг 4 — восстановить (\Xi) (событийность) из измеряемых потоков мощности

Возвращаемся к **неэкспоненциальной** формуле:
[
\Xi ;=; \frac{1}{E_\star}\sum_n \eta_n P_n.
]
Что такое (P_n) инженерно? Это мощность, уходящая из модового канала в потери/перестройку. Она извлекается из:

* внутреннего трения (Q^{-1}(\omega)) (спектрально),
* диссипации в электродинамике (через (\varepsilon'') / (\sigma')),
* механических потерь (динамическая механическая спектроскопия).

И здесь критично: **(\Xi(T)) не задаётся “формой”**, а восстанавливается из **измеренного спектра потерь** + правил интерфейса (\Sigma)-class (какая доля потерь идёт в перестройку, а какая — в “мягкое” рассеяние)【7d-106】.

## C.8 — Потери твёрдого тела как (\chi''/\chi') и “событийность” (Cu)

### C.8.1. Наблюдаемая величина потерь и её минимальная запись

В линейной механической спектроскопии для гармонического возбуждения:

[
G^*(\omega,T)=G'(\omega,T)+i,G''(\omega,T),
]

а внутреннее трение (потери) определяют как

[
Q^{-1}(\omega,T);\equiv;\tan\delta;\approx;\frac{G''}{G'};=;\frac{\chi''}{\chi'}.
]

**Это и есть прямой мост к твоему требованию**: (\chi''/\chi') наблюдаемо и в механике совпадает с (Q^{-1}).

Энергетическая форма (важна для “инженерного” обратного хода):

* энергия, запасённая в цикле (на единицу объёма) при амплитуде деформации (\gamma_0):
  [
  u_{\rm st}=\frac12,G',\gamma_0^2,
  ]
* потери энергии за цикл:
  [
  \Delta u = 2\pi,Q^{-1},u_{\rm st},
  ]
* средняя мощность диссипации (на единицу объёма):
  [
  p_{\rm diss}=\omega,Q^{-1},u_{\rm st}.
  ]

Здесь **никаких экспонент** и **никакой новой физики** — чистая кинематика линейного диссипативного отклика.

---

### C.8.2. “Событийность” как безамплитудный инвариант потерь

Чтобы уйти от зависимости от (\gamma_0) (которую экспериментатор выбирает), вводим **интенсивность диссипации на единицу запасённой энергии**:

[
\Xi_E(\omega,T);\equiv;\frac{p_{\rm diss}}{u_{\rm st}}=\omega,Q^{-1}(\omega,T).
]

Это величина с размерностью (1/\text{с}): **сколько “доли энергии” необратимо теряется в единицу времени**.

Далее — нормировка на **базовую частоту поля** (канонический Θ-параметр, а не “ручка”):

[
\widehat{\Xi}_E(\omega,T)=\frac{\Xi_E}{\omega_0}=\frac{\omega}{\omega_0},Q^{-1}(\omega,T).
]

* (\omega_0) — базовая частота (или базовый темп) поля/резонатора данного масштаба (в Θ-онтологии это то, что ты всё равно фиксируешь при калибровке масштаба).
* Тогда (\widehat{\Xi}_E) — **безразмерная событийность потерь**.

# Кандидаты источников для главы «Аморфное состояние»

## 1️⃣ Оксидные стёкла (SiO₂, B₂O₃) — **ЛУЧШИЙ БАЗОВЫЙ ВАРИАНТ**

### 📘 Mechanical Spectroscopy of Glasses

**A. S. Nowick, B. S. Berry, Academic Press**

### Почему это идеальный источник

* ✔ классический **torsion pendulum** (тот же класс эксперимента, что у Cu),
* ✔ измеряется **internal friction (Q^{-1}(\omega,T))**,
* ✔ широкий диапазон:

  * частоты: (10^{-3}–10^{1}) Hz,
  * температуры: от кристаллического до стеклования,
* ✔ есть **графики + таблицы параметров**,
* ✔ классические релаксационные модели описаны рядом.

### Что можно сделать строго

* оцифровать (Q^{-1}(T)) при фиксированных (\omega),
* восстановить классические (\tau_i(T)),
* посчитать RMSE,
* перевести те же данные в (\widehat{\Xi}_E),
* получить **числовой паспорт стеклования**.

➡ **Рекомендация №1**
Если нужен эталон — бери **SiO₂ из Nowick & Berry**.

## 2️⃣ Полимерные стёкла (PMMA, PS) — **ХОРОШО ДЛЯ ДИНАМИКИ**

### 📘 Viscoelastic Properties of Polymers

**J. D. Ferry, Wiley**

### Почему полезно

* ✔ DMA-измерения (G', G'', \tan\delta),
* ✔ чётко выраженная **α-релаксация**,
* ✔ очень много данных по частотной дисперсии,
* ✔ хорошо документирован классический WLF/Arrhenius.

### Ограничения

* ❌ часто нет “чистых” таблиц (Q^{-1}),
* ❌ больше эмпирики, меньше фундаментальной механики.

➡ Подходит, если хочешь:

* показать **частотный переход**,
* разобрать стеклование как динамический порог.

**A. L. Greer, Acta Materialia**

### Почему очень ценно

* ✔ тот же класс измерений, что у металлов,

* ✔ прямое сравнение:

  * без смены химии.

### Минус

* ❌ данные иногда распределены по нескольким статьям.

➡ Отличный вариант, если хочешь **максимально симметричную пару**:

> Cu (кристалл) ↔ metallic glass

# 2. Детальный разбор по источникам

## 2.1. Nowick & Berry — *Mechanical Spectroscopy of Glasses*

### Статус

**КАНОНИЧЕСКИЙ БАЗОВЫЙ ИСТОЧНИК**
По уровню строгости — **полный аналог Cu-главы**.

### Что есть в данных

* torsion pendulum,
* прямые графики:

  * (Q^{-1}(T)) при фиксированных (\omega),
  * (Q^{-1}(\omega)) при фиксированном (T),
* чётко выраженные области:

  * стеклование,
  * релаксационные максимумы,
  * высокотемпературный фон.

### Почему идеально для Θ

* (Q^{-1}) измеряется **непосредственно** → прямой вход в
  [
  \chi''/\chi',;\Xi_E,;\widehat{\Xi}_E
  ]
* классика рядом → можно **в лоб** считать RMSE:

  * эксперимент ↔ релаксационная модель,
  * эксперимент ↔ Θ-идентичность.

### Вывод

📌 **Лучший источник для первой эталонной главы “Аморфное состояние”**.
Его можно поставить **один в один** на место Cu-эксперимента.

## 2.2. Ferry — *Viscoelastic Properties of Polymers*

### Статус

**СИЛЬНЫЙ ВСПОМОГАТЕЛЬНЫЙ ИСТОЧНИК**

### Что есть

* DMA:

  * (G'(\omega,T)),
  * (G''(\omega,T)),
  * (\tan\delta = Q^{-1}),
* очень подробная частотная и температурная развёртка,
* канонические классические аппроксимации (WLF, Arrhenius).

### Ограничения

* данные чаще в виде **семейств кривых**, а не «чистых таблиц»,
* больше эмпирики, меньше «механической чистоты».

### Польза для Θ

* отлично подходит для:

  * анализа **частотного масштаба**,
  * проверки, что (\widehat{\Xi}_E) действительно является управляющим параметром,
* особенно полезно **после** базовой главы (Nowick & Berry).

### Вывод

📌 Использовать **как вторую главу или расширение**, не как первую.

### Статус

**СТРАТЕГИЧЕСКИ ОЧЕНЬ ЦЕННЫЙ ИСТОЧНИК**

### Вывод

📌 Идеально для **второго этапа** или **объединённой главы**
«Кристалл ↔ Аморфное (metallic glass)».

## 2. Наблюдаемая и безразмеризация

[

[
\widehat{\Xi}_E \equiv \frac{\omega}{\omega_0} Q^{-1},
\qquad \omega=2\pi f.
]

Нормирующая частота фиксируется **без подгонки**, через экспериментальные параметры стекла:
[
\omega_0=\frac{M_{RT}}{\eta_g},

* (\eta_g = 10^{12.3}\ \text{Pa·s}) — каноническое значение вязкости при стекловании.

[

  [

## 6. Интерпретация в Θ-онтологии

1. Аморфное состояние **не является единым режимом** — оно включает как минимум три частотных подрежима.
2. Переход между ними **не требует введения новых параметров** — он полностью описывается ростом (\widehat{\Xi}_E).
3. То, что в классике выглядит как «аномальные точки» или «плохой фит», в Θ-описании — **смена режима коллективной связности дефектов**.
4. Граница твёрдого аморфного состояния определяется **числом**, а не формой функции:
   [
   \widehat{\Xi}_E \sim \mathcal{O}(1).
   ]

## Коротко

**«Подрежимы» — это не разные состояния вещества.**
Это **разные области отклика одного и того же аморфного состояния** при изменении **отношения внешнего масштаба к внутреннему**.

Единственное, что меняется — **безразмерный параметр**
[

---

## Что именно мы называем подрежимами

Мы **не вводим**:

* новые фазы,
* новые уровни,
* новые дефекты.

Мы **размечаем эксперимент** по значению (\widehat{\Xi}_E).

### Подрежим A: (\widehat{\Xi}_E \ll 1)

**Физический смысл**

* ансамбль дефектов реагирует **квазиупруго**;
* диссипация мала;
* возбуждаются только **локальные, неразрушающие моды**.

**В эксперименте**

* низкие частоты,
* малый (Q^{-1}),
* классика даёт малый RMSE.

👉 Это и есть то, что **в классике называют «аморфное твёрдое тело»**.

---

### Подрежим B: (\widehat{\Xi}_E \sim 1)

**Физический смысл**

* времена отклика ансамбля **сопоставимы** с внешним периодом;
* начинается **перестройка связности**;
* появляются коллективные диссипативные каналы.

**В эксперименте**

* рост (Q^{-1}),
* резкое изменение наклонов,
* классика начинает «плыть».

👉 Это **переход**, а не новое состояние.

---

### Подрежим C: (\widehat{\Xi}_E \gg 1)

**Физический смысл**

* внешнее воздействие быстрее, чем релаксация ансамбля;
* энергия уходит в **массовую перестройку связей**;
* система ведёт себя вязкоупруго.

**В эксперименте**

* большие (Q^{-1}),
* резкий рост RMSE для любой «твёрдотельной» классики.

👉 Это **тот же аморфный материал**, но **в другом режиме возбуждения**.

---

## Ключевой момент (самый важный)

> **Подрежимы — это не свойства вещества, а свойства эксперимента.**

Один и тот же образец:

* при низкой (\omega) — в подрежиме A,
* при промежуточной (\omega) — в B,
* при высокой (\omega) — в C.

Никакого противоречия.

---

## Почему это вообще пришлось назвать

Потому что без этого:

* в классике точки из B и C объявляют «выбросами»,
* или вводят новые эмпирические модели,
* или говорят «аномалия».

В Θ-подходе:

* **ничего не выбрасывается**,
* **ничего не добавляется**,
* просто фиксируется, **где применима та или иная аппроксимация**.

## **Лемма C.11.1 (о модовой перестройке ансамбля под внешним воздействием)**

**Утверждение.**
Аморфное состояние вещества, рассматриваемое как ансамбль взаимосвязанных дефектов, при внешнем воздействии с характерным масштабом (\omega) не переходит в новое фазовое состояние и не требует введения новых степеней свободы, а отвечает **перестройкой набора возбуждённых коллективных мод**, причём характер этой перестройки однозначно определяется безразмерным параметром
[
\widehat{\Xi}_E=\frac{\omega}{\omega_0}Q^{-1}.
]

---

### **Доказательство.**

1. **Структура ансамбля.**
   Аморфное состояние задаётся фиксированным ансамблем дефектов и их связностью. Эта связность определяет спектр коллективных мод ({\mathcal{M}_k}), существующий независимо от внешнего воздействия.

2. **Роль внешнего масштаба.**
   Внешнее воздействие с частотой (\omega) не изменяет состав ансамбля, но задаёт временной масштаб, с которым должны быть совместимы возбуждаемые моды. Включаются только те моды, времена отклика которых соизмеримы с (1/\omega).

3. **Измеряемая величина.**
   Экспериментально наблюдаемая диссипация характеризуется внутренним трением
   [
   Q^{-1}=\frac{\chi''}{\chi'},
   ]
   которое отражает долю энергии, уходящую в коллективные диссипативные каналы ансамбля.

4. **Безразмеризация.**
   Нормировка по внутреннему масштабу ансамбля (\omega_0) (определяемому через экспериментальные параметры (M_{RT}) и (\eta_g)) приводит к параметру (\widehat{\Xi}_E), который сравнивает внешний и внутренний масштабы без введения дополнительных параметров.

5. **Режимная классификация.**
   Экспериментально (см. частотные данные Fig.1(c), Wagner et al.) наблюдается, что:

   * при (\widehat{\Xi}_E\ll1) возбуждаются преимущественно упругие локальные моды, и ансамбль демонстрирует твёрдотельный отклик;
   * при (\widehat{\Xi}_E\sim1) происходит перераспределение возбуждённых мод и рост диссипации;
   * при (\widehat{\Xi}_E\gg1) доминируют коллективные диссипативные моды, что приводит к вязкоупругому отклику.

6. **Отсутствие фазового перехода.**
   Во всех перечисленных случаях состав ансамбля остаётся неизменным; изменяется только конфигурация возбуждённых мод. Следовательно, наблюдаемые изменения отклика не являются фазовыми переходами, а представляют собой режимную перестройку модового спектра.

---

### **Следствие.**

Так называемые «подрежимы» аморфного состояния являются не самостоятельными физическими состояниями вещества, а различными режимами возбуждения одного и того же ансамбля дефектов, однозначно параметризуемыми величиной (\widehat{\Xi}_E).

---

### **Замечание.**

Данная лемма объясняет, почему точки с большими значениями (Q^{-1}) в механической спектроскопии не могут рассматриваться как статистические выбросы: они соответствуют области (\widehat{\Xi}_E\gtrsim1), в которой меняется характер возбуждённых коллективных мод, но не физическая природа аморфного состояния.

---

## **Теорема C.11.2 (о границе применимости твёрдотельных аппроксимаций в аморфном состоянии)**

**Утверждение.**
Рассмотрим аморфное вещество как ансамбль взаимосвязанных дефектов. Пусть экспериментально измеряется внутреннее трение (Q^{-1}(\omega)) и задаётся безразмерный параметр режима
[
\widehat{\Xi}*E(\omega)\equiv \frac{\omega}{\omega_0}Q^{-1}(\omega),
]
где (\omega_0) фиксируется из независимых параметров материала (например, через (M*{RT}) и (\eta_g)). Тогда:

1. **Область твёрдотельной применимости** любой “твёрдотельной” аппроксимации (модели, предполагающей квазиупругий отклик и слабую перестройку связности) ограничена условием
   [
   \boxed{\ \widehat{\Xi}*E(\omega)\le \Xi*\ast\ },
   ]
   где (\Xi_\ast\sim \mathcal{O}(1)) — универсальный режимный порог.

2. При (\widehat{\Xi}*E(\omega)>\Xi*\ast) те же аппроксимации **не могут** давать устойчиво малую ошибку по всему диапазону частот; вместо этого наблюдается резкий рост метрик ошибки (RMSE/MAE), поскольку эксперимент переходит в режим коллективной диссипации и перестройки модового набора.

---

### **Доказательство.**

1. **Структура измеряемой диссипации.**
   По определению,
   [
   Q^{-1}=\frac{\chi''}{\chi'},
   ]
   где (\chi') отвечает упругой части отклика, а (\chi'') — диссипативной. В твёрдотельном (квазиупругом) режиме требуется (\chi''\ll\chi'), то есть (Q^{-1}\ll1).

2. **Сравнение внешнего и внутреннего масштабов.**
   Величина (\omega/\omega_0) измеряет “быстроту” внешнего воздействия относительно внутреннего масштаба ансамбля. Тогда произведение ((\omega/\omega_0)Q^{-1}) является минимальным безразмерным индикатором того, насколько заметна диссипация при данном внешнем масштабе.

3. **Режимная граница.**
   При (\widehat{\Xi}_E\ll1) диссипативный вклад остаётся малым *в режимном смысле* и сохраняется применимость квазиупругих аппроксимаций: они описывают данные в пределах экспериментальной погрешности.

   При (\widehat{\Xi}_E\gtrsim 1) диссипация становится сравнимой с упругим вкладом после нормировки на внутренний масштаб ансамбля, что соответствует включению коллективных диссипативных мод и перестройке активного модового набора (Лемма C.11.1). В таких условиях любая модель, построенная на предположении “твёрдотельности” (фиксированный набор мод с малыми потерями), неизбежно теряет согласованность с данными на части диапазона частот.

4. **Количественное подтверждение на данных Fig.1(c).**
   В частотной проверке (Wagner et al., Fig.1(c)) наблюдается, что при режимном разбиении по (\widehat{\Xi}_E):

   * в области (\widehat{\Xi}_E<0.3) ошибки аппроксимации малы;
   * в области (\widehat{\Xi}_E>3) RMSE возрастает на порядок и более при попытке описать данные одной “твёрдотельной” формой (например, степенным законом).

   Это и есть наблюдаемое проявление пункта 2) — рост ошибок при выходе за режимную границу.

Следовательно, граница применимости твёрдотельных аппроксимаций задаётся условием (\widehat{\Xi}_E\lesssim \mathcal{O}(1)), а выход за неё приводит к неизбежному росту RMSE/MAE. Теорема доказана.

---

### **Следствие C.11.2a (о статусе “выбросов”).**

Экспериментальные точки с большими (Q^{-1}) не являются статистическими выбросами, если для них (\widehat{\Xi}*E>\Xi*\ast). Они принадлежат другому режиму коллективной диссипации, и должны учитываться отдельно при фитировании и паспортизации.

---

[

\quad \Xi_\ast\in[0.3,,3].
]

# **Стандарт Θ-блок: режимная проверка агрегатного состояния**

## (Теорема → алгоритм → паспорт → эксперимент)

---

## **I. Теоретическое основание (канон)**

**Исходная теорема (C.11.2).**
Применимость твёрдотельных (квазиупругих) аппроксимаций ограничена условием
[
\widehat{\Xi}*E(\omega)=\frac{\omega}{\omega_0}Q^{-1}(\omega)\le \Xi*\ast,
\qquad \Xi_\ast\sim\mathcal{O}(1),
]
где (\omega_0) — внутренний масштаб ансамбля, определяемый **из независимых экспериментальных параметров**.

---

## **II. Алгоритм режимной проверки (обязательный)**

### **Входные данные**

* экспериментальная таблица:
  ({\omega_i,\ Q^{-1}_i});
* параметры материала:

  * нерасслабленный модуль (M_{RT}),
  * вязкость (\eta) (или эквивалентный релаксационный параметр).

---

### **Шаг 1. Определение внутреннего масштаба**

[
\boxed{\ \omega_0=\frac{M_{RT}}{\eta}\ }
]
*(никакой подгонки, только экспериментальные значения)*

---

### **Шаг 2. Расчёт режимного параметра**

Для каждой точки:
[
\boxed{\ \widehat{\Xi}_{E,i}=\frac{\omega_i}{\omega_0}Q^{-1}_i\ }
]

---

### **Шаг 3. Режимная классификация**

Используется **универсальное** разбиение:

| Условие                     | Режим | Физический смысл             |
| --------------------------- | ----- | ---------------------------- |
| (\widehat{\Xi}_E<0.3)       | A     | квазиупругий (твёрдотельный) |
| (0.3\le\widehat{\Xi}_E\le3) | B     | переходный                   |
| (\widehat{\Xi}_E>3)         | C     | коллективно-диссипативный    |

⚠️ **Точки не удаляются**, а **размечаются**.

---

### **Шаг 4. Сравнение с классикой**

Для **каждого режима отдельно**:

1. выбирается классическая модель (например, (Q^{-1}\sim f^\alpha));
2. проводится фит;
3. **обязательно** считаются:

   * RMSE,
   * MAE (или другие метрики при необходимости).

---

### **Шаг 5. Проверка применимости**

Твёрдотельная аппроксимация считается применимой **только если**:
[
\text{RMSE}_A \ll \text{характерного масштаба } Q^{-1}.
]

Рост RMSE при переходе в B, C:

* **не ошибка эксперимента**,
* **не выбросы**,
* а признак выхода за область применимости модели.

---

---

## 1. Введение и мотивация

Аморфные твёрдые тела (стёкла) демонстрируют одновременно:

* жёсткость твёрдого тела,
* релаксационные свойства жидкости,
* и богатый спектр внутренних потерь.

Классические описания (Maxwell, KWW, VFT) **фиксируют отдельные аспекты**, но не дают **единого объяснения**, связывающего:

* частотный канал,
* температурный канал,
* разные релаксационные режимы (α, β′, β, γ),
* и наблюдаемую универсальность степенных законов.

Цель данной главы — **строго и количественно** показать, что:

> аморфное стекло корректно описывается как **ансамбль взаимосвязанных коллективных мод**,
> каждая из которых соответствует перестройке связности в L1-ансамбле,
> и что эта модель **одновременно и точно** описывает все экспериментальные каналы.

## 3. Теоретическая основа: ансамбль и связность

### 3.1. Аморфное состояние как ансамбль

Рассматривается система из (N) локальных L1-дефектов (узлов), соединённых в неупорядоченный, но устойчивый граф.

Состояние системы определяется:

* матрицей связности (A_{ij}),
* распределением локальных жёсткостей (k_i),
* набором коллективных мод ({\lambda_k}).

---

### 3.2. Коллективные моды и релаксации

Каждая коллективная мода (k) характеризуется:

* жёсткостью (G_k),
* временем перестройки (\tau_k),
* добротностью (Q_k).

Комплексный модуль:
[
G^*(\omega,T)=\sum_k \frac{G_k(T)}{1+i\omega\tau_k(T)}
]

Это **не модель**, а стандартная линейная суперпозиция релаксационных вкладов.

* одинаков для всех температур,
* наблюдается на 3–4 декадах по частоте,
* **не является параметром подгонки** — он задан штриховой линией в эксперименте.

Если бы существовала одна характерная релаксация:

* (Q^{-1}(f)) имело бы максимум (Debye).

Наблюдаемый степенной закон означает:
[
P(\tau)\sim \tau^{-(1-p)}
]

то есть **масштабно-инвариантное распределение времён**, что естественно возникает **только в ансамбле взаимосвязанных мод**.

# Практический блок для кристалла (РЕАЛЬНЫЕ ДАННЫЕ)

## C.9.P — Кристалл: теплоёмкость (C_p(T)) монокристалла Si и Debye-подгонка

### Источник экспериментальных/референс-данных

**NIST Chemistry WebBook**: Silicon (solid phase thermochemistry, Shomate coefficients; reference: Chase 1998, NIST-JANAF Tables). ([webbook.nist.gov][1])

**Что именно берём из источника** (прямо с страницы NIST):

* “Solid Phase Heat Capacity (Shomate Equation)”, диапазон **298–1685 K**
* коэффициенты:
  (A=22.81719,;B=3.899510,;C=-0.082885,;D=0.042111,;E=-0.354063) ([webbook.nist.gov][1])

---

## 1) Таблица исходных данных (формально)

Здесь “данные” — это **референс-табличная зависимость (C_p(T))**, заданная NIST-JANAF (в форме Shomate). Мы используем её как эталонный канал для проверки модели.

**Shomate (NIST):**
[
C_p(T)=A + B t + Ct^2 + Dt^3 + \frac{E}{t^2},\quad t=\frac{T}{1000}
]
(коэффициенты см. выше) ([webbook.nist.gov][1])

---

## 2) Модель для сравнения (классика как частный случай)

**Debye heat capacity (молярная), приближение (C_p\simeq C_v):**
[
C_v(T;\Theta_D)=9R\left(\frac{T}{\Theta_D}\right)^3 \int_0^{\Theta_D/T}\frac{x^4 e^x}{(e^x-1)^2},dx
]
Параметр один: (\Theta_D) (Debye temperature).

---

## 3) Инженерные формулы перевода (что можно считать “паспортом” канала)

* Вход: (T) (K)
* Выход: (C_p(T)) (J/mol·K)
* Безразмеривание:
  [
  \theta=\frac{T}{\Theta_D},\qquad c(\theta)=\frac{C_v}{3R}
  ]
  где (0\le c(\theta)\le 1).

---

## 4) Результат подгонки и точность (RMSE)

Я построил набор из **80 точек** по (T\in[298,1685]) K (из NIST-Shomate) и минимизировал RMSE между NIST и Debye.

**Оптимум:**

* (\Theta_D^{*}=492.0) K
* RMSE (=2.322) J/mol·K
* MAE (=1.931) J/mol·K
* MaxAbsErr (=4.392) J/mol·K
* RelRMSE (к среднему (C_p) на диапазоне) (=8.96%)

> Это аккуратный, воспроизводимый “кристаллический” канал с честными метриками (и видно, где Debye начинает систематически расходиться из-за (C_p\neq C_v) и высокотемпературных эффектов).

# Шаблон “как для аморфного” — что ещё обязательно закрыть для кристалла

Чтобы кристалл был “доделан” **по-настоящему** (как стекло Wagner’04), нужны **ещё 2 экспериментальных канала**, каждый со своими RMSE:

1. **Фононный спектр (D(\omega)) / дисперсии (INS/IXS)**

   * данные: ( \omega(\mathbf{q})) или (D(\omega))
   * модель: решёточная динамика / эффективная матрица жёсткости
   * метрики: RMSE по (\omega(\mathbf{q})), согласие по (D(\omega))

2. **Анеластичность / внутреннее трение (Q^{-1}(f,T))** для кристалла

   * данные: (Q^{-1}(f)) на резонансах (ультразвук/механические резонаторы)
   * модель: узкий релаксационный пик(и), дефекты решётки
   * метрики: RMSE по (Q^{-1}), положение (f_{peak}(T))
