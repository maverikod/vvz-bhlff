# 1) $T^{\mu\nu}$ на сетке: пошаговый расчёт (FFT/символ)

## 1.0. Входные данные

* Решётка: куб $\Omega=[0,L)^3$, узлы $N_x\times N_y\times N_z$, шаги $h_i=L_i/N_i$.
* Поле: комплексная огибающая $a(\mathbf x,t)$ (либо стационарное $a(\mathbf x)$ и гармоника $\propto e^{-i\omega t}$).
* Параметры: $Z,\ \mu>0,\ k_0,\ \chi'(\mathbf x)\ge0$.
* Метрика пространства: $\gamma^{ij}(\mathbf x)$ (изотропно $\gamma^{ij}=\delta^{ij}$ или анизотропно/слоисто).
* Порядок: $0<\beta\le1$. Для $\beta<1$ — спектральный (Riesz) оператор.

## 1.1. Спектральный символ

* Для **изотропии**: $|\mathbf k|_h^2=\sum_{i=x,y,z}\Big(\frac{2}{h_i}\sin\frac{\pi n_i}{N_i}\Big)^2$.
* Для **анизотропии**: $(k\!\cdot\!G\!\cdot\!k)_h=\sum_{i,j} G^{ij} k_i k_j$, где $G^{ij}\equiv\gamma^{ij}$.
* Символ: $\sigma_\beta(\mathbf k)=\big[(k\!\cdot\!G\!\cdot\!k)_h\big]^{\beta}$.

> **Замечание.** Если $\gamma^{ij}$ зависит от $\mathbf x$, применяйте псевдоспектральную схему: шаг (i) FFT → (ii) умножение на $|\mathbf k|^{2\beta}$ для **локально постоянных** блоков/ячеек → (iii) обратная FFT; на стыках — через ABCD/локальные матрицы передачи.

## 1.2. Энергетические величины (плотности)

В гармоническом режиме $a(\mathbf x,t)=\Re\{a(\mathbf x)\,e^{-i\omega t}\}$:

* $T^{00}(\mathbf x)=\dfrac{Z}{2}\,\omega^2|a|^2+\dfrac{\mu}{2}\,\mathcal Q_\beta[a](\mathbf x)+\dfrac{k_0^2\chi'(\mathbf x)}{2}|a|^2$.
* Поток: $T^{0i}(\mathbf x)=\Re\big[Z\,(\partial^0 a)^*\,\nabla^i a\big]= Z\,\omega\,\Im\big[a^*\,\nabla^i a\big]$.
* Стресс (локальный $\beta=1$):

  $$
  T^{ij}=\mu\,\Re(\nabla^i a^*\,\nabla^j a)-\gamma^{ij}\left(\frac{\mu}{2}|\nabla a|_\gamma^2+\frac{k_0^2\chi'}{2}|a|^2+\frac{Z}{2}\omega^2|a|^2\right).
  $$
* Стресс (нелокальный $0<\beta<1$, **спектрально**):

  $$
  T^{ij}(\text{спектр})=\mu\sum_{\mathbf k}\beta\,\big(k\!\cdot\!G\!\cdot\!k\big)^{\beta-1}\,k^i k^j\,|\hat a(\mathbf k)|^2
  -\gamma^{ij}\,\mathcal L_{\rm sp} -\gamma^{ij}\frac{Z}{2}\omega^2\langle|a|^2\rangle,
  $$

  где $\mathcal L_{\rm sp}=\dfrac{\mu}{2}\sum_{\mathbf k}(k\!\cdot\!G\!\cdot\!k)^{\beta}|\hat a|^2+\dfrac{k_0^2}{2}\langle\chi'|a|^2\rangle$.
  Для поля в реальном пространстве восстановите $T^{ij}(\mathbf x)$ обратным преобразованием «моментов» спектра или используйте билокальную форму при пост-обработке на под-объёмах.

## 1.3. Псевдокод (скелет)

```
INPUT: grid Ω, a(x), β, G (γ^ij), Z, μ, k0, χ'(x), ω
# FFT нормировка: a_hat = FFT(a)  # периодическая Т^3-схема

# 1) Символ
for each k in K:
    kGk = k^T * G * k                 # скаляр (или k·k для изотропии)
    sigma[k] = (kGk)^β

# 2) Qβ и L_sp
Qβ = sum_over_k( sigma[k] * |a_hat[k]|^2 )
Lsp = (μ/2)*Qβ + (k0^2/2)*mean_over_x( χ'(x)*|a(x)|^2 )

# 3) Энергия и поток
T00(x) = (Z/2)*ω^2*|a(x)|^2 + (μ/2)*qβ_density(x) + (k0^2/2)*χ'(x)*|a(x)|^2
S_i(x) = Z*ω*Im( a*(x) * ∂_i a(x) )   # через спектр или разностные ∂_i

# 4) Стресс
if β == 1:
    grad_a = ∇a(x)
    Tij(x) = μ*Re(∂_i a* ∂_j a) - γ_ij*( (μ/2)|∇a|_γ^2 + (k0^2/2)χ'|a|^2 + (Z/2)ω^2|a|^2 )
else:
    # спектральная форма «момента»
    Mij = sum_over_k( μ*β*(kGk)^(β-1) * k_i*k_j * |a_hat[k]|^2 )
    Tij = IFFT_to_local(Mij) - γ_ij*( Lsp + (Z/2)*ω^2*mean|a|^2 )

OUTPUT: T00(x), S_i(x)=T0i(x), Tij(x)
```

**Проверки (должны выполняться):**
(i) $\partial_t\int T^{00} dV = -$ потери $-$ утечки $+$ поток через границу;
(ii) для однородия $\Re Y(\omega)\ge0$;
(iii) для плоской волны — совпадение с аналитикой.

# 2) Отклонение луча в многослойной сфере: формулы слоями

Пусть $n(r)=\sqrt{\kappa(r)/\chi'(r)}$ — «оптический индекс». В сферической симметрии инвариант Снеллиуса:

$$
J=b = n(r)\,r\,\sin\psi=\text{const},\qquad r_{\min}\ \text{из}\ n(r_{\min})\,r_{\min}=b.
$$

Полный поворот (вход↔выход):

$$
\Delta\varphi(b)=2\int_{r_{\min}}^{R_{\rm out}} \frac{b\,dr}{r\,\sqrt{(n(r)r)^2-b^2}} - \pi.
$$

## 2.1. Слои с постоянным $n_i$ (радиусы $r_{i-1}\to r_i$)

Если на всём слое $b<n_i r$:

$$
\boxed{\ \Delta\varphi_i(b)=\arccos\!\frac{b}{n_i r_i}\ -\ \arccos\!\frac{b}{n_i r_{i-1}}\ }.
$$

Если в слое $b=n_i r_{\min}$ достигается поворотная точка, интеграл обрывается на $r_{\min}$ и дальнейший проход вглубь отсутствует (полное внутреннее отражение).

**Сумма по слоям:**

$$
\Delta\varphi(b)=\sum_i \Delta\varphi_i(b) - \pi.
$$

## 2.2. Тонкая высокоимпедансная оболочка (контраст $n$ велик)

Пусть на $[r_H,r_H+d]$: $n(r)=n_0\,\eta$, $\eta\gg1$. Малая-толщина, медленный ход:

$$
\delta\psi \simeq \int_{\Sigma}\nabla_\perp\ln n\,ds
\approx \int_{r_H}^{r_H+d}\frac{b}{r\,\sqrt{(n r)^2-b^2}}\,\frac{d\ln n}{dr}\,dr.
$$

При $\eta\to\infty$ знаменатель растёт ⇒ **захват**: для широкого диапазона $b$ луч не проходит (в ABCD — $|T|\to0,\ Y_{\rm out}\to0$).

## 2.3. WKB-волновой вид (удобно для ABCD согласования)

$$
k_r^2(r)=k_0^2 n^2(r) - \frac{b^2}{r^2},\qquad
\Delta\varphi(b)=2\int_{r_{\min}}^{R_{\rm out}} \frac{b\,dr}{r^2\,k_r(r)} - \pi.
$$

На слое $i$ с $n_i=$ const и $r\in[r_{i-1},r_i]$ интеграл берётся в элементарных функциях (через $\arccos$), что и даёт формулу из §2.1. На границе, где $k_r^2<0$, возникает эванесцент — **стоп-диапазон** (тот же критерий, что и в матрице передачи).

**Практика:**
— вычисляйте $\Delta\varphi(b)$ по слоям;
— критический $b_c=\min_r n(r)r$ задаёт границу проходящих траекторий;
— для оболочки с «чёрным» поведением $b_c$ поднимается, область прохода сужается.

# 3) Регрессия $Y(\omega)\rightarrow \{C_{\rm eff},M_{\rm eff}\}$ + импульс

## 3.1. Модель низких частот

$$
Y(\omega)= i\,C_{\rm eff}\,\omega + (i\,C_1 - M_{\rm eff})\,\omega^2 + O(\omega^3).
$$

Следствия: $\Re Y(\omega)\approx -M_{\rm eff}\,\omega^2$, $\Im Y(\omega)\approx C_{\rm eff}\,\omega + C_1\,\omega^2$.

## 3.2. Съёмка

* Частоты: $\omega_j\in(0,\ \omega_{\min}]$, где $\omega_{\min}\approx 0.1$ первой моды объёма (чтобы «ниже резонансов»).
* На каждой $\omega_j$: рассчитайте $Y_j=Y(\omega_j)$, проверьте пассивность $\Re Y_j\ge0$ и баланс энергии (невязка $\le1\%$).

## 3.3. Оценка параметров (закрытые формулы)

**(а) $M_{\rm eff}$ из $\Re Y$** — линейная регрессия без свободного члена:

$$
\Re Y_j \approx -M_{\rm eff}\,\omega_j^2 \quad\Rightarrow\quad
\boxed{\ M_{\rm eff} = -\frac{\sum_j w_j\,\omega_j^2\,\Re Y_j}{\sum_j w_j\,\omega_j^4}\ },
$$

где веса $w_j$ — по дисперсии/доверительным интервалам (или $w_j=1$).

**(б) $C_{\rm eff},C_1$ из $\Im Y$** — линейная регрессия по $\omega, \omega^2$:

$$
\Im Y_j \approx C_{\rm eff}\,\omega_j + C_1\,\omega_j^2
\Rightarrow
\begin{bmatrix} \sum w\omega^2 & \sum w\omega^3 \\ \sum w\omega^3 & \sum w\omega^4 \end{bmatrix}
\begin{bmatrix} C_{\rm eff}\\ C_1 \end{bmatrix}
=
\begin{bmatrix} \sum w\,\omega\,\Im Y \\ \sum w\,\omega^2\,\Im Y \end{bmatrix}.
$$

Решите 2×2 систему, получите $C_{\rm eff},C_1$. Проверьте знак $C_{\rm eff}\ge0$.

**(в) Устойчивость (Padé-апгрейд, опционально).** Для более шумных данных используйте рациональную аппроксимацию 2-го порядка, совместимую с пассивностью:

$$
Y(\omega)\approx \frac{iC\,\omega + iD\,\omega^2}{1 + \tau\,i\omega} - M\,\omega^2,
$$

с ограничениями $C,D,\tau,M\ge0$. Оценка — через нелинейный наименьших квадратов с проекцией на допустимую область.

## 3.4. Импульсная проверка (ступень/дельта)

* Воздейcтвие: $f(t)=f_0\,H(t)$ (или короткий импульс).
* Наблюдаемая медленная координата $\xi(t)$ (фаза/сдвиг ядра) подчиняется:

  $$
  K_{\rm eff}\,\xi + \Gamma_{\rm eff}\,\dot\xi + M_{\rm eff}\,\ddot\xi \simeq f(t).
  $$
* Подгоните $M_{\rm eff}$ по ранней части переходной характеристики (например, по минимуму функционала энергии $\int_0^{T}(\dot\xi^2+\omega_0^2\xi^2)\,dt$ при фиксированных $K_{\rm eff},\Gamma_{\rm eff}$). Сравните с частотной оценкой (расхождение $\le5\textrm{–}10\%$).

## 3.5. Контроль качества (итоговые критерии)

* Пассивность: $\Re Y(\omega)\ge0$ во всей полосе съёмки.
* Радиус-инвариантность: $Y(\omega)$ не зависит от $R$ (при корректном PML).
* Баланс: $|P_{\rm in}-P_{\rm leak}-P_{\rm loss}-\Delta W|/P_{\rm in}\le1\%$.
* Сходимость регрессии: условные числа матриц $\kappa_2<10^6$ (при необходимости — нормируйте $\omega$ и масштабируйте).

## Мини-примеры

**P1. $M_{\rm eff}$ вариационно (фаза).**
Если $a(\mathbf x,t)=A(\mathbf x)e^{i\phi(t)}$, то
$\displaystyle M_{\rm eff}=Z\int |A|^2\,d^3x$.
Для сферически локализованной моды с хвостом $|A|\sim r^{2\beta-3}$ интеграл конечен при $\beta<3/4$, иначе нужна отсечка (PML/«чёрная» оболочка).

**P2. Слой-за-слоем.**
Три слоя: ядро $n_1$ на $[0,r_1]$, оболочка $n_2\gg n_1$ на $[r_1,r_2]$, наружный $n_0$.

$$
\Delta\varphi(b)=\big[\arccos\frac{b}{n_0 R_{\rm out}}-\arccos\frac{b}{n_0 r_2}\big]
+\big[\arccos\frac{b}{n_2 r_2}-\arccos\frac{b}{n_2 r_1}\big]
+\big[\arccos\frac{b}{n_1 r_1}-\arccos\frac{b}{n_1 r_{\min}}\big]-\pi.
$$

Если $b\ge n_2 r_2$, луч заворачивает у внешней границы оболочки — **геометрический барьер**.

**P3. Регрессия $Y(\omega)$.**
Для низких $\omega$ возьмите 15 точек $\le 0.15\,\omega_1$, рассчитайте
$M_{\rm eff}=-\frac{\sum \omega^2\Re Y}{\sum \omega^4}$,
$C_{\rm eff},C_1$ — из 2×2 системы по $\Im Y$. Сверьте импульсом.
