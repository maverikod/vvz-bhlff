# G) Радиальный поток для $(-\Delta)^\beta$ через вариацию энергии в сферическом слое

**Цель.** Дать строгую форму радиального потока через сферу $S_R$ для фракционного оператора $(-\Delta)^\beta,\ 0<\beta\le1$, согласованную с энергетикой БВП и удобную для численной реализации.

## G.1. Энергетика и форма Дирихле

Для стационарной гармоники $a(\mathbf x,t)=\Re\{a(\mathbf x)\,e^{-i\omega t}\}$ энергетический функционал:

$$
\mathcal E[a;\Omega]=\frac{\mu}{2}\,\mathcal Q_\beta[a;\Omega]+\frac{k_0^2}{2}\!\int_{\Omega}\!\chi'(\mathbf x)\,|a|^2\,d\mathbf x,
$$

где **нелокальная** форма Дирихле на $\Omega\subset\mathbb R^3$:

$$
\mathcal Q_\beta[a;\Omega]=C_{3,\beta}\!\left[\frac{1}{2}\!\iint_{\Omega\times\Omega}\!\frac{|a(x)-a(y)|^2}{|x-y|^{3+2\beta}}\,dx\,dy+\!\iint_{\Omega\times\Omega^c}\!\frac{|a(x)-a(y)|^2}{|x-y|^{3+2\beta}}\,dx\,dy\right]\!.
$$

Константа $C_{3,\beta}>0$ — стандартная для $(-\Delta)^\beta$. Во всём пространстве $\Omega=\mathbb R^3$:

$$
\mathcal Q_\beta[a;\mathbb R^3]=\frac{C_{3,\beta}}{2}\iint_{\mathbb R^3\times\mathbb R^3}\frac{|a(x)-a(y)|^2}{|x-y|^{3+2\beta}}\,dx\,dy.
$$

## G.2. «Поток как производная по радиусу» (формула Хадамара)

Пусть $\Omega_R=\{x:\ |x|<R\}$, $S_R=\partial\Omega_R$, $\mathbf n=\mathbf e_r$.
Рассмотрим вариацию $R\mapsto R+\delta R$. Используя формулу Хадамара (вариация области) для нелокальной формы, получаем:

$$
\frac{d}{dR}\,\mathcal Q_\beta[a;\Omega_R]
=\ C_{3,\beta}\, \int_{S_R}\!\! dS_x\ \int_{\mathbb R^3}\!\frac{|a(x)-a(y)|^2}{|x-y|^{3+2\beta}}\,dy,
$$

то есть производная энергии по «выдуванию» сферы равна поверхностному интегралу от **плотности взаимодействия** точки $x\in S_R$ со всем пространством.

**Определение радиального потока (мощности)** через $S_R$ для нашего БВП:

$$
\boxed{\ \mathcal P_R\ \equiv\ \frac{\omega}{2}\,\Im\Big[\mu\,\frac{d}{dR}\,\mathcal Q_\beta[a;\Omega_R]\Big]
=\frac{\mu\,\omega\,C_{3,\beta}}{2}\int_{S_R}\!\!\int_{\mathbb R^3}\frac{\Im\big[(a(x)-a(y))(\overline{a(x)}-\overline{a(y)})\big]}{|x-y|^{3+2\beta}}\,dy\,dS_x.}
$$

Так как $\Im(z\bar z)=0$, удобно перейти к **двулинейной** форме для пары полей $a,b$ и затем положить $b=a$ после выделения потокового множителя (см. ниже).

Практически (для гармоники), поток через $S_R$ удобно считать через **двулинейный** функционал:

$$
\mathscr B_\beta[a,b;S_R]\ :=\ C_{3,\beta}\!\int_{S_R}\!\!\int_{\mathbb R^3}\!\frac{(a(x)-a(y))\,(\overline{b(x)}-\overline{b(y)})}{|x-y|^{3+2\beta}}\,dy\,dS_x,
$$

и тогда

$$
\boxed{\ \mathcal P_R\ =\ \frac{\mu\,\omega}{2}\,\Im\,\mathscr B_\beta[a,a;S_R].\ }
$$

Это строгая **градиентная (shape)**–формула потока для $(-\Delta)^\beta$. Для $\beta=1$ она деградирует к локальной формуле (см. G.4).

## G.3. «Нелокальная нормальная производная» на сфере

Альтернативно вводят **нелокальную нормальную производную** порядка $\beta$ на $\partial\Omega_R$:

$$
\mathcal N_\beta a(x)\ :=\ C_{3,\beta}\ \text{PV}\!\int_{\mathbb R^3\setminus\Omega_R}\!\frac{a(x)-a(y)}{|x-y|^{3+2\beta}}\,dy,\qquad x\in S_R,
$$

тогда двулинейная форма переписывается как

$$
\mathscr B_\beta[a,b;S_R]=\int_{S_R}\!\big(\mathcal N_\beta a(x)\big)\,\overline{b(x)}\,dS_x+\int_{S_R}\!\overline{\mathcal N_\beta b(x)}\,a(x)\,dS_x.
$$

Отсюда (для $b=a$):

$$
\boxed{\ \mathcal P_R\ =\ \mu\,\omega\ \Im\int_{S_R}\overline{a(x)}\ \mathcal N_\beta a(x)\ dS_x.}
$$

Это — прямая «Пойнтинговская» запись для $(-\Delta)^\beta$, удобная в коде (см. G.5).

## G.4. Проверка: локальный предел $\beta=1$

При $\beta=1$: $\mathcal Q_1=\int |\nabla a|^2$, $\mathcal N_1 a=\partial_r a$ на $S_R$.
Получаем классическое:

$$
\mathcal P_R=\mu\,\omega\,\Im \int_{S_R}\overline{a}\ \partial_r a\ dS,
$$

что совпадает с привычным потоком для эллиптического оператора второго порядка.

## G.5. Реализация (алгоритм)

1. **Дискретизация $S_R$** (треугольная/икосаэдральная сетка на сфере).
2. **Квадратура по $y$**: для каждого $x\in S_R$ вычислять (в PV-смысле)
   $\mathcal N_\beta a(x)=C_{3,\beta}\sum_{y\in\mathbb R^3\setminus\Omega_R} w_y \dfrac{a(x)-a(y)}{|x-y|^{3+2\beta}}$.

   * При вычислении на конечной области — добавить PML/внешний слой.
   * Сингулярность обрабатывается исключением шара $|x-y|<\epsilon$ c компенсационным членом (стандартная PV-квадратура).
3. **Поток**: $\mathcal P_R=\mu\,\omega\,\Im \sum_{x\in S_R} w_x\,\overline{a(x)}\,\mathcal N_\beta a(x)$.
4. **Проверки**: (i) **инвариантность по $R$** (при фиксированном PML), (ii) **баланс энергии** в объёме: $P_{\rm in}=P_{\rm leak}+P_{\rm loss}+dW/dt$ с невязкой $\le1\%$.

> Примечание (ускорение). Можно использовать **представление Балакришнана**:
>
> $$
> (-\Delta)^\beta a = \frac{\sin(\pi\beta)}{\pi}\int_0^\infty t^{\beta-1}\,(-\Delta)(-\Delta+tI)^{-1}a\ dt.
> $$
>
> Тогда **поток** есть суперпозиция локальных потоков для сдвинутых задач:
>
> $$
> \boxed{\ \mathcal P_R^{(\beta)}\ =\ \frac{\sin(\pi\beta)}{\pi}\int_0^\infty t^{\beta-1}\,\mathcal P_R^{(1)}[u_t]\ dt,\quad
> u_t=( -\Delta+tI)^{-1}a.}
> $$
>
> Численно: 10–20-точечная Гаусс–Якоби квадратура по $t$; на каждом $t$ — стандартный локальный поток $\mu\,\omega\,\Im\int_{S_R}\bar u_t\,\partial_r u_t\,dS$.

# H) Склейка WKB-геометрии с ABCD по слоям: поворотная точка, $b_c$, «чёрная» оболочка

**Цель.** Соединить лучевую (WKB) картину с реальной волновой передачей через **ABCD** для многослойной сферы, дать условия «тотального стопа» (чёрная оболочка) и явные формулы.

## H.1. Радиальная WKB-динамика

В изотропии $A^{ij}=n^{-2}(r)\delta^{ij}$, где $n^2(r)=\kappa(r)/\chi'(r)$.
Гарантированный интеграл движения (аналог закона Снеллиуса):

$$
b= n(r)\,r\,\sin\psi=\text{const},\qquad
k_r^2(r)=k_0^2 n^2(r)-\frac{b^2}{r^2}.
$$

**Поворотная точка** $r_t$ задаётся $k_r(r_t)=0\iff b = k_0\,n(r_t)\,r_t$.

**Полный угол отклонения** (вход/выход):

$$
\Delta\varphi(b)=2\int_{r_{\min}}^{R_{\rm out}}\!\frac{b\,dr}{r^2\,k_r(r)}-\pi,\quad
r_{\min}:\ b=k_0 n(r_{\min})r_{\min}.
$$

**Критический параметр удара**:

$$
\boxed{\ b_c\ =\ \min_{r\in(0,R_{\rm out}]}\big\{k_0\,n(r)\,r\big\}. }
$$

Если $b<b_c$, то $k_r^2(r)>0$ на всём пути — **поворотной точки нет**, луч проходит. Если $b>b_c$, возникает отражение (есть $r_t$).

> Замечание. В реальной БВП «прохождение» может быть заблокировано не геометрией, а **стоп-диапазоном** (ABCD); ниже даём волновой критерий.

## H.2. ABCD-матрицы для радиальных слоёв

Разобьём $r\in[R_{\rm in},R_{\rm out}]$ на слои $i=1..N$ с почти постоянными $n_i$, толщинами $d_i$. В волновом (1D) описании вдоль радиуса получаем стандартный вид:

* **Проходной слой (пасcбенд)**: $k_{ri}^2>0$, $\phi_i=\int_{r_{i-1}}^{r_i} k_{ri}(r)\,dr$.

  $$
  \mathsf M_i=
  \begin{pmatrix}
  \cos\phi_i & \frac{1}{Y_i}\sin\phi_i\\
  -Y_i\sin\phi_i & \cos\phi_i
  \end{pmatrix}.
  $$

* **Стоп-слой (эвесцент)**: $k_{ri}^2<0$, $q_i=\int_{r_{i-1}}^{r_i} |k_{ri}(r)|\,dr$.

  $$
  \mathsf M_i=
  \begin{pmatrix}
  \cosh q_i & \frac{1}{Y_i}\sinh q_i\\
  Y_i\sinh q_i & \cosh q_i
  \end{pmatrix}.
  $$

Здесь $Y_i(\omega)$ — **характеристический адмиттанс слоя по радиали** (отношение потока к амплитуде). Для $\beta=1$: $Y_i=\mu\,\omega/k_{ri}$. Для общего символа $\omega^2=c_\phi^2 (k\!\cdot\!G\!\cdot\!k)^\beta$ в изотропии получаем WKB-оценку:

$$
\boxed{\ Y_i(\omega)\ \approx\ \mu\,\frac{\omega}{\partial_{\!k_r}\omega}\ =\ \mu\,\frac{\omega^2}{\beta\,c_\phi^2\,|k|^{2\beta-2}\,k_r}\ },
$$

где $|k|^2=k_r^2+b^2/r^2$. (В практике лучше **калибровать $Y_i$** из локальных «микро»-решений в однородном слое и хранить таблицей $Y_i(\omega,b)$.)

**Сборка:** $\mathsf M=\mathsf M_N\cdots \mathsf M_1$. Внешний адмиттанс на $R_{\rm out}$:

$$
Y_{\rm out}(\omega,b)=\frac{M_{21}+M_{22}\,Y_{\rm load}}{M_{11}+M_{12}\,Y_{\rm load}},
$$

где $Y_{\rm load}$ — адмиттанс «наружи» (PML/свободное пространство). **Чёрная оболочка** ⇔ $Y_{\rm out}\to 0$ на полосе $\omega$.

## H.3. Согласование WKB↔ABCD

* **Фаза слоя**: $\phi_i\approx \int_{r_{i-1}}^{r_i}\!k_{ri}(r)\,dr$ (серединная/Гаусс-квадратура; у градиентов — Лангера).
* **Эвесцентная толщина**: $q_i\approx\int_{r_{i-1}}^{r_i}\!|k_{ri}(r)|\,dr$.
* **Стык**: использовать локальные $Y_k^\pm(\omega)$ из микро-задач; интерфейсная матрица (Т/Р) добавляется между $\mathsf M_i$.

**Критерий стопа в оболочке $\Sigma$** (шириной $d$):

$$
\boxed{\ q_{\min}\,d\ \gg\ 1\quad\text{и}\quad \Big|\frac{Y_{\rm in}-Y_\Sigma}{Y_{\rm in}+Y_\Sigma}\Big|\approx 1,\ \ \Big|\frac{Y_{\rm out}-Y_\Sigma}{Y_{\rm out}+Y_\Sigma}\Big|\approx 1. }
$$

Тогда $|T|\sim e^{-2 q_{\min} d}\ll 1$, $Y_{\rm out}\to 0$, «горизонт» реализован на частотной полосе.

## H.4. Быстрые оценки для проектирования «чёрной» оболочки

1. **Геометрия (WKB):** сделайте $k_r^2<0$ на $\Sigma$ для целевого диапазона $b\in[b_1,b_2]$:
   $k_0^2 n^2(r)<b^2/r^2$ ⇒ $n(r) < b/(k_0 r)$ (для заданного $b$).
   Если цель — гасить **все** $b\le b_{\max}$, обеспечьте $\min_{r\in\Sigma} \{k_0 n(r)r\} < b_{\max}$ и большую $q$.

2. **Волна (ABCD):** при заданном $\omega$ добейтесь $q_{\min} d \gtrsim 5\text{–}7$ и сильной импедансной рассогласованности на обоих краях $\Sigma$. Тогда $|T|\lesssim e^{-10}$, $Y_{\rm out}$ — машинный ноль.

3. **Полоса частот:** требование $q_{\min}(\omega)\,d\gtrsim \text{const}$ даёт ширину полосы. Для гладких профилей $n(r,\omega)$ полоса масштабируется с $\sqrt{\kappa/\chi'}$.

4. **Проверка лучами:** для репрезентативного набора $b$ вычислите $\Delta\varphi(b)$. Если имеется **две** поворотные точки (до и после $\Sigma$), лучи **запираются** (многократные внутренние отражения), что согласуется с малым $|T|$ в ABCD.

## H.5. Практический алгоритм (склейка)

1. **Профиль → WKB:** по $\kappa,\chi'$ вычислить $n(r)$, затем $k_r(r;b,\omega)$, $\phi_i,q_i$ послойно.
2. **Калибровка $Y_i$:** решить локальную однородную задачу в слое, снять $Y_i(\omega,b)$ как отношение потока к амплитуде (по формуле G.5). Сохранить таблицей.
3. **Сборка ABCD:** пройти по слоям, перемножить $\mathsf M_i$ и интерфейсы ($Y_k^\pm$). Получить $Y_{\rm out}(\omega,b)$.
4. **Критерии:** $Y_{\rm out}\to0$ на целевой полосе; баланс $<1\%$; инвариантность по $R$; для **всех** $b\le b_{\max}$ — $q_{\min} d \gtrsim 5$.
5. **Кросс-чек:** угол $\Delta\varphi(b)$ и карта поворотных точек $\{r_t(b)\}$ согласуются с зоной стопа из ABCD.

## Итого (кратко)

* Для $(-\Delta)^\beta$ поток через сферу строго задаётся **нелокальной нормальной производной** $\mathcal N_\beta$:
  $\displaystyle \mathcal P_R=\mu\omega\,\Im\int_{S_R}\overline a\,\mathcal N_\beta a\,dS$.
  Удобен и эквивалентен вариант через **формулу Хадамара** по $R$.
* Склейка WKB↔ABCD: фазовые интегралы $\phi_i,q_i$ + калиброванные $Y_i$ дают матричную передачу. «Чёрная» оболочка достигается при $q_{\min}d\gg1$ и сильной импедансной рассогласованности; $Y_{\rm out}\to0$, лучи запираются (две поворотные точки).
