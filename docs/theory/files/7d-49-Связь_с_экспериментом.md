Ок. Делаю каркас «рассеяния на дефекте» строго в логике твоих постулатов: ядро–ПЗ–хвост, длинные хвосты без массовых членов и без экспоненциальных затуханий; динамика — консервативная, масштабно-инвариантная, с топологическим зарядом.

# 1) Поле, заряд, динамика (без «массы»)

Пусть основная переменная — фазовое поле (\Theta(\mathbf x,t)) (хвост) + возможная внутренняя ориентация (\mathbf n(\mathbf x,t)\in S^2) (коротко опущу; хватит (\Theta) для рассеяния).
Топологический ток дефекта (J^\mu) сохраняется: (\partial_\mu J^\mu=0). Никаких массовых членов: только градиентные/высшие производные.

Минимальный Лагранжиан (масштабно-инвариантный, без (m^2\Theta^2)):
[
\mathcal L=\frac{\rho}{2}(\partial_t\Theta)^2-\frac{c_1}{2}|\nabla\Theta|^2-\frac{c_2}{2}(\nabla^2\Theta)^2
+g,\Theta,\partial_\mu J^\mu .
]

* Разрешены **только** производные (никаких экспонент Юкавы).
* Две волновые ветви для хвоста: ( \omega^2=c_1 k^2 + c_2 k^4). При (c_1\to 0) получаем «лифшиц-ветвь» ( \omega\propto k^2) → чисто **алгебраические** дальние хвосты.

Статика (вне ядра): оператор ( \mathcal L_{\text{stat}}=c_1\nabla^2+c_2\nabla^4 ).
При (c_1=0): хвосты задаются (\nabla^4\Theta=0) с алгебраическими решениями и без экспонент.

# 2) Профиль «рифлёного хвоста»

Переходная зона возбуждает на хвосте радиальную модуляцию (твои «слои»):
[
\Theta_0(r);\sim;\frac{A}{r^{p}}\cos!\big(\kappa r+\varphi\big),
\quad p>0,;\kappa=\text{характерная несущая ПЗ}.
]
Это — фундаментальное отличие от «классики»: нет (\mathrm e^{-mr}), только **степенной закон** с радиальной «насечкой».

# 3) Рассеяние волны хвоста на одиночном дефекте

Рассматриваем стационарную волну частоты (\omega), волчисло (k). Две составляющие рассеяния:

**(а) Топологическая (геометрическая) часть**
Дефект создаёт холономию (угол/кручение среды). Волна на «конусе» даёт AB-подобную амплитуду
[
f_{\text{topo}}(\theta);\propto;
\frac{\mathrm e^{-i\pi\Delta}-1}{\sin(\theta/2)},,
]
где (\Delta) — безразмерная мера дислокации/дефицита (задаётся ядром/ПЗ). Это чисто **топологический** вклад: долгобазисный, без экспонент.

**(б) Структурный вклад от радиальной насечки («рифлёный хвост»)**
Пусть малое возмущение индекса/фазовой скорости (\delta n(r)\propto r^{-p}\cos(\kappa r+\varphi)). В борновском/эикональном приближении изотропная часть амплитуды — радиальный Ханкелев трансформ:
[
f_{\text{riff}}(q);\propto;\int_0^\infty r^{2-p}\cos(\kappa r+\varphi),j_0(qr),dr,
\quad q=2k\sin\frac{\theta}{2}.
]
Он даёт **дифракционные кольца** при
[
q_m \approx m,\kappa\quad\Longrightarrow\quad
2k\sin\frac{\theta_m}{2}\approx m,\kappa,;; m=1,2,\dots
]
(строго — с фазой (\varphi) и степенным демпфированием амплитуды (\sim m^{p-3}), но **без экспонент**).

**Итоговая амплитуда:**
[
f(\theta)=f_{\text{topo}}(\theta)+\varepsilon,f_{\text{riff}}(q),\qquad
\frac{d\sigma}{d\Omega}=|f(\theta)|^2.
]
(\varepsilon) задаёт контраст «насечки». В нуле — топологическая вперед-сингулярность; по углам — кольца от (\kappa).

# 4) Квазиклассика для траекторий (к макроуровню)

Для тел (лучевая аппроксимация): фазовый показатель (n(\mathbf r)=1+\delta n(\mathbf r)). Эикональная фазовая функция:
[
\chi(b)=k\int_{-\infty}^{\infty}\delta n!\left(\sqrt{b^2+z^2}\right),dz
;\approx; k,\varepsilon,C_p,b^{1-p},J_{p-1}(\kappa b),
]
где (b) — параметр удара, (J_{\nu}) — Бессель, (C_p) — безразмерный коэффициент (из интеграла).
**Угол отклонения**:
[
\alpha(b);=;-\frac{\partial}{\partial b}\chi(b)
;\sim;k,\varepsilon,b^{-p}\Big[J_{p-2}(\kappa b)-J_{p}(\kappa b)\Big].
]
Т.е. отклонение **осциллирует** по (b) с шагом (\Delta b\approx \pi/\kappa) и затухает степенно (\sim b^{-p}) — никакой экспоненты.

# 5) Взаимодействие дефект–дефект (эффективный «потенциал»)

Энергия перекрытия хвостов:
[
U(R);\propto;\int (\nabla\Theta_1!\cdot!\nabla\Theta_2),d^3x
;\sim; \frac{\cos(\kappa R+\phi)}{R^{\gamma}},
]
(\gamma>0) зависит от (p) и размерности перекрытия. Это даёт **осциллирующе-степенную** силу
(,F(R)\sim \partial_R U,), с «предпочтительными» расстояниями (нули/экстремумы) через (\Delta R\approx \pi/\kappa). На макроуровне это натуральный механизм тонких радиальных структур и слабых секулярных модуляций орбит — **без** классических потенциалов.

# 6) Связь с наблюдаемыми (кратко, по пунктам)

1. **Лабораторное/волновое:**
   дифракционные кольца в (d\sigma/d\Omega) при уголках (\theta_m) из условия (2k\sin(\theta_m/2)=m\kappa).
   Измеряя ({\theta_m}) при известном (k), восстанавливаем (\kappa) и (p) (по убыванию контраста).

2. **Траектории (Solar-/планетный масштаб):**
   малая осциллирующая добавка к отклонению (\alpha(b)) ⇒ **модулированные остатки** в долготах перицентров/анomalиях истинных долгот с характерным радиальным шагом (\Delta r\approx \pi/\kappa). Знак/фаза зависят от (\varphi) и от дипольности системы ядро–ядро (напр. Солнце–Юпитер).

3. **Галактики:**
   из (U(R)\sim \cos(\kappa R)/R^\gamma) следуют **рифления** в кривых вращения (малые «волны» по радиусу) и стоячие/бегущие фазовые узоры. Периодичность по (R) даёт прямую оценку (\kappa).

# 7) Что намеренно НЕ используется

* Никаких (m^2\Theta^2) и (\mathrm e^{-mr}).
* Никаких «ньютоновских» потенциальных паттернов; отклонения и сечения строятся из **топологии + рифлёной хвостовой структуры**.

# 2) Упругие столкновения (EL): простые дет-правила

**Фичи события (минимальный набор):**

* Два трека протонов в *Roman Pots* слева/справа; углы рассеяния микрорадовые, (|t|) из (\theta).
* В T1/T2 и в центральном детекторе — «тишина» (veto на активность сверх фона).
* Баланс по азимуту: (\Delta\phi \approx \pi); (|t_L - t_R|) в пределах калибровки; потери доли импульса (\xi) ~ 0.

**Правило (детерминированно):**
EL = (RP_L && RP_R) && (T1==0 && T2==0 && central==0) && (balance_t & balance_phi).
*(Работает в специальных оптиках β*≈90 м, когда Роты можно подводить близко к пучку и ловить малые (|t|).)* ([ep-news.web.cern.ch][4])

**Согласование с «TOTAL»:** эти же события дают эластическую часть для метода «без люминозности» (см. §4). ([SpringerLink][5])

**Твоё требование (без экспонент):** сверка идёт не через (e^{-B|t|}), а через **спектральные признаки** dσ/dt: ищем **кольцевую (дифракционную) структуру** — ноль/минимум и «бамп» как следствие рифлёного хвоста (см. твой (\kappa)): пиковые положения в (q= \sqrt{|t|}) должны давать шаг (\Delta q \sim \kappa). (Собственно, у TOTEM наблюдались «дипы/бампы» в dσ/dt — это наш «рифлённый» след.)

# 3) Неупругие (INEL): классы и простые признаки

Разбиваем на ND/SD/DD/CEP (central exclusive). Все правила — на детекторных признаках, не на потенциалах.

**(a) ND (не-дифракционные):**
Нет тэггированных протонов (RP_L=RP_R=0), активность повсюду, **нет больших щелей** по η. → ND. ([CERN][6])

**(b) SD (одно-дифракционные):**
RP с одной стороны (например, RP_L=1, RP_R=0), крупная **rapidity-gap** на стороне тэггированного протона; масса разрушающейся системы (M_X^2 \approx \xi,s). → SD. ([CERN][6])

**(c) DD (двойная дифракция):**
RP отсутствуют, **две** фракции активности по краям и центральная щель/оголение; технически TOTEM делал DD-селекцию «veto T1 && tracks T2». → DD. ([CERN Courier][7])

**(d) CEP/эксклюзивная (двойной тэг + центральная система):**
RP_L=RP_R=1, центральная система (X) (лептоны/джеты/…); проверка «эксклюзивности»: (m_X \approx \sqrt{\xi_1\xi_2 s}), малые посторонние отложения. (CMS-TOTEM показали для (\ell^+\ell^-) и искали t={t}.) → CEP. ([CMS Results][3])

**Твоя «дефектная» метка неупругости:**
INEL = наличие эмиссии «хвостовых мод» (частицы/волны) ⇒ **ломается чистая топологическая холономия**, появляются **рифлёные кольца** в распределениях по (|t|) и по η. Критерий без экспонент: детектируем **степенную** убыль амплитуд колец (\sim m^{p-3}) и устойчивый шаг (\Delta q \sim \kappa).

# 4) «TOTAL CERN»: сверка с полной σ без люминозности (чтобы всё сошлось)

TOTEM использует оптическую теорему, объединяя **упругие** (RP) и **неупругие** (T1/T2) счёты в одном Fill’е:
[
\sigma_{\text{tot}}=\frac{16\pi}{1+\rho^2},
\frac{\left.\frac{dN_{\text{el}}}{dt}\right|*{t\to 0}}
{N*{\text{el}}+N_{\text{inel}}},,
]
где (\rho) — Re/Im у упругой амплитуды при (t\to0). Это дало σ(*\text{tot}) на 7, 8, 13 ТэВ. Наш классификатор должен воспроизвести (N*{\text{el}}, N_{\text{inel}}) и корректную экстраполяцию к (t\to0) без априорных экспонент (через твою спектральную модель). ([Physical Review Journals][8])

# 5) Связка с твоей математикой рассеяния на дефекте

* **Упругое:** доминирует топологическая амплитуда (f_{\text{topo}}!\propto 1/\sin(\theta/2)) (форвард-сингулярность) + рифлёная добавка (f_{\text{riff}}(q)) с **кольцами** (q_m!\approx m\kappa). # 6) Масштабирование до макротел (правила + уравнения)

**Опознавание (макро):**

* **Упругое сближение дефектов-тел:** два тела уходят интактными; наблюдаем только угол отклонения (\alpha(b)) с **осциллирующей степенной** зависимостью (по твоей формуле) и **отсутствие** излучённых пакетов хвоста на больших R.
* **Неупругое:** фиксируем вынос энергии в «хвостовые волны» (дальнодействующие пакеты с алгебраикой, не экспонентой) и/или фрагментацию.

**Уравнения движения (две массы (M_{1,2}), относительный вектор (\mathbf R))**
Берём эффективную консервативную часть взаимодействия хвостов
[
U(R)=\frac{A\cos(\kappa R+\varphi)}{R^{\gamma}},\qquad
\mathbf F=-\nabla U,
]
и топологическую геометрию (угловой дефицит → вклад в фазу/траекторию). Тогда
[
\mu,\ddot{\mathbf R}=\mathbf F(\mathbf R) ;(+; \mathbf F_{\text{rad}}[\Theta];\text{только если есть возбуждение хвоста}),
]
где (\mu) — приведённая масса. **Упругое:** (\mathbf F_{\text{rad}}=0). **Неупругое:** энергия уходит в поле (\Theta) (но система в целом консервативна), что даёт наблюдаемый (\Delta E) и слабую секулярную модуляцию орбит.

**Признак границы EL→INEL (без «классики»):** сравнение времени пролёта (\tau_{\text{col}}\sim b/v) со спектром внутренних частот хвоста (\omega(k)) (у тебя (\omega^2=c_1 k^2 + c_2 k^4)):

* Если (\tau_{\text{col}}\gg 1/\omega_{\min}) → адиабатика → **упругое**;
* Если (\tau_{\text{col}}\lesssim 1/\omega_\text{band}) или (b) попадает в резонансные «рифлёные» узлы (b!\approx! m\pi/\kappa) → **неупругое**.

# Задание 40. FFTLog-Ханкель для (a(b)) (узлы (b_m))
