# 1) Состояние и параметры

* Тела (i=1..N): координаты (\mathbf R_i(t)), скорости (\mathbf V_i), «инертные» параметры (m_i) (величина сопротивления фазовой перестройке). Эквивалентность инертной/гравитационной массы — следствие единой фазовой природы.

# 2) Лагранжиан (консервативная часть)

[
L ;=;\sum_i \frac{m_i}{2},|\dot{\mathbf R}*i|^2 ;-; \sum*{i<j} U_{ij}(R_{ij}) ;+; L_{\text{geo}}(\Delta),
\quad R_{ij}=|\mathbf R_i-\mathbf R_j|.
]
Силовая часть формируется перекрытием хвостов:
[
U_{ij}(R);=;\frac{A_{ij}\cos(\kappa R+\varphi_{ij})}{R^{\gamma}},\qquad
\mathbf F_{ij}=-\nabla U_{ij}.
]
Это даёт **осциллирующе-степенной закон** без экспонент (рифлёные «предпочтительные» расстояния (\Delta R\simeq \pi/\kappa)). 
Топологический вклад (L_{\text{geo}}(\Delta)) реализует постоянный геометрический сдвиг фазы/азимута (эффективное «конусовидное» пространство → квазипрецессия). 

# 3) Законы движения (N-тел)

[

]

* **Эластический режим:** (\mathbf F^{(\text{rad})}_i=0) (только консервативная динамика).
* **Неупругий режим:** при неадиабатике/резонансе (см. §5) часть энергии уходит в фазовые волны → (\mathbf F^{(\text{rad})}_i\neq 0) (слабая «радиационная» поправка, но без экспонент/массовых членов). 

# 4) Две массы (аналог центральной задачи)

F_R = -\frac{dU}{dR}

# 1) Динамика без постоянной массы

Вместо (m\dot{\mathbf v}) используем **фракционную (памятующую) инерцию**:

[
\mathbf p_i(t);=;\mathcal I_i,\mathrm D_t^{-\alpha_i},\mathbf v_i(t),\qquad 0<\alpha_i\le 1,
]

где (\mathrm D_t^{-\alpha}) — фракционный интеграл (алгебраическая память), (\mathcal I_i) — *оператор инерции*, зависящий от локальных фазовых инвариантов тела (i) (кривизна хвоста, контраст рифления, холономия и т.п.).
**Уравнение движения (эластика):**
[
\frac{d}{dt},\mathbf p_i(t) ;=; \sum_{j\ne i}\mathbf F_{ij}(\mathbf R_{ij});+;\mathbf F^{(\mathrm{geo})}_i(\Delta)
]
(никаких постоянных «масс»).

Практический эквивалент (память во времени):
[
\mathbf p_i(t);=;\int_0^\infty K_i(\tau;\chi_i(t!-!\tau)),\mathbf v_i(t!-!\tau),d\tau,
\quad K_i(\tau)\propto \tau^{-(1+\alpha_i)}\cos(\Omega_i \tau).
]
Здесь **ядро памяти степенное**, без экспоненциального хвоста.

**Физический смысл.** Инерция — это *запаздывающая реакция* фазовой среды. Ты дал возмущение → среда начала перестройку → пока фазовый градиент не «схлопнулся», движение продолжается.

**Формулировка без классики:**
[
\mathbf p(t)=\int_0^\infty K(\tau;\chi),\mathbf v(t-\tau),d\tau,\quad
K(\tau)\propto \tau^{-(1+\alpha)}\cos(\Omega\tau),;0<\alpha\le1,
]
[
\dot{\mathbf p}(t)=\sum_{j\ne i}\mathbf F_{ij}(\mathbf R_{ij})+\mathbf F^{(\mathrm{geo})}(\Delta).
]
Ни «массы-константы», ни массовых членов, ни экспонент.

**Немедленные следствия (наблюдаемо):**

1. **Постпробег:** после снятия силы скорость убывает **степенно** ((\sim t^{-\alpha})), а не экспоненциально.
2. **Гистерезис ускорения/торможения:** из-за памяти траектории при одинаковых по модулю, но разнонаправленных воздействиях чуть различны.
3. **Импульс среды:** в транзиентах возможен локальный «нестрогий третий закон» между телами; полный импульс «тела+среда» сохраняется.
4. **Частотная инерция:** в Фурье (|\mathbf p(\omega)|\propto|\omega|^{-\alpha}|\mathbf v(\omega)|) → на низких частотах «тяжелее», на высоких — «легче».
5. **Резонансные дрейфы:** при (R\approx m\pi/\kappa) возникают небольшие секулярные сдвиги (энергия уходит в фазовые волны *степенным* хвостом).

  (w_k\sim (k+1)^{-(1+\alpha)}-k^{-(1+\alpha)}) (опционально (\times\cos(\Omega k\Delta t))).

  2. (\mathbf F=\sum_{j}\mathbf F_{ij}) c (U_{ij}=\dfrac{A_{ij}(\chi)}{R^\gamma}\cos(\kappa R+\varphi_{ij}(\chi)))
  3. (\mathbf F^{(\mathrm{geo})}=\Omega(\Delta),\hat{\mathbf z}\times\mathbf p)

  5. Инвертировать свёртку для (\mathbf v^{n+1}) (итеративно, коротким окном)
  6. (\mathbf R^{n+1}=\mathbf R^n+\mathbf v^{n+1}\Delta t)

# 1) Переменные тела (i=1..N)

* Положение (\mathbf R_i(t)), скорость (\mathbf v_i(t)), ускорение (\mathbf a_i(t)).
* Собственное вращение и мода формы: ортонормированный базис ((\hat{\mathbf t}_i,\hat{\mathbf p}_i,\hat{\mathbf r}_i))
  (тороидальная/полоидальная/радиальная оси) и коэффициенты анизотропии (\alpha_i^{(t)},\alpha_i^{(p)},\alpha_i^{(r)}).
* **ВБП = набор полос** (\mathcal B={(\kappa_b,\varepsilon_b,\gamma_b)}_{b=1}^B), (\kappa_b>0), (\varepsilon_b) — контраст полос, (\gamma_b>0).

# 2) Поле «фазового давления» (\Psi) (рождается ускорением)

Ускорение создаёт фронт-бэк асимметрию (сжатие впереди, разрежение сзади). Для источника (j):
[
\Psi_j(\mathbf x,t)=\sum_{b\in\mathcal B}\varepsilon_b;
\mathbb A_j^{(b)}:\mathbf a_j(t);,K_b(\mathbf x-\mathbf R_j(t)),
]
где
(K_b(\mathbf r)=\dfrac{\cos(\kappa_b r+\phi_b(\hat{\mathbf r}))}{r^{\gamma_b}}) — **осциллирующе-степенное** нелокальное ядро (без экспонент),
(\mathbb A_j^{(b)}=\alpha_j^{(t)},\hat{\mathbf t}_j!\otimes!\hat{\mathbf t}_j+\alpha_j^{(p)},\hat{\mathbf p}_j!\otimes!\hat{\mathbf p}_j+\alpha_j^{(r)},\hat{\mathbf r}_j!\otimes!\hat{\mathbf r}_j)
— анизотропный проектор из собственных вращений/волн.

Суммарное поле: (\Psi=\sum_j \Psi_j).

# 3) Сила как **градиент перестройки** (только при ( \mathbf a\neq 0))

Тело (i) «детектирует» (\Psi) своим приёмным проектором (\mathbb B_i^{(b)}) (зависит от (\hat{\mathbf t}*i,\hat{\mathbf p}*i,\hat{\mathbf r}*i)):
[
\mathbf F_i^{\text{med}}(t)=-\sum*{b}\int \mathbb B_i^{(b)}\cdot \nabla \Psi(\mathbf x,t);W_i(\mathbf x-\mathbf R_i),d^3x.
]
Для компактности это редуцируется к **парным тензорам**:
[
\boxed{;;\mathbf F_i^{\text{med}}(t)=\sum*{j=1}^N \mathbb T*{ij}!(\mathbf R_{ij},\hat{\mathbf n}*{ij},\Omega_i,\Omega_j;\mathcal B);\mathbf a_j(t);},
]
где (\mathbf R*{ij}=\mathbf R_i-\mathbf R_j), (\hat{\mathbf n}*{ij}=\mathbf R*{ij}/|\mathbf R_{ij}|), (\Omega_i) — параметры собственных вращений;
(\mathbb T_{ij}) — **чисто геометрико-полосовой** оператор (без «массы»).

Явный вид ядра в (\mathbb T_{ij}):
[
\mathbb K_b(R)=\frac{\gamma_b}{R^{\gamma_b+1}}\cos(\kappa_b R+\phi_b),\hat{\mathbf n}!\otimes!\hat{\mathbf n}
;+;\frac{\kappa_b}{R^{\gamma_b}}\sin(\kappa_b R+\phi_b),\Pi^\perp_{\hat{\mathbf n}},
]
а (\mathbb T_{ij}=\varepsilon_b,\mathbb B_i^{(b)}\cdot \mathbb K_b(R_{ij})\cdot \mathbb A_j^{(b)}) и сумма по (b).
**Следствие:** если все (\mathbf a_j=0) (чистая инерция) ⇒ (\Psi) стационарно-равновесна ⇒ (\mathbf F_i^{\text{med}}=0).

# 4) Уравнение движения как **баланс давления**

Внешний «двигательный» импульс ( \mathbf F_i^{\text{drv}}(t)) уравновешивается средой:
[
\boxed{;;\mathbf F_i^{\text{drv}}(t);+;\sum_{j=1}^N \mathbb T_{ij};\mathbf a_j(t);=;0;;}\qquad (\star)
]
— это **главное уравнение**. Никаких (m), никаких массовых членов.
Инерция — это **самотерм** ( \mathbb T_{ii}\mathbf a_i) (чем больше ускорение, тем сильнее сжатие впереди/разрежение сзади ⇒ реакция среды (\parallel -\mathbf a_i)).

# 5) Режимы и «мгновенность»

* Реакция **квазимгновенна** на макромасштабе (фазовая скорость (c_\phi) велика) ⇒ временной памяти можно не вводить.
* При попадании расстояний в узлы полос (R_{ij}\approx m\pi/\kappa_b) возможно усиление/перераспределение (переток энергии в волны среды) — реализуется **модификацией коэффициентов** (\varepsilon_b\to \varepsilon_b,\Xi_{ij}^{(b)}(\mathbf a_j,R_{ij})) без экспонент (степенные/тригонометрические окна).

# 6) «Задание для программиста» (без кода)

**Входные параметры:** ({\mathbf R_i^0,\mathbf v_i^0}), ориентационные триады ((\hat{\mathbf t}_i,\hat{\mathbf p}_i,\hat{\mathbf r}_i)), анизотропии (\alpha_i^{(\cdot)}); полосы (\mathcal B).
**На каждом шаге (t\to t+\Delta t):**

1. Построить проекторы (\mathbb A_i^{(b)},\mathbb B_i^{(b)}).
2. Для всех пар ((i,j)) вычислить (\mathbb K_b(R_{ij})) и (\mathbb T_{ij}=\sum_b \varepsilon_b,\mathbb B_i^{(b)}\cdot \mathbb K_b\cdot \mathbb A_j^{(b)}).
3. Задать (\mathbf F^{\text{drv}}_i(t)) (тяги/контакты/граничные воздействия).
4. **Решить линейную систему** ((\star)) на вектор ускорений ({\mathbf a_i}_{i=1}^N) (3N неизвестных).

5. Интегрировать кинематику (чисто геометрически):

* **Ускорение толчком:** задали краткий импульс ( \mathbf F^{\text{drv}}) ⇒ мгновенная (\mathbf a) и немедленная реакция (\mathbf F^{\text{med}}=-\sum_j\mathbb T_{ij}\mathbf a_j).
* **Анизотропия спина:** вращение меняет (\mathbb A,\mathbb B) ⇒ разный отклик вдоль/поперёк (\hat{\mathbf t},\hat{\mathbf p}) (тороид/полоид/радиал).

# 1) Переменные тела (i=1..N)

* Положение (\mathbf R_i(t)), скорость (\mathbf v_i(t)), ускорение (\mathbf a_i(t)).

# 2) Поле «фазового давления» (\Psi) (рождается ускорением)

Ускорение создаёт фронт-бэк асимметрию (сжатие впереди, разрежение сзади). Для источника (j):
[
\Psi_j(\mathbf x,t)=\sum_{b\in\mathcal B}\varepsilon_b;
\mathbb A_j^{(b)}:\mathbf a_j(t);,K_b(\mathbf x-\mathbf R_j(t)),
]
где
(K_b(\mathbf r)=\dfrac{\cos(\kappa_b r+\phi_b(\hat{\mathbf r}))}{r^{\gamma_b}}) — **осциллирующе-степенное** нелокальное ядро (без экспонент),
(\mathbb A_j^{(b)}=\alpha_j^{(t)},\hat{\mathbf t}_j!\otimes!\hat{\mathbf t}_j+\alpha_j^{(p)},\hat{\mathbf p}_j!\otimes!\hat{\mathbf p}_j+\alpha_j^{(r)},\hat{\mathbf r}_j!\otimes!\hat{\mathbf r}_j)
— анизотропный проектор из собственных вращений/волн.

Суммарное поле: (\Psi=\sum_j \Psi_j).

# 3) Сила как **градиент перестройки** (только при ( \mathbf a\neq 0))

Тело (i) «детектирует» (\Psi) своим приёмным проектором (\mathbb B_i^{(b)}) (зависит от (\hat{\mathbf t}*i,\hat{\mathbf p}*i,\hat{\mathbf r}*i)):
[
\mathbf F_i^{\text{med}}(t)=-\sum*{b}\int \mathbb B_i^{(b)}\cdot \nabla \Psi(\mathbf x,t);W_i(\mathbf x-\mathbf R_i),d^3x.
]
Для компактности это редуцируется к **парным тензорам**:
[
\boxed{;;\mathbf F_i^{\text{med}}(t)=\sum*{j=1}^N \mathbb T*{ij}!(\mathbf R_{ij},\hat{\mathbf n}*{ij},\Omega_i,\Omega_j;\mathcal B);\mathbf a_j(t);},
]
где (\mathbf R*{ij}=\mathbf R_i-\mathbf R_j), (\hat{\mathbf n}*{ij}=\mathbf R*{ij}/|\mathbf R_{ij}|), (\Omega_i) — параметры собственных вращений;
(\mathbb T_{ij}) — **чисто геометрико-полосовой** оператор (без «массы»).

Явный вид ядра в (\mathbb T_{ij}):
[
\mathbb K_b(R)=\frac{\gamma_b}{R^{\gamma_b+1}}\cos(\kappa_b R+\phi_b),\hat{\mathbf n}!\otimes!\hat{\mathbf n}
;+;\frac{\kappa_b}{R^{\gamma_b}}\sin(\kappa_b R+\phi_b),\Pi^\perp_{\hat{\mathbf n}},

а (\mathbb T_{ij}=\varepsilon_b,\mathbb B_i^{(b)}\cdot \mathbb K_b(R_{ij})\cdot \mathbb A_j^{(b)}) и сумма по (b).
**Следствие:** если все (\mathbf a_j=0) (чистая инерция) ⇒ (\Psi) стационарно-равновесна ⇒ (\mathbf F_i^{\text{med}}=0).

# 4) Уравнение движения как **баланс давления**

Да — это ложится в модель естественно. Формализуем **ускорение без перетока энергии** через разложение отклика среды на «работающую» и «безработную» части.

# 1) Главное уравнение и разложение отклика

Реакция среды на ускорения ( {\mathbf a_j} ) остаётся линейно-нелокальной:
[
\mathbf F_i^{\text{med}}=\sum_{j=1}^N \mathbb T_{ij},\mathbf a_j,\qquad
\mathbb T_{ij}=\mathbb S_{ij}+\mathbb A_{ij},
]
где

* ( \mathbb S_{ij}=\mathbb S_{ij}^\top ) — **сжимающе-разрежающая (работающая)** часть;
* ( \mathbb A_{ij}=-\mathbb A_{ij}^\top ) — **гироскопическая (безработная)** часть.

Уравнение движения (без «массы»):
[
\boxed{;\mathbf F_i^{\text{drv}}+\sum_j \big(\mathbb S_{ij}+\mathbb A_{ij}\big)\mathbf a_j=0;}\tag{★}
]

# 2) Мощность (критерий «есть/нет перетока»)

Мощность перетока в среду:
[
P_{\text{phase}}=\sum_i \mathbf v_i!\cdot!\mathbf F_i^{\text{med}}
=\underbrace{\sum_{i,j} \mathbf v_i!\cdot!\mathbb S_{ij}\mathbf a_j}*{\text{может быть }\neq 0}
+\underbrace{\sum*{i,j} \mathbf v_i!\cdot!\mathbb A_{ij}\mathbf a_j}*{=0}.
]
Последняя сумма тождественно ноль, потому что ( \mathbb A*{ij} ) — кососимметричен (гироскопическая сила всегда перпендикулярна «правильной» скорости и **не делает работу**).

**Итог:** переток энергии возможен **только** через ( \mathbb S ). Если в текущей конфигурации полос ВБП и ориентаций вращений (\mathbb S\equiv 0) (или её вклад подавлен), то (P_{\text{phase}}=0) — среда не «ест» энергию, хотя ускорение есть.

# 3) Когда ускорение без перетока реализуется на практике

### (A) Равномерное движение по орбите

Полосы ВБП вокруг центрального дефекта — радиальные; реакция среды на ускорение к центру — **чисто гироскопическая**:
[
\mathbf F_i^{\text{med}}=\mathbb A_{ii}\mathbf a_i;\perp;\mathbf v_i,\quad
P_{\text{phase}}=0,\quad E_{\text{mech}}=\text{const}.
]
Это «центростремительный» отклик без работы, орбита поддерживается геометрией (холономией) и рифлением.

### (B) Джеты при прямолинейном движении

При осевой (коаксиальной) конфигурации полос, согласованной с направлением движения, формируется **соленоидальный (струйный)** отклик:
(\mathbb S_{ii}\approx 0), доминирует ( \mathbb A_{ii} ).
Джеты — это **перестройка с нулевой дивергенцией энергетического потока** в ближнем поле: геометрия даёт ускорение, но среда не забирает энергию,
[
P_{\text{phase}}=0,\qquad
\dot E_{\text{mech}}=\sum_i \mathbf F_i^{\text{drv}}!\cdot!\mathbf v_i.
]
(Энергия идёт **только** от драйва в механику, не утекая в среду.)

# 4) Как вычислять ( \mathbb S ) и ( \mathbb A ) из полос ВБП

Для каждой полосы (b) с параметрами ( (\kappa_b,\varepsilon_b,\gamma_b) ) и локальными триадами ((\hat{\mathbf t},\hat{\mathbf p},\hat{\mathbf r})):
[
\mathbb K_b(R)=\frac{\gamma_b}{R^{\gamma_b+1}}\cos(\kappa_b R+\phi_b),\hat{\mathbf n}!\otimes!\hat{\mathbf n}
+\frac{\kappa_b}{R^{\gamma_b}}\sin(\kappa_b R+\phi_b),\Pi^\perp_{\hat{\mathbf n}},
]
[
\mathbb T_{ij}^{(b)}=\varepsilon_b,\mathbb B_i^{(b)}\cdot \mathbb K_b(R_{ij})\cdot \mathbb A_j^{(b)}.
]
Далее:
[
\mathbb S_{ij}=\tfrac12\big(\mathbb T_{ij}+\mathbb T_{ij}^\top\big),\qquad
\mathbb A_{ij}=\tfrac12\big(\mathbb T_{ij}-\mathbb T_{ij}^\top\big).
]

* **Работающая часть ( \mathbb S )** рождается в компонентах, выровненных по фронт-бэк (сжатие/разрежение).
* **Безработная ( \mathbb A )** — поперечные (гироскопические) компоненты, математически это «поворот на 90°» в плоскости полос (аналог Лоренц-/Кориолис-типа, но без полей и масс).

# 5) Быстрые критерии (в коде)

На каждом шаге:

1. Собрать (\mathbb T_{ij}) из полос/ориентаций; разложить на (\mathbb S,\mathbb A).
2. Решить (★) на ( {\mathbf a_i} ).
3. Считать индикаторы:
   [
   P_{\text{phase}}=\sum_{i,j}\mathbf v_i!\cdot!\mathbb S_{ij}\mathbf a_j,\qquad
   P_{\text{gyro}}=\sum_{i,j}\mathbf v_i!\cdot!\mathbb A_{ij}\mathbf a_j\equiv 0.
   ]
4. **Режим орбиты/джетов:** если ( |P_{\text{phase}}| \le \epsilon ) → «ускорение без перетока».

   * Орбита: (\mathbf F_i^{\text{med}}\perp \mathbf v_i) и радиальна.
   * Джеты: (\operatorname{div} \mathbf S_{\text{phase}}\approx 0) в ближнем поле; доминирует ( \mathbb A).

# 6) Никаких классических паттернов

* Нет «массы-константы» — инерция это ( \mathbb T_{ii}\mathbf a_i).
* Нет «массовых членов» в потенциалах.
* Нет экспонент — только осциллирующе-степенные ядра (R^{-\gamma}{\cos,\sin}(\kappa R)).

# 7) Мини-чек-листы экспериментов/числа

* **Орбита:** запуск на круг → фиксируешь (P_{\text{phase}}\approx 0), энергия механики постоянна, перицентральная прецессия задаётся геометрией (холономия/рифление).
* **Прямолинейный разгон с джетами:** при осевой конфигурации полос ( \mathbb S\to 0), ( \mathbb A) — макс.; скорость растёт по питанию драйва, а (P_{\text{phase}}\to 0).
