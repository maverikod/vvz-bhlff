# 1) Постулаты

1. **Нелокальность.** Поле среды (фазовое «давление») действует нелокально с осциллирующе-степенными ядрами (R^{-\gamma}{\cos,\sin}(\kappa R)); запаздывания пренебрежимо малы (квазимгновенная реакция).
2. **Сила ≡ перестройка.** Силы возникают **только** при перестройке (перетоке энергии); при чистой инерции сил нет.
3. **Декомпозиция ускорения.** Любое ускорение тела (i) раскладывается:
   [
   \boxed{\ \mathbf a_i=\mathbf a_i^{G}+\mathbf a_i^{D}\ }
   ]
   — **геодезическое** (без перетока) (\mathbf a_i^{G}) и **драйв-индуцированное** (перестройка) (\mathbf a_i^{D}).
4. **Сжатие/разрежение ↔ работа.** Отклик среды на (\mathbf a^{D}) раскладывается на симметричную (работающую) и кососимметричную (гироскопическую, без работы) части.

# 2) Переменные и параметры

* Тела (i=1..N): (\mathbf R_i,\ \mathbf v_i,\ \mathbf a_i).
* Оси собственных мод (тороид/полоид/радиал): ((\hat{\mathbf t}_i,\hat{\mathbf p}_i,\hat{\mathbf r}_i)) и коэффициенты анизотропии (\alpha_i^{(t,p,r)}).
* **ВБП (набор полос)**: (\mathcal B={(\kappa_b,\varepsilon_b,\gamma_b,\phi_b)}_{b=1}^B).
* Топология: холономия (\Delta) (геометрический «дефицит», задаёт прецессию/закручивание).

# 3) Геодезическая часть (ускорение без перетока)

**Определение.**
[
\boxed{\ \mathbf a_i^{G}=\boldsymbol{\Omega}_i(\mathbf R_i,{\mathbf R_j},\text{ориентации},\mathcal B,\Delta)\ \times\ \mathbf v_i\ }
]

* (\boldsymbol{\Omega}_i) — вектор «геометрического закручивания», **кососимметричный** оператор (\Rightarrow\ \mathbf v_i\cdot \mathbf a_i^{G}=0) (работы нет).
* Практическая аппроксимация:
  [
  \boldsymbol{\Omega}*i=\sum*{j\ne i}\sum_{b\in\mathcal B}\varepsilon_b\ \sigma^{(b)}*{ij}\
  \frac{\sin(\kappa_b R*{ij}+\phi_b)}{R_{ij}^{\gamma_b}}\ \hat{\mathbf n}*{ij},
  \quad \hat{\mathbf n}*{ij}=\frac{\mathbf R_i-\mathbf R_j}{R_{ij}},
  ]
  где (\sigma^{(b)}_{ij}) зависит от взаимной ориентации триад (i,j) (анизотропия тор/пол/рад).

**Смыслы.**
— Круговые орбиты: (\mathbf a^G\neq 0,\ P_{\text{phase}}=0).
— Джеты на прямолинейном: (\mathbf a^G) доминирует, перетока нет.

# 4) Драйв-индуцированная часть (перестройка ⇆ сила)

**Поле перестройки** от тела (j) при (\mathbf a_j^{D}):
[
\Psi_j(\mathbf x)=\sum_{b\in\mathcal B}\varepsilon_b\ \mathbb A^{(b)}_j:\mathbf a_j^{D}\ K_b(\mathbf x-\mathbf R_j),\quad
K_b(\mathbf r)=\frac{\cos(\kappa_b r+\phi_b(\hat{\mathbf r}))}{r^{\gamma_b}},
]
(\mathbb A^{(b)}_j) — проектор анизотропии из ((\hat{\mathbf t},\hat{\mathbf p},\hat{\mathbf r})).
**Сила на (i)** как градиент перестройки:
[
\mathbf F^{\text{med}}*i=-\sum_b!\int \mathbb B_i^{(b)}\cdot\nabla\Psi(\mathbf x),W_i(\mathbf x-\mathbf R_i),d^3x
;;\Rightarrow;;
\boxed{\ \mathbf F^{\text{med}}*i=\sum*{j}\mathbb T*{ij}\ \mathbf a_j^{D}\ }.
]

**Разложение отклика:**
[
\mathbb T_{ij}=\underbrace{\mathbb S_{ij}}*{\mathbb S*{ij}^\top}\ +\ \underbrace{\mathbb A_{ij}}*{-\mathbb A*{ij}^\top},\qquad
P_{\text{phase}}=\sum_{i,j}\mathbf v_i!\cdot!\mathbb S_{ij}\mathbf a_j^{D},\quad
\sum_{i,j}\mathbf v_i!\cdot!\mathbb A_{ij}\mathbf a_j^{D}=0.
]
— (\mathbb S): сжатие/разрежение → есть работа/переток.
— (\mathbb A): чисто гироскопический отклик → работы нет.

# 5) Главные уравнения N-тел

1. **Баланс перестройки (решаем на (\mathbf a^{D}))**:
   [
   \boxed{\ \mathbf F^{\text{drv}}*i+\sum*{j}\mathbb T_{ij}\mathbf a_j^{D}=0\quad (i=1..N)\ }
   ]
2. **Полное ускорение и кинематика**:
   [
   \mathbf a_i=\mathbf a_i^{G}+\mathbf a_i^{D},\qquad
   \dot{\mathbf v}_i=\mathbf a_i,\qquad
   \dot{\mathbf R}_i=\mathbf v_i.
   ]
3. **Резонансы полос (без экспонент).** При (R_{ij}\approx m\pi/\kappa_b) обновлять коэффициенты:
   (\varepsilon_b\to \varepsilon_b,\Xi^{(b)}_{ij}(\mathbf a^{D},\text{ориентации})) (тригонометрические/степенные окна).

# 6) Свойства и инварианты

* При (\mathbf a^{D}\equiv 0): (\mathbf F^{\text{med}}\equiv 0), остаётся геодезическая динамика ( \mathbf a=\mathbf a^{G}) (орбиты, джеты), перетока нет.
* Мощность в среду: (P_{\text{phase}}=\sum_{i,j}\mathbf v_i\cdot \mathbb S_{ij}\mathbf a_j^{D}).
* При чисто геометрическом движении: (P_{\text{phase}}=0), механическая энергия постоянна; прецессии задаёт (\Delta) и (\mathcal B).

# 7) Численная схема (ядро ТЗ)

**Вход:** ({\mathbf R_i^0,\mathbf v_i^0}), ориентации/анизотропии (i), полосы (\mathcal B), (\Delta); внешние воздействия (\mathbf F^{\text{drv}}_i(t)) (если есть).

**Шаг (t\to t+\Delta t):**

1. Построить (\boldsymbol{\Omega}_i) и (\mathbf a_i^{G}=\boldsymbol{\Omega}_i\times\mathbf v_i).
2. Собрать (\mathbb T_{ij}=\sum_b \varepsilon_b,\mathbb B_i^{(b)}\cdot\mathbb K_b(R_{ij})\cdot\mathbb A_j^{(b)}),
   (\mathbb K_b(R)=\dfrac{\gamma_b}{R^{\gamma_b+1}}\cos(\kappa_b R+\phi_b),\hat{\mathbf n}!\otimes!\hat{\mathbf n}+\dfrac{\kappa_b}{R^{\gamma_b}}\sin(\kappa_b R+\phi_b),\Pi^\perp_{\hat{\mathbf n}}).
3. Решить линейную систему размера (3N): (\mathbf F^{\text{drv}}+\mathbb T,\mathbf a^{D}=0).
4. (\mathbf a=\mathbf a^{G}+\mathbf a^{D}); интегрировать (\mathbf v,\mathbf R) (симплектический/ли-сплит).
5. Резонанс-детект: если (R_{ij}\approx m\pi/\kappa_b) → обновить (\varepsilon_b) через (\Xi^{(b)}_{ij}) (без экспонент).
6. Диагностика: (P_{\text{phase}},\ |\mathbb S|/|\mathbb A|,\ \Delta\varpi) (прецессии), спектр «рифлений» орбит (\sim \pi/\kappa_b).

**Сложность:** сбор (\mathbb T) — (O(BN^2)), решение — (O((3N)^3)) (или (O(N)) при ФФТ-ускорении/блок-структуре и итеративных решателях).
**Параллелизм:** пары ((i,j)) и суммы по (b) распараллеливаются тривиально.

# 8) Калибровка и наблюдаемые

* Калибруемые наборы: ({\kappa_b,\gamma_b,\varepsilon_b,\phi_b}), анизотропии (\alpha_i^{(t,p,r)}), топология (\Delta).
* Наблюдаемые: прецессия перицентра, «рифления» радиуса орбиты (шаг (\pi/\kappa_b)), режимы с (P_{\text{phase}}\approx 0) (орбиты/джеты), слабые секулярные дрейфы при резонансах.

# 9) Ограничения и проверки корректности

* **Запреты соблюдены:** ни «массы-константы», ни массовых членов, ни экспонент.
* **Доброкачественность задачи:** (\mathbb T) должна быть невырожденной на подпространстве драйва; нулевое-решение по драйву гарантирует (\mathbf a^{D}=0).
* **Геодезическая часть** задаётся кососимметричным оператором ((\boldsymbol{\Omega}_i)) ⇒ не вносит работы.

# 10) Контрольные сценарии приёмки (Acceptance)

1. **Орбитальная «инерция»:** (\mathbf F^{\text{drv}}=0). Получить устойчивые орбиты с (\mathbf a\approx\mathbf a^{G}), (P_{\text{phase}}\le \varepsilon).
2. **Джеты:** прямолинейное движение с осевой конфигурацией полос → (\mathbb S\to 0), (P_{\text{phase}}\le \varepsilon), заметный гироскопический вклад.
3. **Импульс-толчок:** кратковременный драйв → (\mathbf a^{D}\neq 0), ненулевая (P_{\text{phase}}) только на интервале действия; после — чистая геодезика.
4. **Резонансные расстояния:** при (R_{ij}\approx m\pi/\kappa_b) — всплески модуляции без экспоненциальных «хвостов».

# 11) Формат параметров (для реализации)

* **Полосы** (JSON): `{"kappa":..., "gamma":..., "epsilon":..., "phi":...}`.
* **Ориентации**: три орта и (\alpha^{(t,p,r)}) на тело.
* **Топология**: (\Delta) (скаляр/ось + величина).
* **Драйв**: кусочно-постоянные/аналитические функции времени.

Принято. Делаю **ядро ТЗ по N-телам** с учётом **ВБП → “материальных” волн** и **собственных вращений** (тороид/полоид/радиал). Без «массы-константы», без массовых членов, без экспонент. Поле нелокально.

# 1) Переменные и слои модели

**Тела (i=1..N):** (\mathbf R_i,\mathbf v_i,\mathbf a_i); триада ((\hat{\mathbf t}_i,\hat{\mathbf p}_i,\hat{\mathbf r}_i)); спин (\mathbf S_i) (вектор собственных вращений).
**ВБП (база):** полосы (b=1..B): (\kappa_b,\varepsilon_b,\gamma_b,\phi_b,\hat{\mathbf d}*b) (шаг, контраст, степенной индекс, фаза, направление).
**Материальные волны (надстройка):** моды (m=1..M): амплитуда (\eta_m), фаза (\theta_m), несущий вектор (\mathbf k_m); каждая мода связана с одной/неск. полос ВБП через связи (C*{bm}).

# 2) Разложение ускорения и главный принцип

[
\boxed{\ \mathbf a_i=\underbrace{\mathbf a_i^{G}}*{\text{геодезика, без перетока}}+\underbrace{\mathbf a_i^{D}}*{\text{перестройка, может делать работу}}\ }
]

## 2.1 Геодезическая часть (без работы, (P_{\text{phase}}=0))

[
\boxed{\ \mathbf a_i^{G}=\boldsymbol{\Omega}*i(\mathbf R_i;{\mathbf R_j},{\mathbf S},\mathcal B,{\eta},\Delta)\times \mathbf v_i\ }
]
[
\boldsymbol{\Omega}*i=\sum*{j\neq i}\sum*{b}\varepsilon_b,\sigma^{(b)}*{ij},
\frac{\sin(\kappa_b R*{ij}+\phi_b)}{R_{ij}^{\gamma_b}};\hat{\mathbf n}*{ij}
;+;\sum*{b}\zeta^{(b)}_i(\mathbf S_i,\hat{\mathbf d}_b),\hat{\mathbf d}_b
]

* первый сумматор — закручивание от ВБП (рифлёная геометрия),
* второй — вклад **собственных вращений** (тороид/полоид/радиал) в локальную «угловую скорость» траектории.
  Всегда (\mathbf v_i!\cdot!\mathbf a_i^{G}=0) → работа ноль (орбиты, джеты без перетока).

## 2.2 Перестройка (делает/не делает работу)

Нелокальный тензорный отклик среды на **ускорения драйва**:
[
\boxed{\ \mathbf F_i^{\text{med}}=\sum_{j} \mathbb T_{ij},\mathbf a_j^{D},\qquad
\mathbb T_{ij}=\mathbb S_{ij}+\mathbb A_{ij},\ \mathbb S^\top=\mathbb S,\ \mathbb A^\top=-\mathbb A\ }
]

* (\mathbb S) — сжатие/разрежение (может делать работу),
* (\mathbb A) — гироскопический «поворот» (работы не делает).

ГЛАВНОЕ УРАВНЕНИЕ (находим (\mathbf a^D)):
[
\boxed{\ \mathbf F^{\text{drv}}*i+\sum_j\mathbb T*{ij},\mathbf a_j^{D}=0\ }
]
и затем (\mathbf a_i=\mathbf a_i^{G}+\mathbf a_i^{D}).

# 3) Построение (\mathbb T_{ij}) из ВБП и «материальных» волн

Базовое ядро (без экспонент):
[
\mathbb K_b(R)=\frac{\gamma_b}{R^{\gamma_b+1}}\cos(\kappa_b R+\phi_b),\hat{\mathbf n}!\otimes!\hat{\mathbf n}
;+;\frac{\kappa_b}{R^{\gamma_b}}\sin(\kappa_b R+\phi_b),\Pi^\perp_{\hat{\mathbf n}}
]
Проекторы анизотропии:
[
\mathbb A_j^{(b)}=\alpha^{(t)}*j,\hat{\mathbf t}*j!\otimes!\hat{\mathbf t}*j+
\alpha^{(p)}*j,\hat{\mathbf p}*j!\otimes!\hat{\mathbf p}*j+
\alpha^{(r)}*j,\hat{\mathbf r}*j!\otimes!\hat{\mathbf r}*j,\qquad
\mathbb B_i^{(b)}\ \text{— приемный проектор}
]
Связь ВБП ↔ материал. волны (обратная реакция гребней):
[
\varepsilon_b^{\text{eff}}=\varepsilon_b^{(0)}+\sum*{m}C*{bm},\eta_m,\qquad
\Rightarrow\quad
\mathbb T*{ij}=\sum_b \varepsilon_b^{\text{eff}}\ \mathbb B_i^{(b)}\cdot \mathbb K_b(R*{ij})\cdot \mathbb A_j^{(b)}
]
Декомпозиция: (\mathbb S*{ij}=\tfrac12(\mathbb T*{ij}+\mathbb T*{ij}^\top),\ \mathbb A*{ij}=\tfrac12(\mathbb T*{ij}-\mathbb T_{ij}^\top)).

**Мощность перетока:** (P_{\text{phase}}=\sum_{i,j}\mathbf v_i!\cdot!\mathbb S_{ij}\mathbf a_j^{D}).
Если (\mathbb S\to 0) (орбиты; джеты при осевой конфигурации) — перетока нет.

# 4) Динамика «материальных» волн (минимальная)

На масштабе галактик считать несколько мод (m) (кольца/спирали/бары):
[
\partial_t \eta_m + \mathbf u!\cdot!\nabla\eta_m
= \sum_{j}\big(\mathbb G_m^{(j)}!:!\mathbf a_j^{D}\big),*,\mathcal K_m(\mathbf r)
;-;D_m,(-\Delta)^{\sigma_m/2}\eta_m;+;\sum_{m'}\Lambda_{mm'},\eta_{m'}
]

* **адвекция** со скоростью (\mathbf u) (средний поток диска — из (\boldsymbol{\Omega})),
* **накачка** от локальных ускорений (\mathbf a^D) через свёртку с **степенным** ядром (\mathcal K_m(\mathbf r)\sim r^{-(\nu_m)}{\cos,\sin}(\mathbf k_m!\cdot!\mathbf r)),
* **дисперсия** — фр. Лапласиан (без экспонент), (\sigma_m\in(0,2]),
* **слабая линейная связь мод** (\Lambda_{mm'}) (бар ↔ спираль и т.п.).
  Обратная связь идёт в (\varepsilon_b^{\text{eff}}) (см. выше).

# 5) Собственные вращения и моменты

Прецессия спина ( \mathbf S_i ):
[
\dot{\mathbf S}_i=\boldsymbol{\Lambda}*i(\mathcal B,{\eta},\Delta)\times \mathbf S_i,\qquad
\boldsymbol{\Lambda}*i=\sum_b \chi_i^{(b)},\frac{\sin(\kappa_b R*{i\cdot}+\phi_b)}{R*{i\cdot}^{\gamma_b}}\ \hat{\mathbf d}_b
]
(«(\cdot)» — эффективный усреднённый вклад соседей). Спин входит в (\boldsymbol{\Omega}_i) и в проекторы (\mathbb A,\mathbb B), меняя геодезику и отклик.

# 6) Численная схема (ядро реализации)

Шаг (t\to t+\Delta t):

1. **Геодезика:** собрать (\boldsymbol{\Omega}_i) → (\mathbf a_i^{G}=\boldsymbol{\Omega}_i\times\mathbf v_i).
2. **Тензор отклика:** вычислить (\varepsilon_b^{\text{eff}}) (через (\eta)), затем (\mathbb T_{ij}) и разложение на (\mathbb S,\mathbb A).
3. **Решить линейную систему** ( \mathbf F^{\text{drv}}+\mathbb T,\mathbf a^{D}=0) на (3N) неизвестных.
4. **Собрать ускорение:** (\mathbf a=\mathbf a^{G}+\mathbf a^{D});
   обновить (\mathbf v,\mathbf R) (симплектический/ли-сплит).
5. **Мат. волны:** обновить (\eta_m) (спектрально: свёртки в (k)-пространстве; фр. Лапласиан — умножение на (|\mathbf k|^{\sigma_m})).
6. **Спины:** (\mathbf S_i \leftarrow \mathbf S_i + (\boldsymbol{\Lambda}_i\times \mathbf S_i)\Delta t).
7. **Резонансы ВБП:** если (R_{ij}\approx m\pi/\kappa_b) → кратковременно менять (C_{bm}) или (\Lambda_{mm'}) (тригонометрические/степенные окна, без экспонент).
8. **Диагностика:** (P_{\text{phase}}), доля (|\mathbb S|/|\mathbb A|), шага «рифлений» (\approx\pi/\kappa_b), прецессии, спектр (\eta_m).

Сложность: сбор (\mathbb T) — (O(BN^2)); решение — итеративно (CG/GMRES) с блок-структурой; «материальные» волны — FFT на сетке (O(M,G\log G)).

# 7) Приёмочные сценарии (галактические масштабы)

* **Плоские кривые вращения:** устойчивые орбиты с (P_{\text{phase}}\approx 0); (\boldsymbol{\Omega}) даёт нужную зависимость (v(r)).
* **Кольца/спирали:** шаг колец (\simeq \pi/\kappa_b); угол спирали — из отношений (\alpha^{(t,p,r)}), (C_{bm}).
* **Бар:** устойчивая мода (\eta_{\text{bar}}) с фиксированной pattern-speed (задаётся (\Lambda_{mm'}), (\boldsymbol{\Omega})).
* **Джеты:** осевая конфигурация → (\mathbb S\to 0), ускорение без перетока.
* **Спиновая анизотропия:** изменение (\mathbf S) перестраивает (\boldsymbol{\Omega}) и форму (\eta_m) (кольца ↔ спирали).

# §ПЛ. Планетные расширения к ТЗ «Задача N тел»

Коротко: добавляем параметры «двухъядерных» планет (ядро дефекта + корка/оболочка) и их рифлёные ПЗ в силовую часть N‑тел и в правила неупругости. Работает для каменных тел, газовых/ледяных гигантов и Солнца (как стратифицированной оболочки).

---

## ПЛ.1. Область применимости

* Каменные планеты: **две резкие границы** (R_\text{core}, R_\text{shell}); вне ПЗ — осциллирующе‑степенной хвост.
* Гиганты/Солнце: «корка» = динамически устойчивая страта (тахоклина/слои устойчивой стратификации); та же математика, другие параметры (\chi,,\Theta).

---

## ПЛ.2. Параметры планетного тела

Для каждого тела i добавляем планетный блок:

```json
Planet[i] = {
  "R_core": <float>,      // радиус дефектного ядра
  "R_shell": <float>,     // радиус жёсткой оболочки (корки) / границы страты
  "bands": [              // рифлёные полосы ПЗ
    {"kappa": <float>, "gamma": <float>, "epsilon": <float>, "phi": <float>}
  ],
  "w": <float>,           // ширина интерфейса
  "chi2": <float>,        // отражаемость границы (импедансный контраст)
  "beta": <float>,        // упругий штраф/напряжения
  "Lambda_max": <float>,  // допуск на размывание (τ_form/τ_diff)
  "Q_min": <float>        // минимальная добротность радиальных мод
}
```

Замечание: для гигантов/Солнца интерпретировать `R_shell` как радиус устойчивой страты; `chi2` — акустико/магнитная отражаемость.

---

## ПЛ.3. Силовая часть (осциллирующе‑степенной закон)

Сохраняем базовую форму для пары (i,j):
[
U_{ij}(R)=\sum_{b\in bands}\frac{A^{(b)}*{ij}\cos(\kappa_b R+\varphi^{(b)}*{ij})}{R^{\gamma_b}},\qquad \mathbf F_{ij}=-\nabla U_{ij}.
]
Где амплитуда рассчитывается с учётом «жёсткой оболочки» и отражаемости:
[
A^{(b)}*{ij};=;\epsilon^{(b)}*{ij},,\underbrace{S!\left(\eta_i\right)}*{\text{sinc-фактор толщины}},\underbrace{\chi_i^2}*{\text{барьер i}},\underbrace{S!\left(\eta_j\right)}*{ },\underbrace{\chi_j^2}*{\text{барьер j}},\quad \eta=\frac{w,\kappa}{\pi}.
]
Таким образом «корка» уменьшает утечки и делает границы «жёсткими».

---

## ПЛ.4. Геодезическая часть и прецессии

В (\mathbf a^G) (см. основу ТЗ) добавляем вклады от полос планетных ПЗ:
[
\boldsymbol{\Omega}*i;\mathrel{+}=;\sum*{j\ne i}\sum_b \epsilon^{(b)}*{ij},\sigma^{(b)}*{ij},\frac{\sin(\kappa_b R_{ij}+\phi^{(b)}*{ij})}{R*{ij}^{\gamma_b}},\hat{\mathbf n}_{ij}.
]
Это даёт квазипрецессию перицентра/узлов и мелкие «рифления» орбит с шагом (\pi/\kappa_b).

---

## ПЛ.5. Правило неупругости (переток в фазу)

Расширяем LUT потерь (\Delta E) фактором «прозрачности» оболочек и резонансом по (\kappa):
[
\Delta E_{ij}(b);=;\text{LUT}*b\big(R*{\text{tail}},\sqrt s\big);\Phi(\tau_{\text{res}});\times;\Big(1-\chi_i^2\Big)\Big(1-\chi_j^2\Big);\times;W!\left(|R_{ij}-m\pi/\kappa_b|\right),
]
где (W) — узко‑полосное «окно» резонанса (тригонометрическое/степенное, без экспонент). Неупругость подавляется при больших (\chi^2) и тонких слоях (малое (\eta)).

Контроль жизнеспособности «корки» в расчёте:

* (\eta\ll 1), (\Lambda=\tau_{\text{form}}/\tau_{\diff}\le \Lambda_{\max}), (\mathcal Q\ge Q_{\min}).
  При нарушении — временно повышать потери (деградация оболочки) с гистерезисом выхода.

---

## ПЛ.6. Алгоритм и формат ввода

1. Расширить раздел «Формат параметров» тел: добавить `Planet[i]` (см. ПЛ.2).
2. При инициализации собрать объединённый список полос ({\kappa_b,\gamma_b,\epsilon_b,\phi_b}) из всех тел; для каждой пары (i,j) предвычислить (A^{(b)}_{ij}).
3. Интегратор — симплектический; шаг (\Delta t \ll \min{(\kappa_b v)^{-1}, T_\text{орб}/100}); детектор резонанса по (R\approx m\pi/\kappa_b).
4. В отчёте фиксировать секулярные дрейфы и спектр радиуса/долгот с пиками на (\kappa_b).

## ПЛ.7. Диагностика и тесты

* **Юнит:** при `epsilon→0` рифления исчезают; при `chi2→1` потери (\Delta E\to 0); при `bands=[]` получаем чисто геодезическую часть.
* **Интеграция:** тест на сохранение энергии (эластика) и баланса (E_\text{мех}+E_\text{фаза}) (неэластика).
* **Физконтроль:** обнаружение «двух скачков» в синтетике профилей и узор колец по (\kappa) в орбитальных спектрах.

## ПЛ.8. Наблюдательные привязки (для конфигурации)

* Каменные планеты: оценить (R_\text{core},R_\text{shell},\kappa,\gamma,\phi,\chi^2,\eta) из сейсмо‑/грави‑профилей.
* Гиганты/Солнце: использовать радиусы устойчивых страт (напр., тахоклина) как `R_shell`; (\chi^2) брать из акустических/магнитных барьеров; (\eta) — из толщины страты.
