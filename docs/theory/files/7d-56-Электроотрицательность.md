# 0) Конвенция (важно про коэффициент 2)

Дальше **использую стандарт Парра–Пирсона**:
[
\boxed{\ \chi \equiv -\mu,\qquad
\eta \equiv \frac{I-A}{2},\qquad S\equiv \eta^{-1}\ }
]
Тогда локально (около целого (N_0)) квадратичное разложение по (\Delta N=N-N_0) удобно писать как
[
E(N);\approx;E_0;+;\mu_0,\Delta N;+;\eta,(\Delta N)^2,
]
и потому (\dfrac{\partial^2 E}{\partial N^2}\big|_{N_0}=2\eta). (Если предпочитаете (\eta=\partial^2E/\partial N^2), просто умножайте мои (\eta) на 2.)

# 1) Электроотрицательность через (I,A) и производные по (N)

**Тезис 1 (Mulliken ↔ химпотенциал).** При фиксированном (v):
[
\mu=\Big(\frac{\partial E}{\partial N}\Big)_v,\qquad
\chi:=-\mu\ \ \Rightarrow\ \ \boxed{\ \chi\approx \frac{I+A}{2}\ } .
]

**Доказательство.** Возьмём (E(N)\approx E_0+\mu_0\Delta N+\eta(\Delta N)^2). Тогда
[
I=E(N-1)-E(N)=-\mu_0+\eta,\qquad
A=E(N)-E(N+1)=-\mu_0-\eta.
]
Складывая: (I+A=-2\mu_0\Rightarrow \chi=-\mu_0=(I+A)/2). Разность даёт (\eta=(I-A)/2). (\square)

**Замечание (точная ДФТ-картинка).** Из кусочно-линейности (E(N)) и разрывности производной: (\mu^-=-I,\ \mu^+=-A). Наше ( \mu_0 ) — среднее ((\mu^-+\mu^+)/2), т.е. формула Мулликена.

---

# 2) Принцип выравнивания электроотрицательности и перенос заряда

Рассмотрим два фрагмента (A,B), обменивающихся зарядом (\Delta N) (положительное (\Delta N) — к (A)). Энергия (с учётом взаимного кулоновского ядра (J_{AB}>0)):
[
E_{\rm tot}(\Delta N)=
\underbrace{E_A^0+\mu_A,\Delta N+\eta_A(\Delta N)^2}*{A}+
\underbrace{E_B^0-\mu_B,\Delta N+\eta_B(\Delta N)^2}*{B}+
J_{AB}(\Delta N)^2 .
]
Минимизация:
[
\frac{dE_{\rm tot}}{d\Delta N}= (\mu_A-\mu_B)+2(\eta_A+\eta_B+J_{AB}),\Delta N=0,
]
[
\boxed{\ \Delta N^\star=\frac{\mu_B-\mu_A}{2(\eta_A+\eta_B+J_{AB})}
=\frac{\chi_A-\chi_B}{2(\eta_A+\eta_B+J_{AB})}\ }.
]
В оптимуме (\mu_A^\star=\mu_B^\star=\mu_{\rm eq}) — **выравнивание электроотрицательности**.

Энергия стабилизации за счёт переноса заряда:
[
\boxed{\ \Delta E_{\rm CT}=
-\frac{(\mu_A-\mu_B)^2}{4(\eta_A+\eta_B+J_{AB})}
=-\frac{(\chi_A-\chi_B)^2}{4(\eta_A+\eta_B+J_{AB})}\ } .
]

# 3) Частичные заряды в молекуле (Electronegativity Equalization Method, EEM)

Пусть атомы (i=1..M) с «атомными» параметрами (\chi_i^0,\ \eta_i). Введём заряды (q_i) (в электронных единицах, (\sum_i q_i=Q_{\rm tot})). Квадратичный функционал:
[
E(\mathbf q)=\sum_i\Big(\chi_i^0,q_i+\eta_i,q_i^2\Big)
+\sum_{i<j} J_{ij},q_i q_j,
]
где (J_{ij}) — кулоновское ядро (с экранированием/Робином вашей границы). Лагранжиан (\mathcal L=E-\lambda(\sum q_i-Q_{\rm tot})) даёт систему
[
\boxed{\ \chi_i^0+2\eta_i q_i +\sum_{j\ne i} J_{ij}q_j=\lambda,\quad
\sum_i q_i=Q_{\rm tot}\ }.
]
Левая часть равна (\mu_i(\mathbf q)); (\lambda) — общий (\mu_{\rm eq}) (электроотрицательность выравнена). Это и есть строгая форма EEM в нашей конвенции.

---

# 4) Локальные индексы: (f(\mathbf r),\ s(\mathbf r)), локальная «электроотрицательность»

* **Функция Фукуя**: ( \displaystyle f(\mathbf r)=\Big(\frac{\partial \rho(\mathbf r)}{\partial N}\Big)_v) — куда течёт заряд при изменении (N).
* **Локальная мягкость**: ( s(\mathbf r)=S,f(\mathbf r),\quad S=\eta^{-1}).
* **Локальная «электроотрицательность» (химпотенциал)**:
  [
  \mu(\mathbf r)=\frac{\delta E[\rho]}{\delta \rho(\mathbf r)},,\qquad \chi(\mathbf r)=-\mu(\mathbf r).
  ]
  Интегрально (\int f(\mathbf r),d^3r=1). В реакциях (\Delta q_A\approx \int_{\Omega_A} f(\mathbf r),d^3r;\Delta N).

---

# 5) Электрофильность / нуклеофильность (Parr)

Глобальная **электрофильность**:
[
\boxed{\ \omega=\frac{\mu^2}{2\eta}=\frac{\chi^2}{2\eta}\ } .
]
При слабом взаимодействии с донором/акцептором изменение энергии по «каналу заряда» имеет вид
[
\Delta E_{\rm CT}\approx -\frac{(\Delta\chi)^2}{4(\eta_A+\eta_B)}\quad
(\text{если }J_{AB}\ll \eta_A+\eta_B),
]
и чем **мягче** пара (меньше (\eta)), тем больше выигрыш энергии за счёт переноса заряда. Локальные версии: (\omega(\mathbf r)=\frac{\mu(\mathbf r)^2}{2,\eta(\mathbf r)}) при введении локальной жёсткости (\eta(\mathbf r)) как диагонали ядра (\delta^2E/\delta\rho(\mathbf r)\delta\rho(\mathbf r')).

---

# 6) Полярность связи и диполь

Для двухцентровой связи (A!-!B) с найденным (\Delta N^\star) частичные заряды (q_A=-\Delta N^\star,\ q_B=+\Delta N^\star). Связевой диполь:
[
\mathbf d=\sum_i q_i,\mathbf R_i\ \ \Rightarrow\ \
|\mathbf d_{AB}|\approx |\Delta N^\star|,R_{AB},
]
(R_{AB}) — межъядерное расстояние (единицы: при необходимости умножить на (e)). Полярность растёт с (|\chi_A-\chi_B|) и падает с (\eta_A+\eta_B+J_{AB}).

# 7) Как считать из вашей теории (без феноменологии)

1. **(I,A)**: решить стационары для состояний с (N,N!\pm!1) электронами (электронный сектор из (\Theta)+EM), взять
   (I=E(N-1)-E(N),\ A=E(N)-E(N+1)).
2. **(\chi,\eta)**: (\chi=(I+A)/2,\ \eta=(I-A)/2).
3. **(J_{ij})**: из EM-ядра (зелёновский оператор Пуассона) с вашими граничными условиями Робина/диэлектриком.
4. **EEM**: решить линейную систему из §3 с ограничением (\sum q_i=Q_{\rm tot}) → частичные заряды, (\mu_{\rm eq}).
5. **Локальные индексы**: (f(\mathbf r)) и (s(\mathbf r)) — линейный отклик (\delta\rho/\delta N) вашего солвера; (\chi(\mathbf r)=-\delta E/\delta\rho(\mathbf r)).
6. **Валидация**: тренды по периоду — рост (\chi) слева→вправо (рост (Z_{\rm eff})), падение вниз по группе (рост радиуса); полярности связей и диполи — из (\Delta N^\star).

# 8) Дополнительно: две «стороны» электроотрицательности (лев./прав.)

Из разрывности (\mu^\pm):
[
\chi^+ := -\mu^+ = A,\qquad \chi^- := -\mu^- = I,\qquad
\chi=\frac{\chi^++\chi^-}{2}.
]
Это полезно для направленности процессов «донорство/акцепторство» (прибавление/отнятие электрона): выбор (\chi^\pm) согласован с соответствующим (f^\pm(\mathbf r)) (атакующие/уходящие фронтир-орбитали).

## Итого

* (\chi) — **минус химпотенциал**; практически (\chi=(I+A)/2).
* (\eta=(I-A)/2) задаёт сопротивление изменению (N) (жёсткость); (S=\eta^{-1}).
* Равновесие между фрагментами: (\mu_A^\star=\mu_B^\star), перенос (\Delta N^\star=(\chi_A-\chi_B)/[2(\eta_A+\eta_B+J_{AB})]).
* Молекулярные **частичные заряды** — решение линейной EEM-системы, где левая часть — выровненная электроотрицательность.
* Локальные индексы (f(\mathbf r), s(\mathbf r), \chi(\mathbf r)) — из отклика (\rho).
* Энергетика донор–акцептор: (\Delta E_{\rm CT}=-(\Delta\chi)^2/[4(\eta_A+\eta_B+J_{AB})]).

# 0) Постановка (что именно минимизируем)

Работаем с электронным сектором, полученным из «хвоста» фазового поля: самосопряжённый оператор
[
\hat H[v,\mathbf A]=\frac{1}{2m}(-i\hbar\nabla-q\mathbf A)^2+v(\mathbf r)
]
на пространстве функций с Кулоновым хвостом и стыком Робина. При фиксированных ядрах (R) внешний потенциал (v(\mathbf r;R)) задан EM-сектором. Энергия основного состояния:
[
E[v;N]=\min_{\rho\to N}\Big{T[\rho]+E_{\mathrm{EM}}[\rho]+E^\Theta_{\mathrm{xc}}[\rho]+!\int \rho(\mathbf r),v(\mathbf r),d^3r\Big}.
]
Здесь (N) — число электронов, (\rho=\sum_{\rm occ}|\psi_i|^2).

---

# 1) Глобальные величины: (\chi, \eta, S) и связь с (I,A)

**Определения (конвенция Парра–Пирсона).**
[
\mu=\Big(\frac{\partial E}{\partial N}\Big)_v,\qquad \chi:=-\mu,\qquad
\eta:=\frac{I-A}{2},\qquad S:=\eta^{-1}.
]

**Утверждение 1 (Мулликен ↔ химпотенциал).** При фиксированном (v):
[
\boxed{\ \chi=\frac{I+A}{2},\qquad \eta=\frac{I-A}{2}\ }.
]

**Доказательство.** Точная ДФТ даёт кусочно-линейное (E(N)) с левым/правым производными (\mu^-=-I,\ \mu^+=-A). В окрестности целого (N_0) берём среднее (\mu_0=(\mu^-+\mu^+)/2=-(I+A)/2\Rightarrow\chi=(I+A)/2). Разность даёт (\eta=(I-A)/2). (\square)

**Следствие (строгое неравенство).** (\eta>0) (выпуклость (E(N))), (S>0).

---

# 2) Принцип выравнивания электроотрицательности (две подсистемы)

Рассмотрим два фрагмента (A,B). Положительное (\Delta N) — переток к (A).

**Энергетическая модель второго порядка:**
[
E_{\rm tot}(\Delta N)
= E_A^0+\mu_A\Delta N+\eta_A(\Delta N)^2;+;
E_B^0-\mu_B\Delta N+\eta_B(\Delta N)^2;+;J_{AB}(\Delta N)^2,
]
где (J_{AB}>0) — «электростатическая жёсткость» пары (см. §4).

**Утверждение 2 (равновесие и перенос).**
[
\boxed{\ \Delta N^\star=\frac{\mu_B-\mu_A}{2(\eta_A+\eta_B+J_{AB})}
=\frac{\chi_A-\chi_B}{2(\eta_A+\eta_B+J_{AB})}\ },
]
в равновесии (\mu_A^\star=\mu_B^\star).

**Доказательство.** (\frac{dE_{\rm tot}}{d\Delta N}=(\mu_A-\mu_B)+2(\eta_A+\eta_B+J_{AB})\Delta N=0). (\square)

**Энергия стабилизации за счёт перетока:**
[
\boxed{\ \Delta E_{\rm CT}
=E_{\rm tot}(\Delta N^\star)-E_{\rm tot}(0)
=-\frac{(\mu_A-\mu_B)^2}{4(\eta_A+\eta_B+J_{AB})}
=-\frac{(\Delta \chi)^2}{4(\eta_A+\eta_B+J_{AB})}\ }.
]

---

# 3) Молекула из (M) центров: матричная EEM-модель

Пусть на атомах (i=1..M) частичные заряды (q_i) ((\sum_i q_i=Q_{\rm tot})). Запишем квадратичный функционал:

[
E(\mathbf q)=\sum_i \chi_i^0,q_i + \frac{1}{2}\sum_i H_{ii},q_i^2 + \frac{1}{2}\sum_{i\ne j} H_{ij},q_i q_j
]
где
[
H_{ii}:=2\eta_i\ (>0),\qquad H_{ij}:=J_{ij}=J_{ji}\ (>0).
]

**Утверждение 3 (линейная система EEM).**
[
\boxed{\ H,\mathbf q+\boldsymbol\chi=\lambda,\mathbf 1,\qquad \mathbf 1^\top \mathbf q=Q_{\rm tot}\ },
]
где (\boldsymbol\chi=(\chi_1^0,\dots,\chi_M^0)^\top), а (\lambda) — выровненный химпотенциал (\mu_{\rm eq}).

**Доказательство.** Условие экстремума Лагранжиана (\mathcal L=E-\lambda(\sum q_i-Q)) даёт (H\mathbf q+\boldsymbol\chi=\lambda\mathbf 1) и ограничение. (\square)

**Решение (замкнутая форма через Шура).**
[
\begin{bmatrix} H & \mathbf 1\ \mathbf 1^\top & 0\end{bmatrix}
\begin{bmatrix}\mathbf q\ -\lambda\end{bmatrix}
===============================================

\begin{bmatrix}-\boldsymbol\chi\ Q_{\rm tot}\end{bmatrix}
\ \Rightarrow
\begin{cases}
\lambda=\dfrac{\mathbf 1^\top H^{-1}\boldsymbol\chi+Q_{\rm tot}}{\mathbf 1^\top H^{-1}\mathbf 1},[8pt]
\mathbf q=H^{-1}(\lambda \mathbf 1-\boldsymbol\chi).
\end{cases}
]
Корректность: (H\succ 0) на подпространстве ({\mathbf 1}^\perp) (см. §4), следовательно решение единственно.

---

# 4) Что такое (J_{ij}) и почему матрица (H) положительна

**Определение (кулоновское ядро/«жёсткость связи»).**
[
J_{ij}=\iint w_i(\mathbf r),G(\mathbf r,\mathbf r'),w_j(\mathbf r'),d^3r,d^3r',
]
где (G) — зелёновский оператор (-\nabla!\cdot!\varepsilon(\mathbf r)\nabla G=\delta) с граничными условиями Робина (разделы фаз/диэлектрик), а (w_i\ge0) — весовые функции разбиения единицы (\sum_i w_i\equiv1) (например, локализация по атомным областям).

**Лемма (положительная определённость).** Для любого набора ({a_i}):
[
\sum_{ij} a_i J_{ij} a_j
=\iint \Big(\sum_i a_i w_i(\mathbf r)\Big)G(\mathbf r,\mathbf r')\Big(\sum_j a_j w_j(\mathbf r')\Big),d^3r,d^3r' \ge 0.
]
Равенство только при нулевой суммарной источниковой функции. Значит (J\succeq0).

**Следствие.** (H_{ii}=2\eta_i>0), (J\succeq 0 \Rightarrow H\succeq 0) и строго (H\succ 0) на ({\mathbf 1}^\perp). Отсюда единственность EEM-зарядов и устойчивость минимума.

---

# 5) Связь с поляризуемостью и HSAB

**Определения.** Глобальная мягкость (S=\eta^{-1}), локальная мягкость (s(\mathbf r)=S,f(\mathbf r)), (f(\mathbf r)=(\partial\rho/\partial N)_v). Поляризуемость (\alpha) — второй отклик (E) по (\mathcal E).

**Утверждение 4 (сопряжённость мягкости и поляризуемости).** В линейном отклике
[
\Delta\mu=-\eta,\Delta N,\qquad
\Delta \boldsymbol\mu=\boldsymbol\alpha,\mathbf E,
]
а энергия малой перезарядки и поляризации складываются:
[
\Delta E\approx -\frac{(\Delta\chi)^2}{4(\eta_A+\eta_B+J_{AB})}-\frac{1}{2}\mathbf E^\top(\alpha_A+\alpha_B)\mathbf E.
]
Поскольку (\alpha) и (S) растут при «мягкости», выигрыш максимален, когда **оба** партнёра мягкие (или оба жёсткие при малой (\mathbf E)). Это и есть HSAB как следствие максимизации суммарного выигрыша отклика.

**Доказательство (эскиз).** Оба вклада — квадратичные формы по «источникам» ((\Delta\chi,\mathbf E)) на положительно определённых ядрах отклика. Максимизация выигрыша при данных источниках достигается при согласованности «профилей отклика» двух подсистем. (\square)

---

# 6) Энергетика донор–акцепторной связи и «правило Полинга»

Рассмотрим смешение двух конфигураций: ковалентной (|c\rangle) и ионной (|i\rangle) с энергетическим зазором (\Delta\approx \chi_A-\chi_B). Связь возникает за счёт матричного элемента (V) (перекрытие/перенос).

**Двухуровневая модель (перенос-индуцированная стабилизация).**
[
H=\begin{pmatrix}0&V\V&\Delta\end{pmatrix},\quad
E_-\approx -\frac{V^2}{\Delta}\quad(\Delta\gg |V|).
]
С точки зрения §2, тот же эффект даёт (\Delta E_{\rm CT}\sim-(\Delta\chi)^2/[4(\eta_A+\eta_B+J_{AB})]).

**Утверждение 5 (квадратичная связь с (\Delta\chi)).** В «диэлектрическом» пределе (J_{AB}) и (\eta) мало зависят от пары, добавочная прочность связи
[
D_{AB}-\tfrac12(D_{AA}+D_{BB})\ \propto\ (\chi_A-\chi_B)^2.
]
Это — строгий вывод квадратичной зависимости, лежащей в основе «правила Полинга», теперь выражённой через ((\chi,\eta,J)) нашей модели.

**Доказательство.** Либо через двухуровневое смешение (перелинковка (\Delta\leftrightarrow \Delta\chi)) и (V\approx) константа класса связей, либо через §2, приняв медленную вариацию (J,\eta) по ряду родственных молекул. (\square)

---

# 7) Локальные индексы и конденсация на атомы

**Функция Фукуя.** (f^\pm(\mathbf r)=\rho(\mathbf r;N\pm 1)-\rho(\mathbf r;N)) в односторонних производных; глобально (\int f^\pm=1).

**Локальная мягкость.** (s^\pm(\mathbf r)=S,f^\pm(\mathbf r)).

**Конденсация на атом (i).** Возьмём разбиение единицы (\sum_i w_i(\mathbf r)=1):
(f_i^\pm=\int w_i(\mathbf r)f^\pm(\mathbf r),d^3r,\quad s_i^\pm=S,f_i^\pm.)

**Утверждение 6 (согласованность с EEM).** Для малой перезарядки (\Delta N) получаем приращение атомного заряда
[
\Delta q_i\approx f_i,\Delta N,
]
а при минимизации EEM-энергии в поле внешних источников (\chi_i^0) распределение ({q_i}) совпадает с таковым, которое максимально согласовано с профилем (s(\mathbf r)) (вариационный минимум одной и той же квадратичной формы).

**Доказательство (эскиз).** Линеаризация по (\Delta N) даёт (\delta\rho=f,\Delta N). Конденсация по (w_i) даёт формулу. EEM — минимум квадратичной формы на ({\mathbf 1^\top q=Q}); её непрерывный аналог для (\rho) имеет тем же решением «по профилю» (s). (\square)

# 8) Предельные оценки (полезные в вычислительной практике)

**Оценка 1 (амплитуда перетока).**
[
|\Delta N^\star|\le \frac{|\Delta\chi|}{4\sqrt{\eta_A\eta_B}}\quad
\text{(из } \eta_A+\eta_B+J_{AB}\ge 2\sqrt{\eta_A\eta_B}\text{)}.
]

**Оценка 2 (нижняя граница по стабилизации).**
[
\Delta E_{\rm CT}\ge -\frac{(\Delta\chi)^2}{8\sqrt{\eta_A\eta_B}}.
]

**Оценка 3 (диполь связи).** При межъядерном расстоянии (R)
[
|\mathbf d_{AB}|\approx |\Delta N^\star|,R\le \frac{|\Delta\chi|}{4\sqrt{\eta_A\eta_B}},R.
]
Все оценки достигаются (с точностью до множителя) при (J_{AB}\to 0) и (\eta_A=\eta_B).

# 9) Поверхности и твёрдые тела: работа выхода, контактная разность

**Работа выхода (\Phi) и электроотрицательность.** Для металла/поверхности при (T\approx 0) химпотенциал электронного газа (\mu_{\rm surf}=-\Phi\Rightarrow \chi_{\rm surf}=\Phi).

**Контактная разность потенциалов.** При адсорбции молекулы (M) на поверхности (S) выравнивание (\mu_S^\star=\mu_M^\star) даёт интерфейсный переток (\Delta N^\star) по §2 с (\chi_S=\Phi) и соответствующим (J_{SM}). Интерфейсный диполь (\mathbf d\approx \Delta N^\star \mathbf n,z_0) сдвигает работу выхода (\Delta\Phi\sim -4\pi d/A) (электростатика слоя). Это связывает (\Delta\Phi) с (\chi,\eta,J) без феноменологии.

---

# 10) Температура: электронная свободная энергия

При (T>0) вместо (E(N)) берём свободную энергию (F(N,T)=E-TS_{\rm el}) с
[
\mu(T)=\Big(\frac{\partial F}{\partial N}\Big)*{v,T},\qquad
\eta(T)=\frac{1}{2}\Big(\frac{\partial^2 F}{\partial N^2}\Big)*{v,T},\qquad
S(T)=\eta(T)^{-1}.
]
Ферми-размывание «смягчает» систему: (\eta(T)\searrow) при росте (T). Все выводы §2–§8 остаются в силе с заменой (E\to F), (f(\mathbf r)\to) терм. отклик ((\partial\rho/\partial N)_{v,T}).

---

# 11) Как считать в рамках вашей модели (практический конвейер)

1. **(I,A)**: решаем стационары для (N\pm 1) (тот же (\hat H[v,\mathbf A])) ⇒ (I=E(N-1)-E(N),\ A=E(N)-E(N+1)).
2. **(\chi,\eta,S)**: (\chi=(I+A)/2,\ \eta=(I-A)/2,\ S=1/\eta).
3. **(J_{ij})**: решаем (-\nabla!\cdot!\varepsilon\nabla G=\delta) с вашими Робиновыми границами; интегрируем с весами (w_i).
4. **EEM-заряды**: решаем блок-систему из §3 (устойчиво, (H\succeq 0)).
5. **Локальные индексы**: вычисляем (f^\pm(\mathbf r)) (разность плотностей (N!\pm!1) или линейный отклик), конденсируем (f_i^\pm) по (w_i), получаем (s_i^\pm).
6. **Диполи, полярности, (\Delta E_{\rm CT})**: по формулам §2, §6.
7. **Поверхности**: берём (\chi_S=\Phi) (из вариации по (\mathbf A)/работы выхода), далее §2.
8. **Температура**: повторяем пп. 1–5 для свободной энергии (F).

# 12) Дополнительные доказательства и леммы

**Лемма (инвариантность к сдвигу нуля потенциала).** Пусть (v\to v+C). Тогда (\rho) не меняется, (\mu\to \mu+C), (\chi\to \chi-C), но разности (\Delta\chi) и все наблюдаемые (заряды, диполи, (\Delta E_{\rm CT})) неизменны.

*Доказательство.* Линейность по (v) и нормировка (\int\rho=N). (\square)

**Лемма (сходимость EEM в периодическом случае).** Для кристалла с нейтральной ячейкой (\sum_i q_i=0) и экранированным (G) (или условием Эвальда) матрица (H) остаётся положительной на ({\mathbf 1}^\perp). Решение (\mathbf q) существует, единственно и ограничено.

*Доказательство.* (J\succeq 0) (см. §4), локальные (2\eta_i>0). На ({\mathbf 1}^\perp) добавочный член от периодического нулевого-мода исключается, остаётся коэрцитивность. (\square)

**Лемма (максимум (|\Delta E_{\rm CT}|) при согласовании «жёсткостей»).** При фиксированных (\Delta\chi) и ограничениях на «бюджет мягкости» (\eta_A+\eta_B=\mathrm{const}) модуль (\Delta E_{\rm CT}) максимален при (\eta_A=\eta_B).

*Доказательство.* Из (-(\Delta\chi)^2/[4(\eta_A+\eta_B+J)]) при фиксированной сумме минимум знаменателя и, значит, максимум по модулю достигаются при равенстве (\eta_A=\eta_B) (AM–GM). (\square)

## Короткое резюме

* (\chi=-\mu), почти удобно (\chi=(I+A)/2), (\eta=(I-A)/2), (S=1/\eta).
* Перенос заряда между фрагментами:
  [
  \Delta N^\star=\frac{\chi_A-\chi_B}{2(\eta_A+\eta_B+J_{AB})},\quad
  \Delta E_{\rm CT}=-\frac{(\chi_A-\chi_B)^2}{4(\eta_A+\eta_B+J_{AB})}.
  ]
* Молекулярные частичные заряды — решение (H\mathbf q+\boldsymbol\chi=\lambda \mathbf 1) при (\mathbf 1^\top \mathbf q=Q); (H=\mathrm{diag}(2\eta_i)+J), (J_{ij}=\langle w_i|G|w_j\rangle).
* HSAB и «дополнительная прочность» связи следуют из тех же квадратичных форм отклика; в простейшем пределе (\Delta D\propto (\Delta\chi)^2).
* Все выводы опираются на положительность (H) и вариационный минимум — то есть строго следуют из вашей полевой постановки без феноменологии.

# **13. Электроотрицательность и перераспределение заряда**

## 13.1. Определение через химический потенциал

**Основная формула:**
[
\chi = -\mu = -\left(\frac{\partial E}{\partial N}\right)_v,
]
где (E) — энергия основного состояния при фиксированном внешнем потенциале (v(\mathbf r)), а (N) — среднее число электронов.

**Доказательство.**
Вариация по (N) в ограниченной задаче (\delta(E-\mu N)=0) при фиксированном (v) даёт условие равновесия (\mu = \frac{\partial E}{\partial N}|_v). Поскольку электроотрицательность определяется как «способность притягивать электроны», принимаем (\chi = -\mu).

---

## 13.2. Связь с ионизационным потенциалом и сродством

**Теорема (Мулликен):**
[
\chi = \frac{I + A}{2}, \quad \eta = \frac{I - A}{2}.
]

**Доказательство.**
Используем конечные разности для кусочно-линейного функционала (E(N)):
[
\mu^- = E(N) - E(N-1) = -I, \quad \mu^+ = E(N+1) - E(N) = -A.
]
Среднее значение (\mu_0 = \frac{\mu^- + \mu^+}{2} = -\frac{I + A}{2}), следовательно (\chi = -\mu_0 = \frac{I + A}{2}).
Разность производных (\mu^+ - \mu^- = I - A = 2\eta). (\square)

---

## 13.3. Две подсистемы: равновесие электроотрицательности

Для фрагментов (A) и (B) с электроотрицательностями (\chi_A, \chi_B) и жёсткостями (\eta_A, \eta_B) изменение энергии при передаче (\Delta N) электронов:
[
E(\Delta N) = \chi_A \Delta N + \eta_A (\Delta N)^2 - \chi_B \Delta N + \eta_B (\Delta N)^2 + J_{AB} (\Delta N)^2.
]
Минимизация (\frac{dE}{d\Delta N} = 0) даёт:
[
\boxed{\Delta N^\star = \frac{\chi_A - \chi_B}{2(\eta_A + \eta_B + J_{AB})}}.
]
**Следствие.** В равновесии (\mu_A = \mu_B), то есть электроотрицательности выравниваются.

**Энергия стабилизации:**
[
\boxed{\Delta E = -\frac{(\chi_A - \chi_B)^2}{4(\eta_A + \eta_B + J_{AB})}}.
]
Это выражает энергию донорно-акцепторного взаимодействия как функцию параметров поля.

---

## 13.4. Матричная форма для молекулы

Для системы из (M) атомов с зарядами (q_i), параметрами (\chi_i^0, \eta_i) и кулоновским взаимодействием (J_{ij}):
[
E(\mathbf q) = \sum_i \chi_i^0 q_i + \frac{1}{2}\sum_i 2\eta_i q_i^2 + \frac{1}{2}\sum_{i\ne j} J_{ij} q_i q_j.
]
Минимизация при (\sum_i q_i = Q_{\text{tot}}) даёт:
[
H\mathbf q + \boldsymbol{\chi} = \lambda \mathbf 1, \quad \mathbf 1^\top \mathbf q = Q_{\text{tot}},
]
где (H_{ii} = 2\eta_i, H_{ij} = J_{ij}).

**Решение:**
[
\lambda = \frac{\mathbf 1^\top H^{-1} \boldsymbol{\chi} + Q_{\text{tot}}}{\mathbf 1^\top H^{-1} \mathbf 1}, \quad
\mathbf q = H^{-1} (\lambda \mathbf 1 - \boldsymbol{\chi}).
]
**Физический смысл:** (\lambda) — общий химический потенциал, одинаковый для всех атомов.

---

## 13.5. Положительная определённость матрицы

Кулоновская часть:
[
J_{ij} = \iint \frac{w_i(\mathbf r) w_j(\mathbf r')}{|\mathbf r - \mathbf r'|},d^3r,d^3r' \ge 0,
]
и диагональная жёсткость (2\eta_i > 0). Следовательно, (H) — положительно определённа на подпространстве (\sum_i q_i = 0). Это гарантирует единственность решения и устойчивость энергетического минимума.

---

## 13.6. Локальные функции и распределение мягкости

**Функция Фукуя:**
[
f(\mathbf r) = \left(\frac{\partial \rho(\mathbf r)}{\partial N}\right)_v, \quad \int f(\mathbf r)d^3r = 1.
]
**Локальная мягкость:**
[
s(\mathbf r) = S f(\mathbf r) = \frac{f(\mathbf r)}{\eta}.
]
При слабом возмущении (\delta v) плотность изменяется (\delta\rho = \int \chi(\mathbf r,\mathbf r')\delta v(\mathbf r') d^3r'); из этого следует:
[
f(\mathbf r) = \int \chi(\mathbf r,\mathbf r') d^3r',
]
где (\chi) — функция отклика системы на изменение потенциала.

---

## 13.7. Электрофильность и HSAB

**Глобальная электрофильность:**
[
\omega = \frac{\mu^2}{2\eta} = \frac{\chi^2}{2\eta}.
]
**Теорема (HSAB).**
Максимальная стабилизация достигается при взаимодействии двух систем, обладающих схожей мягкостью.

**Доказательство.**
Из формулы энергии обмена:
[
\Delta E = -\frac{(\Delta\chi)^2}{4(\eta_A + \eta_B + J_{AB})},
]
при фиксированной сумме (\eta_A + \eta_B) минимум знаменателя и максимум (|\Delta E|) достигаются при (\eta_A = \eta_B). (\square)

---

## 13.8. Донорно-акцепторная энергия и «правило Полинга»

Энергия смешения ионной и ковалентной форм:
[
E_{\text{bond}} = -\frac{(\chi_A - \chi_B)^2}{4(\eta_A + \eta_B)}.
]
**Следствие (Полинг):**
[
D_{AB} - \frac{D_{AA} + D_{BB}}{2} \propto (\chi_A - \chi_B)^2.
]
То есть разность электроотрицательностей квадратично повышает прочность связи.

---

## 13.9. Поверхности и твёрдые тела

Для металла:
[
\chi_{\text{surf}} = \Phi,
]
где (\Phi) — работа выхода (энергия, необходимая для удаления электрона из поверхности).
При контакте молекулы (M) с поверхностью (S):
[
\Delta N^\star = \frac{\Phi - \chi_M}{2(\eta_M + \eta_S + J_{SM})},
]
что определяет величину интерфейсного диполя и сдвиг работы выхода:
[
\Delta \Phi = -4\pi d / A, \quad d \approx \Delta N^\star z_0.
]

---

## 13.10. Температурная зависимость

При конечной температуре электронная свободная энергия (F(N,T) = E - TS_{\text{el}}).
[
\mu(T) = \left(\frac{\partial F}{\partial N}\right)*{v,T}, \quad
\eta(T) = \frac{1}{2}\left(\frac{\partial^2 F}{\partial N^2}\right)*{v,T}.
]
Ферми-размывание снижает (\eta(T)) ⇒ повышает мягкость системы (S(T)).

---

## 13.11. Вариационная проверка (из поля)

В рамках общей лагранжевой формулировки:
[
\mathcal L = E_\phi[\Theta] + E_U[U] + E_{\mathrm{EM}}[A_\mu] + \int \rho v,
]
вводим вариацию по источнику (v) и по (N).
Тогда:
[
\frac{\delta \mathcal L}{\delta N} = -\mu, \quad \frac{\delta^2 \mathcal L}{\delta N^2} = 2\eta,
]
и эти производные не зависят от выбора калибровки (A_\mu).

**Физический смысл:** электроотрицательность — это отрицание мультипликатора при ограничении на норму поля (\int\rho = N), а жёсткость — вторая вариация по этому ограничению.

---

## 13.12. Практическая реализация в коде

1. Решить три стационарные задачи: для (N), (N!-!1), (N!+!1).
2. Вычислить (I) и (A).
3. Найти (\chi, \eta, S).
4. Составить матрицу (H) и решить EEM-систему (линейная, симметричная).
5. Проверить положительную определённость (H) (собственные значения (>0)).
6. Вычислить (\Delta N^\star), (\Delta E_{\text{CT}}), диполь (\mathbf d = \Delta N^\star R).
7. При необходимости — повторить с (T>0) и с поверхностным потенциалом (v(z)).

---

**Итог:**
Электроотрицательность естественно вытекает из вариационной формулировки: это отрицание множителя при ограничении числа электронов, жёсткость — вторая производная, мягкость — её обратная.
Из этого единого принципа строго выводятся:

* закон выравнивания электроотрицательности;
* энергетика донорно-акцепторных связей;
* частичные заряды (EEM);
* HSAB;
* правило Полинга;
* связь с работой выхода и термозависимостью.

* Сырые данные: первая энергия ионизации (I) и электронное сродство (A) (газовая фаза). Брать из NIST WebBook/ion energetics. ([webbook.nist.gov][1])

## Единая безразмерная «паспортная карточка» объекта

Для каждой молекулы/реакции собираем вектор наблюдаемых величин (цели лосса), уже приведённых к (T=298.15) K:

[
\mathbf{y}=
\Big(
\tilde{\chi},\tilde{\eta},\tilde{\alpha},\tilde{C}*6,
{\tilde{E}*{\text{int}}^{\text{S66}}},
\tilde{\Delta E},\tilde{E_a},\tilde{k},
\tilde{\Delta G}_{\text{solv}},
\tilde{\phi},
\ldots
\Big)
]

А параметры поля (\boldsymbol{\theta}) (те, что вы вводили ранее: центры/жёсткости/связности/диэлектрические и поверхностные коэффициенты) подбираем из задачи регуляризованной многозадачной регрессии:

[
\min_{\boldsymbol{\theta}} \sum_i w_i,\big|\hat{y}_i(\boldsymbol{\theta})-y_i\big|_2^2
\quad\text{при}\quad
\operatorname{Var}(y_i)=1 ;;(\text{z-скейлинг по датасету})
]

Где веса (w_i) выбираем так, чтобы каждая группа задач (термохимия/НК-взаимодействия/растворение/кинетика/редокс) вносила сопоставимый вклад после нормализации дисперсий.
