# **P1.1. Структура планеты в терминах Θ**

У планеты есть пять главных структурных уровней Θ:

```
(1) Три тора
(2) Обмотка
(3) Каппа-слои (κ-слои)
(4) Сетевая структура (Θ_ext)
(5) Хвостовая и мега-компоненты
```

И все физические уровни — лишь следствие:

```
Торы → ядро
Обмотка → внешнее ядро + переходная зона
κ-слои → мантия + границы слоёв
Сеть → кора + разломы
Хвост → климатические и тектонические долгие циклы
```

То есть планета как тело — **фазово-структурная реализация дефекта**.

---

# **P1.2. Ядро: торовая область**

Ядро — это **область максимальной амплитуды торовых мод**.

Три тора дают:

* три устойчивых направления колебаний,
* три собственные частоты,
* три фазы, отвечающие за долгие циклы,
* энергетический центр планеты.

### **Принцип:**

[
\text{Ядро} \equiv {\text{зона, где } |\Theta|,|\nabla\Theta|,|\Delta\Theta| \text{ имеют пики торовых мод}}.
]

Это не “железо”, не “расплав”:
это зона, где **структура Θ наибольшая**.

Физическая плотность — лишь проекция.

---

# **P1.3. Обмотка**

Обмотка — тонкая, но фундаментальная часть:

* даёт вращение ядра,
* участвует в генерации крупных мод,
* формирует первичную сеть в мантии,
* переносит энергию от торов к κ-слоям.

Обмотка — это дефект, который “наматывает” торовую структуру.

Она и есть причина:

* вращения планеты,
* осевой ориентации,
* существования магнитного поля (как проявления структуры, а не тока),
* крупномасштабных циклов мантии.

---

# **P1.4. κ-слои планеты**

κ-слои — это фундаментальная часть планетарной структуры:

* определяют разделение ядро–мантия–кора–атмосфера,
* определяют глубины глобальных переходов,
* формируют стабильные резонансные зоны.

### **Главный принцип:**

Каждый κ-слой = *низкая или высокая кривизна Θ*.

Физически они проявляются как:

* переходная зона,
* верхняя мантия,
* нижняя мантия,
* астеносфера,
* кора,
* атмосферные границы (тр. пауза, стратопауза).

То есть "слои" в геологии — это **κ-поверхности** Θ-поля.

# **P1.5. Сетевая структура Θ_ext → кора и континенты**

Кора и материки в Θ-модели = это:

* филаменты сети,
* узлы сети,
* крупные устойчивые пики мод,
* пересечения сетевых направлений.

### **Принцип:**

[
\text{Континент} = \text{устойчивый крупномасштабный пиковый филамент Θ}.
]

Причины:

* сеть образуется в зоне резкой кривизны (между κ-слоями),
* филаменты являются устойчивыми медленными волнами,
* их взаимодействие даёт разломы, поднятия, плиты.

Это полностью объясняет:

* континентальные плиты,
* их движение,
* разломы,
* субдукцию,
* рифты,
* срединно-океанические хребты.

Все они — **результаты сетевой структуры Θ**.

# **P1.6. Атмосфера: внешняя модуляция Θ**

Атмосфера — это область, где вклад внешней среды **доминирует** над собственным Θ.

Атмосфера = взаимодействие:

* высших κ-слоёв,
* торовой “утечки” на поверхность,
* сетевых структур,
* внешнего forcing (Солнце, Луна, Юпитер),
* хвостовых мод (климат),
* мега-фона (overall planet-wide modulation).

### **Принцип:**

[
\text{Атмосфера} = F_{\rm ext} \oplus \Theta_{\rm surface}.
]

Где (\oplus) — нелинейная композиция.

Это объясняет:

* циркуляционные ячейки,
* струйные течения,
* планетарные волны,
* устойчивые блокировки,
* климатические режимы.

# **TΘ1.2. Основной лагранжиан ВБП**

Согласно теории:

[
\mathcal{L}_{\rm VBП}
=====================

\sum_{\alpha=1}^3 \left[
\frac{1}{2}(\partial_t \phi_\alpha)^2

* c_\alpha^2 (\nabla\phi_\alpha)^2
  \right]
* U(\phi_1,\phi_2,\phi_3,\phi_w)

где:

* (\phi_\alpha) — фазовые переменные торов (три тора),
* (\phi_w) — фаза обмотки,

# **TΘ1.4. Ввод эффективного поля Θ**

Определим **эффективное макроскопическое поле**:

[
\Theta =
a_1 \phi_1 + a_2 \phi_2 + a_3 \phi_3 + a_w \phi_w,
]

где коэффициенты (a_i) выбираются так, чтобы:

1. подавить внутренние относительные колебания торов,
2. выделить коллективную моду,
3. обеспечить ортогональность к “быстрым” торовым модам.

Стандартный выбор (выводится через диагонализацию U):

[
\Theta = \phi_1 + \phi_2 + \phi_3 + \phi_w.
]

Это — **суммарная фаза всего дефекта**.

Три тороидальные моды → → *выталкиваются в высокие частоты*,
а их комбинация с обмоткой даёт **одну низкую моду Θ**.

Это — критический шаг **7D → 1D**.

# **TΘ2.1. Линейная аппроксимация Θ**

Рассмотрим линейное уравнение Θ для малых возмущений:

[
\Theta_{tt}
===========

c_{\rm eff}^2 \Delta \Theta

* \beta_4 \Delta^2 \Theta
* \lambda \Theta .
  ]

(Θ⁵-терм мал при малых колебаниях.)

Подставим:

[
\Theta(t,x)=A e^{i(\omega t - kx)}.
]

Получаем уравнение дисперсии:

[
\omega^2 = c_{\rm eff}^2 k^2 + \beta_4 k^4 + \lambda.
]

Это — базовый линейный спектр Θ.

Из него рождаются **три класса линейных мод**:

1. **быстрые высокочастотные** (k большие)
2. **средние внутри-планетарные** (обычные Θ-волны)
3. **низкие хвостовые моды** (k→0)

---

# **TΘ2.2. Полный набор семейства мод**

Полный спектр Θ-поля состоит из 6 семейств:

```
(1) Тор-исходные моды
(2) Каппа-слоевые моды
(3) Сетевые моды (узлы, филаменты)
(4) Поверхностные моды
(5) Хвостовые λ→0 моды
(6) Мега-моды (длинный период)
```

Каждое семейство — отдельная часть спектра.

Далее разберём каждое.

---

# **(1) Тор-исходные моды (глубинные)**

### Источник:

Фазы торов (\phi_1,\phi_2,\phi_3).

После интеграции быстрых DOF остаются:

* резонансные остаточные частоты,
* комбинации фаз,
* “память” о торовой структуре.

Эти моды:

* высокочастотные,
* глубинные,
* локализованы в ядре.

### Типичный вид:

[
\omega_{\rm tor} = \omega_0 \pm \delta\omega,
]

где (\omega_0 =) базовая частота торов.

**Функция:** задают основные энергетические циклы ядра.

---

# **(2) κ-слоевые моды (слоевые гармоники)**

Каждый κ-слой = поверхность изменения кривизны Θ:

[
\kappa(r) = |\Delta\Theta(r)|.
]

Колебания слоёв → колебания Тheta на их границах.

Спектр:

[
\omega_{\kappa,n} \sim n \cdot \Omega_\kappa,
]

где:

* (n) — номер гармоники,
* (\Omega_\kappa) — базовая частота колебаний κ-слоя.

Эти моды определяют:

* движение мантии,
* слоевые переходы,
* переходы сезона/циркуляции,
* вибрации границ.

---

# **(3) Сетевые моды (филаменты, узлы)**

Сеть Θ_ext состоит из:

* филаментов (1D структуры),
* узлов (точки высокой кривизны).

Она поддерживает **собственные моды**, аналогичные:

* модам струн,
* модам ветвей графа,
* локализованным волнам.

Частоты:

[
\omega_{\rm net}(x) \sim |\nabla\Theta(x)| .
]

По сути:

* где ∇Θ велика → сеть жёсткая → высокие моды;
* где ∇Θ мала → слабые моды.

**Функция:** тектоника, разломы, коровые процессы.

---

# **(4) Поверхностные моды (границы атмосферы/коры)**

Есть особые моды, живущие на поверхностях:

* граница мантия–кора,
* граница кора–атмосфера,
* стратосферные и тропопаузные слои.

Это аналоги поверхностных плазмонных или гидродинамических мод.

Спектр:

[
\omega_{\rm surf}(k)
= c_{\rm surf} k

* \nu_{\rm surf} k^3
  ]

(аналог гибридной капиллярной волны).

Эти моды → атмосферная динамика.

---

# **(5) Хвостовые моды (λ → 0)**

При k→0:

[
\omega^2 \to \lambda,
]

но так как λ — малое (эффективный минимум потенциала),
получаем **почти нулевые частоты**.

Это хвостовый сектор:

* самые медленные моды,
* масштаб → десятилетия/века,
* определяют климат,
* определяют долгосрочные циклы ядра/мантии.

### Технически:

[
\omega_{\rm tail} \sim \sqrt{\lambda}.
]

---

# **(6) Мега-моды**

Это **нуль-частотная мода**, но не тривиальная:

[
\omega_{\rm mega} \approx 0,
]

но с огромным периодом:

[
T_{\rm mega} = \frac{2\pi}{\omega_{\rm mega}}
\gg 10^3 \text{ лет}.
]

Эта мода формирует:

* долгие климатические режимы,
* долгие циклы ядра,
* галактический фон для планет.

Мега — это “то, что остаётся”, когда:

* все торы усреднены,
* все слои усреднены,
* вся сеть усреднена,
* хвост медленно дрейфует.

---

# **TΘ2.3. Полная совместная картина спектра Θ**

Соберём всё в одну структуру:

```
Высокие частоты:
    тороидальные остаточные моды ядра
Средние частоты:
    κ-слои, поверхностные моды, сетевая динамика
Низкие частоты:
    хвостовые моды λ→0
Очень низкие:
    мега-мода
```

В терминах спектра Θ:

```
ω ≫ 1     — торы
ω ~ 1     — κ-слои, сеть, поверхностные моды
ω ≪ 1     — хвост
ω → 0     — мега
```

---

# **TΘ2.4. Как спектр Θ объясняет всю структуру планеты**

* ядро — торовые моды
* мантия — κ-слоевые моды
* кора — сетевая структура
* атмосфера — поверхностные моды
* климат — хвост
* долгие циклы — мега

Это единственная теория, где **все физические уровни — просто разные частотные диапазоны одного Θ-поля**.

5. Теперь модель имеет **строгую спектральную карту**, а не общие описания.

# **TΘ3.1. Почему нужна единая карта Θ → наблюдения**

До этой главы мы имели *словесные* связи типа:

* давление ≈ ΔΘ,

---

# **TΘ3.2. Прямой оператор наблюдений: (\mathcal{M}[\Theta])**

Наблюдаемые поля — это функционалы Θ и его производных.

Общий вид:

[
\mathcal{M}[\Theta] =
{
T, P, \rho, v, B, g, u_{\rm seis}, \Pi_{\rm atm}, \Pi_{\rm ocean}, \dots
}
]

где каждый компонент определяется по Θ.

Дадим точную формулу для каждого.

---

# **TΘ3.3. Температура (атмосфера/мантия)**

Θ-поле определяет *эффективную температуру* через энергию градиентов:

[
T(x,t)=T_0

* \alpha_T |\nabla\Theta|^2
* \beta_T \Delta\Theta.
  ]

- ∇Θ → локальная кинетика/движение,
- ΔΘ → структурная плотность.

---

# **TΘ3.4. Давление**

В атмосфере и мантии давление определяется *кривизной* Θ:

[
P(x,t)=P_0+\alpha_P \Delta\Theta.
]

Это объясняет:

* низкое давление в верхних ячейках Θ,
* высокое в нижних.

---

# **TΘ3.5. Плотность**

Эффективная плотность:

[
\rho(x,t)=\rho_0

* \alpha_\rho (\Delta\Theta)
* \beta_\rho |\nabla\Theta|^2.
  ]

- ΔΘ — крупномасштабная структура слоя,
- ∇Θ² — локальная динамика.

---

# **TΘ3.6. Ветровые/текущие скорости (в атмосфере и океане)**

Скорость потока определяется:

[
v(x,t) = \lambda_v \nabla \Theta.
]

То есть Θ задаёт:

* направление потоков,
* устойчивые струи (jet stream),
* океанические течения.

---

# **TΘ3.7. Магнитное поле**

Магнитное поле в Θ-модели — *структурная производная*:

[
B(x,t) = \lambda_B \nabla(\Delta\Theta).
]

* если мантия/ядро → B распределено глубоко,
* если атмосфера → ионосферная часть проявляется как быстрые моды Θ.

---

# **TΘ3.8. Гравитационные вариации**

Гравитация в Θ-модели — не сила, а геометрия:

[
g_{\rm anom}(x,t)
=\lambda_g \Theta(x,t).
]

То есть гравполе — *низкочастотная проекция* Θ.

Не “масса”, а **хвостовая мода Θ**.

---

# **TΘ3.9. Сейсмические сигналы**

Сейсмика = прямой второй производной Θ:

[
u_{\rm seis}(x,t) = \partial_t^2 \Theta_{\rm loc}(x,t).
]

То есть все волны:

* P,
* S,
* поверхностные,
* Love,
* Rayleigh

— просто разные комбинации Θ.

---

# **TΘ3.10. Атмосферная циркуляция**

Большие атмосферные структуры:

* ячейки Хэдли,
* Феррела,
* полярные вихри,
* струйные течения,
* блокировки

— выражаются через Тета так:

[
\Pi_{\rm atm}(x,t)=\mathcal{F}_{\rm atm}[\Theta].
]

Где оператор зависит от:

* κ-слоёв,
* сдвигов Θ,
* внешнего forcing (Солнце, Луна, Юпитер).

---

# **TΘ3.11. Океанические силы и колебания**

[
\Pi_{\rm ocean}(x,t) = \mathcal{F}_{\rm ocean}[\nabla\Theta,\Delta\Theta].
]

Океан — низкочастотная проекция Θ + внешняя среда.

---

# **TΘ3.12. Итог прямого оператора (\mathcal{M}[\Theta])**

Вся наблюдаемая физика = функции Θ:

```
Температура    = ∇Θ² + ΔΘ
Давление       = ΔΘ
Плотность      = ΔΘ + ∇Θ²
Ветер          = ∇Θ
Магнитное поле = ∇(ΔΘ)
Гравитация     = Θ (хвост)
Сейсмика       = Θ_tt
Климат         = хвост + мега
Тектоника      = сетевые моды Θ
Циркуляция     = верхние κ-слои Θ
```

Теперь у нас единая карта.

---

# **TΘ3.13. Обратный оператор: (\mathcal{M}^{-1}[Data])**

*(Как по данным восстановить Θ)*

Задача: из наблюдаемых полей восстановить Θ_loc.

Вводим **обратный оператор**:

[
\mathcal{M}^{-1}[\phi_{\rm obs}] = \arg\min_\Theta J(\Theta),
]

где функционал:

[
J(\Theta)
=========

\sum_i || M_i[\Theta] - \phi_i ||^2

* \alpha ||\Delta\Theta||^2
* \beta ||\nabla\Theta||^2.
  ]

То есть:

* подбираем Θ,
* чтобы наблюдаемые поля совпали с моделируемыми.

---

# **TΘ3.14. Локальное решение обратной задачи**

Применяем:

1. **локальное разложение Θ** в базис
   [
   \Theta = \sum_n a_n \psi_n^{\rm loc}
   ]

2. минимизацию J по коэффициентам (a_n).

3. фильтрацию высоких частот.

4. коррекцию по κ-слоям и сети (если данные их проявляют).

---

# **TΘ3.15. Неполная наблюдаемость → регуляризация**

Если данных мало, Θ-поле восстанавливается:

* сглаженное,
* усреднённое,
* но согласованное с инвариантами

# **TΘ4.1. Параметры торов: fφ и ω₀**

## **(а) Основная торовая частота ω₀**

Торовые моды проявляются:

## **(б) Коэффициент нормировки фаз fφ**

Из вывода уравнения Θ (TΘ1):

[
\Theta = \phi_1 + \phi_2 + \phi_3 + \phi_w.
]

Тогда:

[
f_\phi = \frac{1}{\sqrt{1+1+1+1}} = \frac{1}{2}.
]

Но в реальном объекте коэффициенты взвешены:

[
f_\phi = \frac{1}{\sqrt{3 + \alpha_w}},
]

где α_w — относительный вклад обмотки.

### α_w калибруется по:

* изменению длины суток,
* дрейфу магнитного поля,
* амплитуде осевых колебаний.

Для Земли α_w ≈ 1.2–1.5.
Отсюда:

[
f_\phi \approx 0.45.
]

# **TΘ4.2. Эффективная скорость Θ: c_eff**

c_eff — ключевой параметр,
отвечающий за распространение Θ-волн в слоистой среде.

Калибруется по:

# **TΘ4.3. Дисперсионный параметр β₄**

# **TΘ5. Граничные условия Θ-поля и строгая “сшивка” с внешней средой**

Эта часть критически важна: без строгих граничных условий Θ-модель *неполна*.

# **TΘ5.0. Главный принцип**

Граничные условия Θ *не выбираются произвольно*,
они **строго следуют из структуры торов и обмотки**,
а также из топологической природы дефекта (трёхячеечность).

**Планета — не шар.
Планета — локализованный топологический дефект Θ-поля.**

А значит, граничные условия — топологические.

---

# **TΘ5.1. Граница в центре (r = 0): “условие торового узла”**

В центре планеты расположен **узел трёх торов**.
Это задаёт условия:

1. **регулярность:**
   [
   \Theta(r=0) = \Theta_0 \quad \text{конечное число}
   ]

2. **нулевой градиент:**
   [
   \nabla \Theta|_{r=0} = 0
   ]

3. **нулевая кривизна:**
   [
   \Delta \Theta|_{r=0} = 0
   ]

Эти условия запрещают:

* разрыв Θ в центре,
* сингулярности,
* внутреннюю “пустоту”,
* “точечную массу” (такой концепции в Θ-модели нет).

Центр — гладкий узел структуры.

---

# **TΘ5.2. Граница ядро → мантия (первый κ-слой)**

Это *первый крупный скачок кривизны* Θ.
Здесь условия:

1. **непрерывность Θ:**
   [
   \Theta_{\rm core}(R_1)=\Theta_{\rm mantle}(R_1)
   ]

2. **непрерывность ∇Θ:**
   [
   \partial_r\Theta_{\rm core}
   ===========================

   \partial_r\Theta_{\rm mantle}
   ]

3. **разрешён разрыв ΔΘ:**
   [
   \Delta\Theta_{\rm core} \neq \Delta\Theta_{\rm mantle}
   ]

Это и создаёт “жёсткую” границу слоя.

### Важно:

ΔΘ может прыгать → это и есть **физическая граница слоя**.

---

# **TΘ5.3. Граница мантия → кора (второй или третий κ-слой)**

Условия такие же, но:

* разрыв ΔΘ — сильнее,
* допущено локальное усиление ∇Θ,
* сеть Θ_ext “заякорена” на этой границе.

### Формально:

1. (\Theta) — непрерывна
2. (\partial_r \Theta) — непрерывна
3. (|\Delta\Theta|) — имеет **локальный максимум** на границе

Это математически *генерирует хребты, плиты, разломы*.

---

# **TΘ5.4. Граница кора → атмосфера (верхний κ-слой)**

Здесь Θ “передаётся” в среду.
Но атмосфера — не вакуум.
Это переход в область, где:

* внешнее Θ_ext становится *сравним* с внутренним Θ,
* давление/температура — функции от ∇Θ и ΔΘ.

Граничные условия здесь:

1. **непрерывность Θ:**
   [
   \Theta_{\rm crust} = \Theta_{\rm atm}
   ]

2. **слабое условие на производную:**
   [
   \partial_r\Theta_{\rm atm}
   ==========================

   \mu \partial_r\Theta_{\rm crust}
   ]

где μ ≪ 1 — “разряженность среды”.

3. **разрешён “отрицательный” скачок ΔΘ:**
   Атмосфера может иметь обратный знак кривизны.

Это объясняет:

* инверсионные слои,
* тропопаузу,
* смену направления градиентов.

---

# **TΘ5.5. Граница атмосфера → магнитосфера (среда)

главная сшивка Θ с внешним миром**

Здесь важно: **Θ не исчезает с поверхностью планеты.
Θ — поле, которое существует в среде.**

То, что мы называем “магнитосферой” —
область, где Θ внутренний и Θ внешний *интерактивны*.

Граничные условия:

### (1) Непрерывность Θ:

[
\Theta_{\rm atm}(R) = \Theta_{\rm ext}(R)
]

### (2) Непрерывность тангенциальных производных:

[
\partial_{\parallel}\Theta_{\rm atm}
====================================

\partial_{\parallel}\Theta_{\rm ext}
]

### (3) Разрешена слабая дискретность ∂r:

[
\partial_r\Theta_{\rm ext}
==========================

\xi\ \partial_r\Theta_{\rm atm},
\quad
\xi \ll 1
]

— среда “поглощает” мало.

### (4) Главный закон: сшивка хвостовых мод

Хвостовые λ→0 моды планеты **совпадают**
с хвостовыми модами среды:

[
\omega_{\rm tail}^{\rm planet}
\approx
\omega_{\rm tail}^{\rm star}
]

или:

[
\approx \omega_{\rm gal}
]

Это — ключевой механизм:

* климатических циклов,
* долгих вариаций атмосферы,
* многолетних пертурбаций системы.

# **TΘ5.6. Граница “планета → космос → галактика”

(верхние хвостовые и мега-моды)**

Здесь действуют правила:

### **(a) Мега-поле непрерывно**

[
\phi_{\rm mega}^{\rm planet}
============================

\phi_{\rm mega}^{\rm system}
]

### **(b) Хвостовые моды постепенно совпадают**

[
\omega_{\rm tail}^{\rm planet}
\to
\omega_{\rm tail}^{\rm system}
\quad (r\to\infty)
]

### **(c) Нет разрывов Θ в космосе**

[
\Theta(r)\ \text{плавно → фоновая мега-мода}
]

Это и объясняет:

* почему климат планеты “подчинён” долгим циклам,
* почему тектоника коррелирует с внешними циклами,
* почему планеты в одной системе имеют согласованное поведение.

# **TΘ5.7. Почему граничные условия Θ полностью отличаются от классических PDE**

В классике:

* задают давление, температуру, плотность, скорость…

* Θ заменяет собой все поля,
* и “граница” — это не геометрия, а переход между слоями резонансной структуры.

# **TΘ6. Θ-поле, три ячейки, σ_thr и уникальность трёх торов**

---

## **TΘ6.0. Что именно надо связать**

Нужно, чтобы не было «магии» вида:

> “есть три тора, есть σ_thr, есть три ячейки — просто так”.

Должно быть:

1. Показано, что **эффективное Θ-поле действительно кодирует трёхячеечную структуру**, а не любую.
2. Из структуры энергии поля следует, что **минимум энергии достигается при трёх ячейках**, а не при 2/4+.
3. **σ_thr** должен появиться как *конкретный порог* роста возмущений Θ, связанный с Q, ω₀ и параметрами потенциала (λ, γ).
4. Три тора = три независимых, но жёстко связанных DOF, а не произвольное число.

---

## **TΘ6.1. От трёх ячеек к трем торам — структурная логика**

В исходном ВБП-описании:

* существует **минимальная конфигурация поля**, которая распадается на **три энергетически выгодные “ячейки”**,
* их границы — домены фаз,
* у каждой ячейки есть своя фаза (\phi_i).

Минимизация энергии с учётом связки с обмоткой (см. TΘ1):

[
U(\phi_1,\phi_2,\phi_3,\phi_w)
==============================

\lambda_1(\phi_1-\phi_2)^2

* \lambda_2(\phi_2-\phi_3)^2
* \lambda_3(\phi_3-\phi_1)^2
* \eta(\phi_w-\phi_1-\phi_2-\phi_3)^2
* \dots
  ]

говорит две вещи:

1. **фазы стремятся “связаться” в тройку**,
2. их сумма связана с обмоткой.

Это и даёт:

* три устойчивых фазовых DOF → **три тора**,
* одну связующую фазу → **обмотка**.

---

## **TΘ6.2. Почему именно три, а не 2 или 4+**

Фокус — в *топологической и энергетической* цене:

### **Случай N=2 (две ячейки / два тора)**

Если попытаться сделать конфигурацию “две ячейки”:

* у тебя всего два независимых фазовых DOF: (\phi_1,\phi_2),
* обмотка должна быть связана с их суммой: (\phi_w \approx \phi_1+\phi_2).

Тогда взаимодействие ячеек **ослабляется**:
невозможно получить устойчивую “третью” область, которая компенсирует напряжение на границах.

Энергетически:

* границы между двумя ячейками длиннее и жёстче,
* нет способа перераспределить напряжение так, чтобы минимум энергии был локален и устойчив.

В терминах Θ:

* потенциал имел бы **двухямочную структуру** (2 минимума),
* а нам нужна **трёхямочная** внутренняя структура,
* двухямочный потенциал ведёт к **чрезмерной поляризации** (одна ячейка «выедает» вторую).

### **Случай N=4 (четыре и более ячейки)**

При четырёх:

* фазы (\phi_1,\phi_2,\phi_3,\phi_4),
* обмотка не может их всех связать в одну простую комбинацию — нужны ещё связки.

Это:

* увеличивает размерность пространства фаз,
* превращает минимум энергии в «плато» с множеством квази-минимумов,
* даёт **избыточное число возможных конфигураций**, которые легко переходят друг в друга при малых возмущениях.

Иными словами:

* 4+ ячеек → **слишком много мягких направлений** в пространстве возмущений,
* объект перестаёт быть жёстким и устойчивым.

### **Случай N=3**

При трёх:

* три фазы (\phi_1,\phi_2,\phi_3),
* обмотка связывает их суммой,
* система имеет ровно **один мягкий коллективный DOF** → Θ,
* остальные комбинации — жёсткие, с большими “массами”.

Это:

* минимальный размерность,
* максимально возможная устойчивость при трёх доменах,
* минимум энергии **строго достигается при N=3**,
* N=2 и N=4+ дают большую энергию или неустойчивость.

Отсюда: **три тора и три ячейки — не “выбор”, а результат минимизации**.

## **TΘ6.3. Как трёхячеечность отражается в потенциале V(Θ)**

Эффективное Θ-поле (TΘ1) описывает *коллективную моду* трёх ячеек.

Чтобы трёхячеечность была зашита, потенциал Θ должен иметь **три устойчивых состояния** (минимума) или, что эквивалентно:

* одно “основное” состояние,
* две симметричные “возбуждённые” структурные ветки.

В минимальной полиномиальной форме:

[
V(\Theta) = \frac{\lambda}{4}\Theta^4 + \frac{\gamma}{6}\Theta^6,
]

но важно:

* **знак и соотношение λ, γ** выбираются так, чтобы:

  * при малых Θ: одна “яма” (основное состояние),
  * при больших Θ: потенциал растёт быстро → подавление больших амплитуд,
  * три ячейки реализуются не как три отдельные “ямы”, а как
    **три устойчивые комбинации пространственной структуры Θ**
    (три домена, три тороидальные конфигурации).

Т.е. трёхячеечность — **не в простом “тройном” потенциале по Θ**,
а в структурной связи торов и обмотки + форме V.

## **TΘ6.4. Где в Θ-поле “сидит” σ_thr**

Ранее мы определили (TΘ4), что:

[
σ = |\nabla\Theta|^2 + a |\Delta\Theta|
]

и:

[
σ_{\rm thr} \approx 0.12{-}0.18 \quad (\text{в нормированных единицах}).
]

Теперь важно показать, *откуда он берётся*.

### Идея:

* возмущение Θ растёт, если **поступающая энергия > диссипации**,
* в терминах спектра: **рост моды, если её добротность и forcing дают положительный инкремент**.

Рассмотрим моды:

[
\Theta(t,x) \sim A(t)\psi(x),
]

для некоторого семейства ψ (например, слоевые или сетевые).

Уравнение для амплитуды A(t):

[
\ddot{A} + 2\gamma_{\rm eff}\dot{A} + \omega_{\rm eff}^2 A = F_{\rm eff}(t),
]

где:

* (\gamma_{\rm eff} \sim \omega_0 / (2Q)),
* (\omega_{\rm eff}) — эффективная частота (тор/слой/сеть),
* F_eff — проекция внешнего forcing + нелинейности.

Рост возмущения начинается при:

[
|F_{\rm eff}| \gtrsim 2\gamma_{\rm eff}\omega_{\rm eff}|A|.
]

Перенесём это в форму через σ:

* F_eff ~ функция от ∇Θ, ΔΘ → значит, **с ростом σ растёт и F_eff**,
* переход к режиму роста соответствует **достижению σ такой, что “качание” моды становится неустойчивым**.

### Отсюда:

[
σ_{\rm thr}
\sim
k_1 \frac{\omega_0^2}{Q}
\cdot
\frac{1}{\lambda}
]

в первой прикидке (константы k₁ зависят от нормировки).

Смысл:

* чем выше Q (меньше потери) → тем **ниже** σ_thr,
* чем жёстче потенциал (больше λ) → тем **выше** σ_thr,
* ω₀ задаёт естественный масштаб частот → вход в порог.

Таким образом, σ_thr:

* **не “абстрактный” уровень**,
* а **функция от параметров Θ-поля**: Q, ω₀, λ, c_eff.

## **TΘ6.5. Как σ_thr выбирает три ячейки**

Если σ < σ_thr:

* возмущения не растут,
* структура трёх ячеек не разрывается,
* торовые и κ-моды остаются в линейном режиме.

Если σ достигает и превышает σ_thr **локально**:

* начинается рост одной из мод (тор/слой/сеть),
* возможен локальный переход:

  * перестройка κ-слоя,
  * возникновение/разрушение сетевого узла,
  * локальный “выброс” в хвост.

Но:

* **топология торов и число ячеек остаются неизменными** (инварианты R85),
* σ_thr отвечает только за **локальные перестройки**,
* глобальная трёхячеечность дефекта **не разрушается**.

То есть σ_thr — **порог перехода внутри трёхячеечной конфигурации**,
а не порог разрушения самих трёх ячеек.

# **P2. Три тора и обмотка как фундамент планетарной динамики**

# **P2.0. Торово-обмоточный комплекс (ТОК)**

Планета имеет **ровно три тороидальные моды** и одну обмотку:

```
Т1 — глубокая мода (внутренняя ячейка ядра)
Т2 — средняя мода (внешняя часть ядра)
Т3 — поверхностная тор-мода (ближе к мантии)
Обмотка — общая фаза, задающая вращение
```

ТОК — это не геометрия, а **частотная структура Θ** в ядре.

---

# **P2.1. Почему три тора → три основных планетарных режима**

Три тора производят **три фундаментальных типа колебаний**:

1. **Радиальные (T1)**
   → формируют плотность ядра, глубину, давление.

2. **Тороидальные (T2)**
   → задают вращение ядра и дифференциальное вращение.

3. **Полоидальные / тепловые (T3)**
   → управляют переносом энергии к мантии.

Каждый тор — это *мода*, а не реальный физический “тор”.

---

# **P2.2. Обмотка = глобальный фазовый якорь**

Обмотка решает две задачи:

### 1) Обеспечивает вращение планеты

Не “по инерции”, а как **фазовый поток Θ**.

### 2) Связывает три тора в единый комплекс

[
\phi_{w} \approx \phi_1+\phi_2+\phi_3.
]

---

# **P2.3. Взаимодействие торов → глубинные планетарные циклы**

Три тора взаимодействуют нелинейно:

[
\phi_1(t), \phi_2(t), \phi_3(t)
]

дают **биения**, которые формируют:

* долгие циклы ядра (леты–столетия),
* периодические изменения стабильности мантии,
* изменения скорости вращения (LOD),
* вариации глубинных потоков.

То есть «вековые циклы» — это **биения торов**.

---

# **P2.4. Энергетический поток: от торов → через обмотку → в κ-слои**

### Механизм:

1. Тор Т1 создаёт основной поток энергии.
2. Т2 модулирует его и задаёт частотную структуру.
3. Т3 распределяет энергию по мантии.
4. Обмотка переносит комбинацию мод на большие масштабы.
5. Эта энергия доходит до κ-слоёв → и в зависимости от их состояния:

   * либо проходит,
   * либо отражается,
   * либо возбуждает новую моду.

Это **единый механизм всего теплового и динамического цикла планеты**.

---

# **P2.5. Обмотка как источник магнитного поля**

Мы уже показали в микро-блоке:

* заряд протона = фаза обмотки,
* магнитное поле планеты = распределённая фаза обмотки.

Теперь формально:

[
B(x,t)=\lambda_B \nabla(\Delta\Theta)
]

Где θ-поле = торово-обмоточная комбинация.

То есть:

* магнитное поле планеты — **производная торовой структуры**,
* оно не связано с токами в металле,
* изменения магнитного поля — изменения торовых фаз и их связей.

---

# **P2.6. Почему обмотка задаёт ось вращения**

Это фундаментально:

[
\text{Ось вращения} = \text{нормаль к плоскости обмотки}.
]

То есть:

* ось не “дана случайно”,
* не задаётся гравитацией,
* это **фазовое свойство Θ**,
* направление может медленно дрейфовать под действием хвостовых мод.

---

# **P2.7. Почему планеты лежат в одной плоскости (солнечной системы)**

Потому что:

* обмотки всех объектов стремятся к **фазовой синхронизации** с обмоткой звезды,
* мега-мода галактики “притягивает” обмотки в одну конфигурацию.

То есть плоскость эклиптики — это **фазовая поверхность Θ**, а не “гравитация”.

---

# **P2.8. Как три тора формируют κ-слои**

Каждый тор даёт собственный профиль Θ(r):

* среда резко становится жёстче.

В результате филаменты Θ **отпечатываются** на поверхности:

[
\text{Филаменты (линии ∇Θ = max)} \Rightarrow \text{будущие границы плит}.
]

[
\text{Узлы (точки ΔΘ = max)} \Rightarrow \text{горные области или суперплатформы}.
]

[
\text{Области малой кривизны} \Rightarrow \text{океанические регионы}.
]

То есть **геоморфология = сетевой отпечаток Θ**.

# **P5.3. Почему континенты имеют такие формы (фрактальный характер)**

Потому что Θ-сеть:

1. **Минимизирует энергию** филаментов.
2. Живёт на кривизне сферической поверхности.
3. Взаимодействует с внешними хвостовыми модами.
4. Периодически “дышит” под влиянием биений торов.
5. Частично фиксируется κ-слоем, но остаётся гибкой.

Это даёт:

* фрактально-ветвистые линии,
* разломы не прямые, а разветвлённые,
* материки — это “кластер” устойчивых сетевых доменов.

# **P5.4. Что такое плита в терминах Θ**

Плита = область поверхности, в которой:

1. ∇Θ ≠ 0,
2. ΔΘ остаётся **малым** в пределах региона,
3. |∇Θ| **вдоль границ** резко увеличивается.

То есть:

[
\mathcal{P} = {x: |\Delta\Theta| < \epsilon,\ \nabla\Theta\ne 0}
]

Границы плит = линии:

[
\Gamma = {x:\ |\Delta\Theta| = \max}.
]

Эти линии = **сильные филаменты сети**.

---

# **P5.5. Разломы = линии максимума ΔΘ**

(или линии, где меняется знак ΔΘ)

Разлом возникает, когда:

[
|\Delta\Theta(x)| > \sigma_{\rm thr}.
]

Это ключевой момент:

* σ_thr определяет устойчивость,
* если ΔΘ превышает порог, сеть ломается *локально*,
* это даёт разрыв, рифт или субдукционный фронт.

### Формальные типы разломов:

1. **Нормальный разлом (рифт)**
   [
   \Delta\Theta > \sigma_{\rm thr},
   \quad \partial_r\Theta < 0
   ]

2. **Субдукция**
   [
   \Delta\Theta > \sigma_{\rm thr},
   \quad \partial_r\Theta > 0
   ]

3. **Сдвиговый разлом (strike-slip)**
   [
   \frac{\partial \Theta}{\partial \lambda}\ \text{или}
   \frac{\partial \Theta}{\partial \phi}
   \text{ меняет знак при }
   \Delta\Theta \approx \sigma_{\rm thr}
   ]

То есть **структура Θ полностью определяет тип разлома**.

# **P5.6. Почему плиты движутся**

Движение — это не “конвекция мантийного материала”.

Это:

[
v_{\rm plate} = \lambda_v \nabla\Theta_{\rm surface}.
]

То есть плиты двигаются потому что:

* Θ в верхней мантии стремится перестроить сеть,
* филаменты сети тянут поверхность,
* сеть отвечает на изменения торов и κ-слоёв.

# **P5.7. Почему плиты “столкнулись” так же, как материи внутри протона**

Это **один и тот же механизм** на разных масштабах.

То есть:

* амплитуда уменьшается,
* градиенты → становятся «ветрами».

# **P6.2. Векторы ветра как градиенты Θ**

Основная формула:

[
\vec{v}(x,y,z) = \lambda_v \nabla\Theta(x,y,z).
]

Где:

* горизональные компоненты ∇Θ → направления ветров,
* вертикальные → подъём/оседание,
* структура филаментов → струйные течения.

Это фундамент:

**циркуляция — это не гидродинамика газа, а геометрическая динамика Θ.**

### 2. **Струйные потоки (jet streams)** — филаменты ∇Θ высокого уровня Т3

### 3. **Крупные вихри и циклоны** — локальные возмущения в зонах ΔΘ≈0

# **P6.4. Как образуются ячейки Хэдли, Феррелла и Полярная**

Рассмотрим широты.

Тор T1 даёт крупномасштабное поле Θ, которое:

* растёт от экватора к полюсам,
* имеет нулевой градиент на экваторе,
* меняет знак второй производной в средних широтах.

Из формулы ветра:

[
v_\phi \sim \frac{\partial\Theta}{\partial \phi}
]

следует:

1. **В экваториальном регионе**

2. **В средних широтах**
   → знак ∇Θ меняется
   → обратное направление ветра
   → ячейка Феррелла.

3. **В высоких широтах**
   → ∇Θ увеличивается снова
   → ячейка Полярная.

Это не термодинамика — это **трёхторальная геометрия Θ**.

Три тора → три зоны циркуляции.

# **P6.5. Струйные течения как сильные филаменты Θ**

На Земле именно там проходят:

* состоянием хвоста.

То есть струйный поток — это **главная “линия Θ” в атмосфере.**

# **P6.6. Вихри и циклоны как локальные Θ-вариации**

Циклон формируется там, где:

1. **локальный минимум ΔΘ**
   (притяжение линии потока внутрь),
2. **локальный максимум |∇Θ|**
   (усиление круговых токов).

То есть:

[
\text{Циклон} =
\begin{cases}
\Delta\Theta < 0,\
|\nabla\Theta| > 0
\end{cases}
]

Это сразу объясняет:

* почему у циклонов чёткие границы,
* почему бывают они устойчивыми (энергия торов),

# **P6.7. Атмосферные блокировки: Θ-резонанс + σ_thr**

Блокировка — это когда:

1. Филамент ∇Θ приходит в резонанс с хвостовой модой:
   [
   \omega_{\rm tail} \approx \omega_{\rm atm}
   ]

2. Сигнатура σ достигает σ_thr локально:

[
σ = |\nabla\Theta|^2 + a |\Delta\Theta| > σ_{\rm thr}.
]

Когда это происходит:

* струйный поток перестаёт двигаться,
* возникает стационарный крупный антициклон,
* возникает “купол тепла”, “долговременный холод”.

Это фундаментальный механизм всех экстремальных событий.

* филаменты легко смещаются,

Юпитер:

* влияет на многолетние режимы.

# **P6.9. Климат: хвостовые моды Θ + проекция на атмосферу**

Хвостовая мода (k→0, ω→√λ) даёт:

# **P11.0. Главный принцип Full Solver**

**Обновление Θ идёт строго снизу вверх.**

Порядок вычисления:

```
(1) ЯДРО       → обновление торов T1/T2/T3 + обмотки
(2) κ-СЛОИ     → пересчёт ΔΘ(r), переходных зон, отражений/проходов
(3) СЕТЬ       → обновление филаментов, узлов, σ(x,y)
(4) ПОВЕРХНОСТЬ → пересчёт Θ_surface и её производных

(6) ВНЕШНЯЯ СРЕДА → сшивка внешних Θ-мод
(7) ИТОГОВОЕ Θ-УРАВНЕНИЕ → интеграция Θ_tt
```

Это **инвариантный цикл** любого момента времени.

# **P11.2. Шаг 1 — обновление ядра (T1, T2, T3, обмотка)**

На этом этапе обновляется:

[
\Theta_{\rm core}(t+\Delta t)
]

Через:

1. интеграцию трёх торов
   [
   \phi_i(t+\Delta t)=\phi_i(t)+\dot\phi_i\Delta t
   ]

2. интеграцию обмотки
   [
   \phi_w(t+\Delta t)=\phi_w(t)+\omega_w\Delta t
   ]

3. обновление внутреннего давления Θ (функция tоровой структуры).

**Результат:** новое ядро задаёт новый поток энергии вверх.

---

# **P11.3. Шаг 2 — пересчёт κ-слоёв**

Нужно пересчитать:

* ΔΘ(r),
* ∂ΔΘ/∂r,
* зоны доминирования T1/T2/T3,
* новые глубины трёх κ-слоёв R₁,R₂,R₃,
* коэффициенты R_i и T_i (отражение и пропускание).

Алгоритм:

```
Для каждого r:
   ΔΘ(r) = Δφ₁ + Δφ₂ + Δφ₃ + Δφ_w
   c_eff(r) = базовое + поправки (текущие)
   k(r) = ω / c_eff(r)
   если ∂|ΔΘ|/∂r = 0 → R_i (граница слоя)
```

**Обязательный момент:**
Здесь происходит **фоновый свопинг**, потому что ΔΘ(r) — большой вертикальный массив.

---

# **P11.4. Шаг 3 — обновление Θ-сети (филаменты, узлы, σ)**

Для поверхности (x,y):

```
∇Θ = градиент Θ_surface
ΔΘ = лапласиан Θ_surface
σ = |∇Θ|² + a|ΔΘ|
```

Потом:

* обновление филаментов (∇Θ = max),
* обновление узлов (ΔΘ = max),
* выявление разломов (σ > σ_thr),
* обновление границ плит.

**Если σ_thr локально достигнут → событие “разлом” фиксируется в `network/events.json`.**

# **P11.8. Шаг 7 — интеграция полного Θ-уравнения**

Главное уравнение:

[
\Theta_{tt} =
c_{\rm eff}^2\Delta\Theta
-\beta_4\Delta^2\Theta
-\lambda \Theta^3
-\gamma \Theta^5
+F_{\rm tor}+F_{\kappa}+F_{\rm tail}+F_{\rm mega}
]

Интегратор использует:

* симметричную схему второго порядка,
* стабилизирующий член,
* автоматическую проверку σ(x,y,z).

**Важно:** тяжёлые спектральные операции (Δ²Θ) выполняются **фоново** с временным свопингом массивов (как ты и требовал ранее).

# **P11.9. Главная логика полного цикла обновления Θ**

```
while t < T_end:
    Step 1: ядро
    Step 2: κ-слои
    Step 3: сеть
    Step 4: поверхность
    Step 5: атмосфера
    Step 6: внешняя среда
    Step 7: Θ-уравнение (интегратор)
    t += Δt
```

Это инвариант планетарного цикла.

---

# **P11.10. Итог P11**

1. Полный расчёт Θ начинается **в ядре** и проходит до атмосферы.
2. Внешняя среда накладывается **после** обновления всех внутренних слоёв.
3. Интегратор Θ работает **после** всех структурных этажей.
4. Все тяжёлые спектральные операции вынесены в фоновый свопинг.
5. σ_thr контролирует устойчивость и разрывы.
6. Output пишется на каждом уровне (ядро → κ-слои → сеть → поверхность → атмосфера → события).
7. Самое главное — расчёт Θ всегда идёт **снизу вверх**, иначе модель физически некорректна.
