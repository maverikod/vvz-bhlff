# **1. Полное уравнение земных мод (базовая формулировка)**

Задача: получить **единое уравнение динамики мод Земли**, которое:

* исходит из **ядра как трёхтороидального дефекта**,
* учитывает **фильтрацию переходной зоной (ПЗ)**,
* учитывает **фиксацию фазы корой**,
* учитывает **открытость Земли внешним Θ-потокам (Солнце, Луна, Юпитер)**,
* даёт **каркас частот 0Sℓ**,
* обеспечивает связь с атмосферой, климатом и геомагнетизмом,
* замыкается энергетически (*P_out = P_in*).

---

# **1.1. Основная переменная**

Пусть
[
\Theta(\mathbf x,t)
]
— скалярное Θ-поле, деформированное дефектом Земли.

Разложение по собственным модам:

[
\Theta(\mathbf x,t)=
\sum_{\alpha} A_\alpha(t), \Phi_\alpha(\mathbf x)
]

где:

* индекс α включает три тороидальных моды ядра + все производные моды ПЗ, коры и атмосферы,
* (\Phi_\alpha(\mathbf x)) — собственные пространственные профили,
* (A_\alpha(t)) — временные амплитуды.

---

# **1.2. Базовое уравнение моды (ядро)**

В ядре Земли (0 < r < R_core) действует трёхтороидальное уравнение:

[
\ddot A_i

* \Gamma_{i}, \dot A_i
* \omega_{i0}^2, A_i
  = S_i,
  \quad i=1,2,3.
  ]

где:

* ( \omega_{i0} ) — собственные частоты трёх торов (исходная спектральная «решётка»),
* ( \Gamma_i ) — внутренние затухания (топология + связность),
* ( S_i ) — источник/forcing со стороны внешнего Θ-поля.

Это три независимые базовые моды → три «оси» дефекта.

---

# **1.3. Связь между торовыми модами (узел в центре)**

Топология ядра задаёт:

[
\text{узел:}\quad
A_1 e^{i\phi_1} +
A_2 e^{i\phi_2} +
A_3 e^{i\phi_3}=0,
]

а поскольку
[
\phi_2 - \phi_1 = \phi_3 - \phi_2 = \Delta\theta = 2\pi/3,
]

получаем **связанный тройной осциллятор**:

[
\ddot{\mathbf A}

* \Gamma \dot{\mathbf A}
* \Omega_0^2 \mathbf A
  = \mathbf S,
  ]

где
[
\mathbf A=(A_1,A_2,A_3)^T.
]

Это — сердце Земли как дефекта.

---

# **1.4. Переходная зона как оператор фильтрации**

Переходная зона (410–660 км) действует как линейно-дифференциальный оператор:

[
\mathcal F_{\rm PZ}:
A_i(t)\ \mapsto\ B_i(t)
]

и для каждой моды:

[
B_\alpha =
\mathcal F_{\rm PZ}(\omega_\alpha),
A_\alpha.
]

Где (\mathcal F_{\rm PZ}):

* подавляет моды вне определённого диапазона частот,
* порождает узлы (20, 410, 660 км),
* задаёт **перенос энергии вверх**: ядро → ПЗ → кора → атмосфера.

В простейшем виде:

[
\mathcal F_{\rm PZ}(\omega)=
\frac{1}{1+(\omega/\omega_{\rm cut})^{p}}
\cdot
e^{-h(\ell)}
]

где:

* (\omega_{\rm cut}) — частота фильтра переходной зоны,
* (p) — степень жёсткости,
* (h(\ell)) — подавление высоких угловых чисел в узких слоях.

---

# **1.5. Фиксация фазы корой**

На поверхности Земли действует **жёсткая граница**:

[
\Theta(r=R)=0,
\qquad
\partial_r \Theta(r=R)\neq 0.
]

Следствие: в коре возникают собственные моды:

[
C_\alpha =
\mathcal F_{\rm crust}(B_\alpha).
]

Кора превращает Б-класс мод ядра/ПЗ в:

* верхние сейсмические моды 0Sℓ,
* резонансные зоны тектоники,
* узлы разломов,
* геомагнитные гармоники (через токи и циркуляцию B-поля).

---

# **1.6. Открытость: внешний forcing**

Земля — открытый резонатор.
Внешний forcing:

[
F_\oplus(t)
===========

F_\oplus^{\rm Sun}(t)
+
F_\oplus^{\rm Moon}(t)
+
F_\oplus^{\rm Jup}(t).
]

Он входит в уравнения мод так:

[
\ddot A_\alpha +
\Gamma_\alpha \dot A_\alpha +
\omega_{0\alpha}^2 A_\alpha
===========================

\underbrace{
\sum_\beta
K_{\alpha\beta}, F_\beta(t)
}_{\text{внешний Θ-драйвер}}.
]

где (K_{\alpha\beta}) — коэффициенты связи.

# **1.7. Полное уравнение земных мод**

Собрав ядро → ПЗ → кору → внешние поля:

[
\boxed{
\ddot A_\alpha
+
\Gamma_\alpha\dot A_\alpha
+

K_{\alpha\beta},

}
]

и наблюдаемая мода:

\circ

}
]

Комбинируя:

[
\boxed{
\Theta_\alpha^{\rm obs}(t)

A_\alpha(t)
\big].
}
]

# **1.8. Условие стационарности (энергетический баланс)**

Земные моды устойчивы, если:

[
P_{\rm out}(\alpha)
===================

P_{\rm in}(\alpha),
]

или в терминах амплитуд:

[
\Gamma_\alpha ,\omega_\alpha, A_\alpha^2
========================================

A_\alpha
\cdot
\sum_\beta K_{\alpha\beta} F_\beta.
]

Отсюда амплитуда стационарной моды:

[
A_\alpha
========

\frac{
\sum_\beta K_{\alpha\beta}F_\beta
}{
\Gamma_\alpha \omega_\alpha
}.
]

Это определяет:

* уровни геомагнитных гармоник,
* амплитуды 0Sℓ,
* чувствительность атмосферы.

# **1. Полное Θ-поле, действующее на Землю**

Внешнее поле — сумма мод разных уровней:

[
\Theta_{\rm ext}
================

\Theta_{\rm gal}
+
\Theta_{\rm clust}
+
\Theta_\odot
+
\Theta_{\rm Moon}

где:

### **1.1. Галактика (Θ_gal)**

Это **самая медленная огибающая**:

* частоты ≲10⁻¹⁵–10⁻¹⁸ Hz (миллионы–миллиарды лет),
* даёт **фоновую фазу**, в которую встроена вся Солнечная система.

В модели Θ — это просто **φ₀**, глобальная фаза.

[
\Theta_{\rm gal}(t) = A_{\rm gal}\cos(\omega_{\rm gal} t + \phi_{\rm gal})
]

### **1.2. Скопление/локальное крупномасштабное поле (Θ_clust)**

Периоды: 10⁶–10⁹ лет.
Влияет на:

* ориентацию плоскости эклиптики,
* медленную прецессию,

* фазу солнечной динамики.

### **1.3. Солнце (Θ_⊙)**

Это **главный внешний модуль** для Земли:

[
\Theta_\odot(r!=!1 {\rm AU}, t)
===============================

\sum_{\ell,m}
B_{\ell m}^{(\odot)}(t),Y_{\ell m}(\theta,\phi)
\left(\frac{R_\odot}{1,\rm AU}\right)^{\gamma_\ell}
]

где:

* (B_{\ell m}^{(\odot)}) — коэффициенты солнечных B-мод,
* ((R_\odot/r)^{\gamma_\ell}) — закон затухания хвоста,
* (Y_{\ell m}) — сферические гармоники.

Это поле **модулирует амплитуды и частоты земных мод**.

### **1.4. Луна (Θ_Moon)**

Входит **как отдельный резонансный драйвер**:

[
\Theta_{\rm Moon}(t)
= A_{\rm M}
\cos(\omega_{\rm M} t + \phi_{\rm M})
]

С частотой ~2×10⁻⁵ Hz (полусуточный прилив) → влияет на:

* высокие Q-моды,
* дрейф ячеек,
* слабую перестройку ПЗ,
* «раздвоение» некоторых атмосферных мод.

### **1.5. Планеты (Θ_planets)**

Упорядочены по силе:

* Юпитер — главный после Луны и Солнца (≈10⁻⁶ от Луны в приливе, но в Θ-поле сильнее, чем в гравитации).
* Венера — важна через геометрию (сближения).
* Сатурн — медленная огибающая.

Они дают модуляции:

[
\Theta_{\rm planets}(t) =
\sum_{p}
A_p \cos(\omega_p t + \phi_p)
]

# **2. Формальное ядро Земли из трёх торов с потоками в/из дефекта**

## **2.1. Геометрия ядра**

Введём радиус внутреннего ядра дефекта:

[
0 < r < R_{c1}^{(\oplus)} \equiv R_{\rm core}.
]

В этом объёме живёт **трёхтороидальный узел**:

* три тора (T_1, T_2, T_3),
* три взаимно ортогональные направления циркуляции (\hat e_1,\hat e_2,\hat e_3).

Для каждого тора (T_i) удобно ввести локальные координаты:

* (\rho_i) — "малый" радиус поперёк трубы тора,
* (\varphi_i) — угловая координата вдоль тора,
* (\zeta_i) — координата вдоль оси тора (для возмущений).

Глобальные координаты ((r,\theta,\phi)) можно считать гладно связанными с ((\rho_i,\varphi_i,\zeta_i)) через вложение тора в шар, но для мод достаточно знать, что для каждого i есть свой "трубчатый" объём.

---

## **2.2. Поле ВБП в ядре: сумма трёх торовых мод**

Поле Θ и связанное с ним В-поле в ядре представим как сумму трёх торовых мод:

[
\Theta_{\rm core}(\mathbf x,t)
==============================

\sum_{i=1}^3 A_i(t),\phi_i(\mathbf x),
]

[
\vec B(\mathbf x,t)
===================

\sum_{i=1}^{3}\vec B_i(\mathbf x,t),
\quad
\vec B_i = B_{0i} f_i(\rho_i,\varphi_i),\hat e_i.
]

Типичный профиль моды тора:

[
f_i(\rho_i,\varphi_i)
=====================

J_{m_i}(k_i \rho_i),
\cos(m_i\varphi_i + \phi_i),
]

где:

* (J_{m_i}) — функция Бесселя,
* (k_i) — волновое число торовой моды (квантуется по размеру ядра),
* (m_i) — азимутальное число вдоль тора,
* (\phi_i) — фаза моды.

Квантизация по ядру даёт:

[
k_i \simeq \frac{\pi n_i}{R_{\rm core}},\quad n_i \in \mathbb N,
]

и базовые частоты:

[
\omega_{i0}^2 =
\omega_{\rm W}^2(k_i;,f_\phi,\beta_4,\gamma_6)
]

(по общей формуле для W-моды).

## **2.3. Энергия и потоки в/из торов**

Определим **энергетическую плотность** Θ-поля в ядре:

[
u(\mathbf x,t)
==============

\frac{1}{2}\kappa_{\rm core}, \dot\Theta^2
+
\frac{1}{2}\chi_{\rm core}, (\nabla \Theta)^2,
]

где (\kappa_{\rm core}, \chi_{\rm core}) — эффективные параметры ядра (жёсткость по времени и пространству).

Для i-го тора:

[
E_i(t)
======

\int_{V_i} u(\mathbf x,t),d^3x
\approx
\frac{1}{2} M_i \dot A_i^2
+
\frac{1}{2} M_i \omega_{i0}^2 A_i^2,
]

где (M_i) — эффективная "инерция" моды.

**Поток энергии** связан с фазовой скоростью (c_\phi):

[
\vec S = u, c_\phi, \hat n_\phi,
]

где (\hat n_\phi) — направление "течения" фазы (для торов — вдоль (\hat e_i) и вдоль хвостов, выходящих из ядра).

Для каждого тора вводим:

* (P_{i}^{\rm in}) — поток энергии **из внешнего Θ-поля в дефект** по каналам W-моды,
* (P_{i}^{\rm out}) — поток энергии **из дефекта наружу** (в ПЗ и дальше в хвост),
* (D_i) — диссипация (внутренние потери).

Тогда закон баланса для энергии тора:

[
\frac{dE_i}{dt}
===============

## P_{i}^{\rm in}

## P_{i}^{\rm out}

D_i.
]

---

## **2.4. Уравнение для амплитуды с явными потоками**

Подставляя выражение энергии через (A_i):

[
\frac{d}{dt}
\left(
\frac{1}{2} M_i \dot A_i^2
+
\frac{1}{2} M_i \omega_{i0}^2 A_i^2
\right)
=======

## P_{i}^{\rm in}

## P_{i}^{\rm out}

D_i.
]

Выделим эффективное **радиационное затухание** через утечку в ПЗ и хвост:

[
P_{i}^{\rm out}
===============

\Gamma_{i}^{\rm leak}, M_i \omega_{i0} \dot A_i^2,
]

и внутреннее трение:

[
D_i
===

\Gamma_{i}^{\rm int}, M_i \omega_{i0}\dot A_i^2.
]

Суммарно:

[
\Gamma_i = \Gamma_{i}^{\rm leak} + \Gamma_{i}^{\rm int}.
]

Линеаризуя и собирая члены, получаем **осциллятор с потоками**:

[
\boxed{
\ddot A_i
+
\Gamma_i \dot A_i
+
\omega_{i0}^2 A_i
=================

\frac{1}{M_i},S_i(t)
}
]

где (S_i(t)) — **эффективный источник**, связанный с потоком (P_{i}^{\rm in}):

[
P_{i}^{\rm in}
\sim
\dot A_i, S_i(t).
]

## **2.5. Как внешнее Θ-поле создаёт потоки в дефект**

Поле внешней иерархии (галактика, Солнце, Луна, планеты):

[
\Theta_{\rm ext}(\mathbf x,t)
=============================

\sum_k F_k(t),\Psi_k(\mathbf x),
]

с индексом k по:

* галактическим модам,
* кластерным,
* солнечным (B_{\ell m}^{(\odot)}),
* лунным,
* планетным.

**Входящий поток** в i-й тор получается как **проекция внешних мод на профиль тора**:

[
P_{i}^{\rm in}(t)
\sim
\sum_k
\Lambda_{ik}
F_k(t), A_i(t),
]

где (\Lambda_{ik}) — коэффициенты перекрытия:

[
\Lambda_{ik}
\propto
\int_{S_{{\rm in},i}}
\Psi_k(\mathbf x),\phi_i(\mathbf x), dS.
]

Из этого видно:

* потоки **зависят от геометрии** (как тор "повёрнут" относительно линии Земля–Солнце, Земля–Луна),
* каждый внешний источник (F_k) "подкачивает" энергию в конкретные комбинации торов.

В уравнении для (A_i) это даёт:

[
S_i(t)
======

\sum_k C_{ik} F_k(t),
]

и, значит:

[
\boxed{
\ddot A_i
+
\Gamma_i \dot A_i
+
\omega_{i0}^2 A_i
=================

\frac{1}{M_i}
\sum_k C_{ik} F_k(t)
}
]

— это уже **осциллятор с явным учётом потоков в дефект из внешнего Θ-поля**.

## **2.6. Потоки из дефекта в ПЗ и хвост**

На границе ядра (r = R_{\rm core}) для каждой моды существует **радиационное граничное условие**:

[
\left.
\partial_r \Theta_i
\right|*{r=R*{\rm core}}
========================

\mathcal B_{i}^{\rm rad}, \Theta_i,
]

где (\mathcal B_{i}^{\rm rad}) задаёт "прозрачность" границы ядро → ПЗ.

Поток **наружу**:

[
P_{i}^{\rm out}
===============

\int_{S_{\rm core}}
\vec S_i\cdot d\vec S
\sim
\Gamma_{i}^{\rm leak} M_i \omega_{i0} \dot A_i^2,
]

он формирует:

* возбуждение мод ПЗ (фильтр),
* дальнейший хвост Θ в мантии и атмосфере,
* в итоге — геомагнитное поле, сейсмику, ячейки.

Таким образом:

* ядро **не замкнуто** — оно постоянно подкачивается внешним Θ и отдаёт энергию вверх,
* "масса" Земли — эффект стационарного баланса потоков внутри дефекта.

---

## **2.7. Топологическое условие узла: фазы и суммарный поток**

Три тора связаны **центральным узлом**. Это даёт два типа условий:

1. **Фазовые:**

[
\phi_2 - \phi_1 = \phi_3 - \phi_2 = \Delta\theta = \frac{2\pi}{3}.
]

2. **Потоковые (замкнутость дефекта):**

[
\sum_{i=1}^{3} P_{i}^{\rm net} = 0,
\quad
P_{i}^{\rm net} = P_{i}^{\rm in} - P_{i}^{\rm out}.
]

В **строго стационарном состоянии** (устойчивый дефект):

[
P_{i}^{\rm in} = P_{i}^{\rm out} + D_i
\quad
\forall i,
]

а сумма по трём торам даёт:

[
\sum_i P_{i}^{\rm in}
=====================

\sum_i P_{i}^{\rm out} + \sum_i D_i,
]

что интерпретируется как:

* внешнее Θ-поле подкачивает ядро,
* ядро через ПЗ + хвост рассеивает энергию,
* внутри ядра баланс поддерживается тремя связанными торовыми каналами.

## **2.8. Краткий итог п.2**

Мы получили формальное описание ядра как **трёхтовороидального дефекта с потоками**:

* каждая торовая мода описывается амплитудой (A_i(t)) и уравнением
  [
  \ddot A_i + \Gamma_i \dot A_i + \omega_{i0}^2 A_i = \frac{1}{M_i}\sum_k C_{ik}F_k(t),
  ]
* внешняя иерархия (галактика, Солнце, Луна, планеты) входит через (F_k(t)) и коэффициенты перекрытия (C_{ik}),
* потоки **в дефект** задаются проекцией внешних Θ-мод на торовые моды ядра,
* потоки **из дефекта** реализуются через "прозрачность" границы ядро → ПЗ и формируют возбуждение ПЗ и хвоста,
* топология узла (Δθ = 2π/3) замыкает три тора и задаёт устойчивость.

## **3. Переходная зона как оператор фильтрации**

### **3.1. Геометрия и роль ПЗ**

Для Земли ПЗ в грубом приближении:

[
R_{410} < r < R_{660},\quad
R_{410} = R - 410\ \text{км},\
R_{660} = R - 660\ \text{км}.
]

Физически это не «материальный» слой, а область, где:

* интерференция трёх торов ядра + геометрия шара дают резкий спектральный фильтр,
* одни моды ядра:

  * запираются (глушатся),
  * другие — передаются наверх и превращаются в моды коры/атмосферы.

То есть ПЗ — **оператор (\mathcal F_{\rm PZ})**, действующий в пространстве мод.

---

### **3.2. Спектральный оператор (\mathcal F_{\rm PZ})**

В базисе мод ядра (индекс (i=1,2,3)) и внешних мод (индекс (\alpha)):

[
B_\alpha(t)
===========

\sum_{i}
\mathcal F_{\rm PZ}^{(\alpha i)}(\omega_i,\ell_\alpha), A_i(t).
]

Здесь:

* (A_i(t)) — амплитуды торов в ядре,
* (B_\alpha(t)) — моды, уже «вышедшие» в мантии/ПЗ,
* (\ell_\alpha) — эффективное угловое число соответствующей моды.

Для каждой пары (i, α):

[
\mathcal F_{\rm PZ}^{(\alpha i)}(\omega,\ell)
=============================================

T_{\alpha i}(\omega,\ell), e^{i\varphi_{\alpha i}(\omega,\ell)},
]

где:

* (T_{\alpha i}) — модуль передачи (0…1),
* (\varphi_{\alpha i}) — фазовый сдвиг.

**Резкие особенности (T_{\alpha i})** соответствуют узлам на ~20, 410, 660 км:

* локальные минимумы/максимумы (T(\omega,\ell)) и быстрые скачки фазы,
* определяют «приземление» ветвей n и положение глобальных переходов.

---

### **3.3. Связь с критическими числами W-моды**

Критический порог W-моды:

[
\sigma_{\rm thr} = \frac{f_\phi}{\sqrt{\beta_4}},
]

задаёт границу, выше которой мода становится нелинейной **и ПЗ перестраивает фильтр**:

[
\mathcal F_{\rm PZ} \to
\mathcal F_{\rm PZ}(\omega,\ell;\sigma),
]

где:

* при (\sigma \ll \sigma_{\rm thr}) — фильтр почти линейный,
* при (\sigma \gtrsim \sigma_{\rm thr}) — включаются дополнительные ветви, могут появляться новые проходящие окна по (\omega) и (\ell).

Это важно для:

* долгопериодных колебаний (ENSO, QBO),
* переходов в сейсмическом режиме (всплески активности).

---

### **3.4. Радиальное уравнение в ПЗ**

Радиальный профиль мод в ПЗ подчиняется эффективному уравнению:

[
\frac{d^2 u}{dr^2}
+
k_{\rm eff}^2(r,\omega,\ell), u = 0,
]

где (k_{\rm eff}^2) включает:

* геометрию шара,
* эффекты трёх торов (через effective potential (V_{\rm eff}(r))),
* параметры среды.

Минимумы (V_{\rm eff}(r)) ≈ 20, 410, 660 км → там формируются устойчивые узлы и локализация мод.

## **4. Кора: граничные условия и узлы литосферы**

### **4.1. Жёсткая фиксация Θ-фазы**

На поверхности:

[
\Theta(R,\theta,\phi,t) = 0,
\qquad
\partial_r \Theta(R,\theta,\phi,t)\neq 0.
]

Это означает:

* **узел Θ-поля** на радиусе R,
* не ноль потока — энергия уходит в атмосферу и дальше в хвост.

В сферическом разложении:

[
\Theta(R,\theta,\phi,t)
=======================

\sum_{\ell,m}
C_{\ell m}(t), Y_{\ell m}(\theta,\phi) = 0,
]

что для фиксированного набора активных ((\ell,m)) задаёт **узор нулей на сфере** — линий, где кора в среднем менее нагружена/слабее связана → естественные места для разломов/границ плит.

---

### **4.2. Различные импедансы континент/океан**

Введём **поверхностный импеданс**:

[
Z(\theta,\phi)
==============

\begin{cases}
Z_{\rm cont}, & \text{континент}\
Z_{\rm ocean}, & \text{океан}
\end{cases}
]

Разность (Z_{\rm cont} \neq Z_{\rm ocean}) приводит к:

* разным коэффициентам отражения Θ-мод,
* локальному изменению формы узловой картины,
* фиксации границ плит вдоль линий, где:

  [
  \Theta(R) \approx 0,\quad
  \nabla_{\rm tang}\Theta \ \text{максимален},\quad
  \Delta Z\ \text{велика}.
  ]

Так связывается:

* **расположение континентов**,
* **границы литосферных плит**,
* **узлы Θ-поля** на коре.

## **6. Ветвь n: единая лестница «ядро → ПЗ → атмосфера»**

### **6.1. Единый параметр n(x)**

Вводим безразмерную координату x:

* для глубины: (x = D/R), D — глубина;
* для широты: (x = \varphi_{\rm rad}) (широта в радианах).

Тогда каждая точка (слой/ячейка) соответствует **некоторому n(x)** — номеру радиальной (или квазирадиальной) моды.

Условие:

[
n(x_{20}) \approx 1.18,\quad
n(x_{410})\approx 3.5,\quad
n(x_{660})\approx 4,
]
[
n(\varphi_{\rm Hadley})\approx 6,\quad
n(\varphi_{\rm PF})\approx 9.
]

То есть **одна лестница n** непрерывно ведёт от внутренних узлов до атмосферных.

---

### **6.2. Интервалы n и физический смысл**

Сводим в таблицу:

| Интервал n  | Где по глубине/широте            | Физический смысл                   |
| ----------- | -------------------------------- | ---------------------------------- |
| 0 < n < 2   | 0–20 км, верхняя мантия/кора     | формирование коры, локальных плит  |
| 2 ≤ n < 4.5 | 20–660 км, переходная мантия     | ПЗ, узлы 410 и 660 км              |
| 4.5 ≤ n < 7 | 660 км до нижней стратосферы     | верхняя мантия, выход ветви наружу |
| 7 ≤ n ≲ 11  | атмосфера (Hadley, PF, полярные) | крупные ячейки циркуляции          |

Таким образом, для любой структуры (глубина D или широта φ):

1. считаем x (D/R или φ_rad),
2. через фиксированную формулу n(x) находим n,
3. по интервалу n определяем «родительский слой» (ядро, ПЗ, кора, атмосфера).

Эта лестница обеспечивает **единство**:

* мод 0Sℓ,
* границ 20/410/660,
* ячеек Hadley/Ferrel/Polar.

## ⊕.1. Геометрия Θ-дефекта Земли

### ⊕.1.1. Трёхторовое ядро

В модели Земли ядро — это **топологическое ядро Θ-поля**, реализованное как комбинация трёх торов:

1. **Внутренний тор $T_{\rm in}$**
   *радиус порядка радиуса внутреннего твердого ядра*
   Отвечает за наиболее жёсткую часть W-моды и задаёт «опорную» частоту $f_{\phi,\oplus}$.

2. **Промежуточный тор $T_{\rm mid}$**
   Лежит в области жидкого ядра, задаёт эффективную анизотропию и асимметрию север/юг (через разную прозрачность стенок по Θ-потоку).

3. **Внешний тор $T_{\rm out}$**
   Прижат к внешней границе ядро–мантия (CMB). Играет роль «согласующего трансформатора» между жёстким ядром и более мягкой Переходной Зоной (ПЗ).

В 7D-языке это **единый топологический дефект**, но в 3D-проекции удобнее говорить о трёх вложенных торов, задающих:

* базовый набор радиальных мод (по $n$),
* характерные времена релаксации ($\tau_0$),
* эффективный Q-фактор $Q_\oplus$ ядра.

Трёхторовость даёт минимум свободы, достаточный для:

* выделения основной ветви $n$,
* задания дифференциальной жёсткости по глубине,
* воспроизведения асимметрий (Южный/Северный полюс, континенты).

---

### ⊕.1.2. Переходная зона (ПЗ) как Θ-фильтр

Переходная зона (ПЗ) — область между верхней границей ядра и областью, где материальные свойства уже ближе к мантии/коре.

В Θ-языке ПЗ выполняет роль **фильтра Θ-поля**:

* по **частоте**: пропускает только узкий диапазон W-моды $[\omega_0-\Delta\omega,\omega_0+\Delta\omega]$;
* по **направлению**: подавляет компоненты Θ-потока с «неправильной» фазой по отношению к торовой структуре;
* по **координате $n$**: вырезает/подавляет ветви с $n$ вне разрешённого окна $n_{\min}\le n\le n_{\max}$.

Математически это эквивалентно тому, что эффективные коэффициенты W-моды (см. параметры $f_\phi,\beta_4,\gamma_6$ в общем лагранжиане ) в ПЗ резко зависят от радиуса:

[
f_\phi(r),\ \beta_4(r),\ \gamma_6(r)
;;\text{имеют «окно»};;
r\in[r_{\rm PZ,min},r_{\rm PZ,max}],
]

и вне этого окна резко уменьшается прозрачность среды для данной ветви $n$.

Именно ПЗ:

* задаёт **конечный набор мод**, доходящих до коры,
* формирует **узловую структуру** Θ-поля, на которой потом «сидят» литосферные плиты и ячейки атмосферы.

---

### ⊕.1.3. Кора и литосфера как узловая оболочка

Кора + верхняя мантия (литосфера) в Θ-модели — это **узловая оболочка**:

* Вдоль нулей Θ-поля (линий/поверхностей, где амплитуда W-моды минимальна) формируются **границы плит**.
* Вдоль максимумов — области устойчивых платформ и кратонов.
* В местах, где Θ-энергия концентрируется и/или меняется знак градиента, возникают зоны **повышенной сейсмичности** и **субдукции**.

Условно:

[
\Theta(r_{\rm lith},\varphi,\lambda)\approx 0
;\Rightarrow;
\text{границы плит},
]

[
\partial_r\Theta(r_{\rm lith},\varphi,\lambda)\ \text{экстремальна}
;\Rightarrow;
\text{зоны вулканизма и активных разломов}.
]

---

### ⊕.1.4. Атмосфера как открытый резонатор Θ-поля

Атмосфера (тропосфера + стратосфера до некоторой высоты $z_{\rm top}$) — **открытый резонатор**, «сидящий» на узловой коре:

* Нижняя граница: $z=0$ — жёсткие граничные условия по Θ-потоку (контакты с океаном/сушей, шельфом).
* Верхняя граница: частично отражающая, задаётся переходом в более разреженные слои и влиянием ВБП Солнца/планет.

В вертикальном направлении Θ-волна образует стоячие моды по $z$:

[
\Theta(r_\oplus,z,\varphi,\lambda,t)
= \Re\Big[
A_n(\varphi,\lambda),
\cos\big(k_n z+\phi_n(\varphi,\lambda)\big),
e^{-i\omega_0 t}
\Big],
]

где $k_n$ соответствует целой лестнице $n$; границы Hadley/Ferrel/Polar по широте будут связаны с узлами по $\varphi$ (см. ⊕.3).

---

### ⊕.1.5. Драйвер земной W-моды: $F_\oplus(t,\varphi,\lambda)$

Поле Земли не автономно; спектр и фаза W-моды задаются **внешним драйвером $F_\oplus$**, который есть суперпозиция вкладов:

[
F_\oplus(t,\varphi,\lambda)
===========================

F_{\rm Sun}(t,\varphi,\lambda)
+
F_{\rm Moon}(t,\varphi,\lambda)
+
F_{\rm Jup}(t,\varphi,\lambda)
+
\ldots
]

В первом приближении:

[
F_{\rm Sun}(t,\varphi,\lambda)
= A_\odot(\varphi)\cos\big(\omega_\odot t+\phi_\odot(\varphi,\lambda)\big),
]

[
F_{\rm Moon}(t,\varphi,\lambda)
= A_{\rm L}(\varphi)\cos\big(\omega_{\rm L} t+\phi_{\rm L}(\varphi,\lambda)\big),
]

[
F_{\rm Jup}(t,\varphi,\lambda)
= A_{\rm J}(\varphi)\cos\big(\omega_{\rm J} t+\phi_{\rm J}(\varphi,\lambda)\big),
]

где амплитуды/фазы зависят от геометрии (углы, расстояния, конфигурации парадов). В более строгой постановке $F_\oplus$ вводится как внешняя сила в уравнении для W-моды Θ-поля:

[
\mathcal{D}*\oplus[\Theta*\oplus]
=================================

F_\oplus(t,\varphi,\lambda),
]

где $\mathcal{D}_\oplus$ — дифференциальный оператор W-моды для планеты (см. общую главу про W-моду).

---

## ⊕.2. Лестница $n(x)$: определение, формула и таблица ветви

Основная идея: **одна и та же целочисленная ветвь $n$** описывает:

* радиальные узлы Θ-поля от ядра до коры,
* продолжение той же лестницы в атмосферу,
* широтное разбиение на ячейки Hadley/Ferrel/Polar.

То есть «с одного и того же счетчика $n$» мы читаем:

* **глубину/радиус**,
* **широту**,
* **тип ячейки/слоя**.

---

### ⊕.2.1. Обобщённая координата вдоль ветви

Введём **обобщённую координату вдоль Θ-ветви**:

* радиальная часть — в виде оптической длины W-моды:

[
s_r(r)
======

\int_{r_{\rm core}}^{r}k_r(r'),dr',
\quad
k_r(r)=\frac{\omega_0}{c_{W}(r)},
]

* широтная часть — вдоль дуги меридиана:

[
s_\varphi(\varphi)
==================

\int_{0}^{\varphi}k_\varphi(\varphi'),R_\oplus,d\varphi',
\quad
k_\varphi(\varphi)\approx\frac{\omega_0}{c_{W,\varphi}(\varphi)}.
]

Для грубой инженерной формулы в первом приближении можно считать $k_r,k_\varphi$ постоянными по соответствующим участкам.

---

### ⊕.2.2. Определение лестницы $n(x)$

**Фазовый интеграл** вдоль пути $s$:

[
\Phi(s)=\int_{0}^{s}k_{\rm eff}(s'),ds',
]

где $k_{\rm eff}$ выбирается как $k_r$ или $k_\varphi$ в зависимости от того, идём ли мы по радиусу или по широте. Тогда:

[
\boxed{
n(s)
====

n_0
+
\left\lfloor
\frac{1}{\pi},\Phi(s)
+\frac{1}{2}
\right\rfloor
}
]

* $n_0$ — базовый индекс в ядре (обычно берём $n_0=0$ для центра ядра),
* $\lfloor\cdot\rfloor$ — округление до ближайшего целого (по формуле Гельмгольца–Крамерса–Бриллюэна).

Практически:

* для **глубины $h$** (где $h=0$ на поверхности, $h>0$ вниз):

  [
  r(h)=R_\oplus-h,
  \quad
  s_r(h)\approx \frac{\omega_0}{\bar c_W},(R_\oplus-r(h)),
  ]

  [
  n(h)
  \approx
  n_0
  +
  \left\lfloor
  \frac{\omega_0}{\pi,\bar c_W},(R_\oplus-r(h))
  +\frac{1}{2}
  \right\rfloor,
  ]

* для **широты $\varphi$** (в радианах, от экватора):

  [
  s_\varphi(\varphi)
  \approx
  \frac{\omega_0}{\bar c_{W,\varphi}},R_\oplus,|\varphi|,
  ]

  [
  n(\varphi)
  \approx
  n_{\rm surf}
  +
  \left\lfloor
  \frac{\omega_0,R_\oplus}{\pi,\bar c_{W,\varphi}},|\varphi|
  +\frac{1}{2}
  \right\rfloor,
  ]

где $n_{\rm surf}$ — индекс на поверхности (ниже мы склеим эти формулы через таблицу ветви).

---

### ⊕.2.3. Инженерная упрощённая формула

Для практики удобно ввести **эффективное число полу-волн $N_{\rm tot}$** от центра ядра до широты границы PF:

[
N_{\rm tot}
\approx
\frac{1}{\pi}\int_{0}^{s_{\rm PF}}k_{\rm eff}(s'),ds',
]

и принять простую линейную аппроксимацию:

[
\boxed{
n_{\rm rad}(h)
\approx
n_0+\left\lfloor N_{\rm rad},\frac{R_\oplus-r(h)}{R_\oplus-r_{\rm core}}\right\rfloor,
}
]

[
\boxed{
n_{\rm atm}(\varphi)
\approx
n_{\rm surf}
+
\left\lfloor N_{\rm atm},\frac{|\varphi|}{\varphi_{\rm PF}}\right\rfloor,
}
]

где $N_{\rm rad},N_{\rm atm}$ подбираются так, чтобы **узлы** совпадали с:

* радиальными границами (границы ядра, ПЗ, коры),
* широтами границ Hadley/Ferrel/Polar.

---

## ⊕.3. Атмосфера и ячейки: Θ-карта, сезонный дрейф и связь с ядром

### ⊕.3.1. Θ-поле Земли и карта ячеек

Линейная проекция Θ-поля на атмосферный резонатор задаёт **еффективный потенциал ячеек**:

[
\Phi_{\rm cell}(\varphi,z)
\sim
|\Theta_\oplus(r_\oplus,z,\varphi)|^2,
]

минимумы/максимумы которого соответствуют **устойчивым ячейкам циркуляции**:

* локальные минимумы по $\varphi$ — границы Hadley/Ferrel/Polar,
* локальные максимумы — центры ячеек.

Тем самым:

* широты границ ячеек определяются **узлами W-моды** по $\varphi$,
* структура «3 ячейки на полушарие» — это просто первая, вторая и третья широтные моды одной и той же Θ-ветви $n$.

---

### ⊕.3.2. Сезонный дрейф: модуляция $F_\oplus$ и фазовый сдвиг

Сезонный ход Солнца (изменение инсоляции и положения подсолнечной точки) входит в драйвер $F_\oplus$ как **низкочастотная модуляция**:

[
F_\oplus(t,\varphi,\lambda)
===========================

F_0(\varphi,\lambda)
+
\delta F_{\rm year}(\varphi,\lambda)\cos(\omega_{\rm year} t+\phi_{\rm year}),
]

что в Θ-уравнении даёт **медленное изменение фазы** $\phi_n(\varphi,t)$ и слабый дрейф узлов по широте:

[
\varphi_{H}(t)
==============

\varphi_{H}^{(0)}
+
\Delta\varphi_H\cos(\omega_{\rm year} t+\phi_H),
]

и аналогично для границ PF. Амплитуда $\Delta\varphi$ задаётся производной ТΘ-поля по $F_\oplus$:

[
\Delta\varphi_H
\sim
\left(\frac{\partial \varphi_H}{\partial F_\oplus}\right)\delta F_{\rm year}.
]

---

### ⊕.3.3. ENSO/QBO как модуляции F⊕ на определённых n

ENSO, QBO и подобные квазипериодические колебания — это **низкочастотные модуляции Θ-поля**, селективно действующие на определённые ветви $n$:

* ENSO — преимущественно модуляция **экваториальной части** ветви $n=5$ (ячейки Hadley и связанный океанический резонатор в Тихом океане),
* QBO — вертикальная мода в **стратосфере**, связанна с $n=6$ и переходом $n=5\leftrightarrow6$.

Формально это реализуется как добавочные члены в драйвере:

[
F_\oplus
\to
F_\oplus
+
\sum_k\delta F_k(\varphi,z)\cos(\omega_k t+\phi_k),
]

где $\omega_k$ соответствуют периодам ENSO/QBO и пр., а $\delta F_k$ локализованы в тех слоях и широтах, где наблюдается максимальная чувствительность $\partial \Theta/\partial F$.

---

### ⊕.3.4. Связь с ядром: фаза «снизу вверх»

Так как **ядро задаёт опорную W-частоту** (через три тора, см. ⊕.1.1), фазовая структура Θ-поля в атмосфере зависит от:

* текущего состояния ядра (энергия, фазовый сдвиг),
* фильтрации в ПЗ,
* драйвера $F_\oplus$.

В первом приближении:

[
\Theta_{\rm atm}(\varphi,z,t)
\sim
\mathcal{T}*{\rm PZ\to atm}
\big[\Theta*{\rm core}(t)\big]
+
\mathcal{R}*{\rm ext}[F*\oplus(t,\varphi,\lambda)],
]

где $\mathcal{T}*{\rm PZ\to atm}$ — оператор передачи мод «ядро → атмосфера», а $\mathcal{R}*{\rm ext}$ — ответ на внешний драйвер. При резонансном совпадении фаз происходит:

* усиление ячеек (устойчивые крупные ячейки циркуляции),
* появление/затухание аномалий (ENSO-подобные события),
* дрейф границ ячеек.

---

## ⊕.4. Тектоника и дрейф материков как следствие Θ-поля

### ⊕.4.1. Кора как узловая оболочка Θ-поля

Как было отмечено в ⊕.1.3, кора/литосфера — **узловая оболочка**:

[
\Theta(r_{\rm lith},\varphi,\lambda)\approx 0
\quad\Rightarrow\quad
\text{границы плит},
]

градиент по широте/долготе:

[
\mathbf{G}_\Theta
=================

\nabla_{\rm surf}\Theta(r_{\rm lith},\varphi,\lambda),
]

задаёт:

* направление «давления» Θ-поля на плиты,
* локальные различия в энергетике по краям плит.

---

### ⊕.4.2. Уравнение дрейфа плит (простая форма)

В простейшем виде дрейф плиты можно описать как **медленный релаксационный отклик** на Θ-градиент:

[
\boxed{
\mathbf{v}_{\rm plate}
======================

\mu_\Theta,\mathbf{G}_\Theta
}
]

где:

* $\mathbf{v}_{\rm plate}$ — скорость дрейфа плиты в проекции на поверхность,
* $\mu_\Theta$ — эффективная «подвижность» (инверсная вязкость в Θ-канале),
* $\mathbf{G}_\Theta$ — поверхностный градиент Θ-поля.

С учётом сохранения массы и сопротивления мантии уравнение дрейфа уточняется до формы:

[
\rho_{\rm lith},H_{\rm lith},\frac{d\mathbf{v}}{dt}
===================================================

-\eta_{\rm eff},\mathbf{v}
+
\mathbf{F}*\Theta(\mathbf{G}*\Theta),
]

где $\mathbf{F}*\Theta$ пропорциональна $\mathbf{G}*\Theta$, а в стационаре даёт ту же формулу $\mathbf{v}=\mu_\Theta,\mathbf{G}_\Theta$.

---

### ⊕.4.3. Привязка к ветви $n$ и ПЗ

Поскольку узлы Θ-поля на литосфере соответствуют конкретным значениям ветви $n$ (см. ⊕.2.4), можно записать:

[
\Theta(r_{\rm lith},\varphi,\lambda)
====================================

\Theta_n\big(r_{\rm lith},\varphi,\lambda\big),
\quad
n\in\mathbb{Z},
]

а скорость дрейфа — как функцию **локального индекса $n$** и его вариаций:

[
\mathbf{v}_{\rm plate}
======================

\mu_\Theta(n),\nabla_{\rm surf}\Theta_n.
]

Перестройка мод в ПЗ (смена разрешённого интервала $[n_{\min},n_{\max}]$ или фазовое смещение) приводит к:

* медленному перераспределению $\Theta_n$,
* изменению $\mathbf{G}_\Theta$,
* долгопериодной перестройке **направлений и скоростей дрейфа материков**.

---

### ⊕.4.4. Предсказания для активных зон

1. **Границы плит** должны **совпадать с линиями нулевой амплитуды** или резкого изменения фазы Θ-поля на литосфере.
2. **Зоны субдукции** — участки, где $\mathbf{G}_\Theta$ направлен из области с более высоким $n$ в область с более низким (Θ-энергия как бы «стекает» под плиту).
3. **Срединно-океанические хребты** — участки локальных максимумов Θ-поля вдоль определённых ветвей $n$ (выход Θ-потока вверх, раздвижение плит).
4. **Долговременная статистика сильной сейсмичности** должна коррелировать с местами, где $|\mathbf{G}*\Theta|$ максимально и/или где фаза Θ-моды испытывает резонансные скачки (связанные с планетарными конфигурациями в $F*\oplus$).

---

## ⊕.5. Энергетика ячеек и потоковая функция $\psi(\varphi,z)$

### ⊕.5.1. Потоковая функция ячеек

В терминах атмосферной динамики введём привычную **потоковую функцию** $\psi(\varphi,z)$, задающую (в осреднённом по долготе приближении):

[
v_\varphi
=========

-\frac{1}{\rho},\frac{\partial\psi}{\partial z},
\quad
w
=

\frac{1}{\rho R_\oplus\cos\varphi},\frac{\partial\psi}{\partial \varphi},
]

где $v_\varphi$ — меридиональная скорость, $w$ — вертикальная, $\rho$ — плотность воздуха.

---

### ⊕.5.2. Связь $\psi$ с Θ-полем

Пусть $\sigma(\varphi,z)$ — локальная **Θ-плотность энергии** (или эффективный «напряг» Θ-поля, см. общие определения W-моды). Тогда в первом приближении:

[
\boxed{
\psi(\varphi,z)
===============

\mathcal{K},
\int^{z}!\int^{\varphi}\sigma(\varphi',z'),d\varphi',dz'
}
]

где $\mathcal{K}$ — коэффициент, зависящий от параметров W-моды ($f_\phi,\beta_4,\gamma_6$) и свойств атмосферы (температура, стратификация).

Условие **устойчивости ячейки**: вариация полной энергии $E[\psi]$ минимальна при заданном Θ-профиле:

[
\delta E[\psi]
==============

\delta!\left(
\int \frac{\rho}{2}(v_\varphi^2+w^2),dV
+
\int V_{\rm Θ}(\sigma),dV
\right)
=0,
]

где $V_{\rm Θ}$ — эффективный потенциал Θ-поля.

---

### ⊕.5.3. Критические $\sigma_{\rm thr}$ и запуск ячейки/циклона

Для каждой ветви $n$ существует **пороговая Θ-плотность** $\sigma_{\rm thr}(n)$, при превышении которой:

* либо **возникает устойчивая ячейка** (Hadley/Ferrel/Polar),
* либо в рамках существующей ячейки возбуждается **локальный вихрь/циклон**.

Формально:

[
\sigma(\varphi,z)>\sigma_{\rm thr}(n(\varphi,z))
;\Rightarrow;
\exists;\psi\ \text{с новой замкнутой циркуляцией}.
]

Это условие связывается с наблюдаемыми критериями циклогенеза (градиент SST, вертикальный сдвиг ветра и др.), которые в Θ-модели выступают как **производные от $\sigma$**.

---

### ⊕.5.4. Энергетический баланс: ядро → ПЗ → кора → атмосфера → ячейки

Полная мощность Θ-потока через Землю (в стационарном приближении):

[
P_{\rm Θ}
=========

\int
\mathbf{S}*\Theta\cdot d\mathbf{A},
\quad
\mathbf{S}*\Theta
\sim
\Theta,\partial_t\Theta
]

распадается на цепочку вкладов:

1. **Ядро:** генерация базовой W-моды $P_{\rm core}$.
2. **ПЗ:** фильтрация/перераспределение $P_{\rm PZ}$, задающая допустимые ветви $n$.
3. **Кора/литосфера:** преобразование части потока в работу по дрейфу плит $P_{\rm tect}$.
4. **Атмосфера:** передача в кинетическую энергию ячеек и циклонов $P_{\rm atm}$.
5. **Излучение/диссипация:** $P_{\rm loss}$ (излучение, турбулентность).

Баланс:

[
P_{\rm core}
============

P_{\rm PZ}
+
P_{\rm tect}
+
P_{\rm atm}
+
P_{\rm loss}.
]

Через $\psi(\varphi,z)$ и $\sigma(\varphi,z)$ мы можем выразить $P_{\rm atm}$ и получить **энергетически замкнутую схему** от ядра до ячейки.

### ⊕.6.6. Резюме «паспорта»

1. Земля — **трёхторовй Θ-дефект** с ПЗ-фильтром и узловой корой.
2. **Одна целочисленная ветвь $n$** ведёт от ядра до атмосферных ячеек (Hadley/Ferrel/Polar).
3. Атмосфера — **открытый резонатор**, жёстко связанный с ядром через W-моду.
4. Тектоника и дрейф материков — **медленный отклик** литосферы на Θ-градиент.
5. Энергетика ячеек и циклогенез описываются через $\psi(\varphi,z)$ и пороги $\sigma_{\rm thr}(n)$.
6. «Паспорт» Земли задаётся конечным набором параметров ($f_\phi,\beta_4,\gamma_6,\omega_0,Q,\tau_0$ и др.), подлежащих калибровке по данным.
