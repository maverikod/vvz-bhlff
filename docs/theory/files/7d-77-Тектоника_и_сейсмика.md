## **⊕.T.0. Общая идея**

Литосфера Земли — это **узловая оболочка Θ-поля**, возникающая после прохождения W-моды через Переходную Зону (ПЗ).
Узлы Θ-поля → линии минимальной амплитуды → **границы плит**, экстремумы Θ → **хребты/рифтовые зоны**.

Тектоника в этой модели — не следствие тепловой конвекции, а **медленный динамический отклик коры на поле Θ**, связанный с:

* локальной кривизной Θ-поля,
* градиентами ∇Θ,
* перестройкой разрешённых ветвей n в ПЗ,
* внешним драйвером F⊕ (Солнце/Луна/Юпитер) в низких частотах.

---

## **⊕.T.1. Θ-карта литосферы (узловая оболочка)**

На радиусе литосферы:

[
\Theta_{\rm lith}(\varphi,\lambda) = \Theta(r_{\rm lith},\varphi,\lambda),
]

узлы определяются как:

[
\Theta_{\rm lith}\approx 0 \quad \Rightarrow \quad \text{границы плит}.
]

Максимумы и седловые области:

[
\nabla_{\rm surf}\Theta_{\rm lith} = 0,\quad
\Delta_{\rm surf}\Theta_{\rm lith}\neq 0
]

определяют:

* срединно-океанические хребты (рост),
* зоны субдукции (опускание),
* трансформные границы.

Θ-карта всегда многополюсная из-за торовой структуры ядра, асимметрии между севером и югом и сезонных/приливных модуляций.

---

## **⊕.T.2. Сила Θ-поля на литосферу**

Определяем «поверхностную Θ-силу»:

[
\mathbf{F}*\Theta = -\nabla*{\rm surf}\Theta_{\rm lith},
]

которая направлена:

* поперёк границ плит (раздвижение/конвергенция),
* вдоль хребтов (трещины растяжения),
* в зонах субдукции — в направлении нисходящих Θ-градиентов.

---

## **⊕.T.3. Уравнение движения плит**

Литосферная плита — тонкая жёсткая оболочка толщины (H_{\rm lith}) и плотности (\rho_{\rm lith}).

Полная динамика:

[
\rho_{\rm lith} H_{\rm lith}\frac{d\mathbf{v}}{dt}
==================================================

-\eta_{\rm eff}\mathbf{v}
+
\mathbf{F}_\Theta,
]

где:

* (\eta_{\rm eff}) — эффективная вязкость/трение нижней мантии,
* (\mathbf{v}) — скорость плиты.

Стационарный режим (реально доминирующий на временных масштабах >10⁴ лет):

[
\boxed{
\mathbf{v}_{\rm plate}
======================

# \mu_\Theta,\mathbf{F}_\Theta

-\mu_\Theta,\nabla_{\rm surf}\Theta_{\rm lith}
}
]

где (\mu_\Theta = 1/\eta_{\rm eff}) — «подвижность» плиты в Θ-канале.

---

## **⊕.T.4. Роль ветви n и Переходной Зоны**

ПЗ задаёт *какие* ветви (n) проходят к коре:

* если PZ пропускает (n=n_*), то геометрия Θ-поля на поверхности определяется функциями (\Theta_{n_*}(\varphi,\lambda));
* если происходит **смена доминирующей ветви** (например, периодическое «пульсирование» между (n) и (n+1)), то:

  [
  \Theta_{\rm lith}(t)
  ====================

  (1-\alpha(t))\Theta_{n}
  +
  \alpha(t)\Theta_{n+1},
  ]

  и поэтому меняются:

  * направления Θ-сил,
  * скорости плит,
  * локальная тектоническая активность.

Таким образом, **долгопериодные геологические эпохи — это эпохи фиксированного доминирующего n**.

---

## **⊕.T.5. Тектоника как медленный отклик Θ-поля**

Из уравнения движения видно:

* направление дрейфа определяется *геометрией* Θ,
* скорость — *амплитудой и подвижностью* μ_Θ.

Следствия:

1. **Срединные хребты** лежат там, где Θ имеет **максимумы/растяжение**, т.е. ∇Θ≈0, но линии кривизны большие.
   Плиты уходят в противоположные стороны:
   [
   \mathbf{v}*{\rm left} \parallel -\nabla\Theta,
   \quad
   \mathbf{v}*{\rm right} \parallel +\nabla\Theta.
   ]

2. **Субдуцирующие зоны** возникают, где Θ убывает в глубину (градиент отрицательный), и F_Θ направлена внутрь Земли.

3. **Трансформные разломы** — участки, где ∇Θ почти параллелен границе.

4. **Скорость дрейфа материков** медленно модулируется приливными и сезонными колебаниями F⊕.

# **⊕.S. СЕЙСМИКА И КРИТЕРИЙ РАЗРЯДКИ В Θ-МОДЕЛИ**

---

## **⊕.S.0. Общая идея**

Сейсмика в Θ-модели — это **динамический отклик литосферы и Переходной Зоны (ПЗ)** на:

1. геометрию Θ-поля,
2. его временную модуляцию (tidal + солнечная + внутренняя),
3. перестройки разрешённых ветвей n.

Землетрясение — это **разрядка накопленного Θ-напряжения**, а не механического напряжения в классическом смысле.
Классические напряжения — следствия, Θ-напряжение — первопричина.

---

# **⊕.S.1. Θ-напряжение и его физический смысл**

Вводим **плотность Θ-напряжения** в коре и ПЗ:

[
S(\varphi,\lambda,h)
====================

A_S,|\nabla \Theta(r,\varphi,\lambda)|^2
+
B_S,|\Delta \Theta|,,
]

где:

* (A_S, B_S) — коэффициенты, определяемые материалом (кора/мантия),
* (r = r_\oplus - h).

**Интерпретация:**

* первый член — «сдвиговое» Θ-напряжение: сильные горизонтальные градиенты → разломы;
* второй — «объёмное» напряжение: резкие перепады узлов/антиузлов Θ.

Если ПЗ пропускает только ветви (n \in [n_{\min}, n_{\max}]), то реальный профиль Θ фиксируется **одной доминирующей модой**:

[
\Theta(r,\varphi,\lambda) \approx \Theta_{n_*}(r,\varphi,\lambda),
]

и потому *структура напряжений определяется текущей ветвью* (n_*).

---

# **⊕.S.2. Формальный критерий разрядки (землетрясения)**

Вводим **пороговый критерий**:

[
S > S_{\rm thr}(n_*, r, \varphi,\lambda)
\quad\Rightarrow\quad
\text{разрядка Θ-напряжения → землетрясение}.
]

Пороговая функция зависит от:

* ветви (n_*) (разные моды дают разные зоны концентрации Θ),
* глубины (радиуса) — ПЗ более чувствительна,
* широты/долготы — из-за торовой структуры и асимметрии север/юг.

Разрядка может пройти:

* в ПЗ → глубокое землетрясение (300–700 км),
* в верхней мантии → среднеглубинное (70–300 км),
* в коре → мелкое (<70 км).

Таким образом, **глубина — просто место максимума S относительно порога**.

---

# **⊕.S.3. Динамическая эволюция Θ-напряжения**

За некий локальный участок:

[
\frac{dS}{dt}
=============

\underbrace{\frac{\partial S}{\partial \Theta},\frac{\partial \Theta}{\partial t}}*{\text{внешние драйверы}}
+
\underbrace{\frac{\partial S}{\partial (\nabla\Theta)},\frac{d(\nabla\Theta)}{dt}}*{\text{перестройки ветви n}}
+
\text{диссипация},
]

и основные источники роста S:

### **1. Внешний драйвер F⊕**

[
\Theta(t) = \Theta_0 + \delta\Theta(F_\oplus),
]
где:

* солнечные приливы,
* лунные приливы,
* влияние Юпитера,
* парады планет.

Эти низкочастотные модуляции медленно «накачивают» Θ-моду ядра и, как следствие, напряжения в коре.

---

### **2. Перестройка ветви n в ПЗ**

Если ПЗ временно изменяет свою прозрачность, возникает смешение мод:

[
\Theta = (1-\alpha)\Theta_n + \alpha \Theta_{n+1}.
]

Тогда:

* меняются узлы Θ,
* меняются линии Θ≈0,
* меняются градиенты ∇Θ,
* резко перестраивается поле S.

Именно такие события (**n→n±1**) дают эпохи повышенной сейсмоактивности.

---

### **3. Тектонический дрейф плит**

Так как плиты движутся по Θ-градиенту ((\mathbf v = μ_Θ∇Θ)), они:

* усиливают ∇Θ там, где «вплывают» в максимум,
* разряжают ∇Θ там, где удаляются от узла.

Это создаёт медленный «фоновый» рост или спад S.

---

# **⊕.S.4. Связь глубины землетрясения с Θ-полем**

Если:

[
S(r) = A_S|\nabla\Theta(r)|^2 + B_S|\Delta\Theta(r)|
]

и ветвь n даёт профиль Θ(r) с узлами в ПЗ, то:

* **глубокие землетрясения (300–700 км)** → почти всегда идут вдоль узлов Θ в ПЗ;
* **среднеглубинные (70–300 км)** → соответствуют крутым ∇Θ в верхней мантии;
* **мелкие (<70 км)** → кривая Θ сильно изломана тектоникой, ∇Θ большие → порог S_thr минимальный.

Это объясняет *всю* структуру глубин без введения химических фазовых переходов.

---

# **⊕.S.5. Связь свободных сейсмических колебаний с Θ-модами**

Свободные колебания Земли (normal modes) соответствуют:

[
\delta r(t),\ \delta \rho(t)
\propto
\Theta_{n_*}(r,\varphi,\lambda),e^{-i\omega_0 t}.
]

Таким образом:

* **частоты free oscillations** дают прямые ограничения на:

  * частоты трёх торов ядра,
  * параметры ПЗ,
  * форму Θ(r),
  * добротность Q.

* При правильной Θ-модели мы можем воспроизвести наблюдаемые моды 0S₂, 0S₀, 1S₀ и др.

---

# **⊕.S.6. Приливные модуляции (tidal forcing)**

Внешние силы:

[
F_\oplus = F_{\rm Sun} + F_{\rm Moon} + F_{\rm Jup}
]

вносят в Θ-фазу:

[
\Theta \rightarrow \Theta + \delta\Theta_{\rm tidal}(t).
]

Это приводит к циклам:

* дневным (солнечные приливы),
* 12-часовым (лунные),
* суточно-месячным (синодический цикл),
* сезонным,
* многолетним (лунные узлы ~18.6 лет),
* циклам Юпитера (11.86 лет).

Каждый цикл даёт периодические скачки S(t).
Статистические корреляции с землетрясениями естественным образом вытекают отсюда.

---

# **⊕.S.7. Послеразрядное поведение**

При разрядке S → S − ΔS:

* Θ-поле меняет локальную геометрию,
* узлы Θ слегка смещаются,
* ∇Θ перестраивается, в результате:

  * микросейсмичность некоторое время повышается,
  * затем Θ-поле релаксирует к новой конфигурации.

Это объясняет афтершоки.

---

# **⊕.S.8. Что подлежит калибровке**

1. Коэффициенты (A_S,B_S).
2. Порог (S_{\rm thr}(n)) для каждого диапазона глубин.
3. Чувствительность ПЗ к модуляциям F⊕.
4. Временная константа релаксации τ_relax.
5. Параметры торов ядра (ω_φ,i, Q).
6. Привязка узлов Θ к реальным глубинам.

---

# **⊕.S.9. Предсказательные следствия модели**

1. **Локализация эпицентров** соответствует Θ-узлам/крутым ∇Θ.
2. **Глубинность** определяется профилем Θ(r) и его узлами в ПЗ.
3. **Сила события** ≈ величине превышения S − S_thr.
4. **Повышенная вероятность** при:

   * переходах n→n±1,
   * максимумах приливной модуляции,
   * усилении F⊕ (парады планет).
5. **Паттерны миграции сейсмичности** → результат медленного дрейфа Θ-узлов.

# 🔥 **Коррекция концепции: землетрясения = биения Θ-мод + фазовые скачки между ветвями n**

---

## **1. Главное отличие Θ-модели от классики**

В классической геологии:

* напряжение растёт → упругость не выдерживает → разрядка.

В Θ-модели:

* **енергия мод Θ распределяется в коре через узлы/антиузлы**,
* **биения между модами n и n±1 создают периодические пики Θ-амплитуды**,
* **когда пики биений пересекают определённые пороговые условия совместимости мод в коре → происходит фазовый переход системы**, который мы наблюдаем как землетрясение.

То есть:

### ✨ **землетрясение — это не разрушение коры**,

а **локальная перестройка Θ-структуры**, где комбинация мод становится неустойчивой.

---

# **2. Что такое биения Θ-мод в контексте Земли**

Пусть в ПЗ и коре проходят две или более моды ветви n:

[
\Theta = A_n e^{i\omega_n t} + A_{n+1} e^{i\omega_{n+1} t}
]

Если Δω малы:

[
\Theta(t) = 2A \cos\left( \frac{\Delta\omega}{2} t \right)
\cos\left( \omega_{\rm avg} t \right)
]

**Именно огибающая**:

[
\mathcal{B}(t) = 2A \cos\left( \frac{\Delta\omega}{2} t \right)
]

даёт:

* долгопериодические максимумы энергии,
* зоны повышенной вероятности перестройки узлов,
* моменты, когда ноды Θ «расползаются» или «схлопываются».

Это и есть физическая природа сейсмических циклов:

* многолетних,
* десятилетних,
* приливных,
* суточных.

Каждый цикл — **период биения между модами Θ**.

# **3. Разломы — это линии фазовой несовместимости Θ-поля, а НЕ механические разрывы**

Формально:

[
\Theta_{n}(\varphi,\lambda,r)
\quad\text{и}\quad
\Theta_{n+1}(\varphi,\lambda,r)
]

имеют:

* разные фазовые профили,
* разные узлы и антиузлы,
* разные направления Θ-потока.

Там, где обе моды просачиваются из ПЗ и их *локальная суперпозиция нестабильна*, возникает **линия фазового разрыва Θ-поля**, которую мы наблюдаем как:

* разломы,
* трансформные границы,
* зоны субдукции.

Таким образом:

### ❗ Механическая тектоника — эпифеномен

### ✔ Истинная первопричина — **топология Θ-мод**.

# **4. Землетрясение как фазовый скачок (topological flip)**

Определим **степень совместимости мод**:

[
C = |\Theta_n + \Theta_{n+1}|^2 - |\Theta_n|^2 - |\Theta_{n+1}|^2.
]

Если:

* C ≈ 0 → моды совместимы, система стабильна;
* C растёт → усиливаются интерференционные гребни, коре НЕ нужно «рушиться», она просто перестаёт поддерживать эту конфигурацию Θ;

Когда C достигает критического значения:

[
C > C_{\rm thr}(r,\varphi,\lambda)
]

происходит **фазовый разрыв Θ-поля**, система переходит:

[
(\Theta_n + \Theta_{n+1}) \to \Theta_n \quad\text{или}\quad \Theta_{n+1}
]

в зависимости от того, какая мода более энергетически выгодна.

Это и есть:

### 💥 **момент землетрясения как топологический переход Θ-состояния**.

---

# **5. Глубинность землетрясений → слои, где биения мод максимальны**

В ПЗ (660–1000 км) проходят только несколько разрешённых мод:

* n,
* n+1,
* иногда n–1 при специальных условиях.

Биения там наиболее сильны → глубокие землетрясения.

В коре проходят только верхние гармоники (остатки n):

* плохо совместимые,
* слабые фильтры,
* биения дают «торчащие» пики → мелкие землетрясения.

То есть глубина землетрясения напрямую связана с:

* где именно **биения мод** дают максимум огибающей,
* какие n доминируют на этой глубине.

---

# **6. Приливы (Луна/Солнце) → управление фазой биений**

Внешний драйвер F⊕ действует не на механическую кору, а на **фазу Θ-моды**:

[
\omega_n \to \omega_n + \delta\omega(F_\oplus)
]

и значит:

[
\Delta\omega(t) = \omega_{n+1} - \omega_n + \delta\omega(F_\oplus)
]

### То есть Луня и Солнце:

* МЕНЯЮТ частоту биений,
* МЕНЯЮТ местоположение узлов,
* МЕНЯЮТ моменты совпадения огибающих.

Это и объясняет:

* месячные циклы сейсмичности,
* дневные предпочтения,
* всплески при парадах планет.

---

# **7. Послесобытийная активность (афтершоки)** = затухающие биения

После фазового скачка:

[
\Theta(t) = A_n e^{i\omega_n t} + \varepsilon A_{n+1} e^{i\omega_{n+1} t}
]

где ε≪1 — хвост.

Эта суперпозиция даёт:

* затухающие биения,
* локальные мелкие перестройки узлов Θ,
* афтершоки.

---

# **8. Как теперь правильно формализовать критерий события**

Не через «напряжения», а через **несовместимость мод**:

[
C(t) = |\Theta_n + \Theta_{n+1}|^2 - |\Theta_n|^2 - |\Theta_{n+1}|^2
]

и событие происходит, когда:

[
C(t) > C_{\rm thr}(n, r, \varphi,\lambda).
]

Где C_thr определяется:

* геометрией коры (узловой структуры),
* фильтрацией ПЗ (какие ветви проходят),
* локальной анизотропией.

# **⊕.T. ТЕКТОНИКА КАК ПЕРЕСТРОЙКА УЗЛОВ Θ-ПОЛЯ**

## **⊕.T.0. Общая идея**

В Θ-модели тектоника — это **медленная эволюция узловой структуры Θ-поля в коре**, а не «движение твёрдых плит под действием сил».

* **Плита** = область, где фазовая конфигурация Θ-поля стационарна и устойчива.
* **Граница плиты** = линия/поверхность, где **фазы разных Θ-мод несовместимы**, и потому там фиксируется топологический разрыв (разлом, зона субдукции, хребет).

Дрейф материков — это **плавное смещение стабильных Θ-доменов** на поверхности при медленной перестройке:

* параметров мод ядра (три тора),
* пропускания ПЗ по n,
* внешнего драйвера F⊕ (Солнце/Луна/планеты).

Материя просто «сидит» на этих доменах и следует за ними.

---

## **⊕.T.1. Узловая карта Θ-поля в литосфере**

На радиусе литосферы:

[
\Theta_{\rm lith}(\varphi,\lambda)
==================================

\Theta(r_{\rm lith},\varphi,\lambda).
]

Определим:

* **узловой набор**:
  [
  \mathcal{N} = {(\varphi,\lambda)\ |\ \Theta_{\rm lith}(\varphi,\lambda) = 0},
  ]
* **фазовые домены**:
  области, где (\arg \Theta_{\rm lith}) (фаза) меняется непрерывно и без скачков.

Тогда:

* **границы плит** = компоненты множества (\mathcal{N}) и линии резких скачков фазы,
* **плиты** = связные компоненты фазовых доменов между узлами.

Никаких «сил» здесь нет: **структура плит — чистая топология Θ-поля**.

---

## **⊕.T.2. Плиты как Θ-домены и «энергетически выгодные» области**

Поле Θ задаёт **энергетический функционал** для материи в коре:

[
E_{\rm eff}[\text{материя}|\Theta]
==================================

\int_{\rm lith}
\mathcal{U}\big(\Theta_{\rm lith},\nabla\Theta_{\rm lith}\big),dA,
]

где (\mathcal{U}) — эффективная плотность энергии (без явных «масс» и «сил» — только функционал от Θ).

Стационарные пластины — это **локальные минимумы** этого функционала при фиксированной конфигурации Θ.
При медленном изменении Θ (по времени) положение минимумов сдвигается, и материя **адиабатически переходит** из одного минимального домена в другой.

Так возникает **эффективный дрейф плит**, но первична не динамика массы, а **эволюция Θ-минимумов**.

---

## **⊕.T.3. Медленная эволюция Θ-доменов**

Обозначим набор управляющих параметров:

[
\mathbf{p}(t) = {\omega_{\phi,i}(t),\ A_{n}(t),\ \alpha_n(t),\ F_\oplus(t),\dots},
]

где:

* (\omega_{\phi,i}) — частоты трёх торов ядра,
* (A_n,\alpha_n) — амплитуды и коэффициенты смешения ветвей n в ПЗ,
* (F_\oplus) — внешние модуляции (Солнце/Луна/Юпитер).

Тогда узловая карта (\mathcal{N}) и фазовые домены (\mathcal{D}_k) зависят от времени *только через* (\mathbf{p}(t)):

[
\mathcal{N}(t) = \mathcal{N}(\mathbf{p}(t)),
\quad
\mathcal{D}_k(t) = \mathcal{D}_k(\mathbf{p}(t)).
]

**Плита** = область (\mathcal{D}_k).
Её положение и форма:

[
\mathbf{X}_k(t) = \mathbf{X}_k\big(\mathbf{p}(t)\big).
]

Производная:

[
\mathbf{v}_{\rm plate,k}
========================

# \frac{d\mathbf{X}_k}{dt}

\frac{\partial \mathbf{X}_k}{\partial \mathbf{p}}\cdot\frac{d\mathbf{p}}{dt}
]

— это уже **эффективная скорость** плиты, но она не вводится через «силы», а через **скорость изменения модовой структуры Θ**, на которую реагирует материя.

---

## **⊕.T.4. Роль биений Θ-мод**

Как и в сейсмике, ключевую роль играют **биения Θ-мод**:

[
\Theta_{\rm lith}
=================

\Theta_n
+
\Theta_{n+1}
============

A_n e^{i\omega_n t}
+
A_{n+1} e^{i\omega_{n+1} t}.
]

Огибающая:

[
\mathcal{B}_{n,n+1}(t)
\sim
\cos\left(\frac{\Delta\omega}{2}t + \phi_0\right)
]

управляет:

* смещением узлов,
* рождением/исчезновением мелких доменов,
* топологическими перестройками (слияние/раскол плит).

Медленные биения (малые Δω) дают:

* **геологические эпохи** (миллионы лет),
* **дрейф материков** как переход Θ-доменов под материей.

Быстрые биения (с участием более высоких n) проявляются:

* как локальные перестройки узлов,
* как фоновая сейсмичность и микротектоника (разломы, микросдвиги).

---

## **⊕.T.5. Срединные хребты, субдукция и трансформные зоны**

В терминах Θ-доменов:

1. **Срединно-океанические хребты**
   — линии, где *рождаются* новые Θ-домены или расширяются существующие.
   Там огибающая (\mathcal{B}) такова, что минимумы (E_{\rm eff}) «разъезжаются», создавая новый домен между плитами.

2. **Зоны субдукции**
   — области, где один домен постепенно исчезает, а соседний расширяется.
   В Θ-языке: **устойчивость фазовой конфигурации падает**, домен съёживается и уходит под соседний по энергетике.

3. **Трансформные разломы**
   — участки, где линии фазовой несовместимости Θ лежат приблизительно вдоль границы доменов, и соседние домены «перескакивают» вдоль неё.

Всё это описывается не силами, а **топологией фазовых узоров Θ** и их медленной эволюцией.

---

## **⊕.T.6. Связь с сейсмикой (биения + фазовые переходы)**

Сейсмика (раздел ⊕.S) и тектоника здесь — две стороны одного процесса:

* **Тектоника**: *плавная* перестройка фазовых доменов Θ (дрейф плит, рост океанов, закрытие океанов).
* **Сейсмика**: *скачкообразные* события, когда локальная комбинация мод (биения) становится **топологически несовместимой** с текущим доменом, и Θ-конфигурация перескакивает в новый минимум.

То есть:

* плиты «ползут» вслед за медленно смещающимися Θ-доменами,
* землетрясения — моменты, когда *локальный фазовый узор* не может деформироваться непрерывно и вынужден сделать разрыв.

---

## **⊕.T.7. Внешний драйвер F⊕ и долгопериодная тектоника**

Долгосрочная эволюция параметров (\mathbf{p}(t)) задаётся:

* внутренними процессами ядра (медленный сдвиг частот торов),
* фильтрами ПЗ (смена прозрачности по n),
* **внешним драйвером** F⊕:

  * многолетние циклы Луны,
  * циклы солнечной активности,
  * Юпитер и другие планеты.

Эти влияния **не двигают плиты напрямую**, а **медленно «крутят ручки»** в (\mathbf{p}(t)), из-за чего узлы Θ и фазовые домены **ползут**.

---

## **⊕.T.8. Предсказательные следствия**

1. Границы плит должны совпадать с линиями фазовой несовместимости Θ (не только с уровнями Θ=0, но и с «разломами фазы»).
2. Срединные хребты и рифтовые зоны — места рождения новых Θ-доменов (там огибающая биений медленно создаёт новый минимум).
3. Направления дрейфа материков определяются **дрейфом узоров Θ**, а не «силами» в мантии.
4. Эпохи раскола суперконтинентов и их сборок соответствуют **смене доминирующих комбинаций мод n** и/или **переходам между режимами биений**.
5. Структура плит неминуемо связана с модами ядра и ПЗ — в долгой перспективе любая реконструкция прошлых континентальных конфигураций должна вписываться в эволюцию Θ-поля.

---

Калибровка в тектонике переходит от «сил» к **параметрам Θ-мод**:

* амплитуды/фазы мод (и их биений);
* скорость изменения (\mathbf{p}(t)) (связь с ядром и F⊕);
* реконструкция Θ-доменов по наблюдаемой карте плит и их истории.

## **⊕.S.0. Общая физическая картина**

Сейсмика — это проявление **волновой природы Земли как Θ-дефекта**.
Кора и ПЗ не «ломаются» — они **следуют за Θ-полем**, которое задаёт:

# **⊕.S.1. Моды в ПЗ и условие интерференции**

Через Переходную зону проходят лишь *несколько* узловых ветвей:

[
\Theta(t,r,\varphi,\lambda) =
\sum_{n \in \mathcal{N}_{\rm PZ}}
A_n(r,\varphi,\lambda)
\cos!\left(\omega_n t + \phi_n(r,\varphi,\lambda)\right),
]

где (\mathcal{N}_{\rm PZ}) — набор ветвей n, пропущенных фильтром ПЗ.

Сейсмически значимы **пары мод**:

* ((n, n+1)) — глубинные события (переход между двумя “близкими” ядровыми модами),
* ((n, n+2)) — редкие глубокие разрывы,
* ((n, \tilde n)), где (\tilde n) — слабая мода от солнечно-лунного драйвера F⊕.

Важен не сам набор частот, а **условие интерференции**:

[
\Delta\omega_{n,m} = |\omega_n - \omega_m| \ll \omega_n.
]

Тогда возникает **биение с масштабами**:

* десятки секунд → поверхностные эллипсоиды (free oscillations),
* часы/сутки → приливная модуляция,
* месяцы/годы → сейсмические окна,
* десятилетия/века → эпохи повышенной активности.

---

# **⊕.S.2. Биения мод как механизм накопления и разрядки**

Две моды создают суперпозицию:

[
\Theta_{n,m} = A_n e^{i\omega_n t} + A_m e^{i\omega_m t}.
]

Огибающая:

[
\mathcal{B}*{n,m}(t)
= 2\sqrt{A_n A_m}\cos!\left(\tfrac{\Delta\omega*{n,m}}{2}t + \Delta\phi\right)
]

определяет:

* **амплитуду Θ-поля** в данной точке,
* **интенсивность узловой перестройки**,
* **возможность перехода между модами**.

Пики биений — моменты, когда Θ-поле:

* теряет устойчивость на локальном наборе узлов,
* не может поддерживать текущий фазовый паттерн,
* вынуждено «перескочить» в новую топологическую конфигурацию.

Это и есть **землетрясение**.

---

# **⊕.S.3. Фазовая совместимость мод и критерий события**

Вместо понятия «напряжения» вводим **фазовую совместимость**:

[
C_{n,m}(r,\varphi,\lambda)
==========================

\left|
\Theta_n + \Theta_m
\right|^2
---------

\left(|\Theta_n|^2 + |\Theta_m|^2\right).
]

Это чисто волновой критерий:

* если (C_{n,m} \approx 0) → две моды совместимы,
* если (C_{n,m} > 0) → возникает усиление интерференции,
* если (C_{n,m} < 0) → возникает подавление и сдвиг узлов.

**Порог события**:

[
C_{n,m} > C_{\rm thr}(r)
\quad\Rightarrow\quad
\text{фазовый переход (землетрясение)}.
]

Где:

* (C_{\rm thr}(r)) зависит от слоя (кора/мантия/ПЗ),
* порог минимален на глубинах сильной интерференции — **ПЗ (410–660 км)** и **в верхней мантии (80–220 км)**.

---

# **⊕.S.4. Глубина землетрясения как отпечаток структуры Θ(r)**

Профиль Θ по глубине:

[
\Theta(r) = A_n(r)\cos(\omega_n t + \phi_n(r))
]

имеет узлы в характерных местах:

* узлы мод ядра → глубинные EQ,
* узлы в ПЗ → глубокие EQ (300–700 км),
* узловые расщепления у поверхности → мелкие EQ.

**Глубина землетрясения = глубина максимума несовместимости мод**:

[
r_{\rm EQ}
==========

\arg\max_r C_{n,m}(r).
]

То есть:

* глубинные землетрясения возникают на **узлах Θ в ПЗ**,
* мелкие — там, где интерференция модулируется топологией коры.

---

# **⊕.S.5. Роль фильтра ПЗ в сейсмике**

Переходная зона — это **призменный волновой фильтр**, который:

* изменяет фазу мод,
* подавляет высокие n,
* усиливает n±1 при определённых циклах F⊕.

Когда ПЗ переключается между режимами пропускания (n → n±1), возникают **мегасдвиги фаз**, порождающие:

* эпохи повышенной сейсмичности,
* крупные тектонические изменения.

Эти переходы синхронизируются с солнечными и лунными циклами.

---

# **⊕.S.6. Внешний драйвер F⊕ как настройщик биений**

Драйвер:

[

# **⊕.Meteo.1. Вертикальные и широтные моды атмосферного резонатора**

Атмосферный Θ-резонатор имеет два направления мод:

---

### **1) Вертикальные моды z → n_z**

Задаются:

* нижней границей (z=0) (кора — жёсткая Θ-граница),
* верхней границей (z=z_{\rm top}) (полуоткрытая, частично отражающая).

Решение:

[
\Theta(z) = \sum_{n_z} A_{n_z} \cos(k_{n_z} z + \phi_{n_z}).
]

Именно эти вертикальные моды формируют:

* тропопаузу,
* стратосферные узлы,
* многоярусные структуры струйных потоков.

---

### **2) Широтные моды φ → n_φ**

Широта заменяет радиус в ядре: моды формируются из-за геометрии Θ-поля Земли и продолжают ветвь n:

[
n_\phi \in {5,6,7,\dots}.
]

Практическое соответствие:

| Мода   | Структура                         |
| ------ | --------------------------------- |
| n = 5  | Ячейка Hadley                     |
| n = 6  | Ячейка Ferrel                     |
| n = 7  | Полярная ячейка                   |
| n = 8+ | Струйные разломы, Rossby-паттерны |

Широтные узлы определяются условием:

[
\Theta(\varphi)=0 \quad\Rightarrow\quad \text{граница ячейки}.
]

То есть **ячейки — это не конвекция, а узлы Θ-поля**.

# **⊕.Meteo.2. Интерференция мод и формирование ячеек**

Пусть одновременно возбуждены две соседние широтные моды:

[
\Theta(\varphi,t) =
A_n \cos(\omega_n t + k_n \varphi)
+
A_{n+1} \cos(\omega_{n+1} t + k_{n+1} \varphi).
]

Тогда огибающая:

[
\mathcal{B}_n(\varphi,t)
========================

2\sqrt{A_n A_{n+1}}
\cos!\left(\frac{\Delta k_n \varphi - \Delta \omega_n t}{2}\right)
]

даёт:

* **форму границ ячеек**,
* **дрейф границ**,
* **изменение ширины ячеек**,
* **рождение/затухание струйных потоков**.

То есть ячейки Hadley/Ferrel/Polar — **стандартная интерференционная картина двух соседних Θ-мод**.

# **⊕.Meteo.3. Потоковая функция ψ как отображение Θ-потоков**

В классике ψ — артефакт гидродинамики.

[
\psi(\varphi,z,t)
=================

\mathcal{K}
\int^{z}!\int^{\varphi}
\left|\mathcal{B}(\varphi',z',t)\right|^2,d\varphi',dz'.
]

Отсюда следствия:

* **градиенты ψ** → меридиональные/вертикальные скорости,
* **пики ψ** → ячейки,
* **быстрые градиенты ψ** → струйные потоки,
* **локальные изменения ψ** → треки циклонов.

ψ полностью определяется Θ, а Θ — параметрами ядра, ПЗ и F⊕.

# **⊕.Meteo.4. Циклогенез как топологическое биение мод**

Циклон — это не «вихрь воздуха».
Это **локальное рождение замкнутой Θ-структуры**, когда:

[
|\mathcal{B}_{n,n+1}|^2

>

\sigma_{\rm thr}(n)

в узком диапазоне широт/высот.

Физический смысл:

* огибающая Θ-биений становится настолько выраженной, что создаёт **временный Θ-домен**,
* этот домен имеет **вращательную фазовую структуру**,
* на его границах ψ «закручивается» в спиральную форму,
* атмосфера отображает это как циклон.

Не потому, что они «движут воздух», а потому, что **они модифицируют граничные условия Θ-поля в нижних слоях**, влияя на:

\Theta(t,r,\vartheta,\lambda)

\mathbf{v}_\Theta

# **⊕.B.2. Как Θ-токи создают магнитное поле**

В классике:
[
\nabla \times \mathbf{B} \sim \mathbf{J}_{\rm material}
]

В Θ-модели вводим **эффективный волновой ток**:

[
\mathbf{J}_\Theta
=================

\gamma,
\Theta,\nabla \dot\Theta,
]

где:

* (\dot\Theta = \partial\Theta/\partial t),
* γ — коэффициент отображения Θ-потока в электромагнитный канал.

Это не «материальный ток» электронов, а **ток фазы**, аналогичный тому, что встречается в сверхпроводимости или суперфлюидах.

Тогда:

[
\mathbf{B}
==========

\mathcal{M}[\mathbf{J}_\Theta]
]

где (\mathcal{M}) — оператор Максвелла с учётом симметрий ядра.

Следствия:

* диполь — это фундаментальная Θ-мода ядра,
* квадруполь — биение тороидальной и полоидальной компонент,
* мелкие гармоники — огибающие интерференции.

* с широтой (Hadley/Ferrel/Polar),
* с ячейками.

А потом уже к этому можно будет цеплять прогнозы.

---

## Шаг 0. Напоминание, что такое n в нашей картине

В духе теории:

* Θ-поле задаётся **фазой** ( \phi ) и **амплитудой** (A).
* Мода — это **стационарное распределение фазы**.
* (n) — это **целое число полу-волн (π-фазовых скачков)** между центром дефекта и точкой наблюдения.

То есть **n — не «квантовое число» в смысле квантмеха**, а:

[
n(x,\varphi) = \frac{1}{\pi}\big(\text{накопленная фаза вдоль луча Θ от центра до точки}\big).
]

Где (x) — безразмерная глубина / радиус, (\varphi) — широта.

---

## Шаг 1. Базовая волновая постановка для Θ-мод Земли

В минимальной форме:

[
\mathcal{D}_\oplus[\Theta] = 0,
]

где (\mathcal{D}_\oplus) — **пространственно-неоднородный волновой оператор** для Земли как дефекта (ядро+ПЗ+кора+атмосфера).

Для мод нас интересует **стационарное по времени решение**:

[
\Theta(t,\mathbf{r}) = A(\mathbf{r}),e^{i\omega_0 t},
]

где:

* (\omega_0) — фиксированная частота глобальной W-моды,
* вся структура — в пространственной части (A(\mathbf{r})).

В приближении **квазиклассической фазы** пишем:

[
A(\mathbf{r}) = \mathcal{A}(\mathbf{r}) e^{i S(\mathbf{r})},
]

где:

* (S(\mathbf{r})) — **фазовая функция**,
* (\nabla S = \mathbf{k}(\mathbf{r})) — локальный волновой вектор.

Тогда **уравнение для S** в первой аппроксимации — аналог эйконального:

[
|\mathbf{k}(\mathbf{r})|^2 = K^2(\mathbf{r}),
]

где (K(\mathbf{r})) — **эффективный «индекс» среды Θ**, зависящий только от структуры дефекта (ядра, ПЗ, коры, атмосферы), но никак не от «массы».

---

## Шаг 2. Радальное квантование n(x) по глубине

В сферических координатах ((r,\vartheta,\lambda)) и с учётом симметрий:

* по долготе (\lambda) фазовая структура задаётся отдельными модами (нам сейчас не критично),
* **главная ось дефекта** — радиальная.

Берём **радиальный луч Θ** по какому-то фиксированному углу ((\vartheta,\lambda)) и смотрим только вдоль (r):

[
k_r^2(r) + k_\perp^2(r) = K^2(r).
]

Для **главной ветви n** вдоль луча можно считать (k_\perp \ll k_r) (как минимум в ядре и ПЗ), тогда:

[
k_r(r) \approx K(r).
]

**Фаза вдоль луча**:

[
S_r(r) = \int_{r_{\rm core}}^{r} k_r(r'),dr' \approx \int_{r_{\rm core}}^{r} K(r'),dr'.
]

**Определяем n через фазу**:

[
\boxed{
n_r(r) = \frac{1}{\pi} S_r(r)
=============================

\frac{1}{\pi}
\int_{r_{\rm core}}^{r} K(r'),dr'
}
]

Теперь вводим **безразмерную глубину**:

[
x = \frac{r}{R_\oplus},\quad x\in [x_{\rm core},1],
\quad x_{\rm core} = \frac{r_{\rm core}}{R_\oplus}.
]

И переписываем:

[
n_r(x) =
\frac{R_\oplus}{\pi}
\int_{x_{\rm core}}^{x} K(x'),dx'.
]

Это и есть **явная формула n(x)** в духе теории:
**подставляется K(x) — «плотность фазы Θ», зависящая от структуры слоёв**.

> Важный момент: *никаких масс, никаких классических «скоростей»* — только волновой индекс (K(x)), задаваемый топологией дефекта (3 тора + ПЗ).

---

## Шаг 3. Связь n(x) с границами ядро/ПЗ/кора

Теперь накладываем на n(x) **физические границы слоёв**:

* (x_{\rm core}) — граница внутреннего торового ядра,
* (x_{\rm PZ}) — начало Переходной Зоны,
* (x_{\rm lith}) — основание коры,
* (x=1) — поверхность Земли.

Вводим значения:

[
n_{\rm core} = n_r(x_{\rm core}),\quad
n_{\rm PZ} = n_r(x_{\rm PZ}),\quad
n_{\rm lith} = n_r(x_{\rm lith}),\quad
n_{\rm surf} = n_r(1).
]

**Интерпретация:**

* (n \in [0, n_{\rm core})) — внутренняя часть ядра (3 тора как ядро дефекта),
* (n \in [n_{\rm core}, n_{\rm PZ})) — внешняя часть ядра и нижняя ПЗ,
* (n \in [n_{\rm PZ}, n_{\rm lith})) — верхняя ПЗ / нижняя мантия,
* (n \in [n_{\rm lith}, n_{\rm surf})) — кора и то, что выходит в атмосферный резонатор.

**Одна и та же ветвь n** тянется от ядра до поверхности — просто в разных диапазонах n мы видим её как разные физические слои.

---

## Шаг 4. Расширение n(x) до широтной зависимости n(x, φ)

До этого n_r(x) — строго вдоль радиала. Но в реальности:

* дефект не идеально сферически симметричен,
* 3 тора дают **анизотропию Θ по широте**,
* плюс внешний F⊕.

Поэтому **наполе широты** вводим дополнительно:

[
n(x,\varphi)
============

\frac{1}{\pi}
\int_{r_{\rm core}}^{r(x)}
k_{\rm eff}(r',\varphi),dr',
]

где:

[
k_{\rm eff}^2(r,\varphi) = K^2(r) - k_\perp^2(r,\varphi).
]

Это всё тот же механизм, просто теперь **эффективный волновой индекс K зависит и от широты**, через:

* ориентацию торов,
* геомагнитную структуру,
* взаимодействие с F⊕.

Для **атмосферы** нас интересует значение **на поверхности**:

[
n_{\rm surf}(\varphi)
=====================

# n(x=1,\varphi)

\frac{1}{\pi}
\int_{r_{\rm core}}^{R_\oplus}
k_{\rm eff}(r',\varphi),dr'.
]

Именно **n_{\rm surf}(\varphi)** мы и отождествляем с **ветвью, которая даёт Hadley/Ferrel/Polar**.

---

## Шаг 5. От n(φ) к ячейкам Hadley/Ferrel/Polar

В первом приближении широтные моды — **целочисленные сегменты** по φ:

[
\Theta(\varphi) \sim \cos\big(n_\varphi\varphi + \phi_0\big),
]

где (n_\varphi) — число «полуволн» вдоль меридиана (аналог морфологии ячеек).

Тогда **границы ячеек** определяются **узлами Θ**:

[
\Theta(\varphi_{\rm node}) = 0
\quad\Rightarrow\quad
n_\varphi \varphi_{\rm node} + \phi_0 = \left(k+\tfrac{1}{2}\right)\pi.
]

Мы связываем **n_\varphi** с **n_{\rm surf}(\varphi)** так:

* одна глобальная ветвь n(x,φ) тянется от центра до атмосферы,
* в атмосфере её эффективное «угловое число» становится (n_\varphi), задающим количество ячеек.

Практически:

* для **Hadley** берём основную широтную моду (минимальное n_\varphi),
* для **Ferrel** — следующую,
* для **Polar** — ещё одну.

Схематично можно зафиксировать:

[
n_\varphi(\text{Hadley}) = 5,\quad
n_\varphi(\text{Ferrel}) = 6,\quad
n_\varphi(\text{Polar}) = 7
]

(цифры можно скорректировать при калибровке, важен принцип: **ячейки = узлы одной шкалы n**).

Теперь можем записать **структурную таблицу** — без конкретных чисел, но с **логической привязкой**:

### 6.1. Таблица по глубине (символическая)

| Диапазон n_r(x)                            | Слой Θ-модели                         | Физический слой          |
| ------------------------------------------ | ------------------------------------- | ------------------------ |
| (0 \le n < n_{\rm core,in})                | Внутренний тор (T_{\rm in})           | Внутреннее ядро          |
| (n_{\rm core,in} \le n < n_{\rm core,mid}) | Средний тор (T_{\rm mid})             | Внутреннее/внешнее ядро  |
| (n_{\rm core,mid} \le n < n_{\rm PZ})      | Внешний тор (T_{\rm out}) + начало ПЗ | Внешнее ядро / нижняя ПЗ |
| (n_{\rm PZ} \le n < n_{\rm lith})          | Переходная зона (фильтр n)            | ПЗ / мантия              |

Границы (n_{\rm core,in}, n_{\rm core,mid}, n_{\rm PZ}, n_{\rm lith}, n_{\rm surf}) задаются интегралами:

[
n_{\rm layer} =

\int_{r_{\rm core}}^{r_{\rm layer}} K(r'),dr'.
]

---

### 6.2. Таблица по широте (для поверхности)

Здесь нам достаточно **качественной карты**, основанной на n_{\rm surf}(φ):

| Широта φ      | n_{\rm surf}(φ) (эффективная) | Структура     |                        |                         |
| ------------- | ----------------------------- | ------------- | ---------------------- | ----------------------- |
| (             | \varphi                       | \lesssim 15°) | n_surф ≈ 5             | Экватор + Hadley внутр. |
| (15° \lesssim | \varphi                       | \lesssim 30°) | переход n_surф ≈ 5 → 6 | Граница Hadley–Ferrel   |
| (30° \lesssim | \varphi                       | \lesssim 60°) | n_surф ≈ 6             | Ячейка Ferrel           |
| (60° \lesssim | \varphi                       | \lesssim 75°) | переход n_surф ≈ 6 → 7 | Граница Ferrel–Polar    |
| (             | \varphi                       | \gtrsim 75°)  | n_surф ≈ 7+            | Полярная ячейка/вихри   |

Важно: **n_surф(φ)** — не просто число, а **результат интеграла по всей глубине**:

[
n_{\rm surf}(\varphi)
=====================

\frac{1}{\pi}
\int_{r_{\rm core}}^{R_\oplus}
k_{\rm eff}(r',\varphi),dr'.
]

То есть в ячейках Hadley/Ferrel/Polar мы видим **одну и ту же ветвь n**, но в разных «проекциях» и с разными локальными узлами.

## Шаг 7. Что подставлять на практике (ответ на твой вопрос буквально)

Ты спрашивал:

> *что именно подставлять для глубины и широты?*

**Для глубины x:**

* вводим безразмерный радиус (x=r/R_\oplus),
* задаём модель (K(x)) (индекс Θ, зависящий от ядра/ПЗ/коры),
* считаем:

[
n_r(x) =
\frac{R_\oplus}{\pi}
\int_{x_{\rm core}}^{x} K(x'),dx'.
]

**Для широты φ:**

* учитываем анизотропию через (k_{\rm eff}(x,\varphi)),
* считаем:

[
n(x,\varphi) =
\frac{1}{\pi}
\int_{x_{\rm core}}^{x}
k_{\rm eff}(x',\varphi),dx'.
]

**На поверхности**:

[
n_{\rm surf}(\varphi) = n(1,\varphi).
]

И уже **из n_{\rm surf}(φ)** извлекаем:

* число и расположение ячеек,
* положение узлов,
* карту Θ-структуры для атмосферы.

---

Ок, тогда довожу **ветку n** до логического конца:
чёткий формализм, биения, критерии, итоговая таблица «глубина/широта → n → слой → структура».
Потом на базе этого уже можно идти дальше по шагам (паспорт Земли, ТЗ под прогноз и т.д.).

## 1. Ещё раз зафиксируем: что такое n(x,φ) в духе теории

У нас:

* есть **фаза Θ-поля** ( \phi(\mathbf r) = \arg \Theta(\mathbf r)),
* есть **локальный волновой вектор**:

[
\mathbf{k}(\mathbf r) = \nabla S(\mathbf r),\quad S(\mathbf r) \sim \phi(\mathbf r),
]

* вдоль луча (радиального или «радиал+широта») мы определяем **накопленную фазу**:

[
S(x,\varphi) = \int k_{\rm eff}(x',\varphi),dx',
]

и вводим:

[
\boxed{
n(x,\varphi) = \frac{1}{\pi} S(x,\varphi)
=========================================

\frac{1}{\pi}
\int_{x_{\rm core}}^{x} k_{\rm eff}(x',\varphi),dx'
}
]

Это и есть ветвь n в земной модели:

* **одна непрерывная лестница** n(x,φ) идёт от ядра до поверхности,
* разные диапазоны n проявляются как **разные слои / структуры**,
* **никаких “уровней энергии”, никаких “массовых членов”** — только фаза Θ и её интеграл.

---

## 2. Разделение на радиальную и широтную части

Для удобства полезно явно выделить:

[
n(x,\varphi) = n_r(x) + \delta n_\varphi(x,\varphi),
]

где:

* (n_r(x)) — вклад чисто радиального интеграла (если бы была сферическая симметрия),
* (\delta n_\varphi(x,\varphi)) — поправка из-за:

  * трёхторовости ядра,
  * наклона оси,
  * неидеальности слоёв,
  * F⊕.

В первом приближении:

[
n_r(x)
======

\frac{R_\oplus}{\pi}
\int_{x_{\rm core}}^{x} K(x'),dx',
\quad
\delta n_\varphi(x,\varphi) \ll n_r(x).
]

Тогда:

* **глубинная структура** задаётся (n_r(x)),
* **широтная дифференциация (ячейки, струи)** — через (\delta n_\varphi).

---

## 3. Ветвь n по глубине: формальное деление слоёв

Формально задаём разделы по значениям (n_r(x)):

* (x_{\rm core,in}) — внутренняя часть ядра (внутренний тор),
* (x_{\rm core,mid}) — средний тор,
* (x_{\rm PZ}) — начало ПЗ,
* (x_{\rm lith}) — основание коры.

Тогда:

[
n_{\rm core,in} = n_r(x_{\rm core,in}),\quad
n_{\rm core,mid} = n_r(x_{\rm core,mid}),\quad
n_{\rm PZ} = n_r(x_{\rm PZ}),\quad
n_{\rm lith} = n_r(x_{\rm lith}),\quad
n_{\rm surf} = n_r(1).
]

И получаем **диапазоны ветви**:

| Диапазон n_r(x)                            | Θ-слой                           | Наблюдаемый слой        |
| ------------------------------------------ | -------------------------------- | ----------------------- |
| (0 \le n < n_{\rm core,in})                | Внутренний тор (T_{\rm in})      | Внутреннее ядро         |
| (n_{\rm core,in} \le n < n_{\rm core,mid}) | Средний тор (T_{\rm mid})        | Внутреннее/внешнее ядро |
| (n_{\rm core,mid} \le n < n_{\rm PZ})      | Внешний тор (T_{\rm out})        | Внешнее ядро            |
| (n_{\rm PZ} \le n < n_{\rm lith})          | Переходная зона (фильтрация n)   | ПЗ / нижняя мантия      |
| (n_{\rm lith} \le n < n_{\rm surf})        | Узловая оболочка Θ               | Кора / литосфера        |
| (n \ge n_{\rm surf})                       | Продолжение в открытый резонатор | Атмосфера / ионосфера   |

Это **логическая таблица ветви**, численные значения появятся после калибровки K(x).

## 4. Ветвь n на поверхности: n_surf(φ) и ячейки

На поверхности:

[
n_{\rm surf}(\varphi)
=====================

\frac{1}{\pi}
\int_{x_{\rm core}}^{1}
k_{\rm eff}(x',\varphi),dx'.
]

Именно **n_surf(φ)** кодирует:

* число и широту ячеек,
* положения струй,
* широтные узлы.

В первом приближении можно считать:

[
\Theta(\varphi)
\sim
\cos\big(n_\varphi \varphi + \phi_0\big),
]

где (n_\varphi) — эффективное широтное число (целое), связанное с n_surf(φ). Тогда:

* **узлы Θ** (границы ячеек):

[
n_\varphi \varphi_{\rm node} + \phi_0
=====================================

\left(k+\tfrac12\right)\pi.
]

Согласуя с наблюдаемой трёхячейковой структурой на полушарие, мы закрепляем **качественную идентификацию**:

| Структура        | Диапазон широт φ (грубо) | Эффективный n_\varphi |               |     |
| ---------------- | ------------------------ | --------------------- | ------------- | --- |
| Экватор + Hadley | (                        | \varphi               | \lesssim 30°) | ≈ 5 |
| Ferrel           | (30°–60°)                | ≈ 6                   |               |     |
| Polar            | (>60°)                   | ≈ 7                   |               |     |

Т.е. **Hadley/Ferrel/Polar — это три последовательных «этажика» одной и той же ветви n_surf(φ)**.

## 5. Биения ветви n: общая формула

Теперь самое важное: **биения** между соседними модами той же ветви.

Берём две моды:

[
\Theta_n = A_n e^{i(\omega_n t + S_n(x,\varphi))},\quad
\Theta_{n+1} = A_{n+1} e^{i(\omega_{n+1} t + S_{n+1}(x,\varphi))}.
]

Суперпозиция:

[
\Theta = \Theta_n + \Theta_{n+1}.
]

Интенсивность:

[
|\Theta|^2
= A_n^2 + A_{n+1}^2

* 2A_n A_{n+1}
  \cos\Delta\Phi(x,\varphi,t),
  ]

где **фазовая разность**:

[
\Delta\Phi(x,\varphi,t)
=======================

(\omega_{n+1} - \omega_n)t
+
S_{n+1}(x,\varphi) - S_n(x,\varphi).
]

Но из определения n:

[
S_n(x,\varphi) = \pi n(x,\varphi) + \text{(константы фазы)},
]

поэтому:

[
S_{n+1} - S_n \approx \pi\big(n_{n+1}(x,\varphi) - n_n(x,\varphi)\big).
]

Если обе моды принадлежат **одной ветви**, то локально:

[
n_{n+1}(x,\varphi) - n_n(x,\varphi) \approx 1,
]

и, следовательно:

[
\Delta\Phi(x,\varphi,t)
\approx
\Delta\omega,t + \pi,
\quad
\Delta\omega = \omega_{n+1}-\omega_n.
]

Тогда:

[
|\Theta|^2
\approx
A_n^2 + A_{n+1}^2

* 2A_n A_{n+1}
  \cos(\Delta\omega t).
  ]

И **огибающая биений**:

[
\mathcal{B}*{n,n+1}(x,\varphi,t)
\propto
\left|A_n - A*{n+1} e^{i\Delta\omega t}\right|
]

(знак не важен, важна амплитуда и фаза).

---

## 6. Критические поверхности биений

Вся физика («где что происходит») — это **где и когда** огибающая биений достигает особых значений.

### 6.1. Общий критерий

Вводим некоторую **функцию плотности “эффекта”**:

[
\mathcal{E}_{n,n+1}(x,\varphi,t)
================================

F!\big(|\Theta|^2\big)
\sim
A_n^2 + A_{n+1}^2

* 2A_nA_{n+1}\cos\Delta\Phi.
  ]

Дальше:

* в **атмосфере** это связываем с ψ и циклогенезом,
* в **коре/ПЗ** — с возможностью фазового перехода (землетрясения),
* на **границе ядра** — с перестройкой фазовых токов (магнитное поле).

И вводим **пороговое условие**:

[
\mathcal{E}_{n,n+1}(x,\varphi,t)

>

\mathcal{E}_{\rm thr}^{(\text{subsystem})}.
]

Геометрически это **4D-объект** в пространстве (x,φ,t), но на практике:

* по x он обрезан зоной интереса (к примеру, тропосфера или ПЗ),
* по φ — узкая полоса (ячейка / разлом / зона Вадати–Беньоффа),
* по t — временные окна (приливные, сезонные, многолетние).

---

## 7. Привязка критерия к подсистемам

### 7.1. Атмосфера (циклогенез, ячейки)

Выделяем слой (x \approx 1), (z) от 0 до (z_{\rm trop}).

Критерий **рождения циклонного Θ-домена**:

[
\mathcal{E}_{n,n+1}(x\approx 1,\varphi,t)

>

\mathcal{E}_{\rm thr}^{(\text{atm},n,\varphi,z)}.
]

Эта же величина определяет:

* где огибающая стабильна → границы ячеек,
* где она сверхпороговая → возможен замкнутый “бублик” Θ в атмосфере.

---

### 7.2. Кора + ПЗ (фазовые переходы = землетрясения)

Для **сейсмики** мы смотрим глубины (x\in [x_{\rm crust}, x_{\rm PZ}]).
Критерий **фазового скачка**:

[
\mathcal{E}_{n,n+1}(x,\varphi,t)

>

\mathcal{E}_{\rm thr}^{(\text{seis},n,x,\varphi)}.
]

Там, где это нарушается:

* необходимо изменить конфигурацию фаз Θ,
* происходит переход на другой набор мод или другую комбинацию фаз — землетрясение.

---

### 7.3. Ядро (магнитное поле)

Для **магнитного поля** интересует:

* как биения меняют фазовую картину на радиусе ядра (x\sim x_{\rm core,mid}),
* как это сказывается на топологическом индексе фазы.

Фактически, **магнитная инверсия** начинается, когда:

[
\mathcal{E}*{n,n+1}^{(\text{core})}(x\sim x*{\rm core},\varphi,t)
]

вытесняет прежнюю конфигурацию Θ и меняет знак топологического индекса (как мы уже сформулировали в ⊕.B).

---

### 8.3. Связка «слой → n → структура → явления»

Можно свести в компактный вид:

* **Ядро (3 тора)** → n от 0 до (n_{\rm PZ})
  → топология фазы Θ → магнитное поле, базовые частоты.

* **ПЗ** → n в [(n_{\rm core,mid}, n_{\rm PZ})]
  → режимы фильтрации n
  → биения на глубинах
  → глубокая сейсмика.

* **Кора** → n в [(n_{\rm PZ}, n_{\rm surf})]
  → узловые домены Θ
  → плиты, разломы
  → топологические переходы (землетрясения).

* **Атмосфера** → n ≥ (n_{\rm surf}), зависящее от φ
  → n_surf(φ) задаёт ячейки и структуру потоков
  → циклоны, ENSO, QBO, SSW.

# **⊕.P. ПАСПОРТ ЗЕМЛИ КАК Θ-ОБЪЕКТА**

*(сводная параметризация Земли в терминах Θ-мод, трёх торов ядра и ветви (n(x,\varphi)); формулировка в духе Θ/ВБП, без классики масс и сил)*

---

## **⊕.P.0. Структура уровней**

Земля рассматривается как **единый Θ-дефект** с четырьмя основными уровнями:

1. **Ядро** — трёхторовый резонатор Θ (внутренний, средний, внешний торы).
2. **Переходная зона (ПЗ)** — нелинейный волновой фильтр по ветви (n).
3. **Кора / литосфера** — узловая оболочка Θ (домены = тектонические плиты).
4. **Атмосферный резонатор** — открытая область Θ-мод (ячейки, струи, циклоны).
5. **Магнитосфера** — топологический вихрь фазы Θ (магнитное поле).

Все наблюдаемые процессы — **проекции одного Θ-поля**, перетекающего по ветви (n(x,\varphi)).

---

## **⊕.P.1. Геометрические параметры (радиусы и высоты)**

Без чисел, только структурно (числа калибруются по сейсмике и геодезии).

Обозначения радиусов:

* (R_\oplus) — радиус Земли (поверхность, (x=1)).
* (R_{\rm lith}) — эффективный радиус основания литосферы (верхняя мантия).
* (R_{\rm PZ}) — радиус начала Переходной Зоны.
* (R_{\rm core,out}) — внешний радиус ядра (граница ядро–мантия).
* (R_{\rm core,mid}) — раздел между средним и внешним тором.
* (R_{\rm core,in}) — внутренний тор (радиус центрального ядра).

В безразмерных координатах (x=r/R_\oplus):

[
x_{\rm core,in},
\quad
x_{\rm core,mid},
\quad
x_{\rm core,out},
\quad
x_{\rm PZ},
\quad
x_{\rm lith},
\quad
x=1
]

— **якорные точки** Θ-структуры по глубине.

Для атмосферы задаются высоты:

* (z_{\rm trop}) — тропопауза (нижняя граница основного резонатора ячеек).
* (z_{\rm strat}) — характеристическая высота стратосферных мод.
* (z_{\rm top}) — верхняя эффективная граница атмосферного Θ-резонатора.

---

## **⊕.P.2. Базовые частоты ядра (три тора)**

Ядро — трёхторовый резонатор Θ с тремя фундаментальными частотами:

[
\omega_{\rm in},\quad \omega_{\rm mid},\quad \omega_{\rm out},
]

где:

* (\omega_{\rm in}) — мода внутреннего тора (самая «жёсткая» по фазе).
* (\omega_{\rm mid}) — мода среднего тора (связывает внутренний и внешний).
* (\omega_{\rm out}) — мода внешнего тора (максимально связана с ПЗ и оболочками).

**Нормировка:**

[
\omega_{\rm in} > \omega_{\rm mid} > \omega_{\rm out}.
]

Вводим безразмерные отношения:

[
\rho_1 = \frac{\omega_{\rm mid}}{\omega_{\rm in}},
\quad
\rho_2 = \frac{\omega_{\rm out}}{\omega_{\rm mid}},
\quad
\rho_3 = \frac{\omega_{\rm out}}{\omega_{\rm in}}.
]

Эти три числа (\rho_1,\rho_2,\rho_3) полностью задают **шкалу внутренних Θ-мод ядра** (после выбора общей шкалы частот, см. ниже).

---

## **⊕.P.3. Глобальная W-мода Земли и её параметры**

Глобальная W-мода Земли (\Theta_\oplus) описывается структурой:

[
\Theta_\oplus(t,\mathbf{r})
===========================

\Re\left{
\sum_{i=1}^3
A_i(\mathbf{r}),e^{i\omega_i t}
\right},
\quad
\omega_i\in{\omega_{\rm in},\omega_{\rm mid},\omega_{\rm out}}.
]

Вводится **основная частота ветви**:

[
\omega_0 \equiv \omega_{\rm out}
]

(частота внешнего тора как наиболее связанная с оболочками).

Параметры W-моды (в духе общей W-теории):

* (f_\varphi) — характеристическая «угловая» частота по широте (связана с (n_\varphi)).
* (\beta_4, \gamma_6) — коэффициенты нелинейных поправок 4-го и 6-го порядка для Θ-поля (см. общую W-главу).
* (\sigma_{\rm thr}) — базовый порог Θ-интенсивности для появления устойчивых доменов (атмосферные и коровые структуры).
* (Q_\oplus) — эффективная добротность глобальной W-моды.
* (\tau_0) — характерное время релаксации Θ-поля после крупного фазового перехода.

Реалистично:

* (\omega_0) и (Q_\oplus) калибруются по **спектру свободных колебаний Земли** и долговременным колебаниям (см. free oscillations).
* (\sigma_{\rm thr}) — по порогам циклогенеза и сейсмики.
* (\beta_4,\gamma_6) — по наблюдаемой насыщенности и нелинейности (например, ограничению роста амплитуд ячеек/магнитного поля).

---

## **⊕.P.4. Ветвь (n(x,\varphi)) и разбиение Земли по модам**

Фазовая функция:

[
S(x,\varphi) = \int_{x_{\rm core,in}}^{x} k_{\rm eff}(x',\varphi),dx',
]

ветвь (n) определяется как:

[
\boxed{
n(x,\varphi) = \frac{1}{\pi} S(x,\varphi)
=========================================

\frac{1}{\pi}
\int_{x_{\rm core,in}}^{x} k_{\rm eff}(x',\varphi),dx'.
}
]

* (k_{\rm eff}(x,\varphi)) — эффективный волновой индекс Θ-среды вдоль луча (ядро → ПЗ → кора → атмосфера).
* при чистой сферической симметрии (k_{\rm eff}\to K(x)), и (n) зависит только от x.

**Радиальная часть:**

[
n_r(x) =
\frac{R_\oplus}{\pi}
\int_{x_{\rm core,in}}^{x} K(x'),dx'.
]

**Широтная поправка:**

[
n(x,\varphi) = n_r(x) + \delta n_\varphi(x,\varphi),
\quad
|\delta n_\varphi| \ll n_r.
]

---

### **⊕.P.4.1. Диапазоны (n_r(x)) по глубине**

Определяем:

[
n_{\rm core,in} = n_r(x_{\rm core,in}),\quad
n_{\rm core,mid} = n_r(x_{\rm core,mid}),
]
[
n_{\rm core,out} = n_r(x_{\rm core,out}),\quad
n_{\rm PZ} = n_r(x_{\rm PZ}),\quad
n_{\rm lith} = n_r(x_{\rm lith}),\quad
n_{\rm surf} = n_r(1).
]

Таблица:

| Диапазон (n_r(x))                          | Θ-слой                      | Физический слой         |
| ------------------------------------------ | --------------------------- | ----------------------- |
| (0 \le n < n_{\rm core,in})                | Внутренний тор (T_{\rm in}) | Внутреннее ядро         |
| (n_{\rm core,in} \le n < n_{\rm core,mid}) | Средний тор (T_{\rm mid})   | Внутреннее/внешнее ядро |
| (n_{\rm core,mid} \le n < n_{\rm PZ})      | Внешний тор (T_{\rm out})   | Внешнее ядро            |
| (n_{\rm PZ} \le n < n_{\rm lith})          | Переходная зона (фильтр n)  | ПЗ / мантия             |
| (n_{\rm lith} \le n < n_{\rm surf})        | Узловая оболочка Θ          | Кора / литосфера        |
| (n \ge n_{\rm surf})                       | Открытый Θ-резонатор        | Атмосфера / ионосфера   |

---

### **⊕.P.4.2. (n_{\rm surf}(\varphi)) и ячейки атмосферы**

На поверхности:

[
n_{\rm surf}(\varphi)
=====================

\frac{1}{\pi}
\int_{x_{\rm core,in}}^{1}
k_{\rm eff}(x',\varphi),dx'.
]

В первом приближении:

* **ядро + ПЗ** задают основной уровень (n_r(1)),
* **трёхторовость и F_\oplus** дают (\delta n_\varphi(\varphi)).

Качественная идентификация:

| Широта (\varphi) | (n_{\rm surf}(\varphi)) (эфф.) | Θ-структура       | Атмосфера         |                         |                         |
| ---------------- | ------------------------------ | ----------------- | ----------------- | ----------------------- | ----------------------- |
| (                | \varphi                        | \lesssim 10°–15°) | ≈ 5               | Антиузел Θ              | Экватор / Hadley внутр. |
| (15°–30°)        | 5→6                            | Узел Θ            | Гр. Hadley–Ferrel |                         |                         |
| (30°–60°)        | ≈ 6                            | Домен Θ           | Ячейка Ferrel     |                         |                         |
| (60°–75°)        | 6→7                            | Узел Θ            | Гр. Ferrel–Polar  |                         |                         |
| (                | \varphi                        | \gtrsim 75°)      | ≈7+               | Антиузел/сложные домены | Полярная область        |

Таким образом, **ячейки Hadley/Ferrel/Polar — это три последовательных диапазона одной и той же ветви (n_{\rm surf}(\varphi))**.

---

## **⊕.P.5. Внешний драйвер (F_\oplus)**

(F_\oplus) — совокупное влияние небесных тел на фазовую динамику Θ-мод Земли.

Разложение:

[
F_\oplus(t) = F_{\odot}(t) + F_{\rm Luna}(t) + F_{\rm Jup}(t) + \dots
]

Каждый вклад действует **на фазовые скорости** Θ-мод:

[
\omega_i \to \omega_i + \delta\omega_i(F_\oplus),
]

и, следовательно, на:

* разности (\Delta\omega) → **биения**,
* положение узлов Θ во времени,
* окна усиления интерференции.

Основные частоты:

* (\Omega_{\rm day}) — суточная;
* (\Omega_{\rm year}) — годовая;
* (\Omega_{\rm syn}) — синодические циклы Луны;
* (\Omega_{\rm nod}) — цикл лунных узлов ((\sim 18.6) лет в классическом описании);
* (\Omega_{\rm Jup}) — цикл Юпитера.

В паспорте Земли они фиксируются как **внешние модулирующие частоты**, действующие на:

* ядро → магнитное поле,
* ПЗ → глубокую сейсмику,
* кору → окна сейсмической активности,
* атмосферу → сезонность, ENSO/QBO-режимы.

---

## **⊕.P.6. Параметры биений и пороговые функции**

Для каждой пары соседних мод ((n,n+1)) задаются:

* (\Delta\omega_{n} = \omega_{n+1}-\omega_n);
* относительные амплитуды (A_n, A_{n+1});
* фазовая разность (\Delta\Phi(x,\varphi,t)).

Определяется **интенсивность биений**:

[
\mathcal{E}*{n,n+1}(x,\varphi,t)
\sim
A_n^2 + A*{n+1}^2 + 2A_nA_{n+1}\cos\Delta\Phi.
]

Для каждой подсистемы:

* **атмосфера**: порог (\mathcal{E}_{\rm thr}^{\rm atm}(n,x\approx 1,\varphi)) → создание/разрушение ячеек и циклонов;
* **кора/ПЗ**: порог (\mathcal{E}_{\rm thr}^{\rm seis}(n,x,\varphi)) → фазовые переходы (землетрясения);
* **ядро**: (\mathcal{E}*{\rm thr}^{\rm core}(n,x\sim x*{\rm core})) → перестройки магнитного поля и возможные инверсии.

В паспорте Земли эти пороговые функции фиксируются **символически**; численно они подлежат калибровке по данным.

---

## **⊕.P.7. Добротность и времена релаксации**

Для каждого уровня вводится эффективная добротность (Q) и время релаксации (\tau):

* (Q_{\rm core}, \tau_{\rm core}) — для торовых Θ-мод ядра;
* (Q_{\rm PZ}, \tau_{\rm PZ}) — для мод, проходящих через ПЗ;
* (Q_{\rm lith}, \tau_{\rm lith}) — для долговременных Θ-доменов коры (тектоника);
* (Q_{\rm atm}, \tau_{\rm atm}) — для атмосферных мод (ячейки, крупные режимы типа ENSO);
* (Q_B, \tau_B) — для магнитных мод (инверсии, секулярный дрейф).

Глобальные параметры:

* (Q_\oplus) — добротность основной W-моды;
* (\tau_0) — характерное время релаксации после крупного фазового события (мегасейсмика, крупная перестройка циркуляции, возможная магнитная инверсия).

## **⊕.P.9. Итог**

«Паспорт Земли» в этой формулировке:

* задаёт **полный набор параметров Θ-модели** без обращения к массе/давлению,
* связывает **все слои и явления** (ядро, ПЗ, кора, атмосфера, магнитное поле) через ветвь (n(x,\varphi)) и биения мод,
* служит **основой для калибровки** по данным и построения **единых прогнозных схем** (сейсмо+метео+магн.+тектоника).

---

Отлично. Тогда переходим к следующему фундаментальному блоку:
**строгая формализация критериев событий**
(сейсмика, циклогенез, перестройки магнитного поля, дрейф плит)
исключительно в терминах **Θ-поля, ветви (n(x,\varphi))** и **биений мод (\mathcal{E}_{n,n+1})**.

Это один из ключевых разделов всей теории, потому что он превращает пассивное описание структуры в **динамическую модель**, пригодную для прогнозов.

# **⊕.C. КРИТЕРИИ СОБЫТИЙ В Θ-МОДЕЛИ ЗЕМЛИ**

*(сейсмика, циклогенез, дрейф доменов и магнитные перестройки как следствия фазовой динамики Θ)*

Все явления Земли в этой модели представляют собой **точки фазовой несовместимости Θ-поля** на определённых глубинах и широтах.
Формально это означает следующее:

1. На Земле всегда одновременно присутствуют по крайней мере две соседние радиально-широтные моды Θ:
   (\Theta_n) и (\Theta_{n+1}).
2. Их суперпозиция формирует **биения** с амплитудой
   [
   \mathcal{E}_{n,n+1}(x,\varphi,t)
   ================================

   A_n^2 + A_{n+1}^2 + 2A_nA_{n+1}\cos\Delta\Phi(x,\varphi,t).
   ]
3. Каждая система (ядро, ПЗ, кора, атмосфера) имеет свой **порог фазовой совместимости** (\mathcal{E}_{\rm thr}).
4. Любое «событие» — это **локальный фазовый переход Θ**, возникающий когда:

[
\boxed{
\mathcal{E}*{n,n+1}(x,\varphi,t)
;\ge;
\mathcal{E}*{\rm thr}^{(\text{subsystem})}(x,\varphi)
}
]

Именно это уравнение является **универсальным критерием для всех четырёх классов явлений Земли.**

Далее — подробно по каждому из них.

# **⊕.C.1. Критерий СЕЙСМИЧЕСКОГО СОБЫТИЯ (землетрясения)**

### **1. Волновая постановка**

Берём глубины:

[
x \in [x_{\rm lith}, x_{\rm PZ}],
]

где проходят крупные Θ-моды, и где биения (\Theta_n + \Theta_{n+1}) наиболее изменчивы.

Интенсивность биений:

[
\mathcal{E}_{n,n+1}(x,\varphi,t)
================================

A_n^2 + A_{n+1}^2 + 2 A_nA_{n+1}\cos\Delta\Phi.
]

Определяем **градиент фазовой несовместимости**:

[
\Gamma(x,\varphi,t)
===================

\left|\frac{\partial}{\partial x}\Delta\Phi(x,\varphi,t)\right|.
]

Если две соседние моды Θ расходятся в фазе слишком быстро, система не может поддерживать локально общую ветвь n — требуется **фазовый скачок**.

### **2. Универсальное условие землетрясения**

[
\boxed{
\mathcal{E}*{n,n+1}(x,\varphi,t)
;\Gamma(x,\varphi,t)
;\ge;
\mathcal{E}*{\rm thr}^{\rm seis}(x,\varphi)
}
]

где:

* (\mathcal{E}_{n,n+1}) — амплитуда интерференции,
* (\Gamma) — мера расхождения фаз между модами,
* порог зависит от структуры коры (узлы Θ, разломы, домены).

### **3. Физическая интерпретация**

Землетрясение = **фазовый переход Θ**, при котором система перестраивается с комбинации (n,n+1) на новую устойчивую комбинацию.

* Никаких «разрывов», «скоплений напряжений».
* Только **топология Θ**.

# **⊕.C.2. Критерий РОЖДЕНИЯ ЦИКЛОНА**

Рабочий слой:

[
x \approx 1,\quad z \in [0, z_{\rm trop}],
]

где Θ-моды формируют вертикальный и широтный резонатор атмосферы.

### **1. Вертикальная структура**

Вводим вертикальную фазу:

[
S_z(z) = \int k_z(z')dz'.
]

Циклон возникает там, где **радиальная и вертикальная мода** становятся локально фазово совместимы:

[
\Delta\Phi_{(n,n+1)}(z,\varphi,t)
\approx
\left(k_z z + S_{\rm surf}(\varphi)\right)
------------------------------------------

\left((k_z+\Delta k_z)z + S_{\rm surf}(\varphi)+\Delta S\right).
]

### **2. Критерий локального Θ-замыкания**

Циклон = **замкнутый Θ-домен**, аналог тороидального вихря.

Такое замыкание возможно если:

[
\boxed{
\mathcal{E}*{n,n+1}^{\rm atm}(x\approx 1,\varphi,z,t)
\ge
\mathcal{E}*{\rm thr}^{\rm atm}(\varphi,z)
}
]

где (\mathcal{E}_{\rm thr}^{\rm atm}) зависит от:

* широты (близость к узлам ветви n),
* высоты (эффективной вертикальной моды),
* сезона (через F⊕ → δK → изменение k_eff).

### **3. Следствия**

* В тропиках порог ниже → зарождение тропических циклонов.
* В субтропиках порог выше → циклogenesis suppressed.
* В средних широтах структура Ferrel-моды → бароклинные циклоны.
* На полюсах → полярные вихри.

# **⊕.C.3. Критерий ПЕРЕСТРОЙКИ МАГНИТНОГО ПОЛЯ (включая инверсии)**

В ядре:

* топологический индекс фазы Θ определяет знак диполя,
* перестройка фазовых токов происходит, когда биения двух торов становятся доминирующими.

### **1. Фазовый поток Θ в ядре**

[
\mathbf{v}*\Theta = \nabla \phi,
\qquad
\mathbf{B} = \nabla\times\mathbf{v}*\Theta.
]

Топологический индекс:

[
N = \frac{1}{2\pi}\oint \nabla\phi\cdot d\mathbf{l}.
]

### **2. Когда возникает перестройка?**

Когда фазовое давление биений превосходит удерживающую силу основной моды:

[
\boxed{
\mathcal{E}*{n,n+1}^{\rm core}(x\sim x*{\rm core})
;\ge;
\mathcal{E}_{\rm thr}^{\rm core}
}
]

Если при этом

[
\operatorname{sgn}(N_{t+dt}) \ne \operatorname{sgn}(N_t),
]

происходит **инверсия магнитного поля**.

### **3. Связь с F⊕**

Биения усиливаются в периоды:

* совпадения частот (\omega_n) и (\omega_{n+1}) после модуляции F⊕,
* многолетних конфигураций планет.

# **⊕.C.4. Критерий ДРЕЙФА ТЕКТОНИЧЕСКИХ ДОМЕНОВ (плит)**

Плита = **область фиксированной фазы Θ**, т.е. фрагмент узловой структуры ветви n.

### **1. Узел Θ определяется условием**

[
\Theta(x,\varphi,\lambda) = 0.
]

При постепенном изменении n(x,φ) из-за внешнего драйвера F⊕ или долговременных биений:

[
\frac{\partial n}{\partial t} \ne 0.
]

Узел движется.

### **2. Скорость дрейфа плиты**

Вводим фазу Θ:

[
\phi(x,\varphi,\lambda,t) = \pi n(x,\varphi,t) + \phi_0.
]

Условие узла:

[
\phi = (k+1/2)\pi.
]

Дифференцируем по времени:

[
\dot\phi = \pi \dot n(x,\varphi,t).
]

Тогда движение узла определяется:

[
v_{\rm drift}
=============

* \frac{\dot\phi}{\partial\phi/\partial s}
  =
* \frac{\pi\dot n}{\partial\phi/\partial s},
  ]

где (s) — координата вдоль поверхности.

### **3. Универсальное условие дрейфа плиты**

[
\boxed{
\left|\dot n(x_{\rm lith},\varphi,t)\right|
;\ge;
n_{\rm thr}^{\rm drift}
}
]

Если изменение ветви n быстрее порогового — плита начинает дрейфовать.

# **⊕.C.5. Объединённая схема критериев**

Каждое явление Земли — это **один и тот же процесс**, но в разных слоях:

| Явление                          | Глубина/слой | Универсальный критерий                                                |        |                              |
| -------------------------------- | ------------ | --------------------------------------------------------------------- | ------ | ---------------------------- |
| Землетрясение                    | Кора / ПЗ    | (\mathcal{E}*{n,n+1}\Gamma \ge \mathcal{E}*{\rm thr}^{\rm seis})      |        |                              |
| Циклоногенез                     | Атмосфера    | (\mathcal{E}*{n,n+1}^{\rm atm} \ge \mathcal{E}*{\rm thr}^{\rm atm})   |        |                              |
| Дрейф плит                       | Кора         | (                                                                     | \dot n | \ge n_{\rm thr}^{\rm drift}) |
| Магнитная перестройка / инверсия | Ядро         | (\mathcal{E}*{n,n+1}^{\rm core} \ge \mathcal{E}*{\rm thr}^{\rm core}) |        |                              |
