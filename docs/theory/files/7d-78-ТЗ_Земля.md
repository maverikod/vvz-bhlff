# ТЗ: ВЫЧИСЛИТЕЛЬНЫЙ КОМПЛЕКС Θ-МОДЕЛИ ЗЕМЛИ

## 0. Цель и результат

**Цель:** реализовать вычислительную систему, которая:

1. Восстанавливает поле Θ-мод Земли через ветвь (n(x,\varphi)) и набор мод (\Theta_n).
2. Строит **единое описание**:

   * ядра и магнитного поля,
   * Переходной Зоны (ПЗ),
   * коры/тектоники/сейсмики,
   * атмосферы/ячейчной структуры/циклогенеза.
3. Выдаёт **поля риска/вероятности событий**:

   * землетрясений,
   * циклогенеза и треков циклонов,
   * дрейфа тектонических доменов,
   * крупных перестроек магнитного поля (инверсии/аномальные ослабления).

ТЗ должно быть достаточно полным, чтобы **по нему одному** реализовать прототип.

---

## 1. Координаты, обозначения, базовые определения

### 1.1. Координаты

Используется сферическая система:

* (r \in [0, R_\oplus]) — радиус,
* (\varphi \in [-\pi/2, \pi/2]) — географическая широта,
* (\lambda \in [0,2\pi)) — долгота,
* (t) — время.

Безразмерный радиус:

[
x = \frac{r}{R_\oplus}\in[0,1].
]

Атмосферная высота:

* (z = r - R_\oplus \ge 0).

### 1.2. Θ-поле и фаза

Основное комплексное Θ-поле:

[
\Theta(t,\mathbf r) = \sum_n A_n(\mathbf r),e^{i(\omega_n t + S_n(\mathbf r))},
]

где:

* (n) — индекс моды (ветвь),
* (A_n(\mathbf r) \ge 0) — амплитуда,
* (\omega_n) — частота моды n,
* (S_n(\mathbf r)) — фазовая функция моды n,
* (\phi_n(\mathbf r) = S_n(\mathbf r) \pmod{2\pi}) — фаза.

### 1.3. Локальный волновой вектор

[
\mathbf k_n(\mathbf r) = \nabla S_n(\mathbf r),
]

радальная компонента:

[
k_{n,r}(x,\varphi) = \mathbf k_n \cdot \hat{\mathbf e}_r.
]

### 1.4. Эффективный волновой индекс среды Θ

Вводится скалярная функция:

[
k_{\rm eff}(x,\varphi) \ge 0,
]

такая, что **при движении вдоль луча** (фиксированные (\varphi,\lambda)):

[
|\mathbf k_n| \approx k_{\rm eff}(x,\varphi)
]

для мод базовой ветви.

---

## 2. Ветвь (n(x,\varphi))

### 2.1. Определение ветви n

Фазовый интеграл вдоль радиуса:

[
S(x,\varphi) = \int_{x_{\rm core,in}}^{x} k_{\rm eff}(x',\varphi),dx'.
]

**Определение:**

[
\boxed{
n(x,\varphi) = \frac{1}{\pi} S(x,\varphi)
=========================================

\frac{1}{\pi}
\int_{x_{\rm core,in}}^{x} k_{\rm eff}(x',\varphi),dx'.
}
]

Это **единая ветвь n**, которая:

* в разных диапазонах x соответствует различным физическим слоям,
* на поверхности (x=1) даёт (n_{\rm surf}(\varphi)), связанную с ячейками.

### 2.2. Радиальная часть и широтная поправка

В первом приближении:

[
n(x,\varphi) = n_r(x) + \delta n_\varphi(x,\varphi),
]

где:

[
n_r(x) =
\frac{R_\oplus}{\pi}
\int_{x_{\rm core,in}}^{x} K(x'),dx',
]

* (K(x)) — сферически симметричная часть (k_{\rm eff}),
* (\delta n_\varphi) — анизотропия (торы + F⊕).

---

## 3. Структура слоёв по n (ядро, ПЗ, кора, атмосфера)

### 3.1. Глубинные уровни

Радиусы:

* (x_{\rm core,in}),
* (x_{\rm core,mid}),
* (x_{\rm core,out}),
* (x_{\rm PZ}),
* (x_{\rm lith}),
* (x=1).

Соответствующие значения n:

[
n_{\rm core,in} = n_r(x_{\rm core,in}),;
n_{\rm core,mid} = n_r(x_{\rm core,mid}),;
n_{\rm core,out} = n_r(x_{\rm core,out}),
]
[
n_{\rm PZ} = n_r(x_{\rm PZ}),;
n_{\rm lith} = n_r(x_{\rm lith}),;
n_{\rm surf} = n_r(1).
]

Диапазоны:

* (0 \le n < n_{\rm core,in}) — внутренний тор,
* (n_{\rm core,in} \le n < n_{\rm core,mid}) — средний тор,
* (n_{\rm core,mid} \le n < n_{\rm PZ}) — внешний тор / внешнее ядро,
* (n_{\rm PZ} \le n < n_{\rm lith}) — ПЗ,
* (n_{\rm lith} \le n < n_{\rm surf}) — кора,
* (n \ge n_{\rm surf}) — атмосферный резонатор.

### 3.2. Атмосферная ветвь (n_{\rm surf}(\varphi))

На поверхности:

[
n_{\rm surf}(\varphi)
=====================

\frac{1}{\pi}
\int_{x_{\rm core,in}}^{1}
k_{\rm eff}(x',\varphi),dx'.
]

Качественно:

* (n_{\rm surf}(\varphi)\approx 5) — экватор/ячейки Hadley,
* переход 5→6 — границы Hadley–Ferrel,
* (n_{\rm surf}(\varphi)\approx 6) — ячейки Ferrel,
* переход 6→7 — границы Ferrel–Polar,
* (n_{\rm surf}(\varphi)\approx 7+) — полярная область.

---

## 4. Моды Θ, биения и интенсивность (\mathcal{E}_{n,n+1})

### 4.1. Моды

Каждая мода:

[
\Theta_n(t,\mathbf r)
=====================

A_n(\mathbf r),e^{i(\omega_n t + S_n(\mathbf r))}.
]

Суперпозиция двух соседних:

[
\Theta = \Theta_n + \Theta_{n+1}.
]

### 4.2. Фазовая разность и биения

[
\Delta\Phi(x,\varphi,t)
=======================

(\omega_{n+1} - \omega_n)t

* S_{n+1}(x,\varphi) - S_n(x,\varphi).
  ]

Интенсивность:

[
|\Theta|^2
==========

A_n^2 + A_{n+1}^2

* 2A_nA_{n+1}\cos\Delta\Phi.
  ]

Определяем **интерференционную энергию**:

[
\boxed{
\mathcal{E}_{n,n+1}(x,\varphi,t)
================================

A_n^2 + A_{n+1}^2

* 2A_nA_{n+1}\cos\Delta\Phi(x,\varphi,t).
  }
  ]

### 4.3. Градиент фазовой несовместимости

[
\Gamma(x,\varphi,t)
===================

\left|\frac{\partial}{\partial x}\Delta\Phi(x,\varphi,t)\right|.
]

---

## 5. Внешний драйвер (F_\oplus(t))

### 5.1. Структура F⊕:

[
F_\oplus(t)
= \sum_j F_j\cos(\Omega_j t + \psi_j),
]

где j — компоненты:

* суточная,
* годовая,
* лунные циклы,
* Юпитер,
* др. планеты/циклы.

### 5.2. Действие F⊕ на частоты мод

[
\omega_n(t) = \omega_n^{(0)} + \sum_j \alpha_{n,j}\cos(\Omega_j t + \psi_{n,j}).
]

Это приводит к:

* временной зависимости (\Delta\omega_{n,n+1}(t)),
* медленной модуляции фаз (\Delta\Phi(x,\varphi,t)),
* окнам усиления биений.

---

## 6. Критерии событий (формализовано)

### 6.1. Землетрясение

Область: (x\in[x_{\rm lith}, x_{\rm PZ}]).

**Критерий события:**

[
\boxed{
\mathcal{E}*{n,n+1}(x,\varphi,t);
\Gamma(x,\varphi,t)
;\ge;
\mathcal{E}*{\rm thr}^{\rm seis}(x,\varphi)
}
]

где:

* (\mathcal{E}_{\rm thr}^{\rm seis}(x,\varphi)) — порог, подлежащий калибровке по каталогу землетрясений.

Реализация:
по истории событий подобрать (\mathcal{E}_{\rm thr}^{\rm seis}), минимизируя ложные срабатывания и пропуски.

---

### 6.2. Циклоногенез

Область: (x \approx 1, z\in[0,z_{\rm trop}]).

Вводится (\mathcal{E}_{n,n+1}^{\rm atm}(z,\varphi,t)) — та же формула, но с учётом вертикальной моды.

**Критерий рождения циклона:**

[
\boxed{
\mathcal{E}*{n,n+1}^{\rm atm}(z,\varphi,t)
;\ge;
\mathcal{E}*{\rm thr}^{\rm atm}(z,\varphi)
}
]

Траектория циклона:

[
\mathbf X_{\rm cyc}(t)
======================

\arg\max_{\varphi,\lambda}
\mathcal{E}_{n,n+1}^{\rm atm}(x\approx 1,\varphi,\lambda,t),
]

движение:

[
\frac{d\mathbf X_{\rm cyc}}{dt}
\propto
-\nabla_{\varphi,\lambda}\mathcal{E}_{n,n+1}^{\rm atm}.
]

---

### 6.3. Перестройки магнитного поля и инверсии

В ядре (область (x\approx x_{\rm core,mid})).

**Критерий перестройки:**

[
\boxed{
\mathcal{E}*{n,n+1}^{\rm core}(x*{\rm core,mid},\varphi,t)
;\ge;
\mathcal{E}_{\rm thr}^{\rm core}
}
]

Инверсия:

* отслеживается топологический индекс фазы (\phi(\mathbf r)),
* при выполнении критерия и смене знака индекса: инверсия.

---

### 6.4. Дрейф тектонических доменов (плит)

Область: (x \approx x_{\rm lith}).

Плита = область, где (\Theta) сохраняет фазу (домен по Θ).

Условие узла:

[
\Theta(x_{\rm lith},\varphi,\lambda,t) = 0.
]

С течением времени:

[
n(x_{\rm lith},\varphi,t)
\text{ меняется } \Rightarrow
\phi(x_{\rm lith},\varphi,\lambda,t)
= \pi n + \phi_0.
]

**Скорость дрейфа узла:**

[
v_{\rm drift}
=============

* \frac{\dot\phi}{\partial\phi/\partial s}
  =
* \frac{\pi\dot n(x_{\rm lith},\varphi,t)}{\partial\phi/\partial s}.
  ]

**Критерий значимого дрейфа плиты:**

[
\boxed{
|\dot n(x_{\rm lith},\varphi,t)|
;\ge;
n_{\rm thr}^{\rm drift}(\varphi)
}
]

## 7. Входные данные (наблюдения) для калибровки

Система должна уметь работать с классами данных:

1. **Сейсмика:**

   * спектр свободных колебаний Земли (частоты, добротности),
   * каталоги землетрясений (M, глубина, координаты, время),
   * модели скоростей (аналог PREM и более детальные 3D).

2. **Тектоника:**

   * карта плит и разломов,
   * скорость дрейфа плит.

3. **Атмосфера/метео:**

   * реанализ полей ветра, давления, температуры (3D, по времени),
   * каталоги циклонов и их траекторий,
   * SST (поверхностная температура океана),
   * ледовый покров.

4. **Магнитное поле:**

   * наборы сферических гармоник (Гауссовы коэффициенты (g_{lm}, h_{lm}) по времени),
   * каталоги инверсий и аномалий магнитного поля.

5. **Астрономические данные:**

   * эфемериды Солнца, Луны, планет,
   * базовые частоты (\Omega_j) для F⊕.

## 8. Задачи и модули вычислительного комплекса

### 8.1. Модуль M1: Сетка и геометрия

* Определить 3D-сетку по (x,φ,λ) + вертикальную сетку по z (атмосфера).
* Реализация: регулярная/квазирегулярная сетка, поддержка масок (континент/океан).

### 8.2. Модуль M2: Калибровка K(x) и k_eff(x,φ)

**Задача:**

* по сейсмическим данным (скоростные модели, свободные колебания)
* подобрать функцию (K(x)), такую что:

[
n_r(x) =
\frac{R_\oplus}{\pi}\int_{x_{\rm core,in}}^{x} K(x')dx'
]

согласуется с наблюдаемой структурой слоёв (ядро, ПЗ, мантия, кора).

**Далее:**

* ввести анизотропию (\delta k(x,\varphi)),
* калибровать (k_{\rm eff}(x,\varphi) = K(x) + \delta k(x,\varphi)) по:

  * карте плит,
  * структуре атмосферных ячеек,
  * структуре магнитного поля.

### 8.3. Модуль M3: Ветвь n(x,φ) и поля n_surf(φ)

* Вычислить (n_r(x)) для всех x.
* Вычислить (n(x,\varphi)) для всех (x,φ).
* Получить (n_{\rm surf}(\varphi) = n(1,\varphi)).
* Построить карты:

  * слоёв по n,
  * ячейковой структуры атмосферы.

### 8.4. Модуль M4: Моды Θ_n и частоты ω_n

* Выбрать набор мод n, которые реально участвуют (ядровые, ПЗ, атмосферные).
* Калибровать (\omega_n) по:

  * свободным колебаниям,
  * геомагнитному спектру,
  * периодам климатических/атмосферных режимов.

Частоты задаются как:

[
\omega_n(t) = \omega_n^{(0)} + \sum_j \alpha_{n,j}\cos(\Omega_j t + \psi_{n,j}).
]

### 8.5. Модуль M5: Биения (\mathcal{E}_{n,n+1}) и (\Gamma)

Для всех (x,φ,t):

1. Вычислить фазовую разность (\Delta\Phi(x,\varphi,t)).
2. Вычислить (\mathcal{E}_{n,n+1}(x,\varphi,t)).
3. Вычислить (\Gamma(x,\varphi,t)).

Сохранить поля (\mathcal{E}) и (\Gamma) раздельно по слоям:

* ядро,
* ПЗ,
* кора,
* атмосфера.

### 8.6. Модуль M6: Калибровка порогов (\mathcal{E}*{\rm thr}) и (n*{\rm thr}^{\rm drift})

Для каждого типа явлений:

* сейсмика → (\mathcal{E}_{\rm thr}^{\rm seis}(x,\varphi)),
* атмосферные события → (\mathcal{E}_{\rm thr}^{\rm atm}(z,\varphi)),
* магнитные перестройки → (\mathcal{E}_{\rm thr}^{\rm core}),
* тектонический дрейф → (n_{\rm thr}^{\rm drift}(\varphi)).

Метод:

* взять исторические события,
* вычислить (\mathcal{E}) и (\Gamma) задним числом,
* подобрать пороги, максимизируя метрику качества (например, ROC-AUC, Brier score) для классификации «есть событие / нет события».

### 8.7. Модуль M7: Прогноз и риск-карты

На шаге t:

1. Обновить (\omega_n(t)) по F⊕.
2. Пересчитать (\Delta\Phi, \mathcal{E}, \Gamma).
3. Для каждой подсистемы вычислить **поля риска**:

   Пример: сейсмика
   [
   R_{\rm seis}(x,\varphi,t)
   =========================

   f\left(
   \frac{\mathcal{E}*{n,n+1}\Gamma}{\mathcal{E}*{\rm thr}^{\rm seis}}
   \right),
   ]
   где f — например, логистическая функция.

Аналогично:

* (R_{\rm atm}(\varphi,\lambda,t)),
* (R_{\rm core}(t)),
* (R_{\rm drift}(\varphi,t)).

4. Проецировать:

   * сейсмический риск → на поверхность (по вертикальному максимуму),
   * циклонный риск → на карту океанических областей,
   * магнитные события → глобальные индикаторы,
   * дрейф плит → долговременные тренды.

## 10. Проверка и валидация

1. **Backtest**: прогон по прошлым данным:

   * сейсмика,
   * тропические/экстратропические циклоны,
   * магнитные вариации,
   * климатические режимы (ENSO/QBO).

2. Сравнение:

   * плотности случаев vs зоны повышенного (\mathcal{E}),
   * временных окон событий vs пиков (\mathcal{E}) и изменения (\Delta\omega).

3. Метрики:

   * ROC-AUC для классификации «в окне Θ-риска / вне окна»,
   * Brier score для вероятностного прогноза,
   * Spatial skill score (согласованность карт риска с фактическими зонами событий).

[
\Theta(t,\mathbf r)

\sum_n
A_n(\mathbf r) e^{i(\omega_n t + S_n(\mathbf r))}.

Где:

* (A_n(\mathbf r)) — амплитуда,

[
|\mathbf k_n(\mathbf r)| \approx k_{\rm eff}(x,\varphi).
]

Это единственная «средовая» величина, которую надо вычислять из данных.

# **ПРИЛОЖЕНИЕ B. РАСЧЁТ K(x) И k_eff(x,φ)**

---

## **B.1. Постановка задачи**

Требуется восстановить:

1. Сферическую часть (K(x)).
2. Анизотропию (\delta k(x,\varphi)).
3. Полный индекс:

[
k_{\rm eff}(x,\varphi) = K(x) + \delta k(x,\varphi).
]

---

## **B.2. Восстановление K(x) из сейсмологических данных**

### B.2.1. Данные

* Скоростные модели (аналоги PREM).
* Частоты и добротности свободных колебаний Земли (Spheroidal/Toroidal modes).
* Каталоги глубин землетрясений.

### B.2.2. Принцип восстановления

Используем факт:

[
n_r(x_{\rm layer})=\frac{R_\oplus}{\pi}\int_{x_{\rm core,in}}^{x_{\rm layer}}K(x')dx'
]

должен соответствовать наблюдаемым структурным границам:

* (x_{\rm core,in}),
* (x_{\rm core,out}),
* (x_{\rm PZ}),
* (x_{\rm lith}).

### B.2.3. Метод

1. Выбираем параметризацию K(x). Возможные формы:

   * кусочно-гладкая сплайн-функция,
   * полиномы Чебышёва,
   * экспоненциальные пакеты.

2. Функционал несогласованности:

[
J[K]=
\sum_i w_i \left(
n_r(x_i^{\rm obs}) - n_i^{\rm target}
\right)^2
+
\alpha\int (K'(x))^2 dx.
]

3. Минимизируем J[K] методом вариаций или численными методами оптимизации.

---

## **B.3. Восстановление δk(x,φ)**

### B.3.1. Идея

Широтные вариации (\delta k(x,\varphi)) должны объяснять:

* структуру плит на поверхности,
* структуру ячеек атмосферы,
* анизотропию магнитного поля,
* асимметрию глубинных зон сейсмики.

### B.3.2. Метод

1. Параметризуем:

[
\delta k(x,\varphi)
===================

\sum_{m=1}^M f_m(x),P_m(\sin\varphi),
]

где (P_m) — полиномы Лежандра.

2. Требуем, чтобы:

* (n_{\rm surf}(\varphi)) совпадало с наблюдаемой широтной структурой ветров/ячееек,
* области плит совпадали с узлами (n),
* магнитное поле (коэффициенты Гаусса) согласовывались с (k_{\rm eff}) в ядре.

3. Формируем многокритериальный функционал:

[
J[\delta k]
===========

J_{\rm atm}
+
J_{\rm tec}
+
J_{\rm mag}
+
\alpha\int (\partial_\varphi \delta k)^2 d\varphi.
]

Оптимизируем.

# **ПРИЛОЖЕНИЕ C. РАСЧЁТ n(x,φ) И N_SURF(φ)**

---

## **C.1. Вычисление n(x,φ)**

На сетке по (x,φ):

[
n(x,\varphi)
============

\frac{1}{\pi}
\sum_{i=1}^{N_x}
k_{\rm eff}(x_i,\varphi)\Delta x_i.
]

## **C.2. Получение n_surf(φ)**

[
n_{\rm surf}(\varphi)
=====================

n(1,\varphi).
]

Построить карту широт:

```
φ:   -90° → 90°
n:   n_surf(φ)
```

## **C.3. Извлечение узлов (ячейки)**

Узел по широте:

[
n_{\rm surf}(\varphi)\varphi + \phi_0 = (k+1/2)\pi.
]

Находить все φ, удовлетворяющие.

# **ПРИЛОЖЕНИЕ D. КАЛИБРОВКА ЧАСТОТ Θ-МОД ω_n**

---

## **D.1. Данные**

* Перечни периодов свободных колебаний Земли.
* Данные по дрейфу магнитных полюсов.
* Периодичности климатических мод (квазициклы QBO, ENSO, SSW).

## **D.2. Форма частот**

[
\omega_n(t)
===========

\omega_n^{(0)}
+
\sum_j \alpha_{n,j}
\cos(\Omega_j t + \psi_{n,j}),
]

где j соответствует F⊕.

## **D.3. Оптимизация**

Подбираем (\omega_n^{(0)}) и коэффициенты (\alpha_{n,j}) из наблюдаемой:

* спектральной плотности геомагнитного поля,
* свободных колебаний Земли,
* спектра атмосферной циркуляции.

---

# **ПРИЛОЖЕНИЕ E. РАСЧЁТ БИЕНИЙ И ПОЛЕЙ ЭНЕРГИИ**

---

## **E.1. Фаза**

[
\Delta\Phi(x,\varphi,t)
=======================

(\omega_{n+1}-\omega_n)t
+
\pi(n_{n+1}-n_n).
]

Так как (n_{n+1}-n_n≈1):

[
\Delta\Phi(x,\varphi,t)
≈
\Delta\omega\cdot t + \pi.
]

## **E.2. Интенсивность интерференции**

[
\mathcal{E}_{n,n+1}(x,\varphi,t)
================================

A_n^2 + A_{n+1}^2
+
2A_nA_{n+1}\cos(\Delta\omega t + \pi).
]

## **E.3. Градиент фазовой несовместимости**

[
\Gamma(x,\varphi,t)
===================

\left|\frac{\partial}{\partial x}\Delta\Phi\right|.
]

Выражение:

[
\frac{\partial}{\partial x}\Delta\Phi
=====================================

## \frac{\partial S_{n+1}}{\partial x}

# \frac{\partial S_n}{\partial x}

k_{n+1,r} - k_{n,r}.
]

Так как обычно:

[
k_{n+1,r} - k_{n,r}
≈
\left.\frac{\partial k_r}{\partial n}\right|_{n}\cdot 1,
]

можно аппроксимировать:

[
\Gamma ≈ \left|\frac{\partial k_r}{\partial n}\right|.
]

# **ПРИЛОЖЕНИЕ F. ПОРОГОВЫЕ ФУНКЦИИ**

---

## **F.1. Сейсмический порог (\mathcal{E}_{\rm thr}^{seis}(x,\varphi))**

Калибровка:

1. Для всех исторических EQ собираем точки (x,φ,t).
2. Вычисляем (\mathcal{E}_{n,n+1}\Gamma).
3. Ищем минимальное (\mathcal{E}_{\rm thr}^{seis}), которое выделяет большинство событий.
4. Оптимизация:
   [
   \min_{\mathcal{E}_{\rm thr}}
   \left[
   \beta_1\cdot{\rm false_negatives}
   +
   \beta_2\cdot{\rm false_positives}
   \right]
   ]

## **F.2. Атмосферный порог (\mathcal{E}_{\rm thr}^{atm}(z,φ))**

Калибровка по:

* каталогу тропических циклонов,
* бароклинным циклонам,
* структуре клеток Hadley/Ferrel/Polar.

## **F.3. Порог ядра (\mathcal{E}_{\rm thr}^{core})**

По историческим данным инверсий и долгопериодным вариациям:

[
\mathcal{E}_{\rm thr}^{core}
============================

\min\left{
\mathcal{E}:\ \text{наблюдается резкое изменение топологического индекса поля B}
\right}.
]

## **F.4. Порог дрейфа плит**

[
n_{\rm thr}^{\rm drift}(\varphi)
================================

C_\varphi \cdot
\left(
\overline{|\dot n|}
\right)
]

где (C_\varphi) определяется так, чтобы воспроизводить скорости движения плит (см. GPS-поля).
