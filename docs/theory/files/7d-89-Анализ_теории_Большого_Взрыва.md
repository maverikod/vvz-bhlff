## ✔ 2.1. Если мы измеряем ускорение расширения, то мы измеряем:

[
\boxed{;\omega_{\rm edge}^2 \equiv \Lambda = 3,\Omega_\Lambda,H_0^2;}
]

то есть **граничная частота ячейки равна корню из эффективной Λ**.

Это красиво стыкуется с тем, что Λ имеет размерность 1/L², а ( \omega_{\rm edge} ) — 1/T, если работать в единицах c=1 (L≈T).

Тогда:

[
R_{\rm cell} = \frac{\pi}{\omega_{\rm edge}}
= \frac{\pi}{\sqrt{3,\Omega_\Lambda},H_0}.
]

Это уже **чистая рабочая формула**: из (H₀, Ω_Λ) → R_cell.

---

## 2. Динамика: как сюда входит w (параметр состояния)

В обычной FRW-космологии для тёмной энергии:

[
\rho_{\rm de}(a) \propto a^{-3(1+w)},
]

если w ≈ const в интересующем диапазоне z. ([arXiv][2])

В Θ-модели мы считаем, что:

* **ρ_de — это просто эффективная плотность энергии, связанная с краем ячейки**,
* а край описывается параметром ( \omega_{\rm edge}(t) ).

Минимальная (и естественная) гипотеза:

[
\rho_{\rm de} \propto \omega_{\rm edge}^2.
]

(Ровно так же, как в теории для частиц энергия масштабируется с частотой базовой моды — см. разделы про частотный диапазон и базовую частоту Θ. )

Тогда:

[
\boxed{;\rho_{\rm de}(t) = \kappa,\omega_{\rm edge}^2(t);}
]

для некоторой универсальной константы κ (одна и та же для всех ячеек данного уровня).

С другой стороны, по FRW:

[
\rho_{\rm de}(a) \propto a^{-3(1+w)}.
]

Отсюда:

[
\omega_{\rm edge}^2(a) \propto a^{-3(1+w)},
]

или

[
\boxed{;\omega_{\rm edge}(a) \propto a^{-\tfrac{3}{2}(1+w)};}
]

Это **ключевое равенство**:

* при w = −1 → ω_edge ≈ const → настоящий de Sitter;
* при w < −1 → ω_edge **растёт** с масштабным фактором (фантом);
* при w > −1 → ω_edge **падает** (ослабление тёмной энергии).

Современные данные дают w ≈ −1 (с намёком на w < −1). ([arXiv][3])

Значит:

* в реальности мы *очень близко* к режиму **почти постоянной ω_edge**,
* но лёгкий наклон (если w < −1) означает, что **ячейка находится в фазе усиленного роста**.

---

## 3. Рабочие формулы для (ω_edge, R_cell) через (H₀, Ω_Λ, w)

### 3.1. Сегодняшние значения

Сегодня (a=1):

1. Из наблюдаемой Λ:

[
\omega_{\rm edge,0} = \sqrt{3,\Omega_{\Lambda,0}},H_0.
]

2. Из теоретической связки R = π/ω:

[
R_{\rm cell,0} = \frac{\pi}{\omega_{\rm edge,0}}
= \frac{\pi}{\sqrt{3,\Omega_{\Lambda,0}},H_0}.
]

Подставляя Planck-2018 (TT,TE,EE+lowE+lensing+BAO):

* (H_0 \approx 67.8\ \text{км/с/Мпк}),
* (\Omega_\Lambda \approx 0.69). ([Википедия][1])

получаем (в единицах c=1) радиус того же порядка, что и стандартный «радиус видимой Вселенной» ~ десятки Gly.
(Разница в численных коэффициентах π, √3 и т.п. как раз и будет калиброваться при точной подгонке под CMB+BAO — это уже задача отдельного численного блока.)

### 3.2. Эволюция с масштабным фактором

Из (\omega_{\rm edge}(a) \propto a^{-\tfrac{3}{2}(1+w)}):

[
\frac{\dot\omega_{\rm edge}}{\omega_{\rm edge}}
= -\frac{3}{2}(1+w),\frac{\dot a}{a}
= -\frac{3}{2}(1+w),H.
]

Отсюда:

[
\boxed{;\left|\frac{\dot\omega_{\rm edge}}{\omega_{\rm edge}}\right|
= \frac{3}{2}|1+w|,H;}
]

И вот здесь появляется твоя идея:

> «По динамике тёмной энергии судить о частоте.»

Да, именно так:

* измеряя отклонение w от −1,
* мы **напрямую** измеряем относительную скорость изменения ω_edge,
* а через ω_edge и Λ — получаем **скорость динамики края ячейки**.

## 4. Связь с H₀: как свести всё в одну формулу

Если использовать твою интерпретацию:

[
H \approx \left|\frac{\dot\omega_{\rm edge}}{\omega_{\rm edge}}\right|
]

(для текущей эпохи), мы получим:

[
H_0 \approx \frac{3}{2}|1+w|,H_0.
]

Из чего следует, что при точном w = −1 **эта идентификация нестрогая** (обе стороны → 0).
Поэтому правильнее считать, что:

* H₀ задаётся *суммарной* динамикой материи + тёмной энергии,
* а тёмная энергия даёт **добавку** к H через своё w(z).

В рабочей модели:

* мы **фиксируем ω_edge,0 через Λ** (как выше),
* а динамику ( \dot\omega_{\rm edge} ) берём из w(z).

То есть **формально**:

[
\omega_{\rm edge,0} = \sqrt{3,\Omega_{\Lambda,0}},H_0,
]

а

[
\dot\omega_{\rm edge,0}
= -\frac{3}{2}(1+w_0),H_0,\omega_{\rm edge,0}.
]

И тогда:

* **по (H₀, Ω_Λ)** → абсолютное значение ω_edge,0 и R_cell,0;
* **по w₀** → знак и скорость её изменения.

## 5. Родительский резонатор: как из этого вытаскивать следующий уровень

В теории у тебя уже есть общая формула «квантования масштабов»:

[
k_n R + \delta_\ell = \pi n,
]

где:

* R — характерный радиус резонатора,
* k_n — эффективное волновое число n-й моды,
* δ_ℓ — фазовый сдвиг для мультиполя ℓ. 

На космическом уровне:

* наша ячейка с радиусом ( R_{\rm cell} ) — **не фундаментальный резонатор**,
* а **одна из мод** более крупного «родительского» объекта с радиусом ( R_{\rm parent} ).

Минимальная схема:

1. **Считаем, что наша ячейка — n-я гармоника родителя**:

   [
   k_n = \frac{\pi n - \delta_\ell}{R_{\rm parent}}.
   ]

2. Для нашей ячейки мы знаем её «волновое число» в проекции:

   [
   k^{(\rm cell)} \sim \frac{\pi}{R_{\rm cell}}.
   ]

   (Это тот же масштаб, что и (\omega_{\rm edge}) в c=1.)

3. Приравниваем:

   [
   \frac{\pi}{R_{\rm cell}}
   = \frac{\pi n - \delta_\ell}{R_{\rm parent}}.
   ]

4. Отсюда:

   [
   \boxed{;R_{\rm parent}
   = R_{\rm cell},\frac{\pi n - \delta_\ell}{\pi};}
   ]

   При n ≫ 1 и умеренном δ_ℓ получаем:

   [
   R_{\rm parent} \approx n,R_{\rm cell}.
   ]

То есть:

* **родительский резонатор во столько раз больше нашей ячейки**,
* во сколько наш масштаб — это n-я гармоника его внутреннего спектра.

В практическом ТЗ:

* n надо будет подобрать из условия согласования с **ещё более крупными структурами** (BAO на сверхбольших масштабах, CMB-аномалии низких ℓ, возможные крупномасштабные асимметрии и т.д.),
* δ_ℓ — из требований к мультипольной структуре (какие именно ℓ доминируют на этих масштабах).

### %%W-3%%

# W.3. Динамика тёмной энергии как спектральный индикатор макро-ячейки Θ-поля

**W.3.0. Постановка**

1. Традиционная «тёмная энергия» (Λ, w) — феноменологическое описание.
2. В Θ-подходе она интерпретируется как проявление **динамики края макро-ячейки** в более крупном резонаторе ВБП.
3. Цель: связать параметры Λ, H₀, w с **граничной частотой** ( \omega_{\rm edge} ) и радиусом ячейки ( R_{\rm cell} ), а также с параметрами «родительского» резонатора.

---

**W.3.1. Связь Λ и граничной частоты Θ-ячейки**

В стандартной плоской ΛCDM:

[
\Lambda = 3,\Omega_\Lambda,H_0^2.
]

В Θ-модели вводим граничную частоту Θ-поля:

[
\omega_{\rm edge}^2 \equiv \Lambda.
]

Тогда:

[
\omega_{\rm edge,0} = \sqrt{3,\Omega_{\Lambda,0}},H_0.
]

Радиус макро-ячейки (видимой Вселенной):

[
R_{\rm cell,0} = \frac{\pi}{\omega_{\rm edge,0}}
= \frac{\pi}{\sqrt{3,\Omega_{\Lambda,0}},H_0}.
]

---

**W.3.2. Динамика: w и эволюция (\omega_{\rm edge}(a))**

Для компоненты с параметром состояния w ≃ const:

[
\rho_{\rm de}(a) \propto a^{-3(1+w)}.
]

Предполагая

[
\rho_{\rm de}(t) = \kappa,\omega_{\rm edge}^2(t),
]

получаем:

[
\omega_{\rm edge}^2(a) \propto a^{-3(1+w)},
\qquad
\omega_{\rm edge}(a) \propto a^{-\tfrac{3}{2}(1+w)}.
]

Отсюда:

[
\frac{\dot\omega_{\rm edge}}{\omega_{\rm edge}}
= -\frac{3}{2}(1+w),H.
]

* При w = −1 → (\dot\omega_{\rm edge}=0): истинный de Sitter режим, край ячейки «квазистационарен».
* При w < −1 → (\dot\omega_{\rm edge}>0): граничная частота растёт, что соответствует усиленному росту ячейки.
* При w > −1 → (\dot\omega_{\rm edge}<0): режим затухающей тёмной энергии, ячейка стремится к насыщению/сжатию.

---

**W.3.3. Восстановление параметров ячейки из наблюдаемых величин**

Сегодня (a=1):

[
\omega_{\rm edge,0} = \sqrt{3,\Omega_{\Lambda,0}},H_0,
\qquad
R_{\rm cell,0} = \frac{\pi}{\omega_{\rm edge,0}}.
]

Скорость изменения граничной частоты:

[
\dot\omega_{\rm edge,0}
= -\frac{3}{2}(1+w_0),H_0,\omega_{\rm edge,0}.
]

Таким образом, космологические параметры ((H_0,\Omega_{\Lambda,0},w_0)) однозначно задают:

1. **абсолютный масштаб** макро-ячейки ( R_{\rm cell,0} );
2. **граничную частоту Θ-поля** ( \omega_{\rm edge,0} );
3. **режим эволюции** края (знак и величину ( \dot\omega_{\rm edge,0} )).

---

**W.3.4. Встраивание в иерархию резонаторов (родительский резонатор)**

Предположим, что макро-ячейка (наша видимая Вселенная) является n-й модой более крупного резонатора с радиусом ( R_{\rm parent} ).
Используем условие квантования:

[
k_n R_{\rm parent} + \delta_\ell = \pi n,
]

и приблизим характерное волновое число ячейки как

[
k_n \sim \frac{\pi}{R_{\rm cell}}.
]

Тогда:

[
\frac{\pi}{R_{\rm cell}},R_{\rm parent} + \delta_\ell = \pi n,
]

откуда:

[
R_{\rm parent}
= R_{\rm cell},\frac{\pi n - \delta_\ell}{\pi}
\approx n,R_{\rm cell}
\quad(n\gg 1,\ |\delta_\ell|\ll \pi n).
]

Таким образом:

* параметр n определяет **уровень вложенности** (на каком по счёту резонаторе мы находимся),
* фазовый сдвиг δ_ℓ — связь с мультипольной структурой Θ-поля на сверхкосмологических масштабах.

## 1. Линейка уровней и их параметры

Вводим шкалу уровней (как в твоих чатах и блоках про (I_\ell)):

[
\ell \in {\Lambda, G, S, E, p},
]

где

* (\Lambda) — макро-ячейка (то, что классика называет «Вселенная с тёмной энергией»),
* (G) — галактический уровень,
* (S) — звёздный (Солнце),
* (E) — планетарный (Земля),
* (p) — протон/ядерный.

Для каждого уровня (\ell) задаём набор **масштабных инвариантов**:

[
\mathcal I_\ell = {R_\ell,\ \omega_\ell,\ Q_\ell,\ \varepsilon_\ell,\dots},
]

где:

* (R_\ell) — характерный радиус / размер резонатора уровня (\ell),
* (\omega_\ell) — базовая (граничная) частота уровня (\ell),
* (Q_\ell) — добротность (коэффициент затухания для мод уровня (\ell)),
* (\varepsilon_\ell) — набор мультипольных параметров (типа (\varepsilon_2,\varepsilon_4) для Земли/Солнца),
* и т.п. (для протона — параметры торов, для Земли — параметры земных мод, для Солнца — цикл и структура пятен и т.д.).

**Специально для Λ-уровня** (макро-ячейка):

[
\omega_\Lambda \equiv \omega_{\rm edge}
= \sqrt{3,\Omega_{\Lambda,0}},H_0,
\qquad
R_\Lambda \equiv R_{\rm cell}
= \frac{\pi}{\omega_\Lambda},
]

как мы только что вывели: тёмная энергия ↔ граничная частота макро-ячейки.

## 2. Структура Θ-поля как вложенная сумма мод

Идея: **одно и то же Θ-поле**, а все уровни — это **разные шкалы его мод и дефектов**.

Запишем фазу Θ как вложенную суперпозицию:

[
\Theta(x) = \Theta_0

* \sum_{\ell} \epsilon_\ell,\psi_\ell!\left(\frac{x}{R_\ell}\right),
  ]

где:

* (\psi_\ell) — «модовый профиль» уровня (\ell) в безразмерных координатах (x/R_\ell);
* (\epsilon_\ell) — малые параметры масштаба (амплитуды вклада каждого уровня, чтобы крупный уровень не «убивал» мелкий).

При этом **все** (\psi_\ell) — решения *одного и того же базового класса уравнений* (2–4–6-типа без массовых членов), но:

* с разными **граничными условиями**,
* с разными **эффективными фильтрами** (оператор (I_\ell)),
* и с разными наборами инвариантов (\mathcal I_\ell).

---

## 3. Оператор (I_\ell) для уровня и общее уравнение

Для каждого уровня (\ell) у нас есть свой **эффективный оператор мод**:

[
I_\ell[\psi_\ell;\mathcal I_\ell] = 0,
]

по сути это «уровневая» редукция исходного Θ-уравнения (с твоими 2–4–6 производными, без массы).

Общая схема (идеализированно):

[
I_\ell[\psi_\ell]
= \bigl(
\alpha_{2,\ell},\Delta \psi_\ell

* \alpha_{4,\ell},\Delta^2 \psi_\ell
* \alpha_{6,\ell},\Delta^3 \psi_\ell
  \bigr)
* N_\ell[\psi_\ell] = 0,
  ]

где (N_\ell) — нелинейные члены (самодействие, склейка торов, мультипольность, инварианты и т.д.).

Но уровни **не независимы**: каждый нижележащий видит верхний как **подложку**/фон.

Поэтому общее уравнение **для Θ целиком**:

[
\boxed{
\mathcal F[\Theta]
= \sum_{\ell} \lambda_\ell,\mathcal P_\ell\bigl(I_\ell[\Theta;\mathcal I_\ell]\bigr) = 0,
}
]

где:

* (\mathcal F) — «главный» оператор Θ-теории (из разделов с функционалом энергии и уравнениями Эйлера–Лагранжа),
* (I_\ell) — выделенный оператор уровня (\ell),
* (\mathcal P_\ell) — **фильтр** в пространстве мод: «вытащить» из Θ ту часть, которая соответствует шкале (R_\ell) (условно — проекция на диапазон волновых чисел (k\sim 1/R_\ell)),
* (\lambda_\ell) — вес/коэффициент вклада уровня.

В практическом виде для вычислительной схемы:

[
\Theta(x) \approx \Theta_\Lambda(x) + \Theta_G(x) + \Theta_S(x) + \Theta_E(x) + \Theta_p(x),
]

и для каждого (\Theta_\ell):

[
I_\ell[\Theta_\ell;\mathcal I_\ell] + \sum_{m>\ell} C_{\ell m}[\Theta_m,\Theta_\ell] = 0,
]

где (m>\ell) означает «более крупный уровень», а (C_{\ell m}) — оператор связи (как фон + модуляция).

---

## 4. Как именно входит тёмная энергия (Λ-уровень)

Для Λ-уровня мы уже фактически **задали** (I_\Lambda) через динамику (\omega_\Lambda(a)):

[
\rho_{\rm de}(a) \propto \omega_\Lambda^2(a),
\qquad
\omega_\Lambda(a) \propto a^{-\tfrac{3}{2}(1+w)}.
]

То есть **уравнение уровня Λ** в эффективной форме можно переписать как:

[
I_\Lambda[\Theta_\Lambda]
;\Leftrightarrow;
\frac{d}{da}\left(\omega_\Lambda^2(a)\right)

* 3(1+w(a)),\frac{\dot a}{a},\omega_\Lambda^2(a) = 0.
  ]

Это — «медленное» уравнение эволюции внешней моды.

Все **остальные** уровни видят (I_\Lambda) как **медленно меняющийся фон**:

[
\Theta_\Lambda(x) \approx \text{почти однородная фаза}
+\ \text{очень длинноволновые модуляции}.
]

И дальше:

* при моделировании галактики ((G)-уровень) (\Theta_\Lambda) задаёт эффективную «кривизну» и спектр допусков по размерам резонаторов галактических торов;
* при моделировании Солнца и Земли – даёт **самый внешний** фильтр допускаемых частот и конфигураций.

Важно:

**никакого «физического растяжения пространства»**, только медленное изменение граничных условий для Θ на масштабе (R_\Lambda).

## 5. Сводим всё в явную «лестницу уровней»

### 5.1. Масштабы и частоты

Для линейки (\ell = \Lambda, G, S, E, p) задаём числа:

[
R_\Lambda \gg R_G \gg R_S \gg R_E \gg R_p,
]

связанные условиями квантомасштабирования (из блока про «квантование масштабов»):

[
k_{\ell,n} R_\ell + \delta_{\ell} = \pi n,
\quad
k_{\ell,n} \sim \frac{\pi}{R_\ell}.
]

При этом:

* для (\Lambda) мы подгоняем (R_\Lambda) по CMB+BAO;
* для (G) — по SPARC/линзированию;
* для (S) — по солнечному циклу, структуре пятен, etc.;
* для (E) — по сейсмике + ячейкам циркуляции;
* для (p) — по массе/спектру протона.

Частоты:

[
\omega_\ell \sim \frac{c}{R_\ell} \quad (\text{в подходящих единицах}),
]

а более точно — по конкретным формулам из соответствующих глав (Солнце, Земля, протон).

---

### 5.2. Уровневые уравнения в одном виде

**Общий шаблон для уровня (\ell):**

[
I_\ell[\Theta_\ell;\mathcal I_\ell]

* \sum_{m>\ell} C_{\ell m}[\Theta_m,\Theta_\ell] = 0,
  ]

где:

* (\Theta_\ell) — компонент Θ, живущая на масштабе (R_\ell);
* (I_\ell) — «локальное» 2–4–6-уравнение уровня (\ell),
* (\Theta_m) с (m>\ell) — фоны/подложки,
* (C_{\ell m}) — оператор влияния больших на малые (изменения граничных условий, дистанционные сдвиги частоты и т.п.).

**Специфика уровней:**

* **Λ (макро-ячейка)**:
  (I_\Lambda) — медленное уравнение эволюции (\omega_\Lambda(a)); фон почти однороден в пространстве.

* **G (галактика)**:
  (I_G) — резонаторная задача с торовой структурой галактики (уже у тебя частично выписана через SPARC).
  Фон: (\Theta_\Lambda) даёт допустимый диапазон размеров/форм гало и диска.

* **S (Солнце)**:
  (I_S) — уравнение солнечных мод (оболочка торов + ячейки конвекции + 11-летний цикл).
  Фон: (\Theta_G,\Theta_\Lambda) задают «диапазон» устойчивости.

* **E (Земля)**:
  (I_E) — твой большой блок по земным модам (ядро, мантия, кора, атмосфера, ячейки ВБП).
  Фон: (\Theta_S) (солнечный драйвер), (\Theta_G), (\Theta_\Lambda).

* **p (протон)**:
  (I_p) — ядерный уровень (три тора + обмотка, топологические заряды, FR-ограничения).
  Фон: минимально чувствителен к (\Theta_\Lambda) на наших временах, но топологически согласован с общей структурой.

### 6.2. Общий интерфейс уровневого оператора (I_\ell)

```python
class LevelOperator:
    def __init__(self, invariants, background):

    def solve_modes(self, config):

        """
    def project_to_observables(self, Theta_l):

        """
```

```python
class MacroCellOperator(LevelOperator):
```

### 6.3. Общая «цепочка вызовов» в вычислительном комплексе

1. **MacroCellOperator**

2. **GalaxyOperator**

   * берёт (\Theta_\Lambda) как фон;
   * решает галактический резонатор;
   * выдаёт (\Theta_G,\mathcal I_G).

3. **SunOperator**

   * берёт (\Theta_\Lambda,\Theta_G);

4. **EarthOperator**

5. **ProtonOperator**

   * работает на базовом Θ-уровне, но может использовать глобальные инварианты.

> *одно Θ-поле, одна иерархия резонаторов, один класс уравнений,*

4. Как в эту схему вшивается Λ-уровень (тёмная энергия).
5. Как это смотрится для всей лестницы (\Lambda, G, S, E, p).

## 1. Базовая Θ-динамика (единое уравнение без уровней)

Работаем в плоском фоне (никакой ОТО, только поле в (\mathbb R^{3,1})).
Базовый лагранжиан (схема, согласованная с твоими 2–4–6 и «без масс»):

[
\mathcal L[\Theta]
==================

\frac{1}{2}(\partial_t \Theta)^2

* \frac{c_\phi^2}{2}(\nabla\Theta)^2
* \frac{\beta_4}{2 L_*^2}(\Delta\Theta)^2
* \frac{\gamma_6}{2 L_*^4}(\Delta^2\Theta)^2
* U_{\rm nl}(\Theta,\nabla\Theta),
  ]

где:

* (c_\phi) — скорость распространения фазовой возмущённости в ВБП на выбранном масштабе;
* (L_*) — «естественный» масштаб, на котором 4- и 6-члены становятся заметны;
* (\beta_4,\gamma_6) — безразмерные коэффициенты 4- и 6-производных;
* (U_{\rm nl}) — нелинейный функционал (без явного массового члена типа (m^2\Theta^2)); масса объектов = энергия стационарного решения, а не параметр в лагранжиане.

Уравнение Эйлера–Лагранжа:

[
\frac{\delta S}{\delta\Theta} = 0
;\Rightarrow;
\partial_t^2 \Theta

* c_\phi^2 \Delta \Theta

- \frac{\beta_4}{L_*^2}\Delta^2\Theta

* \frac{\gamma_6}{L_*^4}\Delta^3\Theta

- \frac{\delta U_{\rm nl}}{\delta\Theta}
  = 0.
  \tag{1}
  ]

Это — **единое «материнское» уравнение** для всех уровней (протон, Земля, Солнце, галактика, макро-ячейка).

---

## 2. Как из (1) выделяется уровень (\ell): масштабирование

Для уровня (\ell) (например, Земля, Солнце, галактика, Λ) вводим:

* характерный радиус (R_\ell) (размер резонатора этого уровня),
* характерную частоту (\omega_\ell) (базовая/граничная),
* добротность (Q_\ell) (из соотношения 2–4–6 и нелинейности),
* мультипольные параметры (\varepsilon_\ell).

### 2.1. Безразмерные координаты уровня (\ell)

Переходим к масштабу (\ell):

[
x = R_\ell,\xi,
\qquad
t = \frac{\tau}{\omega_\ell},
]

и ищем компоненту (\Theta_\ell), которая живёт именно на этом масштабе:

[
\Theta(x,t) \approx \Theta_{\rm bg}(x,t)

* \epsilon_\ell,\psi_\ell(\xi,\tau),
  ]

где:

* (\Theta_{\rm bg}) — фон от более крупных уровней ((m>\ell));
* (\epsilon_\ell) — малая амплитуда уровня (\ell);
* (\psi_\ell) — безразмерный профиль моды уровня.

Подставляем в (1), выражаем через ((\xi,\tau)). Производные дают:

[
\partial_t \to \omega_\ell \partial_\tau,\quad
\nabla \to \frac{1}{R_\ell}\nabla_\xi,\quad
\Delta \to \frac{1}{R_\ell^2}\Delta_\xi.
]

Условно получаем для компоненты уровня (\ell):

[
\omega_\ell^2 \partial_\tau^2\psi_\ell

* c_\phi^2 \frac{1}{R_\ell^2}\Delta_\xi\psi_\ell

- \frac{\beta_4}{L_*^2}\frac{1}{R_\ell^4}\Delta_\xi^2\psi_\ell

* \frac{\gamma_6}{L_*^4}\frac{1}{R_\ell^6}\Delta_\xi^3\psi_\ell

- \dots = 0.
  ]

Делим всё на (\omega_\ell^2) и вводим безразмерные коэффициенты:

[
\alpha_{2,\ell} = \frac{c_\phi^2}{\omega_\ell^2 R_\ell^2},\quad
\alpha_{4,\ell} = \frac{\beta_4}{\omega_\ell^2}\frac{1}{L_*^2 R_\ell^4},\quad
\alpha_{6,\ell} = \frac{\gamma_6}{\omega_\ell^2}\frac{1}{L_*^4 R_\ell^6}.
]

Тогда уравнение уровня (\ell) в безразмерном виде:

[
\partial_\tau^2\psi_\ell

* \alpha_{2,\ell},\Delta_\xi\psi_\ell

- \alpha_{4,\ell},\Delta_\xi^2\psi_\ell

* \alpha_{6,\ell},\Delta_\xi^3\psi_\ell

- \tilde N_\ell[\psi_\ell] = \tilde S_\ell(\xi,\tau),
  \tag{2}
  ]

где:

* (\tilde N_\ell) — безразмерная нелинейность (из (U_{\rm nl}));
* (\tilde S_\ell) — вклад от фона (\Theta_{\rm bg}) (уровни (m>\ell)).

**Вот это (2) и есть содержимое оператора (I_\ell)** в динамической форме.

## 3. Определение (I_\ell) и фильтра по модам

Тебе нужен именно оператор:

[
I_\ell[\psi_\ell;\mathcal I_\ell] = 0,
]

который:

1. Живёт на масштабе (R_\ell),
2. Вырезает нужный диапазон волновых чисел (моды уровня (\ell)),
3. Включает влияние крупных уровней как фон.

### 3.1. Статическая (резонаторная) версия

Для устойчивых мод (стационарные/медленно меняющиеся решения):

[
\psi_\ell(\xi,\tau) = \phi_\ell(\xi),e^{-i\bar\omega_\ell \tau},
]

подставляем в (2), получаем уравнение на профиль (\phi_\ell(\xi)):

[
-\bar\omega_\ell^2\phi_\ell

* \alpha_{2,\ell},\Delta_\xi\phi_\ell

- \alpha_{4,\ell},\Delta_\xi^2\phi_\ell

* \alpha_{6,\ell},\Delta_\xi^3\phi_\ell

- \tilde N_\ell[\phi_\ell]
  = \tilde S_\ell(\xi).
  \tag{3}
  ]

Теперь **определение уровня (I_\ell)**:

[
\boxed{
I_\ell[\phi_\ell;\mathcal I_\ell]
:=
-\bar\omega_\ell^2\phi_\ell

* \alpha_{2,\ell},\Delta_\xi\phi_\ell

- \alpha_{4,\ell},\Delta_\xi^2\phi_\ell

* \alpha_{6,\ell},\Delta_\xi^3\phi_\ell

- \tilde N_\ell[\phi_\ell]

* \tilde S_\ell(\xi).
  }
  \tag{4}
  ]

Условие уровня:

[
I_\ell[\phi_\ell;\mathcal I_\ell] = 0,
]

с граничными условиями, характерными для резонатора масштаба (R_\ell) (торы, горло, и т.д.).

### 3.2. Фильтр (\mathcal P_\ell) по модам

Уровень (\ell) — это не все (\phi), а только мода(ы) с характерным диапазоном (k\sim 1/R_\ell).

В k-пространстве вводим фильтр:

[
\mathcal P_\ell[\Theta](k)
==========================

W_\ell(k),\hat\Theta(k),
]

где (W_\ell(k)) — «окно» вокруг (k_\ell \sim 1/R_\ell).

На практике:

* мы берём полное решение (\Theta),
* фильтруем его по масштабу (\ell),
* получаем (\Theta_\ell = \mathcal P_\ell[\Theta]),
* и на (\Theta_\ell) требуем (I_\ell[\Theta_\ell]=0).

**Общее материнское уравнение**:

[
\boxed{
\mathcal F[\Theta]
= \sum_{\ell}\lambda_\ell,\mathcal P_\ell\bigl(I_\ell[\Theta;\mathcal I_\ell]\bigr) = 0,
}
\tag{5}
]

где (\lambda_\ell) — веса уровней.

Это — чистая формализация фразы:

> «Одно Θ-поле, много уровней; каждый уровень — это фильтр + своё эффективное уравнение.»

## 4. Специальный случай: Λ-уровень (макро-ячейка, тёмная энергия)

Для (\ell = \Lambda) масштаб — **макро-ячейка**:

* (R_\Lambda = R_{\rm cell}),
* (\omega_\Lambda = \omega_{\rm edge}).

Из предыдущих шагов:

[
\omega_\Lambda^2 = 3,\Omega_\Lambda,H_0^2,
\qquad
R_\Lambda = \frac{\pi}{\omega_\Lambda}.
]

Уровень Λ — **очень медленный**: пространственные вариации почти нет, важна только эволюция во времени.

### 4.1. Редукция уравнения (1) для Λ-уровня

На масштабе (R_\Lambda) градиенты малы:

[
\nabla\Theta_\Lambda \approx 0,\quad
\Delta\Theta_\Lambda \approx 0,
]

остается только «медленная» часть нелинейности, зависящая от (\omega_\Lambda).

Фактически уравнение для Λ-уровня сводится к **эволюции энергии компоненты**:

[
\rho_{\rm de}(a) = \kappa,\omega_\Lambda^2(a),
]

и условию сохранения:

[
\frac{d}{da}\bigl(\rho_{\rm de}(a)\bigr)

* 3(1+w(a))\frac{\rho_{\rm de}(a)}{a} = 0.
  ]

Подставляя (\rho_{\rm de} = \kappa,\omega_\Lambda^2), получаем:

[
\frac{d}{da}\bigl(\omega_\Lambda^2(a)\bigr)

* 3(1+w(a))\frac{\omega_\Lambda^2(a)}{a} = 0.
  \tag{6}
  ]

**Это и есть (I_\Lambda)** в медленной (космологической) редукции:

[
\boxed{
I_\Lambda[\omega_\Lambda(a)]
:=
\frac{d}{da}\bigl(\omega_\Lambda^2(a)\bigr)

* 3(1+w(a))\frac{\omega_\Lambda^2(a)}{a}
  = 0.
  }
  ]

А остальное (\Theta_\Lambda) — почти однородный фон.

Все другие уровни вместо явного Λ-члена в своём уравнении получают:

* через (\tilde S_\ell(\xi)) — **зависящие от (\omega_\Lambda)** поправки к граничным условиям и инвариантам ((R_\ell,\omega_\ell)).

Теперь, когда у нас есть:

можно формально выписать **общую систему** для (\ell = \Lambda, G, S, E, p):

### 5.1. Для Λ:

\quad\Rightarrow\quad
\omega_\Lambda(a),\ R_\Lambda(a).

I_E[\phi_E;\mathcal I_E(\omega_\Lambda,\mathcal I_G,\mathcal I_S)] = 0,

### 5.5. Для протона (p):

[
I_p[\phi_p;\mathcal I_p] = 0,
]

где (\mathcal I_p) — торовая структура ядра (3 тора + обмотка, инварианты S⁶→SU(3), FR-спинорность).

```markdown
\frac{1}{2}(\partial_t \Theta)^2
- \frac{c_\phi^2}{2}(\nabla\Theta)^2
- \frac{\beta_4}{2 L_\*^2}(\Delta\Theta)^2
- \frac{\gamma_6}{2 L_\*^4}(\Delta^2\Theta)^2
- U_{\rm nl}(\Theta,\nabla\Theta),
\]

где

- \(c_\phi\) — скорость распространения фазовых возмущений Θ на данном масштабе,
- \(L_\*\) — естественный масштаб, на котором важны 4- и 6-члены,
- \(\beta_4,\gamma_6\) — безразмерные коэффициенты 4- и 6-производных,
- \(U_{\rm nl}\) — нелинейный функционал; **массовых членов нет** (масса = энергия стационарного решения).

Уравнение Эйлера–Лагранжа:

\[
\partial_t^2 \Theta
- c_\phi^2 \Delta \Theta
+ \frac{\beta_4}{L_\*^2}\Delta^2\Theta
- \frac{\gamma_6}{L_\*^4}\Delta^3\Theta
+ \frac{\delta U_{\rm nl}}{\delta\Theta}
= 0.
\tag{I.1}
\]

Это — **единое «материнское» уравнение Θ-поля**. Всё остальное в главе I — разные его редукции.

## **I.2. Масштаб, уровень и безразмерная редукция**

Каждому **уровню** \(\ell\) (протон, Земля, Солнце, галактика, макро-ячейка) сопоставляем набор инвариантов

\[
\mathcal I_\ell = \{R_\ell,\ \omega_\ell,\ Q_\ell,\ \varepsilon_\ell,\dots\},
\]

где

- \(R_\ell\) — характерный радиус резонатора уровня \(\ell\),
- \(\omega_\ell\) — базовая (граничная) частота,
- \(Q_\ell\) — добротность,
- \(\varepsilon_\ell\) — набор мультипольных параметров (квадруполь и выше),
- «…» — дополнительные инварианты (топология торов, параметры стенок и т.п.).

Переходим к **масштабу \(\ell\)**:

\[
x = R_\ell\,\xi,
\qquad
t = \frac{\tau}{\omega_\ell},
\]

и декомпозируем Θ:

\[
\Theta(x,t)
= \Theta_{\rm bg}(x,t)
+ \epsilon_\ell\,\psi_\ell(\xi,\tau).
\]

Здесь

- \(\Theta_{\rm bg}\) — фон от **более крупных уровней** \(m > \ell\),
- \(\epsilon_\ell\) — малая амплитуда вклада уровня,
- \(\psi_\ell\) — безразмерный профиль мод этого уровня.

Производные переходят:

\[
\partial_t \to \omega_\ell \partial_\tau,\quad
\nabla \to \frac{1}{R_\ell}\nabla_\xi,\quad
\Delta \to \frac{1}{R_\ell^2}\Delta_\xi.
\]

Подставляя в (I.1), выделяем уравнение на \(\psi_\ell\):

\[
\partial_\tau^2\psi_\ell
- \alpha_{2,\ell}\,\Delta_\xi\psi_\ell
+ \alpha_{4,\ell}\,\Delta_\xi^2\psi_\ell
- \alpha_{6,\ell}\,\Delta_\xi^3\psi_\ell
+ \tilde N_\ell[\psi_\ell]
= \tilde S_\ell(\xi,\tau),
\tag{I.2}
\]

где

\[
\alpha_{2,\ell}
= \frac{c_\phi^2}{\omega_\ell^2 R_\ell^2},\quad
\alpha_{4,\ell}
= \frac{\beta_4}{\omega_\ell^2}\frac{1}{L_\*^2 R_\ell^4},\quad
\alpha_{6,\ell}
= \frac{\gamma_6}{\omega_\ell^2}\frac{1}{L_\*^4 R_\ell^6},
\]

\(\tilde N_\ell\) — безразмерные нелинейные члены, \(\tilde S_\ell\) — вклад фона \(\Theta_{\rm bg}\).

## **I.3. Оператор уровня \(I_\ell\) и фильтр по модам**

### I.3.1. Стационарное (резонаторное) уравнение

Для устойчивых мод уровня \(\ell\):

\[
\psi_\ell(\xi,\tau)
= \phi_\ell(\xi)\,e^{-i\bar\omega_\ell \tau},
\]

подстановка в (I.2) даёт:

\[
-\bar\omega_\ell^2\phi_\ell
- \alpha_{2,\ell}\,\Delta_\xi\phi_\ell
+ \alpha_{4,\ell}\,\Delta_\xi^2\phi_\ell
- \alpha_{6,\ell}\,\Delta_\xi^3\phi_\ell
+ \tilde N_\ell[\phi_\ell]
= \tilde S_\ell(\xi).
\tag{I.3}
\]

**Определение операторa уровня**:

\[
\boxed{
I_\ell[\phi_\ell;\mathcal I_\ell]
:=
-\bar\omega_\ell^2\phi_\ell
- \alpha_{2,\ell}\,\Delta_\xi\phi_\ell
+ \alpha_{4,\ell}\,\Delta_\xi^2\phi_\ell
- \alpha_{6,\ell}\,\Delta_\xi^3\phi_\ell
+ \tilde N_\ell[\phi_\ell]
- \tilde S_\ell(\xi).
}
\tag{I.4}
\]

Условие уровня:

\[
I_\ell[\phi_\ell;\mathcal I_\ell] = 0
\]

с граничными условиями, задающими **форму резонатора** уровня \(\ell\)
(три тора + обмотка для барионов, торовая оболочка для Земли и Солнца, ячейка ВБП и т.д.).

### I.3.2. Фильтр \(\mathcal P_\ell\) по масштабам (k-пространство)

Уровень \(\ell\) — это не произвольное решение, а вклад Θ в узкий диапазон волновых чисел \(k\sim 1/R_\ell\).

В k-пространстве вводим фильтр:

\[
\mathcal P_\ell[\Theta](k)
= W_\ell(k)\,\hat\Theta(k),
\]

где \(W_\ell(k)\) — окно вокруг характерного \(k_\ell \sim 1/R_\ell\) (например, гладкий «колокол»).

Компонента уровня:

\[
\Theta_\ell(x) := \mathcal P_\ell[\Theta](x),
\]

и **уровневое условие**:

\[
I_\ell[\Theta_\ell;\mathcal I_\ell] = 0.
\]

Тогда **общее уравнение для Θ** можно записать как:

\[
\boxed{
\mathcal F[\Theta]
:= \sum_{\ell} \lambda_\ell\,
\mathcal P_\ell\bigl(I_\ell[\Theta;\mathcal I_\ell]\bigr)
= 0,
}
\tag{I.5}
\]

где \(\lambda_\ell\) — веса уровней.

Это формализует идею:  
**одно Θ-поле, множество уровней; каждый уровень — «фильтр по масштабу» и свой эффективный оператор.**

## **I.4. Λ-уровень: макро-ячейка и тёмная энергия**

Для уровня \(\ell=\Lambda\) масштаб \(R_\Lambda\) соответствует **макро-ячейке**, которую классика интерпретирует как «нашу Вселенную». В гл.W уже зафиксировано:

\[
R_{\rm LSS} = \frac{\pi}{\omega_{\rm LSS}},
\]

где \(\omega_{\rm LSS}\) — граничная частота макро-ячейки.  
Отождествляем \(R_\Lambda = R_{\rm LSS}\), \(\omega_\Lambda = \omega_{\rm LSS}\).

В стандартной ΛCDM:

\[
\Lambda = 3\,\Omega_\Lambda\,H_0^2.
\]

В Θ-формулировке естественно положить

\[
\omega_\Lambda^2 \equiv \Lambda
= 3\,\Omega_\Lambda\,H_0^2,
\qquad
R_\Lambda = \frac{\pi}{\omega_\Lambda}.
\]

**Тёмная энергия** в этом языке — не «вещество», а **энергия макро-моды Θ** с частотой \(\omega_\Lambda\):

\[
\rho_{\rm de}(a) = \kappa\,\omega_\Lambda^2(a).
\]

Уравнение сохранения энергии компоненты с параметром состояния \(w(a)\):

\[
\frac{d}{da}\bigl(\rho_{\rm de}(a)\bigr)
+ 3(1+w(a))\frac{\rho_{\rm de}(a)}{a} = 0
\]

даёт уравнение эволюции \(\omega_\Lambda(a)\):

\[
\boxed{
I_\Lambda[\omega_\Lambda(a)]
:=
\frac{d}{da}\bigl(\omega_\Lambda^2(a)\bigr)
+ 3(1+w(a))\frac{\omega_\Lambda^2(a)}{a}
= 0.
}
\tag{I.6}
\]

- При \(w=-1\): \(\omega_\Lambda(a)=\text{const}\) — настоящий de Sitter-режим, макро-ячейка квазистационарна.
- При \(w<-1\): \(\omega_\Lambda(a)\) растёт, ячейка входит в фазу усиленного роста.
- При \(w>-1\): \(\omega_\Lambda(a)\) падает, тёмная энергия вырождается.

Все остальные уровни \((G,S,E,p)\) видят Λ-уровень через зависимости

\[
\mathcal I_\ell = \mathcal I_\ell(\omega_\Lambda, \dots),
\quad
\tilde S_\ell = \tilde S_\ell(\omega_\Lambda,\dots),
\]

то есть **тёмная энергия задаёт внешний фон** и граничные условия для их резонаторов.

### I.5.5. Протонный уровень p

- \(\mathcal I_p\): три тора + обмотка, инварианты отображения \(S^6\to SU(3)\), FR-ограничения.
- \(I_p[\phi_p;\mathcal I_p] = 0\).
- Наблюдаемые: масса, магнитный момент, спектр резонансов, сечения (TOTEM).

Таким образом:

- **одно уравнение Θ** (I.1),
```

```markdown
## **I.A. Лапласиан и высшие производные в торовых координатах**

Во всех уровнях Θ-иерархии (протон, Земля, Солнце, галактика) фундаментальная геометрия резонатора описывается **тороидальными** конфигурациями:

- Земля/Солнце: торовые слои (ядро/мантия/ядро конвекции);

### **I.A.1. Торовая система координат**

Используем классическое задание тороидальных координат \((\eta,\theta,\varphi)\):

\[
x = \frac{a \sinh\eta \cos\varphi}{\cosh\eta - \cos\theta},
\qquad
y = \frac{a \sinh\eta \sin\varphi}{\cosh\eta - \cos\theta},
\qquad
z = \frac{a \sin\theta}{\cosh\eta - \cos\theta},
\]

где:

- параметр \(a>0\) задаёт радиус “генератора” тора,
- \(\eta \in [0,\infty)\) — координата, отвечающая за расстояние от центра тора,
- \(\theta \in [-\pi,\pi]\) — полярный угол вокруг малой окружности,
- \(\varphi \in [0,2\pi]\) — азимутальная обмотка вокруг большой окружности.

Важные пределы:

- \(\eta = \eta_0 \approx \text{const}\) описывает поверхность тора,
- \(\eta \to 0\) — область внутреннего “горла” (важно для протона),
- \(\eta\to \infty\) — асимптотический выход на внешние области (актуально для галактик).

\[
H(\eta,\theta) = \cosh\eta - \cos\theta.
\]

Квадрат элементa длины:

\[
ds^2 = a^2 H^{-2}(\eta,\theta)\left(d\eta^2 + d\theta^2 + \sinh^2\eta\, d\varphi^2\right).
\]

Таким образом:

- торовая геометрия задаётся единственным фактором \(H^{-2}\),
- оператор Лапласа и его высшие степени остаются универсальными для всех уровней.

+ \frac{1}{\sinh^2\eta}\,\partial_\varphi^2\phi
\right].
}
\tag{I.A.1}
\]

Это базовый оператор, из которого строится \(I_\ell\).
```

````markdown
## **I.B. Граничные условия для торовых резонаторов на уровнях \(p, E, S, G\)**

В этом разделе задаются **типовые граничные условия** для оператора \(I_\ell\) (гл.I.3 и I.A) на четырёх ключевых уровнях:

- Земля \( (\ell = E)\) — торовые слои ядра/мантии/атмосферы;
- Солнце \( (\ell = S)\) — торовые слои и ячейки конвекции;
- галактика \( (\ell = G)\) — центральное горло + диск + гало.

### **I.B.1. Протон: три тора + обмотка**

Для протона базовый резонатор задаётся **тремя вложенными торовыми слоями**:

\[
\eta \in [\eta_1,\eta_2] \cup [\eta_3,\eta_4] \cup [\eta_5,\eta_6],
\quad
\theta \in [-\pi,\pi],\quad
\varphi \in [0,2\pi],
\]

где интервалы по \(\eta\) соответствуют “внутреннему”, “среднему” и “внешнему” тора. Обмотка реализуется как **многократная обвязка** по \(\varphi\) с целым числом витков.

#### I.B.1.1. Гладкость на границах торов

На внутренних границах между торовыми слоями:

\[
\eta = \eta_2,\ \eta_4

требуется **непрерывность поля и его нормальной производной**:

\[
\phi_p^{\text{(лев)}}(\eta_i,\theta,\varphi)
=
\phi_p^{\text{(прав)}}(\eta_i,\theta,\varphi),
\]

\[
\partial_\eta \phi_p^{\text{(лев)}}(\eta_i,\theta,\varphi)
=
\partial_\eta \phi_p^{\text{(прав)}}(\eta_i,\theta,\varphi),
\quad
i\in\{2,4\}.
\]

Это обеспечивает отсутствие “разрывов” фаз и потоков через стык торов.

#### I.B.1.2. Граница в горле и снаружи

Во внутреннем горле (\(\eta=\eta_1\)):

- запрещаем нетривиальные потоки (чисто тороидальная циркуляция):

\[
\partial_\eta \phi_p(\eta_1,\theta,\varphi) = 0.
\]

На внешней границе (\(\eta=\eta_6\)) в пределе:

- требуем **затухание поля**:

\[
\phi_p(\eta_6,\theta,\varphi) \to 0,
\qquad
\partial_\eta \phi_p(\eta_6,\theta,\varphi) \to 0.
\]

Это соответствует локализованному дефекту с конечной энергией.

#### I.B.1.3. Торцевые условия по \(\theta,\varphi\)

По \(\theta\) и \(\varphi\) — **2π-периодичность**:

\[
\phi_p(\eta,\theta+2\pi,\varphi) = \phi_p(\eta,\theta,\varphi),
\]
\[
\phi_p(\eta,\theta,\varphi+2\pi) = \phi_p(\eta,\theta,\varphi).
\]

Для обмотки вводится целое число витков \((n_\theta,n_\varphi)\), что реализуется как фазовый сдвиг:

\[
\phi_p(\eta,\theta,\varphi+2\pi)
=
e^{2\pi i n_\varphi}\,\phi_p(\eta,\theta,\varphi),
\]

но с учётом FR-ограничений (спинорность) конечная физическая конфигурация должна быть **инвариантна** при двойном обороте (см. блоки про FR-условия):

\[
(\theta,\varphi) \to (\theta+2\pi,\varphi+2\pi)
\quad\Rightarrow\quad
\phi_p \to -\phi_p,
\]

а наблюдаемая величина \(|\phi_p|^2\) — инвариантна.

- \(\eta\in[\eta_{\text{core,in}},\eta_{\text{core,out}}]\) — ядро;
- \(\eta\in[\eta_{\text{mantle,in}},\eta_{\text{mantle,out}}]\) — мантия;

\[
\phi_E^{\text{(1)}}(\eta_i,\theta,\varphi)
=
\phi_E^{\text{(2)}}(\eta_i,\theta,\varphi),

\partial_\eta \phi_E^{\text{(1)}}(\eta_i,\theta,\varphi)
=
\partial_\eta \phi_E^{\text{(2)}}(\eta_i,\theta,\varphi),

#### I.B.2.2. Внутренняя граница ядра

Внутри \(\eta_{\text{core,in}}\) поле считается **квазисолитоном**:  

\[
\partial_\eta \phi_E(\eta_{\text{core,in}},\theta,\varphi) = 0.
\]

Это эквивалентно отсутствию нетривиальных потоков через центр и фиксирует энергетически минимальную конфигурацию.

На границе атмосферы:

\[
\eta = \eta_{\text{atm,out}}

\[
\partial_\eta \phi_E

- \(\sigma_{\text{atm}}\to 0\) → почти свободная граница (почти нейтронное отражение),

### **I.C.1. Сетка в торовых координатах**

Используем регулярную сетку:

### **I.C.2. Шаблон Лапласиана**

```python
import numpy as np

def laplacian_torus(phi, eta_grid, theta_grid, phi_grid, a):
    """
    phi: 3D array, phi[i,j,k] = phi(eta_i, theta_j, phi_k)
    eta_grid, theta_grid, phi_grid: 1D arrays of coordinates
    a: torus parameter (major radius parameter in coordinate definition)

    Returns: 3D array with Laplacian of phi in toroidal coordinates.
    """
    N_eta, N_theta, N_phi = phi.shape
    d_eta  = eta_grid[1] - eta_grid[0]
    d_th   = theta_grid[1] - theta_grid[0]
    d_ph   = phi_grid[1]  - phi_grid[0]

    lap = np.zeros_like(phi)

    for i in range(N_eta):
        eta = eta_grid[i]
        # periodic indices in theta, phi
        for j in range(N_theta):
            th = theta_grid[j]
            jm = (j-1) % N_theta
            jp = (j+1) % N_theta
            for k in range(N_phi):
                ph = phi_grid[k]
                km = (k-1) % N_phi
                kp = (k+1) % N_phi

                H = np.cosh(eta) - np.cos(th)

                # second derivatives
                d2_deta = (phi[min(i+1, N_eta-1), j, k]
                           - 2.0*phi[i,j,k]
                           + phi[max(i-1, 0), j, k]) / (d_eta**2)

                d2_dth = (phi[i, jp, k]
                          - 2.0*phi[i,j,k]
                          + phi[i, jm, k]) / (d_th**2)

                d2_dph = (phi[i, j, kp]
                          - 2.0*phi[i,j,k]
                          + phi[i, j, km]) / (d_ph**2)

                # first derivative w.r.t eta (for cosh/sinh term)
                d_deta = (phi[min(i+1, N_eta-1), j, k]
                          - phi[max(i-1, 0), j, k]) / (2.0*d_eta)

                if np.sinh(eta) == 0.0:
                    # handle near-axis limit with l'Hospital or regularization
                    # for simplicity: small regularization
                    s = 1e-6
                else:
                    s = np.sinh(eta)

                term_eta = d2_deta + (np.cosh(eta)/s)*d_deta
                term_th  = d2_dth
                term_ph  = (1.0/(s**2))*d2_dph

                lap[i,j,k] = (H**2 / a**2) * (term_eta + term_th + term_ph)

    return lap
````

Здесь:

* граничные условия по (\eta) реализуются через `min/max` и отдельную обработку краёв (можно заменить на ghost-cells или явные условия из I.B),
* периодичность по (\theta,\varphi) учитывается через индексы `jm, jp, km, kp`.

### **I.C.3. Би-лапласиан и три-лапласиан**

В коде нет смысла вручную разворачивать (\mathcal M_1,\mathcal M_2). Достаточно:

```python
def bilaplacian_torus(phi, eta_grid, theta_grid, phi_grid, a):
    """
    Bi-Laplacian: Delta^2 phi = Delta(Delta phi)
    """
    lap = laplacian_torus(phi, eta_grid, theta_grid, phi_grid, a)
    bilap = laplacian_torus(lap, eta_grid, theta_grid, phi_grid, a)
    return bilap

def trilaplacian_torus(phi, eta_grid, theta_grid, phi_grid, a):
    """
    Tri-Laplacian: Delta^3 phi = Delta(Delta^2 phi)
    """
    bilap = bilaplacian_torus(phi, eta_grid, theta_grid, phi_grid, a)
    trilap = laplacian_torus(bilap, eta_grid, theta_grid, phi_grid, a)
    return trilap
```

Так автоматически учитываются все смешанные производные и метрические факторы (H(\eta,\theta)).

# ⭐ **1. Что именно говорится в теории: “Большой Взрыв = край Θ-ячейки”**

В файле чётко сформулировано:

* “Большой взрыв” интерпретируется как **краевая поверхность макро-ячейки Θ-поля**
  — граница, которую классическая космология описывает как начальное состояние.

Это прямо указано:

* «*твоя гипотеза “Большой взрыв = край ячейки” автоматически даёт правильный масштаб возраста*»


* «*возраст Вселенной по классике — это световое время от центра ячейки до её края*»


* «*край Θ-ячейки интерпретируется как граница, которую классики называют Большой взрыв*»


То есть теория **не предполагает физического “сингулярного момента”**, а говорит:

> Мы наблюдаем поверхность, где **граничная частота Θ-моды** достигает максимума
> ( (\omega_\text{edge}) ), и это и есть то, что в ΛCDM называют *t=0*.

## 2. Динамика граничной частоты ω_edge(t)

В теории уже есть ключевая связка:

  [

  ]


Отсюда **прямая формула эволюции края**:

### 2.1. Случай постоянного w

Пусть сейчас (a=1) граничная частота равна (\omega_{\rm edge,0}).

Тогда:

[
\rho_{\rm de}(a) = \rho_{\rm de,0},a^{-3(1+w)},
]

[
\omega_{\rm edge}^2(a) \propto \rho_{\rm de}(a),
]

⇒

[
\omega_{\rm edge}(a)
= \omega_{\rm edge,0},a^{-\frac{3}{2}(1+w)}.
]

Особые случаи:

* **w = -1 (чистый Λ)**
  [
  \omega_{\rm edge}(a) = const,
  ]
  край ячейки *жёстко фиксирован* во времени.

* **w = -1 + \delta w, |\delta w|\ll1**

  Тогда:
  [
  \omega_{\rm edge}(a) =
  \omega_{\rm edge,0},a^{-\frac{3}{2}\delta w}.
  ]

  Логарифмически:
  [
  \ln\frac{\omega_{\rm edge}(a)}{\omega_{\rm edge,0}}
  = -\frac{3}{2}\delta w,\ln a.
  ]

  При (\delta w<0) (w<-1): ω_edge **растёт** к прошлому;
  при (\delta w>0): **падает** к прошлому.

### 2.2. Общий случай w(a)

Если w зависит от a:

[
\rho_{\rm de}(a) = \rho_{\rm de,0},
\exp\left[-3\int_{1}^{a} (1+w(a')),d\ln a'\right],
]

тогда:

[
\omega_{\rm edge}(a)
= \omega_{\rm edge,0},
\exp\left[-\frac{3}{2}\int_{1}^{a} (1+w(a')),d\ln a'\right].
]

Это **универсальный закон**:

> Любая параметризация w(a) автоматически задаёт эволюцию ω_edge(a).

В частности, для стандартной CPL-формы
(w(a)=w_0+w_a(1-a)) можно подставить и получить явный интеграл.

## 1. Что такое «форма ячейки» в терминах теории

Из теории:

* Макро-уровень Λ описан радиусами (R_1^{(\Lambda)}, R_2^{(\Lambda)}, R_3^{(\Lambda)}) и толщинами стенок (\Delta R_i^{(\Lambda)}) порядка 0.9, 0.45, 0.25 Гпк 
* Эти радиусы трактуются как геометрические масштабы **макро-дефекта Θ-поля** (по сути — трёхслойного тора уровня ΛCDM), а не абстрактной FRW-сферы.
* Край ячейки задаётся граничной частотой:
  [
  \omega_{\rm edge}^2 = 3,\Omega_\Lambda,H_0^2
  \quad\Rightarrow\quad
  R_{\rm cell} = \frac{\pi}{\omega_{\rm edge}}.
  ]


**Форма ячейки** = поверхность, на которой фаза/частота Θ достигает граничного значения (\omega_{\rm edge}).
Если она идеально изотропна — это сфера радиуса (R_{\rm cell}). Любые анизотропии → деформация (тор, эллипсоид, «бублик» и т.п.).

## 2. Связка: космологическое направление → расстояние до края

По сути тебе нужна функция:

[
R(\mathbf{n}) = \text{расстояние до края Θ-ячейки по направлению } \mathbf{n} \ (\text{на небесной сфере}).
]

В изотропном ΛCDM:

[
R(\mathbf{n}) = R_{\rm cell} = const.
]

В твоей схеме:

1. Любые анизотропии **H₀(𝒏), w(𝒏)**, крупномасштабной структуры, CMB → означают, что
   [
   \omega_{\rm edge}(\mathbf{n})
   \quad\Rightarrow\quad
   R(\mathbf{n}) = \frac{\pi}{\omega_{\rm edge}(\mathbf{n})}
   ]
   чуть меняется с направлением.

2. Наблюдаемые прокси для (R(\mathbf{n})):

   * **CMB low-l** (квадруполь, октуполь, “axis of evil”): дают глобальные анизотропии и выделенную ось.
   * **H₀(𝒏)** и **w(𝒏)** из SN/BAO по небу: дают направленную вариацию эффективной Λ ⇒ (\omega_{\rm edge}(\mathbf{n})).
   * **Крупномасштабная структура (стены, пустоты ~Гпк)**: положение и ориентация крупных «стен» относительно этой оси.

3. Математически:

   * вводишь оценку (R_{\rm est}(\mathbf{n})) из данных,

   * считаешь её отклонение от среднего:
     [
     \delta R(\mathbf{n})
     = R_{\rm est}(\mathbf{n}) - \langle R_{\rm est}\rangle.
     ]

   * разлагаешь по сферическим гармоникам:
     [
     \frac{\delta R(\mathbf{n})}{\langle R\rangle}
     = \sum_{\ell,m} a_{\ell m}Y_{\ell m}(\mathbf{n}).
     ]

   Это и есть **форма ячейки в гармоническом представлении**.

## 3. Как ухватить именно тор (а не просто «чуть неровную сферу»)

Тор (или семейство торов) даёт **отдельный паттерн анизотропии**:

* есть выделенная ось (ось тора);
* расстояние до стенки зависит в основном от угла к этой оси, а не от азимута.

В простейшей 3D-картинке:

* Возьми кольцевой тор с большим радиусом (R_{\rm maj}) и малым (r_{\rm min}).
* Положение наблюдателя внутри фиксировано.
* Расстояние до поверхности тора вдоль луча под углом (\theta) к оси можно выписать аналитически как (R_{\rm tor}(\theta)).

Тогда процедура:

1. **Определяешь ось**:

   * по совокупности CMB-квадруполя/октуполя и крупномасштабных структур;
   * это как раз то, что в данных называют выделенной «космической осью».

2. **Фитируешь параметризацию**:

   * берёшь модель торовой поверхности с параметрами ((R_{\rm maj}, r_{\rm min},) положение наблюдателя());
   * подгоняешь их так, чтобы
     [
     R_{\rm tor}(\mathbf{n};\text{параметры})
     \approx R_{\rm est}(\mathbf{n})
     ]
     в среднем по небу.

3. **Проверяешь согласованность уровней**:

   * полученные (R_{\rm maj}, r_{\rm min}) должны быть согласованы по порядку с уже описанными
     [
     R_1^{(\Lambda)}, R_2^{(\Lambda)}, R_3^{(\Lambda)}
     ]
     и толщинами (\Delta R_i^{(\Lambda)} \sim 0.9, 0.45, 0.25\ \text{Гпк}). 
   * а крупные «стены» ~1–3 Гпк должны естественно ложиться на локальные максимумы/минимумы этой поверхности.

Если торовая модель даёт:

* правильный масштаб (R_1^{(\Lambda)}),
* правильную толщину стенок,
* правильные осевые анизотропии CMB и распределения структур,

то ты **фактически восстановил форму ячейки как трёхслойный макро-тор**.

### **W.7. Восстановление формы Θ-ячейки из космологических направлений**

---

#### **W.7.1. Определение направленного радиуса до края**

Пусть наблюдатель находится в фиксированной точке внутри макро-ячейки уровня ΛCDM.

Определим:

* (\mathbf{n}\in S^2) — единичный вектор направления на небесной сфере;
* (R(\mathbf{n})) — **геометрическое расстояние** от наблюдателя до края Θ-ячейки вдоль направления (\mathbf{n});
* (\omega_{\rm edge}(\mathbf{n})) — **граничная частота Θ-поля** в этом направлении.

В локальной квазистационарной аппроксимации (когда изменение параметров поля вдоль луча определяется только направлением, а не временем наблюдения) естественно положить:

[
R(\mathbf{n}) = \frac{\pi}{\omega_{\rm edge}(\mathbf{n})}.
]

Изотропный предел ΛCDM соответствует:

[
\omega_{\rm edge}(\mathbf{n}) = \omega_{\rm edge,0} = const
\quad\Rightarrow\quad
R(\mathbf{n}) = R_{\rm cell} = const.
]

Любые наблюдаемые крупномасштабные анизотропии (CMB, направленная вариация (H_0, w), распределение гигапарсекных структур) трактуются как **отклонения (\omega_{\rm edge}(\mathbf{n})) от константы**, а значит — как геометрическая деформация поверхности края Θ-ячейки.

---

#### **W.7.2. Гармоническое описание формы края**

Нормируем радиус на его среднее значение:

[
\bar R = \frac{1}{4\pi}\int_{S^2} R(\mathbf{n}), d\Omega,
\quad
\delta R(\mathbf{n}) = R(\mathbf{n}) - \bar R.
]

Тогда **форма края** задаётся безразмерной функцией:

[
\varepsilon(\mathbf{n})
= \frac{\delta R(\mathbf{n})}{\bar R}
= \frac{R(\mathbf{n})}{\bar R} - 1.
]

Разложим её в сферических гармониках:

[
\varepsilon(\mathbf{n})
= \sum_{\ell=0}^{\infty}\sum_{m=-\ell}^{\ell}
a_{\ell m} Y_{\ell m}(\mathbf{n}).
]

Здесь:

* (\ell=0) даёт тривиальный сдвиг радиуса (уже учтён в (\bar R));
* (\ell=1) — диполь (смещение наблюдателя относительно центра);
* (\ell=2) — квадруполь (эллипсоидальные/тороидальные деформации);
* (\ell\ge3) — более мелкие детали формы.

**Задача космологии формы:**
по космологическим наблюдениям построить оценку (\varepsilon(\mathbf{n})) и её гармонические коэффициенты (a_{\ell m}).

#### **W.7.3. Как получить (R(\mathbf{n})) из наблюдаемых величин**

Направленная функция (R(\mathbf{n})) не измеряется напрямую, но восстанавливается через набор наблюдаемых прокси. Концептуально:

1. **CMB-анизотропии (особенно low-l)**
   дают направление и амплитуду крупных гармоник (\ell=2,3) (квадруполь, октуполь), т.е. выделяют глобальную ось симметрии/дефекта.

2. **Направленная вариация (H_0(\mathbf{n}))**
   (из суперновых, BAO и др.) даёт угловую зависимость эффективной Λ, а значит и (\omega_{\rm edge}(\mathbf{n})), через связь:

   [
   \omega_{\rm edge}^2(\mathbf{n})
   \sim 3,\Omega_\Lambda(\mathbf{n}),H_0^2(\mathbf{n}).
   ]

3. **Направленная вариация параметра состояния (w(\mathbf{n}))**
   (если она детектируется) задаёт **динамику края** (\omega_{\rm edge}(a,\mathbf{n})) и позволяет отличить статическую деформацию формы от чисто временной эволюции.

4. **Крупномасштабная структура (LSS)**
   — положение и ориентация гигантских стен и пустот (масштабы (\sim1\text{–}3\ \mathrm{Гпк})) сопоставляются с характерными радиусами и толщинами слоёв (R_i^{(\Lambda)},\Delta R_i^{(\Lambda)}). Локальные максимумы/минимумы (\varepsilon(\mathbf{n})) должны коррелировать с положением крупнейших структур.

Практический рецепт:

* из наблюдений строится набор направленных полей:

  * (H_0(\mathbf{n})),
  * (w(\mathbf{n})),
  * статистики LSS((\mathbf{n})) (контраст, число кластеров и т.п.),
  * CMB-мультиполи (a^{\rm CMB}_{\ell m});

* подбирается такая функция (R_{\rm est}(\mathbf{n})), что:

  * её крупномасштабная гармоника совпадает с CMB-осью,
  * направленная зависимость (1/R_{\rm est}(\mathbf{n})) согласуется с вариацией (H_0(\mathbf{n}), w(\mathbf{n})),
  * крупные максимумы/минимумы коррелируют с LSS.

Получив (R_{\rm est}(\mathbf{n})), считаем:

[
\varepsilon_{\rm est}(\mathbf{n})
= \frac{R_{\rm est}(\mathbf{n})}{\bar R_{\rm est}} - 1,
]

и по нему — набор коэффициентов (a_{\ell m}).

#### **W.7.4. Тороидальная параметризация формы**

В Θ-теории макро-ячейка уровня ΛCDM трактуется как **трёхслойный тор**. Это означает наличие **выделенной оси** (оси макро-тора) и специфического типа анизотропии: форма зависит в основном от полярного угла (\theta) относительно этой оси.

Введём систему координат так, что:

* ось макро-тора совпадает с осью (z);
* наблюдатель находится в точке (\mathbf{r}_O=(x_O,y_O,z_O)).

Имплицитное уравнение стандартного кольцевого тора в декартовых координатах:

[
\Big(\sqrt{x^2+y^2} - R_{\rm maj}\Big)^2 + z^2 = r_{\rm min}^2,
]

где:

* (R_{\rm maj}) — **большой радиус тора** (расстояние от оси до центров “трубки”),
* (r_{\rm min}) — **малый радиус** (радиус “трубки”).

Луч, идущий от наблюдателя вдоль направления (\mathbf{n}=(n_x,n_y,n_z)), параметризуется:

[
\mathbf{r}(s) = \mathbf{r}_O + s,\mathbf{n},\quad s\ge0.
]

**Расстояние до края тора по направлению (\mathbf{n})** — это такое (s=s_{\rm tor}(\mathbf{n})), при котором (\mathbf{r}(s)) впервые пересекает поверхность тора:

[
\Big(\sqrt{(x_O + s n_x)^2 + (y_O + s n_y)^2} - R_{\rm maj}\Big)^2 +
(z_O + s n_z)^2
= r_{\rm min}^2.
]

Это нелинейное уравнение относительно s, но оно **одномерное** и решается численно (для каждого направления (\mathbf{n})).

Так получаем **модельный направленный радиус**:

[
R_{\rm tor}(\mathbf{n}; R_{\rm maj}, r_{\rm min}, \mathbf{r}*O)
= s*{\rm tor}(\mathbf{n}).
]

При необходимости трёхслойная структура (три тора) описывается аналогично, с разными ((R_{\rm maj}^{(k)}, r_{\rm min}^{(k)})) для слоя k, либо через эффективную “толстую” оболочку.

#### **W.7.5. Процедура подгонки торовой формы к данным**

**Шаг 1. Выбор оси.**
По CMB-квадруполю/октуполю и крупномасштабным анизотропиям LSS выбирается ось (\hat{\mathbf{z}}), вдоль которой:

* максимизируется вклад (\ell=2,3) в (\varepsilon(\mathbf{n})),
* или минимизируется “размазанная” анизотропия.

В этой системе координат торомодель имеет минимальное число независимых параметров.

**Шаг 2. Построение наблюдательного (R_{\rm est}(\mathbf{n})).**
Из полей (H_0(\mathbf{n})), (w(\mathbf{n})), CMB и LSS с учётом связи

[
\omega_{\rm edge}^2(\mathbf{n})
\sim 3,\Omega_\Lambda(\mathbf{n}),H_0^2(\mathbf{n}),
\quad
R(\mathbf{n}) \sim \frac{\pi}{\omega_{\rm edge}(\mathbf{n})},
]

строится оценка (R_{\rm est}(\mathbf{n})) (возможно, в виде набора значений по произвольной сетке на сфере).

**Шаг 3. Фит торовой модели.**
Подбираются параметры тора ((R_{\rm maj}, r_{\rm min}, \mathbf{r}_O)) как минимум функционала:

[
\mathcal{F}(R_{\rm maj}, r_{\rm min}, \mathbf{r}_O)
===================================================

\int_{S^2}
\left(
\frac{R_{\rm tor}(\mathbf{n}; R_{\rm maj}, r_{\rm min}, \mathbf{r}*O)}
{\bar R*{\rm tor}}
------------------

\frac{R_{\rm est}(\mathbf{n})}
{\bar R_{\rm est}}
\right)^2 d\Omega,
]

где (\bar R_{\rm tor}, \bar R_{\rm est}) — усреднения по сфере.

**Шаг 4. Проверка согласованности с радиусами уровней.**
Полученные параметры (R_{\rm maj}, r_{\rm min}) должны:

* по порядку величины согласовываться с (R_1^{(\Lambda)}, R_2^{(\Lambda)}, R_3^{(\Lambda)}) и (\Delta R_i^{(\Lambda)});
* обеспечивать естественную интерпретацию гигапарсекных “стен” как локальных максимумов/минимумов направленного радиуса.

Если:

* минимум (\mathcal{F}) мал,
* угол между осью тора и выделенной CMB-осью невелик,
* и масштабные параметры согласованы с радиусами из разделов W.1–W.4,

то можно считать, что **форма Θ-ячейки** на уровне ΛCDM **восстановлена как макро-тороидальная оболочка**, а классические “изотропия и однородность” — это лишь приближение к этой торовой геометрии для наблюдателя, находящегося внутри.
