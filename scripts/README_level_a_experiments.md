# Level A Numerical Experiments Runner

## Описание

Скрипт `run_level_a_experiments.py` выполняет все валидационные тесты Level A согласно спецификации **7d-31-БВП_план_численных_экспериментов_A.md** и генерирует итоговый отчёт о приёмке.

## Запуск

```bash
cd /home/vasilyvz/Desktop/Инерция/7d/progs/bhlff
source .venv/bin/activate
python scripts/run_level_a_experiments.py
```

## Что выполняется

Скрипт запускает все обязательные тесты Level A:

- **A0.1**: Плоская волна (стационар) - проверка формулы â = ŝ / D
- **A0.2**: Периодический многочастотный источник - проверка суперпозиции
- **A0.3**: Нулевая мода при λ=0 - корректная обработка сингулярности
- **A0.4**: Временной решатель с гармоническим источником
- **A0.5**: Энергетический баланс/невязка
- **A1.1**: Инвариантность к масштабу длины
- **A1.2**: Инвариантность к выбору базовых единиц

## Результаты

После выполнения скрипт создаёт:

1. **`output/level_a_experiments/acceptance_report.json`** - полный JSON отчёт со всеми результатами
2. **`output/level_a_experiments/acceptance_summary.txt`** - текстовый отчёт для чтения

## Критерии приёмки

- A0.1–A0.2: E₂ ≤ 10⁻¹², анизотропия ≤ 10⁻¹²
- A0.3: корректный FAIL/сообщение при нарушении ŝ(0)=0 при λ=0参加工作
- A0.4: ошибка амплитуды и фазы ≤ 10⁻⁸
- A0.5: ‖r‖₂/‖s‖₂ ≤ 10⁻¹², ортогональность в допуске
- A1.1–A1.2: инвариантность ≤ 10⁻¹²
